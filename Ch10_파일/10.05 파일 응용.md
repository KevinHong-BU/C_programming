### 10.05 파일 응용

#### ● 예제 10-05-1 : html파일을 읽어 html 태그를 모두 제거한다.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 
4 int main()
5 {
6 	int i=0, is=0;
7 	FILE *fpi, *fpo;
8 	char c;
9 
10 	if( (fpi = fopen("test.html", "r")) == NULL) {
11 		printf("File open error");
12 		return 1;
13 	}
14 	if( (fpo = fopen("test_h.txt", "w")) == NULL) {
15 		printf("File create error");
16 		return 1;
17 	}
18 
19 	do {
20 		c = fgetc(fpi);
21 		if( c == '<' && is == 0) {
22 			is = 1;
23 		} else if (is == 0)
24 			fputc(c, fpo);
25 		else if (c == '>')
26 			is = 0;
27 	} while (c != EOF);
28 	fclose(fpi);
29 	fclose(fpo);
30 }
```

#### ● 해설
* 7: 입출력 파일 포인터 선언
* 10\~13: 입력 파일에 대한 처리
* 14\~17: 출력 파일에 대한 처리
* 19\~27: 파일을 읽어서 태그는 버리고 태그가 아닌 부분만 출력파일에 저장한다.
* 20: 입력 파일에서 한 문자 읽는다.
* 21\~22: 읽은 문자가 태그의 시작 문자인 ‘<’이고, is 가 0이면, is를 1로 한다. is가 0이란 현재 파일 포인터가 태그 밖에 있다는 의미이며, is가 1이면 태그 속에 있다는 의미이다.
* 23\~24: 파일 포인터가 태그의 밖(is가 0이면)에 있으면, 현재 문자를 파일로 저장한다.
* 25\~26: 읽은 문자가 ‘>’이면, 즉, 태그가 끝나는 문자이면 is를 0, 즉, 태그 밖으로 나간다는 의미이다.
* 27: End Of File(EOF)까지 반복한다.
> [!NOTE]
> 태그 안에 있으면 저장하지 않고, 태그 밖에 있으면 저장한다.

#### ● 예제 10-05-2 : 우편번호 검색 프로그램
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <string.h>
4 
5 int main()
6 {
7 	FILE *zt;
8 	char zipcode[10], size, temp[150], d[5], find[20];
9 	char si_do[10], si_gun_gu[20], up_myon_dong[80];
10 	char rhee[80], full_addr[80];
11 
12 	zt = fopen("zipcode2.txt", "r");
13 
14 	while(1) {
15 		printf("\n\n동(리) 입력(q:끝) : ");
16 		scanf("%s", find);
17 		if(*find == 'q')
18 			break;
19 		size = strlen(find);
20 		while(1) {	 
21 			fgets(temp, 150, zt);	// 엔터 문자까지 읽어 들임. 
22 			sscanf(temp, "%[^;]%[;]%[^;]%[;]%[^;]%[;]"
23 				    "%[^;]%[;]%[^;]%[;]%[^\n]",
24 				zipcode, d, si_do, d, si_gun_gu, d, 
25 				up_myon_dong, d, rhee, d, full_addr);
26 			if(strlen(rhee) > 2) {
27 				if(strncmp(rhee, find, size)==0)
28 					printf("%s, %s\n", zipcode, full_addr);
29 			} else if(strlen(up_myon_dong) > 2) {
30 				if(strncmp(up_myon_dong, find, size)==0)
31 					printf("%s, %s\n", zipcode, full_addr);
32 			} else if(strlen(si_gun_gu) > 2) {
33 				if(strncmp(si_gun_gu, find, size)==0)
34 					printf("%s, %s\n", zipcode, full_addr);
35 			}
36 			if(feof(zt))
37 				break;
38 		}
39 		rewind(zt);		// 파일 포인터 위치를 맨 처음으로 옮김.
40 	}
41 	fclose(zt);
42 }
```

#### ● 해설
* 7: 입력 파일 포인터 선언
* 8\~10: 각종 데이터를 저장할 변수 선언
* 12: 입력 파일 열기
* 14\~40: 파일에서 데이터를 읽어서 검색한 다음 해당하는 동(리)가 있으면 출력하기
* 16: 찾을 문자열 입력
* 17: 문자열의 첫 글자가 ‘q’이면 끝낸다.
* 19: 문자열의 길이를 구한다.
* 21: 한 줄을 읽어 temp 문자열에 저장한다.
* 22\~25: 한 줄 읽은 문자열에서 ‘;’을 기준으로 각 필드를 분리한다.
* 26\~35: 가장먼저 리(rhee)의 길이가 2보다 크면(공백일 수 있음) 리에서 찾고, 공백이면 읍면동에서 찾고, 읍면동도 공백이면 시군구에서 찾는다. 우편번호에 보면 공백이 있는 부분이 있음을 알 수 있다.
* 27,30,33: 문자열에 해당하는 문자가 있는지 검사한다.
* 36\~37: 파일 끝까지 해당하는 문자열이 있는지 검색한다. 끝이면 다시 루프를 반복한다.
* 39: 파일 포인터를 다시 파일의 맨 처음 위치로 옮김.
> [!NOTE]
> 이 우편번호는 우체국 사이트에 가면 구할 수 있다. 원본은 엑셀로 되어 있는데, 텍스트 파일로 바꾸었다. 텍스트 파일로 바꿨지만 실제 내용은 아래의 zipcode.txt와 같이 된다. 이 파일을 한번 더 처리하여 zipcode2.txt로 새로 만들었다. 원본 텍스트 파일에서는 공백이 일정하지 않아 정확하게 처리하기 위하여 새로 만든 것이다. 여러분도 프로그램으로 한번 작성해 보기 바란다. 파일이 매우 커서 메모장에서는 읽기가 힘들어 워드패드로 읽는 것이 좋다.
