### 10.02 파일에 데이터 쓰기

#### ● 예제 10-02-1 : 19×19단을 만들어 파일에 쓰기
```C
1 #include <stdio.h> 
2 #include <stdlib.h>
3 #include <assert.h>
4 
5 int main()
6 {
7 	int dan, cnt, col;
8 	FILE *fp;
9 	fp = fopen("구구단.txt", "w");
10 	assert(fp);
11 	for(dan = 2; dan<20; dan+=4) {		// 한 줄에 4단씩 출력하기
12 		for(col=dan; col<dan+4; col++)	// 단의 머리글 쓰기
13 			fprintf(fp, "%7d 단%7s", col, ""); 
14 		fprintf(fp, "\n");
15 		for(cnt = 1; cnt<20; cnt++) {		// 각 단은 19까지 계산하기
16 			for(col=dan; col<dan+4; col++)	// 4칸씩 쓰기
17 				fprintf(fp, "%3d * %3d = %3d  ", col, cnt, col*cnt);
18 			fprintf(fp, "\n");
19 		}
20 		fprintf(fp, "\n");
21 	}
22 	fclose(fp);
23 }
```

#### ● 해설
* 9: 파일 “구구단.txt”를 쓰기 모드로 열어 파일 포인터 fp에 대입
* 10: 파일을 열 수 없을 때 assert함수를 이용하여 에러 메시지 출력하고 끝낸다.
* 11\~21: 파일에 한 줄에 4개씩 출력하여 21단까지 출력한다.
* 12\~13: 각 단이 시작할 때 4개의 단 제목을 한 줄에 출력한다.
* 14: 줄을 바꾼다.
* 15\~19: 1부터 19까지 1씩 변화시키면서 계산하여 출력하기 위한 루프
* 16\~17: 한 줄에 4단씩 출력한다.
* 18: 한 줄이 출력되면 줄을 바꾼다.
* 20: 19까지 모두 계산하여 출력한 다음 줄을 바꾼다.
* 22: 파일을 닫는다.
> [!NOTE]
> 파일의 출력 결과는 메모장이나 워드패드를 이용하여 확인해 본다.

#### ● 예제 10-02-2 : RGB색상 표를 HTML형식의 파일로 만들어 저장한다.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 
4 FILE *fp;
5 int rgb, font;
6 
7 void PrintHead(void);
8 void PrintBody(int, int, int);
9 void PrintTail(void);
10 
11 int main()
12 {
13 	int r, g, b;
14 	fp = fopen("rgb.html", "w");
15 	PrintHead();
16 	for(r=0; r<0x100; r+=0xf) {
17 		for(g=0; g<0x100; g+=0xf) {
18 			fprintf(fp, "\n<TR>\n");
19 			for(b=0; b<0x100; b+=0xf)
20 				PrintBody(r, g, b);
21 		}
22 	}
23 	PrintTail();
24 	fclose(fp);
25 }
26 
27 void PrintHead(void) /* print head */ 
28 {
29 	fprintf(fp, "<HTML>\n");
30 	fprintf(fp, "<HEAD></HEAD>\n");
31 	fprintf(fp, "<BODY>\n");
32 	fprintf(fp, "<H1> RGB 테이블</H1>\n");
33 	fprintf(fp, "<TABLE CELLPADDING=0, CELLSPACING=0 BORDER=0>");
34 }
35 
36 void PrintBody(int r, int g, int b) 
37 {
38 	char *format0 = "<TH HEIGHT=65 WIDTH=200 BGCOLOR=#%06x>";
39 	char *format1 = "<FONT SIZE=-1 COLOR=#%06X>";
40 	char *format2 = "R:%02X<br>G:%02X<br>B:%02X</FONT></TH>";
41 
42 	rgb = (r<<16)+(g<<8)+b;
43 	font = rgb ^ 0xffffff; // 보색을 만들어 주는 부분, 바탕색과 글자색 구분을 위해서 
44 	fprintf(fp, format0, rgb);
45 	fprintf(fp, format1, font);
46 	fprintf(fp, format2, r, g, b);
47 	fprintf(fp, "\n");
48 }
49 
50 void PrintTail(void) /* print tail */ 
51 { 
52 	fprintf(fp, "</TR>");
53 	fprintf(fp, "</TABLE>");
54 	fprintf(fp, "</BODY>\n");
55 	fprintf(fp, "</HTML>\n");
56 }
```

#### ● 해설
* 4: 파일 포인터 선언, 모든 함수에서 공유하기 위해 전역변수로 선언한다.
* 5: 전역 변수 선언, 다른 함수들에서도 공유한다.
* 7\~9: 함수 원형 선언
* 14: 파일 “rgb.html”를 쓰기 모드로 열어 파일 포인터 fp에 대입. 
* 15: html파일의 앞 부분인 head를 출력하기 위한 함수 호출
* 16\~22: 파일의 body부분을 하나씩 만들어 출력하기 위한 루프
* 16: red 색상을 0부터 255까지 15씩 증가시킨다.
* 17: green 색상을 0부터 255까지 15씩 증가시킨다.
* 18: 테이블 레코드 부분이 시작됨을 알린다.
* 19: blue 색상을 0부터 255까지 15씩 증가시킨다.

* 23: html파일의 마무리 부분인 tail을 출력하기 위한 함수 호출
* 24: 함수의 사용이 끝났으므로 닫는다.
* 27\~34: 파일의 헤더 부분 출력하는 함수
* 29\~33: 파일의 헤더부분은 한 번만 출력하게 된다. html의 기본 태그를 조금만 익히면 누구나 알 수 있는 부분이다.
* 36\~48: 실제 색상 테이블을 만들어 출력하는 함수이다.

* 43: 폰트(font)의 색상을 RGB의 색상과 반대로 만든다.
* 44\~46: 바탕색과 폰트의 색상, r, g, b의 값을 html의 표로 출력한다.
* 47: 줄을 바꾼다.
* 50~56: html의 꼬리 부분을 출력한다.
> [!NOTE]
> 파일의 길이가 매우 길기 때문에 출력 내용은 워드패드를 이용하여 확인해 본다. 결과를 확인하기 위해 인터넷 브라우저를 이용한다. HTML 태그들은 그대로 fprintf를 이용하여 출력하면 된다. 다만 값의 변화가 있는 부분만 적절히 처리하여 출력한다.

#### ● 예제 10-02-4 : 정수 데이터를 무작위로 발생하여 정렬하여 파일에 저장하기.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 
5 int cmpare(const void *a, const void *b) { 
6 	return (*(int *)a - *(int *)b);
7 }
8 
9 int main()
10 {
11 	FILE *fp;
12 	int i, j, *d, Number;
13 	char filename[30];
14 	printf("파일이름 입력: ");
15 	scanf("%s", filename);
16 	printf("생성할 데이터 개수 : ");
17 	scanf("%d", &Number);
18 	d = (int *) malloc(Number*4);		// 메모리 공간 할당
19 
20 	fp = fopen(filename, "w");
21 	srand(time(0));
22 
23 	for (i=0; i<Number; i++)
24 		d[i] = rand() + rand(); // 0~65535까지 난수 발생
25 
26 	qsort(d, Number, sizeof(int), cmpare);
27 
28 	for (i=0; i<Number; i++)
29 		fprintf(fp, "%d\n", d[i] );
30 
31 	free(d);
32 	fclose(fp);
33 }
```

#### ● 해설
* 5\~7: 정수 비교함수
* 15: 저장할 파일 이름 입력
* 17: 생성할 데이터의 개수 입력
* 18: 저장할 정수의 배열 공간 확보
* 20: 파일 생성
* 21: 난수 초기화 함수 실행
* 23\~24: 난수를 입력한 수의 개수만큼 발생하여 배열 메모리에 저장한다. 포인터를 써도 된다.
* 26: qsort함수를 이용하여 발생한 난수를 정렬한다.
* 28\~29: 파일에 배열의 내용을 모두 저장한다. 즉, 정렬되어 있는 데이터를 저장한다.

#### ● 예제 10-02-5 : 문자열 데이터를 무작위로 발생하여 정렬하여 파일에 저장하기.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 #include <string.h>
5 #define MAX 20 // max number of characters of string
6 int cmpare(const void *a, const void *b) { 
7 	return strcmp(*(char **)a, *(char **)b);
8 }
9 
10 int main()
11 {
12 	FILE *fp;
13 	int i, j, Number, Nos; // nos = number of characters of string
14 	char **s, **t, *p, filename[30];
15 
16 	printf("파일이름 입력: ");
17 	scanf("%s", filename);
18 	printf("생성할 데이터 개수 : ");
19 	scanf("%d", &Number);
20 	fp = fopen(filename, "w");
21 	srand(time(0));
22 	// 2차원 포인터 확보
23 	t = s = malloc(sizeof(char *) * Number); 
24 	for (i=0; i<Number; i++, t++)	// n개의 문자열 공간 확보 
25 		*t = (char *) malloc(sizeof(char) * (MAX+1) );
26 
27 	t = s;
28 	for (i=0; i<Number; i++, *t++) {
29 		// 발생시킬 문자의 개수를 지정 
30 		Nos = rand() % (MAX -1) + 2; 
31 		for (j=0, p=*t; j<Nos; j++, p++)
32 			*p = rand()%26 + 'a'; //문자화 
33 		*p = 0;	 // NULL 삽입 
34 	}
35 
36 	qsort(s, Number, sizeof(char *), cmpare);
37 	t = s;
38 	for (i=0; i<Number; i++, *t++)
39 		fprintf(fp, "%s\n", *t);
40 
41 	t = s;
42 	for (i=0; i<Number; i++, *t++)
43 		free(*t);
44 
45 	free(s);
46 	fclose(fp);
47 }
```

#### ● 해설
* 6: 한 문자열의 최대 길이 설정
* 7\~9: 문자열 비교함수
* 17: 저장할 파일 이름 입력

* 23: 2차원 포인터 확보, 문자열의 배열이므로 2차원 포인터이다.
* 24\~25: 개수만큼 문자열 공간 확보
* 28\~34: 개수만큼 문자열 생성
* 30: 한 개의 문자열의 길이 무작위로 생성
* 31\~32: 문자열의 길이만큼 무작위로 문자 생성
* 32: 생성될 문자는 ‘a’부터 26개이다. 즉, 소문자 ‘a’부터 소문자 ‘z’까지 중의 하나를 발생한다.

* 38\~39: 발생한 문자열 모두 파일에 저장

#### ● 유제 10-02-2 : 예금 통장 관리 프로그램을 작성하라. 파일은 2 종류를 만든다. 하나는 고객관리 파일이고, 다른 하나는 고객의 통장 입출금 내역을 관리하는 파일이다. 먼저 고객을 찾고, 그 고객에 대한 입출금 내역을 출력하는 프로그램을 작성한다. 따라서 통장 입출금 내역은 고객별로 하나씩 만들어야 한다. 고객을 3명 만들고 각 고객에 따라 통장의 내역을 보여주어야 한다. 단, 고객은 3명만 초깃값으로 등록되어 있다고 가정하고, 입출금 내역은 입금과 출금에 대해서 입력하여 관리할 수 있도록 한다. 구조체로 만들고 관리한다. 구조체도 적당히 설계하라.
