### 10.03 바이너리 파일 입출력

#### ● 예제 10-03-1 : 바이너리 파일로 저장하기(fwrite)
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 struct score {
4 	unsigned char name[20];
5 	unsigned char Kor, Eng, Math, Sci;
6 	unsigned int Tot;
7 	float Avg;
8 };
9 
10 int main()
11 {
12 	int i=0;
13 	struct score st[20] = {
14 		   {"강인한", 95, 89, 87, 90, }, {"장윤아", 90, 91, 93, 89, },
15 		   {"김수영", 76, 87, 89, 79, }, {"민효연", 92, 91, 87, 77, },
16 		   {"성유리", 87, 90, 78, 70, }, {"한태연", 88, 77, 68, 78, },
17 		   {"곽서현", 88, 99, 66, 77, }, {"박규리", 79, 87, 97, 99, },
18 		   {"강지영", 67, 69, 79, 78, }, {"한지영", 98, 78, 97, 88, },
19 		   {"한준수", 57, 67, 65, 76, }, {"홍준호", 78, 73, 69, 79, },
20 		   {"임찬성", 68, 78, 67, 65, }, {"강우영", 80, 90, 89, 77, },
21 		   {"박택훈", 47, 65, 55, 67, }, {"조성권", 76, 56, 67, 87, },
22 		   {"이창훈", 88, 89, 85, 92, }, {"정진석", 89, 90, 78, 80, },
23 		   {"정운태", 91, 93, 89, 90, }, {"홍민수", 89, 99, 87, 93, }};
24 	struct score *sp;
25 	FILE *fp;
26 
27 	if( (fp = fopen("score.bin", "wb")) == NULL) {
28 		printf("File open error");
29 		return 1;
30 	}
31 	for(i=0; i<20; i++) {
32  		st[i].Tot = st[i].Kor + st[i].Eng + st[i].Math + st[i].Sci;
33  		st[i].Avg = st[i].Tot / 4.0;
34 		sp = &st[i];
35 		fwrite(sp, sizeof(struct score), 1, fp);
36 	}
37 	fclose(fp);
38 }
```

#### ● 해설
* 3\~8: 구조체 선언
* 13\~23: 구조체에 초깃값 설정
* 24: 구조체 포인터 선언
* 25: 파일 포인터 fp 선언.
* 27\~30: 파일을 생성할 수 없는 경우 처리과정
* 31\~36: 구조체 하나씩 계산하여 파일에 저장
* 32: 합계 계산
* 33: 평균 계산
* 34: 포인터에 현재 구조체 배열의 주소를 대입
* 35: 바이너리 파일에 데이터 출력, 블록단위의 데이터를 출력할 때는 fwirte함수를 사용한다.
* 37: 파일 닫기
> [!NOTE]
> 블록 단위의 파일은 메모리 블록을 그대로 파일로 출력하므로 2진 형태로 저장된다. fwrite함수의 첫 번째 매개변수는 블록의 주소, 두 번째 매개변수는 블록의 크기, 세 번째 매개변수는 저장할 블록의 개수, 네 번째 매개변수는 파일 포인터이다.

#### ● 실행 결과: score.bin

#### ● 예제 10-03-2 : 바이너리 파일에서 읽기(fread)
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 
4 struct score {
5 	unsigned char name[20];
6 	unsigned char Kor, Eng, Math, Sci;
7 	unsigned int Tot;
8 	float Avg;
9 };
10 
11 int main()
12 {
13 	int i=0;
14 	struct score *sp=(struct score*)malloc(sizeof(struct score));
15 	FILE *fp;
16 
17 	if( (fp = fopen("score.bin", "rb")) == NULL) {
18 		printf("File open error");
19 		return 1;
20 	}
21 	while(1) {
22 		fread(sp, sizeof(struct score), 1, fp);
23 		if(feof(fp))
24 			break;
25 		printf("%s, 국어:%u, 영어:%u, 수학:%u, 과학:%u, 합계:%u, 평균:%f\n",
26 			sp->name, sp->Kor, sp->Eng, sp->Math, sp->Sci, sp->Tot, sp->Avg);
27 	}
28 	fclose(fp);
29 }
```

#### ● 해설
* 4\~9: 구조체 선언
* 14: 구조체를 저장할 메모리 공간확보 및 포인터 대입, 메모리 공간은 구조체의 크기만큼 확보
* 15: 파일 포인터 fp 선언.
* 21\~27: 파일에서 구조체를 하나씩 읽어서 화면에 출력
* 22: 구조체 크기만큼의 블록을 파일에서 하나 읽는다(fread). 블록단위로 파일에서 읽는 함수는 fread이다.	첫 번째 매개변수는 블록의 주소, 두 번째 매개변수는 블록의 크기, 세 번째 매개변수는 읽을 블록의 개수, 네 번째 매개변수는 파일 포인터이다.
* 23,24: 파일의 끝에 도달하면 끝낸다.
* 25,26: 파일에서 읽은 데이터를 모니터에 출력한다.
* 27: 파일 닫기

#### ● 예제 10-03-3 : 비트필드 구조체를 2진 파일로 저장 및 읽기

* 51\~52: 파일의 끝에 도달하면 끝낸다.
