### 10.07 랜덤 파일

#### 랜덤(random) 파일이란?
파일의 위치 이동을 마음대로 할 수 있도록 한다.

* 위치 이동 함수
  `fseek, fgetpos, fsetpos, ftell`

* 기타 관련 함수
	`freopen, fflush, ferror, ungetc`

* fseek : 지정된 위치로 이동한다.
  
	int fseek ( FILE * stream, long int offset, int origin );
	offset : 상대적인 위치, +:순방향, -:역방향
	origin : 이동 기준 위치
		SEEK_SET : 파일의 맨 처음 위치
		SEEK_CUR : 파일에서의 현재 위치
		SEEK_END : 파일의 맨 끝 위치

* fgetpos : 파일에서의 현재 포인터의 위치 값 획득
  
	int fgetpos ( FILE * stream, fpos_t * position );

* fsetpos : 파일에서 지정된 포인터로 이동

	int fsetpos ( FILE * stream, const fpos_t * position );

* ftell : 파일에서의 현재 위치를 알려 줌

	long int ftell ( FILE * stream );

* freopen : 파일 포인터 재설정한다.

	FILE * freopen ( const char * filename, const char * mode, FILE * stream );

* fflush

	int fflush ( FILE * stream );

	입력 파일일 경우 : 버퍼를 지운다. 출력파일일 경우 저장되지 않은 버퍼에 있는 데이터를 모두 저장한다.

* ferror : 파일에 에러가 있는지 검사한다.

	int ferror ( FILE * stream );

* ungetc : 한 글자를 가상으로 파일에 다시 저장

	int ungetc ( int character, FILE * stream );  \\\\즉, 한 글자를 안 읽은 것으로 한다.

#### ● 예제 10-07-1 : 파일을 읽어서 거꾸로 출력하기
```C
1 // 파일 거꾸로 읽기
2 #include <stdio.h> 
3 #define   MAXSTRING   100 
4 
5 int main()
6 { 
7 	char   fname[MAXSTRING]; 
8 	int    c; 
9 	FILE   *ifp; 
10 
11 	fprintf(stderr, "\nInput a filename:  "); 
12 	scanf("%s", fname); 
13 	ifp = fopen(fname, "rb");	// 2진 모드로 파일 읽기 
14 	fseek(ifp, -1, SEEK_END);	// 끝에서 역방향으로 한 글자 이동 
15 	while ( 1 ) {
16 		c = getc(ifp);	// 한 글자 읽음 : 한 글자 순방향으로 이동 
17 		putc(c, stdout);
18 		fseek(ifp, -2, SEEK_CUR);   // 역방향으로 두 글자 이동
19 		if ( ftell(ifp) <= 0 )
20 	 		break;
21 	} 
22 	c = getc(ifp);	// 마지막 문자(맨 앞 글자) 읽기 
23 	putc(c, stdout);	// 마지막 문자 출력 
24 	printf("\n");
25 }
```

#### ● 해설
* 9: 읽기 파일 포인터 선언
* 11: 표준 에러 장치, 즉 모니터로 메시지 출력, printf와 동일하지만, stderr이란 장치로 방향 전환.
* 12: 입력할 파일 이름을 입력한다.
* 13: 파일을 2진 읽기 모드로 연다.
* 14: 파일의 끝으로 파일 포인트 이동한다.
* 15\~21: 파일의 처음으로 이동할 때까지 하나씩 읽어서 화면에 출력한다.
* 16: 한 글자 읽는다.
* 17: 모니터에 읽은 문자 출력한다.
* 18: 현재 파일 포인트에서 역방향으로 2칸 이동한다. 파일에서 한문자 읽으면 순방향으로 한 칸 이동하므로 거꾸로 읽기 위해서는 역방향으로 다시 2칸 이동하여야 한다.
* 19\~20: 파일의 처음 위치까지 갔는지 확인하고 처음 위치 보다 작으면 루프를 끝낸다.
* 22: 2바이트씩 거꾸로 이동하므로 파일의 맨 처음 문자는 읽지 못하므로 마지막에 하나 더 읽는다.
* 23: 마지막 읽은 문자를 출력한다.
> [!NOTE]
> 파일에서 데이터를 읽으면 자동적으로 순방향으로 읽은 바이트 수만큼 포인트가 이동한다.

#### ● 예제 10-07-2: 한글 파일을 읽어서 거꾸로 출력하기
```C
1 #include <stdio.h> 
2 #include <stdlib.h>
3 #include <ctype.h>
4 
5 FILE *gfopen(char *filename, char *mode);
6 
7 int main()
8 {
9 	FILE *fp, *tmp_fp;
10 	int c, c2, size=0;
11 	char fn[30];
12 	printf("파일이름 입력 : ");
13 	scanf("%s", fn);
14 	fp = gfopen(fn, "r+");
15 	tmp_fp = tmpfile();	// 임시파일은 2진 파일이다. 
16 	while( (c=getc(fp)) != EOF) {
17 		putc(c, tmp_fp);
18 		size++;
19 	}
20 
21 	fprintf(fp, "---\n");
22 	fseek(tmp_fp, -1, SEEK_CUR);
23 	while( size > 0 ) {
24 		c = getc(tmp_fp);
25 		size--;
26 		if( c & 0x80 ) {
27 			fseek(tmp_fp, -2, SEEK_CUR);
28 			c2 = getc(tmp_fp);
29 			size--;
30 			putc(c2, fp);
31 			putc(c, fp);
32 		}
33 		else
34 			putc(c, fp);
35 		fseek(tmp_fp, -2, SEEK_CUR);
36 	}
37 	fclose(fp);
38 }
39 
40 FILE *gfopen(char *filename, char *mode) {
41 	FILE *fp;
42 	if( (fp = fopen(filename, mode)) == NULL) {
43 		fprintf(stderr, "Cannot open %s - bye!\n", filename);
44 		exit(0);
45 	}
46 	return fp;
47 }
```

#### ● 해설
* 9: 읽기 파일 포인터 및 임시 파일 포인터 선언
* 13: 읽을 파일 이름 입력
* 14: 파일 열기 함수 호출
* 15: 임시 파일 포인터 만들기
* 16\~19: 읽기 파일에서 임시 파일로 모든 데이터 읽기
* 21: 원 파일의 내용과 구분하기 위한 구분선 넣기
* 22: 임시 파일 포인터를 거꾸로 2바이트 이동
* 23\~36: 글자를 읽어 원 파일에 거꾸로 저장하기
* 24: 한 글자 읽기
* 25: 파일의 크기에서 1 빼기, 즉, 읽은 바이트 수만큼 감소
* 26\~32: 읽은 글자의 최상위비트가 1이면 한글이므로 한글에 대한 처리를 하는 루프
* 27: 한글은 2바이트이므로 한 바이트 더 읽음.
* 30\~31: 두 바이트를 순서대로 저장. 한글 한 글자를 저장한다.
* 33\~34: 한글이 아니면 그냥 저장한다.
* 35: 파일 포인터를 거꾸로 2바이트 이동한다.
> [!NOTE]
>  여기서 ASCII코드는 7비트라고 가정하였다. 물론 확장 ASCII코드가 있으면 이 프로그램을 정상적으로 동작하지 않을 수도 있다. 만약 확장 ASCII코드 즉, 그래픽 문자가 있다면 처리 방식을 달리해야 할 것이다. 그 프로그램도 작성해 보라.
