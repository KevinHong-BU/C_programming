#### ● 유제 10-02-1 : 10진수 0부터 255까지를 10진, 16진, 8진, 2진수로 변환하고, 파일에 저장하는 프로그램을 작성하라.
```C
1 #include <stdio.h> 
2 
3 int main()
4 {
5 	FILE *n_sys;
6 	int dec;
7 	char bin[33];
8 	
9 	n_sys = fopen("num_system.txt", "w");
10 	fprintf(n_sys, "%5s%6s%6s%6s\n", "10진", "16진", "8진", "2진");
11 	for(dec = 0; dec<256; dec++) {
12 		itoa(dec, bin, 2);
13 		fprintf(n_sys, "%4d | 0x%02X | %04o | %08s\n", dec, dec, dec, bin);
14 	}
15 	fclose(n_sys);
16 }
```

#### ● 해설
* 5: 파일 포인터 n_sys 선언.
* 7: 2진수로 변환하여 저장할 문자열 선언
* 9: 파일 “num_system.txt”를 쓰기 모드로 열어 파일 포인터 n_sys에 대입. 
* 10: 파일의 머리글 출력
* 11\~14: 10진수 0부터 255까지 변화하면서 루프 수행
* 12: 10진수를 2진수 문자열로 변환하여 bin에 저장
* 13: 10, 16, 8, 2진수를 파일로 저장
* 15: 파일 닫기
> [!NOTE]
> 16진수는 앞에 0X, 8진수는 0으로 시작하여 저장하고, 2진수는 앞의 빈자리를 0으로 채운다.

#### ● 유제 10-04-1 : BMP 컬러 파일을 읽어 흑백(그레이)로 바꾸는 프로그램을 작성하라.

63

71 	fwrite(ihd.block, 40, 1, fpo); 
72 
73 	//padding 계산

45 	}
