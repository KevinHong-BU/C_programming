### 10.08 기타 파일 관련함수

* 기타 관련 함수
-	freopen, fflush, ferror, ungetc

---
#### ● 예제 10-08-1: fgetpos 함수 예
```C
1 /* fgetpos example */
2 #include <stdio.h>
3 int main ()
4 {
5 	FILE * pFile;
6 	int c, n;
7 	fpos_t pos;
8 	
9 	pFile = fopen ("myfile.txt", "r");
10 	if (pFile==NULL)
11 		perror ("Error opening file");
12 	else {
13 		c = fgetc (pFile);
14 		printf ("1st character is %c\n", c);
15 		fgetpos (pFile, &pos);
16 		for (n=0;n<3;n++) {
17 			fsetpos (pFile, &pos);
18 			c = fgetc (pFile);
19 			printf ("2nd character is %c\n", c);
20 		}
21 		fclose (pFile);
22 	}
23 }
```

#### ● 해설
* 5: 파일 포인터 선언
* 7: fpos_t 데이터 형은 fgetpos함수를 이용하여 파일의 위치 정보를 가져오고, fsetpos함수를 사용하여 파일내의 위치를 설정한다.
* 9: 파일 열기
* 10\~11: 파일이 없을 때 에러메시지 출력
* 13: 파일에서 문자하나 읽기
* 15: 파일내의 현재 위치 pos 변수에 읽어오기
* 16\~20: 3번 반복하여 작업하기
* 17: 위치를 전에 읽어온 곳으로 변경
* 18: 문자하나 읽기, 문자를 하나 읽으면 위치가 변경된다. 변경된 위치를 다시 17행에서 원위치로 되돌린다.


#### ● 예제 10-08-2: freopen 함수 예
```C
1 /* freopen example: redirecting stdout */
2 #include <stdio.h>
3 
4 int main ()
5 {
6 	int dan, iteration; 
7 	freopen ("freopen.txt", "w", stdout);
8 	printf ("이 출력은 파일로 방향을 바꾸어 출력된다.\n");
9 	printf ("즉, 모니터(표준출력장치)가 아닌 파일로 출력하게 된다.\n\n");
10 	for(dan=2; dan<10; dan++) {
11 		printf("%5d 단\n", dan); 
12 		for(iteration=1; iteration<10; iteration++)
13 			printf("%2d * %2d = %2d\n", dan, iteration, dan*iteration);
14 		printf("\n");
15 	}
16 	fclose(stdout);
17 }
```

#### ● 해설
* 7: freopen은 파일을 세 번째 매개변수와 관련된 파일을 닫고 새로이 열게 된다. 여기서는 표준출력장치 파일을 닫고 방향을 바꾸어 “freopen.txt”란 파일에 데이터를 저장하기 위해서 연다.
* 8: 이후부터 표준출력장치로 출력되는 모든 내용은 지정된 파일로 저장된다.
* 16: 지정된 파일포인터를 닫는다. 즉, 표준출력장치에서 지정된 방향(파일)으로 출력되는 것을 해제한다.


#### ● 예제 10-08-3: ftell 함수 예
```C
1 /* ftell example : getting size of a file */
2 #include <stdio.h>
3 int main ()
4 {
5 	FILE * pFile;
6 	long size;
7 	pFile = fopen ("myfile.txt","rb");
8 	if (pFile == NULL)
9 		perror ("Error opening file");
10 	else {
11 		fseek (pFile, 0, SEEK_END);
12 		size = ftell (pFile);
13 		fclose (pFile);
14 		printf ("Size of myfile.txt: %ld bytes.\n", size);
15 	}
16 }
```

#### ● 해설
* 5: 파일 포인터 선언
* 7: 파일 열기
* 11: 파일의 맨 끝으로 이동한다.
* 12: ftell함수를 이용하여 그 위치를 알아낸다.
* 14: 그 값을 출력한다. 위치 값이란 파일의 처음부터 그 위치까지의 바이트 수를 의미한다.


#### ● 예제 10-08-4: fflush 함수 예
```C
1 /* fflush example */
2 #include <stdio.h>
3 char mybuffer[80];
4 int main()
5 {
6 	FILE * pFile;
7 	pFile = fopen ("myfile.txt","r+");
8 	if (pFile == NULL)
9 		perror ("Error opening file");
10 	else {
11 		fputs ("test", pFile);
12 		fflush (pFile);   // flushing or repositioning required
13 		fgets (mybuffer, 80, pFile);
14 		puts (mybuffer);
15 		fclose (pFile);
16 	}
17 }
```

#### ● 해설
* 6: 파일 포인터 선언
* 7: 파일을 “r+”(읽기+추가)형식으로 열기
* 11: 파일의 현재 위치에 “test”를 출력한다.
* 12: 다음 데이터를 읽기 전에 현재 위치를 상기시킨다. 이 함수가 없다면 다음의 동작들이 제대로 일어나지 않을 수 있다.
* 13: 현재 포인터 뒤의 문자들을 읽는다.
* 14: 그것을 화면에 출력한다.
> [!NOTE]
> fflush는 현재 파일 스트림이 쓰기 상태이고 마지막 동작이 출력이면 아직 저장되지 않은 내용이 출력 버퍼에 있으면 파일로 저장한다. 만약 읽기 전용으로 열려 있고 마지막 동작이 읽기였다면 이 동작을 컴파일러의 라이브러리에 따라 달라진다. 어떤 컴파일러에서는 이 동작이 입력 버퍼를 지우는 역할을 하는 경우도 있는데, 이는 표준이 아니다. 만약 매개변수가 없다면(NULL) 모든 열려 있는 파일을 상기시킨다.


#### ● 예제 10-08-5: ferror 함수 예
```C
1 /* ferror example: writing error */
2 #include <stdio.h>
3 int main ()
4 {
5 	FILE * pFile;
6 	pFile=fopen("myfile.txt", "r");
7 	if (pFile==NULL)
8 		perror ("Error opening file");
9 	else {
10 		fputc ('x', pFile);
11 		if (ferror (pFile))
12 			printf ("Error Writing to myfile.txt\n");
13 		fclose (pFile);
14 	}
15 }
```

#### ● 해설
* 5: 파일 포인터 선언
* 6: 파일을 “r”(읽기)형식으로 열기
* 10: 파일에 문자 ‘x’를 기록한다. 읽기 모드로 열었으므로 쓰기가 되지 않음을 알 수 있다.
* 11\~12: 이 때 에러가 발생하므로 에러메시지를 출력하도록 한다.


#### ● 예제 10-08-6: perror 함수 예
```C
1 /* perror example */
2 #include <stdio.h>
3 
4 int main ()
5 {
6 	FILE * pFile;
7 	pFile=fopen ("unexist.ent", "rb");
8 	if (pFile==NULL)
9 		perror ("The following error occurred");
10 	else
11 		fclose (pFile);
12 }
```

#### ● 해설
* 6: 파일 포인터 선언
* 7: 파일을 “rb”(2진 읽기)형식으로 열기, 파일 읽기에 문제가 있으면 에러 번호가 발생하게 된다.
* 8~9: 파일이 열리지 않으면 파일 포인터는 NULL이 되며 이 때 perror를 이용하여 에러 번호에 해당하는 에러메시지를 출력하도록 한다. 출력 문자열 뒤에 콜론(:)과 공백이 출력되고 메시지가 출력되고, 메시지 출력한 다음에는 줄 바꿈이 이루어진다.
* Try: 에러 번호는 0번부터 42번까지 지정되어 있다. 컴파일러마다 다를 수도 있다. 에러번호에 따른 에러 메시지를 출력해 보도록 하라. 에러 번호에 따른 에러 메시지 출력은 strerror 함수를 이용한다. 에러 번호는 errno.h에 정의되어 있다. 
	for(i=0; i<43; i++) printf("%d : %s\n", i, strerror(i));
