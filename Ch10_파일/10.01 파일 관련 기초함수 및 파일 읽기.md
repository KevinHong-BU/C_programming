### 10.01 파일 관련 기초함수 및 파일 읽기

#### 파일 포인터 : 파일을 논리적인 장치로 인식하도록 포인터로 연결하여 액세스한다.
*	파일을 열기 전에 파일포인터를 선언한다: FILE *file_pointer;
*	파일에서 데이터를 읽은 바이트 수만큼 파일 포인터가 이동한다.

#### 파일 열기 : 파일을 사용하려면 파일을 열어야 한다.
*	file_pointer = fopen(“파일이름”, “액세스방법 지정”);
  
#### 파일 닫기 : 파일 사용이 끝나면 파일을 닫는다.
	fclose(file_pointer);

● 예제 10-01-1 : 파일 읽기 예(한 줄씩 읽기)
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	char temp[255];	// 파일에서 읽은 데이터를 저장할 문자열 변수
6 	FILE *in;	// 파일 포인터 선언
7 	if( (in = fopen("score1.txt", "r")) == NULL) {		// 파일읽기
8 		printf("File open error");		// 파일이 없으면 에러메시지 출력
9 		return 0;			// 프로그램의 실행을 끝냄
10 	}
11 	while( fgets(temp, 255, in) != NULL )	// 한 번에 최대 255문자씩 읽음
12 		printf("%s", temp);
13 	fclose(in);
14 }
● 읽을 파일

● 해설
6: 파일 포인터 in 선언
7: “scoret1.txt”라는 파일을 읽기(r) 전용 모드로 열어서 file pointer “in”에 파일의 포인터를 저장한다. 파일이 없으면 NULL포인터가 반환된다. 따라서 이 파일 포인터가 널이면 프로그램을 끝낸다.
11: 파일 포인터 “in”에서 한 줄을 읽어서 temp문자열에 저장한다. 최대 255바이트를 읽을 수 있도록 지정한다. 255바이트가 되기 전에 줄 바꿈 문자를 만나면 거기까지만 읽는다. 더 이상 없으면 NULL을 읽게 된다.
12: 읽은 문자열을 모니터에 출력한다.
13: 파일을 다 사용하였으면 닫는다. 닫아주지 않으면 간혹 문제가 발생할 수가 있다.
Notice: 여기서는 데이터를 한 줄씩 읽어서 바로 출력하였다.

● 예제 10-01-2 : 파일 읽기 예(데이터를 구분하여 fscanf로 읽기)
1 #include <stdio.h>
2 int main()
3 {
4 	int x, kor, math, sci;
5 	char temp[256], name[30];
6 	FILE *in;
7 	if( (in = fopen("score1.txt", "r")) == NULL) {
8 		printf("File open error");
9 		return 1;
10 	}
11 	while( !feof(in) ) {
12 		fscanf(in, "%s %d %d %d", name, &kor, &math, &sci);

● 해설
6: 파일 포인터 in선언
7: “scoret1.txt”라는 파일을 읽기(r) 전용 모드로 열어서 file pointer “in”에 파일의 포인터를 저장한다. 파일이 없으면 NULL포인터가 반환된다. 따라서 이 파일 포인터가 널이면 프로그램을 끝낸다.
11: feof함수는 file 포인터가 파일의 끝(NULL)이면 참(1)을 반환하고 그렇지 않으면 0을 반환한다.즉, 파일의 끝이 아니면 루프를 계속 수행하라는 의미이다.
12: 파일에서 데이터를 하나씩 읽는다. fscanf는 scanf함수와 거의 사용법이 같으며, 단 하나 파일 포인터를 첫 번째 매개변수로 사용한다는 것만 다르다. 파일에서 공백으로 데이터가 구분되어 있으므로 여기서 공백을 구분 문자로 사용하였다.
13: 읽은 파일의 내용을 한 줄씩 출력한다.
15: 파일을 닫는다.
Notice: 파일에서 데이터를 제대로 읽기가 어렵다. 연습을 많이 해서 제대로 읽는 방법을 익히기 바란다. 특히 구문문자를 잘 이용하고, 문자열 읽는 방법 등을 잘 알아둔다.

27 	fclose(fp);
