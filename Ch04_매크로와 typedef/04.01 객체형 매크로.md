### 04.01 객체형 매크로

매크로는 객체형과 함수형으로 나눌 수 있다. 이 절에서는 객체형 매크로에 대해서 살펴본다. 매크로 이름(식별자:identifier)은 일반적으로 모두 대문자로 쓰는 것을 권장한다. 다른 식별자와 구분하기 위해서이다. 매크로는 정해진 기호와 그 기호에 해당하는 값을 프로그램이 시작하기 전에 선언하는 것이 일반적이다. 이 매크로는 프로그램이 컴파일 될 때 본문중의 내용을 대체하게 된다. 매크로는 하나의 기호로써 프로그램 중에 선언된 기호가 나오게 되면 그 부분에 기호를 없애고 선언된 값으로 대체한다. 따라서 실제 컴파일된 프로그램 내에서는 매크로의 흔적이 없어지게 된다. 매크로를 사용하는 이유는 동일한 내용이 여러 번 나타나게 될 때, 또는 다른 팀원의 프로그램과 같은 값을 공용하여 사용할 때이다. 매크로의 장점은 프로그램 수정이 쉽다는 것이다. 매크로를 프로그램 시작 전에 선언하고, 이 매크로를 이용하여 프로그램을 작성하게 되면 나중에 매크로 값을 수정할 필요가 생겼을 때 매크로 하나만 수정하면 된다. 주의할 점은 매크로 선언 시 문장 끝에 세미콜론은 쓰지 않는다. 써도 되지만 나중에 문제가 생길 소지가 매우 많으므로 안 쓰도록 한다.
#으로 시작하는 문장을 전처리기(preprocessor)라고 한다. 그 의미는 컴파일하기 전에 미리 점검하여 컴파일 옵션을 지정할 수 있도록 한다. #include는 미리 컴파일된 라이브러리를 포함하는 것이고, #define 매크로를 설정하고 그것을 컴파일 할 때에 적용한다. 그 외에도 조건에 따른 컴파일을 할 수 있는 것들이 몇 개 있다.

#### ● 예제 04-01-1 : 매크로 PI 선언
```C
1 #include <stdio.h>
2 #define PI 3.141592		// 객체형 매크로 선언
3 int main()
4 {
5 	double radius;
6 	printf("원의 반지름 입력 : ");
7 	scanf("%lf", &radius);
8 	printf("원의 면적은 : %f\n", radius*radius*PI); 
9 }
```

#### ● 해설
* 2: 매크로 PI를 3.141592라고 선언한다.
* 8: radius*radius*PI는 radius*radius*3.141592로 대체된다.

#### ● 예제 04-01-2 : 매크로 OPEN, CLOSE를 이용한 계산. 호텔의 방에 1번부터 100번까지 번호가 붙어있다. 또, 사람이 100명 있는데 마찬가지로 번호가 1번부터 100번까지 붙여져 있다. 모든 사람이 호텔 방을 1번부터 100번까지 차례대로 지나가면서 자기 번호의 배수에 해당하는 방문을 열려 있으면 닫고, 닫혀있으면 열도록 한다. 100명의 사람이 방을 모두 스쳐 지나가면 방문은 몇 개나 열려 있을까? 단, 처음에는 모두 닫혀있다고 가정한다. 즉, 1번 사람이 방문을 모두 지나가면 모든 방문이 열리게 된다. 모든 정수는 1의 배수이기 때문이다. 2번이 지나가면 짝수는 모두 닫히게 된다. 이런 식으로 100명이 모두 지나간다.
```C
1 #include <stdio.h>
2 #define OPEN 1
3 #define CLOSE 0
4 int main()
5 {
6 	int man, room[101] = {CLOSE, }, room_no=1;
7 	for(man=1; man<=100; man++)
8 		for(room_no=1; room_no<=100; room_no++)
9 			if(room_no % man == 0)
10 				room[room_no] = !room[room_no];
11 
12 	for(room_no=1; room_no<=100; room_no++)
13 		if(room[room_no] == OPEN)
14 			printf("%d ", room_no);
15 }
```

#### ● 해설
* 2,3: 매크로 OPEN, CLOSE를 1, 0으로 선언한다.
* 6: 방을 101개 배열로 선언한다. 배열은 0부터 시작하므로 0은 사용하지 않기로 한다. 초깃값은 모두 CLOSE로 만든다.
* 7~10: 사람이 1번부터 방 100개를 지나가면서 방문의 상태를 현재 상태의 반대로 만든다.
* 9,10: 방의 번호가 사람의 번호로 나누어지면 방문의 상태를 반대로 만든다.
* 12~14: 열려 있는 방문을 출력한다.
> [!NOTE]
> 거꾸로 생각해 보면 방문을 건드리고 지나가는 사람의 수를 세어보면 알 수 있다. 제곱수를 제외한 모든 정수는 약수의 개수가 짝수 개이므로 제곱수만 열려 있게 된다. 즉, 제곱수의 약수는 홀수 개이다.

#### ● 예제 04-01-3 : 매크로 ON, OFF 이용한 계산. 실내 온도에 맞춰 에어컨과 난방기를 작동하도록 프로그램한다. 25도를 초과하게 되면 에어컨을 ON하고, 10도 미만이면 난방기를 ON한다. 그 사이의 값일 때는 둘 다 OFF로 한다. 2초 간격으로 온도를 점검한다. 여기서는 온도를 난수를 이용하여 자동으로 발생하게 만든다. 온도의 범위는 –40도에서 +40도 까지로 하였다. while 문은 5장에서 설명한다. 여기서 while은 무한히 반복한다.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 #define ON 1
5 #define OFF 0
6 int main()
7 {
8  	int degree=1, aircon = ON, hitter = OFF;
9  	int state = -1, start;
10  	srand(time(0));
11  	while(1) {
12 		start = time(0);
13 		while(time(0) - start < 2);
14 		degree = rand()%80 - 40;
15 		printf("온도 : %3d\n", degree); 
16 		if(degree > 25 && state != 1) {
17 			aircon = ON;
18 			hitter = OFF;
19 			state = 1;
20 			printf("aircon ON, hitter OFF\n");
21 		}
22 		else if(degree < 10 && state != 2) {
23 			aircon = OFF;
24 			hitter = ON;
25 			state = 2;
26 			printf("aircon OFF, hitter ON\n");
27 		}
28 		else if(degree >= 10 && degree <= 25 && state != 0) {
29 			aircon = OFF;
30 			hitter = OFF;
31 			state = 0;
32 			printf("aircon OFF, hitter OFF\n");
33 		}
34 	}
35 }
```

#### ● 해설
* 4,5: 매크로 ON, OFF를 선언한다.
* 8: 초기 상태를 지정한다.
* 9: state는 현재 에어컨과 난방기 중 무엇이 작동하고 있는지를 나타낸다. 1이면 에어컨이 켜진 상태이고, 2이면 난방기가 켜진 상태, 0이면 둘다 꺼진 상태를 나타낸다. start는 시간차이를 계산하기 위한 변수이다.
* 10: 난수를 초기화 한다.
* 11~34: 무한 루프를 수행하면서 현재 상황을 계속 점검한다.
* 12,13: 2초 간격으로 점검하기 위한 구문
* 14: 난수를 이용하여 온도를 발생한다.
* 16~21: 에어컨을 켜기 위한 동작
* 22~27: 난방기를 켜기 위한 동작
* 28~33: 에어컨과 난방기 둘 다 끄기 위한 동작, 상태가 변하지 않으면 이들 동작은 그 상태를 계속 유지한다.
> [!NOTE]
> 온도의 범위는 여러분이 적절하게 바꾸어 사용하라.

#### ● 예제 04-01-4 : 두 문자열 결합(##) 사용 
```C
1 #include <stdio.h>
2 #define output(x) printf("%d %d\n", x ## 1, x ## 2)
3 int main()
4 {
5 	int count1, count2, i1, i2;
6 
7 	count1 = 10;
8 	count2 = 20;
9 	i1 = 99;
10 	i2 = -10;
11 
12 	output(count);
13 	output(i);
14 }
```

#### ● 해설
* 2: output(x)는 12, 13번 행을 대체한다. x 대신에 count, i를 체하고 ##뒤의 문자와 결합하여 새로운 문자열을 생성하게 된다. 즉, count1, count2로 문자열이 만들어지고, i1, i2로 문자열이 만들어져	대체된다.
* 5: 변수 선언
* 7~10: 변수에 값 저장
* 12,13: 매크로를 대체한다. 다만 count, i를 사용하고 매크로에서 이들 문자 뒤에 1, 2를 붙여서 새로운 문자열을 만들어 그에 해당하는 변수명으로 만들게 된다.
> [!NOTE]
> 아주 제한적으로 사용될 수 있기 때문에 크게 이용할 일은 별로 없을 것이다.

#### ● 예제 04-01-5 : 문자열로 만들기(#) 사용 
```C
1 #include <stdio.h>
2 #define MKSTRING(str) # str
3 int main()
4 {
5 	int value;
6 	value = 10;
7 	printf("%s is %d\n", MKSTRING(value), value);
8 }
```

#### ● 해설
* 2: 문자열 만들기, 매크로에 식별자가 입력되면 그것을 문자열로 만들어 준다.
* 7: MKSTRING(value)는 식별자 value를 문자열 “value”로 만든다.
> [!NOTE]
> 이 방법 또한 잘 사용하지는 않는다. 크게 이용할 일은 없을 것으로 보인다.
