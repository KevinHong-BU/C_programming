### 04.03 typedef
typedef는 기본 데이터형의 이름에 별명을 붙여서 다르게 사용하고자 할 때 사용할 수 있다. 또 다른 사용법으로는 8장에서 설명할 사용자 정의 데이터형(구조체, 공용체, 열거형)을 새로운 데이터 형으로 선언할 때도 사용할 수 있다.

● typedef의 용도는 2가지이다.<br>
● 기본 자료형의 이름에 별명을 붙여 다르게 사용하고자 할 때<br>
● 사용자 정의 자료형을 선언할 때 : 뒤에 배울 구조체, 공용체, 열거형 등의 자료형을 선언할 때<br>

#### ● 예제 04-03-1 : typedef를 이용하여 double을 real로 이름을 바꾸어 사용
```C
1 #include <stdio.h> 
2 #include <math.h>
3 #ifndef PI
4 #define PI 3.141592
5 #endif
6 typedef double real;
7 int main()
8 {
9 	int i;
10 	real s=0.0, r=10.0;
11 	s = 2.*PI*r;
12 	printf("s = %f\n", s);
13 }
```

#### ● 해설
* 6: typedef를 이용하여 double을 real로 이름을 바꾸어 사용한다. 따라서 double이나 real은 같은 것이다. 별명을 사용한다고 생각하면 된다.
* 10: real 데이터 형으로 s와 r 변수를 선언한다.
> [!NOTE]
> 실제로 size_t나 다른 몇몇 데이터 형이 이와 같이 지정되어 사용되고 있다.

#### ● 예제 04-03-2 : 구조체를 typedef를 이용하여 간단한 데이터 형 이름을 붙여서 사용하기
```C
1 #include <stdio.h> 
2 #include <math.h>
3 
4 typedef struct {
5 	double real;
6 	double imag;
7 } complex;        // new type is complex
8 
9 int main()
10 {
11 	complex c;
12 	double mag;
13 	c.real = 3.5;
14 	c.imag = 5.0;
15 
16 	mag = sqrt(c.real*c.real + c.imag*c.imag);
17 	printf("복소수의 크기 = %f\n", mag);
18 }
```

#### ● 해설
* 3~6: 구조체를 정의하고 새로운 데이터 형의 이름을 complex로 지정한다.
* 9: 새로운 데이터 형을 이용하여 변수 선언
> [!NOTE]
> 구조체는 이렇게 사용한다는 정도만 알아두고 자세한 내용은 다음 9장에서 살펴보자. C99에는 복소수형이 표준으로 정의되어 있다.
