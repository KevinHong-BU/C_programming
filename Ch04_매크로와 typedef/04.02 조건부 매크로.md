### 04.02 조건부 매크로
조건부 매크로는 매크로의 선언 여부를 판단하여 새로 지정하거나 기존에 지정된 매크로를 해제할 때 사용할 수 있다. #if, #ifdef, #ifndef, #undef 등이 있다. 매크로의 존재 여부에 따라 컴파일 조건이 달라진다는 의미이다.

#### ● 예제 04-02-1 : 매크로의 선언 여부에 따른 컴파일
```C
1 #include <stdio.h>
2 #define DEBUG 1
3 int main()
4 {
5 	int i, sum=0;
6 	for( i = 1; i<=10; i++) {
7 		sum += i;
8 		#if DEBUG
9 			printf("%d ", sum);
10 		#endif
11 	}
12 	printf("sum = %d\n", sum);
13 }
```

#### ● 해설
* 2: 매크로 DEBUG를 1로 선언
* 8~10: 매크로 DEBUG의 선언 여부에 따라 이 부분이 컴파일 될 수도 안 될 수도 있다. 여기서는 DEBUG가 1이므로 9번 행을 컴파일하게 된다. 따라서 실행결과를 보면 9번 행이 컴파일 되어 실행됨을 알 수 있다.
> [!NOTE]
> <예제 04-02-2>는 반대의 상태이다. 참고하라.

---
#### ● 예제 04-02-2 : 매크로의 선언 여부에 따른 컴파일
```C
1 #include <stdio.h>
2 #define DEBUG 0
3 int main()
4 {
5 	int i, sum=0;
6 	for( i = 1; i<=10; i++) {
7 		sum += i;
8 		#if DEBUG
9 			printf("%d ", sum);
10 		#endif
11 	}
12 	printf("sum = %d\n", sum);
13 }
```

#### ● 해설
* 2: 매크로 DEBUG를 0으로 선언
* 8~10: 매크로 DEBUG의 선언 여부에 따라 이 부분이 컴파일 될 수도 안 될 수도 있다. 여기서는 DEBUG가 0이므로 9번 행을 컴파일하지 않는다. 따라서 8~10번 행은 없는 것이다.
> [!NOTE]
> <예제 04-02-1>과 비교해 보라. 이러한 예는 프로그램 개발 시 중간 중간에 필요한 데이터를 출력해 봄으로써 디버깅 하는데 유용하게 사용할 수 있다. 간단한 디버깅을 할 수 있다. 완전히 개발한 다음에 매크로 값만 변경해서 재 컴파일 하면 해당되는 부분은 실행에서 빠지게 된다.

---
#### ● 예제 04-02-3 : 조건부 매크로 #ifdef의 사용
```C
1 #include <stdio.h> 
2 #include <math.h>
3 int main()
4 {
5 	int i;
6 	double s=0.0, r=10.0;
7 	#ifdef PI
8 		s = 2.*PI*r;
9 	#else
10 		s = 2.*3.14*r;
11 	#endif
12 	printf("s = %f\n", s);
13 }
```

#### ● 해설
* 2: math.h의 수학함수는 사용하지 않지만 혹시 PI라는 매크로가 선언되었는지를 모르기 때문에 포함하여 컴파일한다. 
* 7~11: PI라는 매크로가 math.h에 선언되어 있는지를 검사하고, 그 여부에 따라 적절하게 컴파일할 문장이 달라지게 된다.
> [!NOTE]
> math.h에는 PI가 선언되지 않았다는 것을 알 수 있다. 즉, 결과를 보면 10번 행이 실행됨을 알 수 있다.


---
#### ● 예제 04-02-4 : 조건부 매크로 #undef의 사용
```C
1 #include <stdio.h> 
2 #include <math.h>
3 int main()
4 {
5 	int i;
6 	double s=0.0, r=10.0;
7 	#undef 	PI
8 	#define	PI	3.141592
9 	s = 2.*PI*r;
10 	printf("s = %f\n", s);
11 }
```

#### ● 해설
* 7: PI라는 매크로가 정의되어 있더라도 없는 것으로 만든다.
* 8: 다시 PI를 정의한다.
* 9: 정의된 PI를 이용하여 계산한다.
> [!NOTE]
> 미리 정의되어 있어도 없는 것으로 하고 새로 정의한다. 기존에 매크로가 정의되어 있다고 하더라도 무시하고 새로 정의한다.

---
#### ● 예제 04-02-5 : 조건부 매크로 #ifndef의 사용
```C
1 #include <stdio.h> 
2 #include <math.h>
3 #ifndef PI
4 #define PI 3.141592
5 #endif
6 int main()
7 {
8 	int i;
9 	double s=0.0, r=10.0;
10 	s = 2.*PI*r;
11 	printf("s = %f\n", s);
12 }
```

#### ● 해설
3~5: 매크로 PI가 정의되어 있지 않다면 새로 정의한다. 미리 정의되어 있다면 기존의 매크로를 사용하고 그렇지 않으면 새로 정의한다.

#### ● 예제 04-02-6 : 표준 매크로 __DATE__, __FILE__, __LINE__, __STDC__, __TIME__의 사용
```C
1 #include <stdio.h> 
2 int main()
3 {
4 	printf("%s%s\n%s%s\n%s%d\n%s%d\n%s%s\n",
5 		"__DATE__=", __DATE__,
6 		"__FILE__=", __FILE__,
7 		"__LINE__=", __LINE__,
8 		"__STDC__=", __STDC__,
9 		"__TIME__=", __TIME__);
10 	printf("C version = %d\n", __STDC_VERSION__);
11 }
```

#### ● 해설
* 5: 표준 매크로 __DATE__는 프로그램이 실행되는 날짜를 문자열로 나타낸다.
* 6: 표준 매크로 __FILE__은 프로그램의 이름과 디렉터리를 문자열로 나타낸다.
* 7: 표준 매크로 __LINE__은 현재 프로그램 내에서의 행의 번호를 정수로 나타낸다.
* 8: 표준 매크로 __STDC__은 현재 C컴파일러가 표준인지 아닌지를 나타낸다. 표준이면 1이고 그렇지 않으면 다른 값을 나타낸다.
* 9: 표준 매크로 __TIME__은 현재 프로그램이 실행될 때의 시스템의 시간을 문자열로 나타낸다.
> [!NOTE]
> Notice: 실행결과와 같이 간단하게 프로그램 내에서 정보를 표시한다.
