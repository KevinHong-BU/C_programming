#### 비교연산자(관계연산자: 참, 거짓 판단)
| 연산자 이름 | 문법 | 예: a=10, b=20일 때 |
|:---:|:---|:---|
| 작음 | a < b | x = a < b; // a가 b보다 작으므로 참이다. 그러므로 x=1 |
| 작거나 같음 | a <= b | x = a <= b; // a가 b보다 작거나 같으므로 참이다. 그러므로 x=1 |
| 큼 | a > b | x = a > b; // a가 b보다 크지 않으므로 거짓이다. 그러므로 x=0 |
| 크거나 같음 | a >= b | x = a >= b; // a가 b보다 크거나 같지 않으므로 거짓이다. 그러므로 x=0 |
| 같지 않음 | a != b | x = a != b; // a는 b와 같지 않으므로 참이다. 그러므로 x=1 |
| 같음 | a == b | x = a != b; // a는 b와 같지 않으므로 거짓이다. 그러므로 x=0 |


#### 논리연산자(참, 거짓 판단)

<table>
  <tr>
    <th>연산자 이름</th>
    <th>문법</th>
    <th>예: a=10, b=20일 때</th>
  </tr>
  <tr>
    <td>논리부정(NOT)</td>
    <td>!a</td>
    <td>!( x < y); // x가 y보다 작으므로 참이다. 따라서 !참이므로 결과는 거짓이다.</td>
  </tr>
  <tr>
    <td>논리AND(logical AND)</td>
    <td>a && b</td>
    <td>x<10 && y<100; // 두 조건이 모두 참이므로 논리 AND는 참이다.</td>
  </tr>
  <tr>
    <td>논리OR(logical OR)</td>
    <td> a || b</td>
    <td>x<10 || y<10; // 하나의 조건이라도 만족하므로 논리 OR는 참이다.</td>
  </tr>
</table>


#### 비트연산자(비트 단위 연산: 정수형의 데이터에만 적용)
<table>
  <tr>
    <th>연산자 이름</th>
    <th>문법</th>
    <th>예: x=10, y=20일 때 (2진수로 10=000010102, 20=000101002이다)</th>
  </tr>
  <tr>
    <td>왼쪽시프트</td>
    <td>a << b</td>
    <td>x = x << 2;<br> // x를 왼쪽으로 2자리 시프트(곱하기 22)한다. 따라서 x는 40(001010002)이 된다.</td>
  </tr>
  <tr>
    <td>왼쪽시프트 대입</td>
    <td>a <<= b</td>
    <td>x <<= 2;<br> // x를 왼쪽으로 2자리 시프트(곱하기 22)한다. 따라서 x는 40(001010002)이 된다.</td>
  </tr>
  <tr>
    <td>오른쪽시프트</td>
    <td>a >> b</td>
    <td>y = y >> 2;<br> // y를 오른쪽으로 2자리 시프트 (나누기 22)한다. 따라서 y는 5(000001012)가 된다.</td>
  </tr>
  <tr>
    <td>오른쪽시프트 대입</td>
    <td>a >>= b</td>
    <td>y >>= 2;<br> // y를 오른쪽으로 2자리 시프트(나누기 22)한다. 따라서 y는 5(000001012)가 된다.</td>
  </tr>
  <tr>
    <td>비트부정(1의 보수, NOT)</td>
    <td>~a</td>
    <td>a = ~x;<br> // a는 11110101이 된다.</td>
  </tr>
  <tr>
    <td>비트AND</td>
    <td>a & b</td>
    <td>a = x & y;<br> // 각 자리수를 모두 AND한다. 따라서 0이 된다.</td>
  </tr>
  <tr>
    <td>비트AND 대입</td>
    <td>a &= b</td>
    <td>x &= y;<br> // 각 자리수를 모두 AND한다. 따라서 000000이 된다.</td>
  </tr>
  <tr>
    <td>비트OR</td>
    <td>a | b</td>
    <td>a = x | y;<br> // 각 자리수를 모두 OR한다. 따라서 000111102(=30)이 된다.</td>
  </tr>
  <tr>
    <td>비트OR 대입</td>
    <td>a |= b</td>
    <td>x |= y;<br> // 각 자리수를 모두 OR한다. 따라서 000111102(=30)이 된다.</td>
  </tr>
  <tr>
    <td>비트XOR</td>
    <td>a ^ b</td>
    <td>a = x ^ y;<br> // 각 자리수를 모두 XOR한다. 따라서 000000002이 된다.</td>
  </tr>
  <tr>
    <td>비트XOR 대입</td>
    <td>a ^= b</td>
    <td>x ^= y;<br> // 각 자리수를 모두 XOR(다르면 1, 같으면 0)한다. 따라서 000000002이 된다.</td>
</table>

#### 기타연산자(포인터, 멤버, 배열, 3항 조건연산자, 형 변환 등)
<table>
  <tr>
    <th>연산자 이름</th>
    <th>문법</th>
    <th>예: x=10, y=-20, *xp, array[10]; 일 때</th>
  </tr>
  <tr>
    <td>단순대입</td>
    <td>a = b</td>
    <td>x = y; // x는 -20이 된다.</td>
  </tr>
  <tr>
    <td>함수호출</td>
    <td>a()</td>
    <td>x = abs(y); // x는 20이 된다.</td>
  </tr>
  <tr>
    <td>배열첨자</td>
    <td>a[b]</td>
    <td>x = array[5]; // x에는 배열 array의 6번째 요소의 값이 저장된다.</td>
  </tr>
  <tr>
    <td>포인터(간접연산:dereference)</td>
    <td>*a</td>
    <td>xp = &x; y = *xp; // y는 -20이 된다.</td>
  </tr>
  <tr>
    <td>주소연산자(reference)</td>
    <td>&a</td>
    <td>xp = &x; // xp는 x의 주소가 저장된다.</td>
  </tr>
  <tr>
    <td>포인터에 의한 멤버호출</td>
    <td>a->b</td>
    <td>a가 구조체 포인터 변수이고, b는 구조체의 멤버 일 때, 구조체의 멤버를 호출한다.</td>
  </tr>
  <tr>
    <td>멤버</td>
    <td>a.b</td>
    <td>a가 구조체 변수이고, b는 구조체의 멤버 일 때, 구조체의 멤버를 호출한다.<br> C++에는 없음</td>
  </tr>
  <tr>
    <td>형변환</td>
    <td>(type) a</td>
    <td>(double) a; // a의 데이터 형을 double로 변환한다.</td>
  </tr>
  <tr>
    <td>콤마</td>
    <td>a, b</td>
    <td>여러 개의 변수나 상수를 나열할 때 쓴다.</td>
  </tr>
  <tr>
    <td>3항 조건연산자</td>
    <td>a ? b : c</td>
    <td>a = x<10 ? 0 : 1;  // x가 10보다 작으면 a에 0을 그렇지 않으면 a에 1을 저장한다.<br> C++에는 없음</td>
  </tr>
  <tr>
    <td>크기반환연산자</td>
    <td>sizeof(a)</td>
    <td>sizeof(int); // 데이터 형의 크기를 바이트 단위로 반환한다. int형은 4바이트이다.<br> C++에는 없음</td>
</table>
