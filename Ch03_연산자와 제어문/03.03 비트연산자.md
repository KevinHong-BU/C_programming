### 03.03 비트연산자

이 절에서는 비트(bitwise)연산자가 무엇이며, 어떤 용도로 사용하는지 예제를 통해 살펴보자. 비트연산자는 다른 연산자와 달리 비트 하나하나를 일일이 비교하여 연산하는 것이다. 비트연산자는 정수형 데이터에서만 가능하며, 2진수에 대해서 잘 알고 있어야 그 결과를 이해할 수 있을 것이다.

#### ● 예제 03-03-1 : 시프트 연산
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=56, y;		// 56=00000000 00000000 00000000 00111000
5 	y = x << 2;			// y= 00000000 00000000 00000000 11100000 = 224
6 	printf("%d\n", y);
7 	y = x >> 2;			// y= 00000000 00000000 00000000 00001110 = 14
8 	printf("%d\n", y);
9 	x = -56;			// -56=11111111 11111111 11111111 11001000
10 	y = x << 2;			// y= 11111111 11111111 11111111 00100000 = -224
11 	printf("%d\n", y);
12 	y = x >> 2;			// y= 11111111 11111111 11111111 11110010 = -14
13 	printf("%d\n", y);
14 }
```

#### ● 해설
* 5: `x`를 왼쪽으로 2비트 시프트한다. 오른쪽의 2비트 빈자리는 0으로 채운다.
* 6: 결과는 (=4)을 곱한 값인 56*4=224가 된다. 왼쪽으로 한번 시프트하면 2(), 두 번 시프트하면 4(=), 3번 시프트하면 8(=)을 곱한 결과가 된다. 즉 번 시프트하면 을 곱하는 결과가 된다.
* 7: `x`를 오른쪽으로 2비트 시프트한다. x가 양수이므로 왼쪽의 2비트 빈자리는 0으로 채운다.
* 8: 결과는 (=4)을 나눈 값인 56/4=14가 된다. 오른쪽으로 한번 시프트하면 1/2(=), 두 번 시프트하면 1/4(=), 3번 시프트하면 1/8(=)을 곱한 결과가 된다. 즉 번 시프트하면 을 곱한 결과가 된다.
* 10: 음수인 –56을 왼쪽으로 2자리 시프트한다. 결과는 –224가 된다.
* 12: 음수인 –56을 오른쪽으로 2자리 시프트한다. 이때는 왼쪽의 빈자리는 1로 채운다. 결과는 –14가 된다.
> [!NOTE]
> 컴퓨터에서는 2의 보수를 사용한다. 따라서 부호가 0이면 양수, 1이면 음수이며, 오른쪽으로 시프트할 때 부호를 채우게 된다.

#### ● 예제 03-03-2 : 비트단위 논리연산
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char x=56, y=49, z; 	// 56: 00111000, 49: 00110001
5 	z = x & y;			// 00111000&00110001=00110000=16+32=48
6 	printf("%d\n", z);
7 	z = x | y;			// 00111000|00110001=00111001=16+32+8+1=57
8 	printf("%d\n", z);
9 	z = x ^ y;			// 00111000^00110001=00001001=8+1=9
10 	printf("%d\n", z);
11 	z = ~x;			// ~00111000=11000111=-128+64+4+2+1=-57
12 	printf("%d\n", z);
13 }
```

#### ● 해설
* 5: 비트별로 AND 연산을 한다. 따라서 56과 49를 AND하면 48이 된다.
* 7: 비트별로 OR 연산을 한다. 따라서 56과 49를 OR하면 57이 된다.
* 9: 비트별로 XOR 연산을 한다. 따라서 56과 49를 XOR하면 9가 된다.
* 11: `x`의 각 비트를 반대의 상태로 바꾼다. 즉, 0은 1로, 1은 0으로 바꾼다. 이것을 1의 보수라 한다. 따라서 –57이 된다.
> [!NOTE]
> 여기서 `x`는 ASCII 문자이므로 8비트 데이터이다. 16비트나 32 정수 데이터에서도 동일하게 적용된다. XOR는 비트가 서로 다른 값이면 1, 같은 값이면 0이다.
