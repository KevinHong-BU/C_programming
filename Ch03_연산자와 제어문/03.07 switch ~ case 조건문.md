### 03.07 switch ~ case 조건문
이 절에서는 조건이 여러 가지 일 때 사용하는 `switch ~ case`를 보자. `if` 조건 중에서 특별한 경우 `switch ~ case`를 사용하면 좋은 경우가 있다. `if`보다 한 눈에 구문을 파악할 수 있지만 한 가지 조건이 따른다. 이 조건은 `switch`속의 조건이 반드시 정수형을 나타내는 식 또는 변수이어야 한다.

<br>switch ~case 문

`case`에 지정하는 값은 정수형 상수이다. 각 `case`에는 `break;`를 사용하여 다음 조건에 해당하는 문장을 실행하지 않도록 하는 것이 보통이며, 가끔은 `break;`를 사용하지 않고 여러 개의 조건을 논리적 OR로 실행하는 경우도 있다. 즉, 조건이 여러 개일 때는 `break;`를 필요한 곳에만 넣을 수도 있다(<예제03-07-2>).

---
#### ● 예제 03-07-1 : switch ~ case를 이용한 짝수, 홀수 판별
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수: ");
6 	scanf("%d", &x);
7 	switch(x%2) {
8 		case 0:
9 			 printf("짝수\n");
10 			 break;
11 		case 1: // default:로 하여도 된다.
12 			 printf("홀수\n");
13 	} 
14 }
```

#### ● 해설
* 7: `x%2`는 정수식이다. 이 식의 결과(0 또는 1)에 따라서 `case` 실행문이 달라진다.
* 8: `x%2`의 결과가 0일 경우 9번 문장을 실행한다.
* 10: `break`를 만나면 `switch ~ case`를 빠져나간다.
* 11: `x%2`의 결과가 1일 경우 12번 문장을 실행한다. 이 예에서는 조건이 0, 1 두 가지 밖에 없으므로 `case 1:` 대신에 `default:`를 사용해도 무방하다.
> [!NOTE]
> 조건은 반드시 정수형이어야 하며, `case`에는 정수형 상수가 와야 한다. `break;`는 더 이상  `switch ~ case` 속의 다음 문장을 실행하지 않게 한다. 즉, `switch ~ case` 블록을 빠져나간다.

---
#### ● 예제 03-07-2 : `switch ~ case`를 이용한 점수 평가
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("점수: ");
6 	scanf("%d", &x);
7 	switch(x/10) {			// 10으로 나눈 몫을 계산한다.
8 		case 10:
9  		case 9:
10 			printf("A\n");
11 			break;
12 		case 8:
13 			printf("B\n");
14 			break;
15 		case 7:
16 			printf("C\n");
17 			break;
18 		case 6:
19 			printf("D\n");
20 			break;
21 		default :
22 			printf("F\n");
23     }
24 }
```

#### ● 해설
* 7: `x/10`은 정수식이다. x가 0부터 100까지가 입력되면 이 식의 결과는 0부터 10까지 값을 가지게 된다.
* 8: 결과가 10이면 이어서 실행한다. 즉, `case 9:`가 `case 10:`에 포함된 것처럼 동작한다. 즉, A를 출력한다.
* 9: 결과가 9이면 A를 출력하고 `switch ~ case`를 끝낸다.
* 12: 결과가 8이면 B를 출력하고 `switch ~ case`를 끝낸다.
* 21: 결과 값이 5이하이면 F를 출력하고 `switch ~ case`를 끝낸다.
> [!NOTE]
> 조건이 여러 개 일 때는 `break;`를 쓰지 않고 이어지는 다른 문장을 나열한다.


---
#### ● 예제 03-07-3 : case에 여러 개의 문장이 포함되는 경우
```C
1 #include <stdio.h>
2 int main()
3 {
4     float tc, tf;
5     int select;
6     printf(" 1 : 화씨 -> 섭씨\n 2 : 섭씨 - > 화씨\n");
7     printf("선택 : "); 
8     scanf("%d", &select);
9     switch (select) {
10 		case 1:
11 			printf("화씨온도를 입력하시오 : ");
12 			scanf("%f", &tf);
13 			tc = ( tf -32 ) * 5. / 9.;
14 			printf("화씨온도 %5.1f는 섭씨온도 %5.1f입니다.\n", tf, tc);
15 			break;
16 		case 2:
17 			printf("섭씨온도를 입력하시오 : ");
18 			scanf("%f", &tc);
19 			tf = 9. / 5. * tc + 32.;
20 			printf("섭씨온도 %5.1f는 화씨온도 %5.1f입니다.\n", tc, tf);
21 			break;
22 		default:
23 			printf("잘못된 선택입니다.\n"); 
24 	}
25 }
```

#### ● 해설
* 6,7: 입력에 대한 설명을 출력한다.
* 8: 선택할 번호를 입력한다.
* 9: select는 정수형 변수이다.
* 10: 선택한 번호가 1일 때 11~14까지 실행한다.
* 15: 선택한 번호 1에 대한 명령이 모두 실행되면 `switch ~ case`를 벗어난다.
* 16: 선택한 번호가 2일 때 17~20까지 실행한다.
* 21: 선택한 번호 2에 대한 명령이 모두 실행되면 `switch ~ case`를 벗어난다.
* 22: 선택한 번호가 1 또는 2가 아닐 때 에러 메시지 23번을 실행하고 끝낸다.
> [!NOTE]
> 한 조건에 대한 명령이 여러 개일 때 `break;` 이전까지 계속 실행한다.

#### ● 예제 03-07-4 : 오늘은 올해의 몇 번째 날인가?
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int year, month, date, dates, leap=0;
5 
6 	printf("년 :");
7 	scanf("%d", &year);
8 	printf("월 :");
9 	scanf("%d", &month);
10 	printf("일 :");
11 	scanf("%d", &date);
12 
13 	dates = date;
14 	leap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0); // 윤년 판단 
15 	switch(month) {
16 		case 2: dates += 31; break;
17 		case 3: dates += (leap+59); break;
18 		case 4: dates += (leap+90); break;
19 		case 5: dates += (leap+120); break;
20 		case 6: dates += (leap+151); break;
21 		case 7: dates += (leap+181); break;
22 		case 8: dates += (leap+212); break;
23 		case 9: dates += (leap+243); break;
24 		case 10: dates += (leap+273); break;
25 		case 11: dates += (leap+304); break;
26 		case 12: dates += (leap+334); break;
27 	}
28 
29 	if( (month <0 || month >12) || (date < 0 || date > 31) )
30 		printf("Error\n"); 
31 	else 
32 		printf("%d년 %d월 %d일은 %d년의 %d번째 날입니다.\n",
33 			year, month, date, year, dates);
34 }
```

#### ● 해설
* 4: 각종 변수 선언
* 6~11: 년, 월, 일 입력
* 14: 윤년인지 판단한다.
* 15~27: 해당 월이 입력되면 그 월 이전까지 며칠인지 `switch ~ case`로 계산한다. 단 2월 이후부터는 윤년이면 0 또는 1(leap)을 더한다.
* 29~30: 잘못된 입력이 있을 경우 에러메시지 출력. 완벽하진 않지만 단순하게 써 보았다.
* 31~33: 정상적인 입력일 경우 오늘까지의 날짜를 출력한다.
