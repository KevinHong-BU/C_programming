● 예제 03-04-1 : 임시변수를 이용한 변수 교환
1 #include <stdio.h>
2 int main()
3 {
4 	int x=450, y=341, temp;
5 	printf("교환 전: x=%d, y=%d\n", x, y);
6 	temp = x;
7 	x = y;
8 	y = temp;
9 	printf("교환 후: x=%d, y=%d\n", x, y);
10 }
● 해설
6: x를 temp에 저장한다.
7: y를 x에 저장한다.
8: temp를 y에 저장한다. 이런 과정을 거치면 두 개의 값, 즉, x, y가 바뀐다.
Notice: 보통 이와 같은 방법으로 두 개의 변수의 값을 바꾼다. 일단 하나의 변수에 다른 값이 저장되면 먼저 있던 값은 없어진다. 그러므로 다른 임시 변수에 잠시 보관해 둔다.

● 예제 03-04-2 : XOR를 이용한 정수 변수 교환
1 #include <stdio.h>
2 int main()
3 {
4 	int x=50, y=41;
5 	printf("교환 전: x=%d, y=%d\n", x, y);
6 	x = x ^ y; // 정수연산만 가능하다.
7 	y = x ^ y;
8 	x = x ^ y;
9 	printf("교환 후: x=%d, y=%d\n", x, y);
10 }
● 해설
6: 50:00110010, 41:00101001, 
	x = x^y = 50^41 = 00110010^00101001=00011011(27), 
	y = x^y = 27^41 = 00011011^00101001=00110010(50),
	x = x^y = 27^50 = 00011011^00110010=00101001(41)
Notice: XOR(^)연산은 피연산자가 정수일 때만 가능하다.

● 예제 03-04-3 : 연산 순서와 대입을 통한 변수 교환
1 #include <stdio.h>
2 int main()
3 {
4 	int x=450, y=341;
5 	printf("교환 전: x=%d, y=%d\n", x, y);
6 	x = x + y - ( y = x );
7 	printf("교환 후: x=%d, y=%d\n", x, y);
8 }
● 해설
6: x와 y를 계산한 뒤, x를 y에 대입한 다음 뺀다. 
Notice: 이 예는 앞에서도 나왔던 예이다. 결합 순서를 잘 따져보자.

● 예제 03-04-4 : 축약형 대입연산자의 이용
1 #include <stdio.h>
2 int main()
3 {
4 	int x=5, y=3;
5 	x += y+10;					// x = x + (y*10);
6 	printf("x=%d\n", x);
7 	x=5; y=3; x -= 5*y;			// x = x - (5*y);
8 	printf("x=%d\n", x);
9 	x=5; x *= 3;				// x = x * 3;
10 	printf("x=%d\n", x);
11 	x=5; x /= 2;				// x = x / 2;
12 	printf("x=%d\n", x);
13 	x=5; x %= 3;				// x = x % 3;
14 	printf("x=%d\n", x);
15 	x=5; x <<= 2;				// x = x << 2;
16 	printf("x=%d\n", x);
17 	y=3; y >>= 4;				// y = y >> 4;
18 	printf("y=%d\n", y);
19 	x=5; y=3; x &= y;			// x = x & y;
20 	printf("x=%d\n", x);
21 	x=5; x |= 255;				// x = x | 255;
22 	printf("x=%d\n", x);
23 	x=5; x ^= 0x00ff;			// x = x ^ 0x00ff;
24 	printf("x=%d\n", x);
25 }
● 해설
5: y에 10을 더해서 그것을 x에 더한다.
15: x를 두 번 왼쪽을 시프트 한 다음에 다시 x에 저장한다.
Notice: 이들 복합 대입연산자는 오른쪽의 식을 모두 평가한 다음에 마지막에 결과를 좌측의 변수와 다시 계산한다.
