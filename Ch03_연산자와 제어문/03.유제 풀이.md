```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수 입력 : ");

7 	x%2 ? printf("홀수\n") : printf("짝수\n");

```

#### ● 해설
* x를 2로 나누면 그 결과는 0또는 1의 값을 가진다. C언어에서는 0은 거짓이며, 0 이외의 어떤 값은 모두 참으로 간주한다. 따라서 여기서는 x%2가 1(참)이면, 즉 홀수이면 printf("홀수\n")이 실행되고, x%2가 0(거짓)이면, 즉 짝수이면 printf("짝수\n")가 실행된다.

#### ● 유제 03-03-1 : 비트연산자를 이용하여 어떤 양의 정수를 0으로 만드는 방법을 제시하라.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x = 450;
5 	x = x & 0;
6 	printf("%d\n", x);
7 	x = 450;
8 	x = x ^ x;
9 	printf("%d\n", x);
10 	x = 450; //음수이면 안 된다. 음수일 때는 결과가 -1이 된다.
11 	x = x >> 31;
12 	printf("%d\n", x);
13 	x = 450;
14 	x = x + ~x + 1;
15 	printf("%d\n", x);
16 }
```

#### ● 해설
* 5: 어떤 값 x와 0을 비트 AND하면 0이 된다. 0은 모든 비트가 0이므로 어떤 값과 AND를 해도 결과는 0이다.
* 8: x와 x를 XOR하면 0이 된다. 같은 값을 XOR하면 모든 비트가 같으므로 결과는 0이 된다.
* 11: 양의 정수 x를 오른쪽으로 31비트 시프트하면 모든 비트가 0으로 채워진다. 즉, 부호가 0이므로 이동한 자리는 부호 0으로 채워서 모두 0이 된다. 만약에 음수이면 모두 부호 1로 채워져 –1이 된다.
> [!NOTE]
> 2의 보수체계에 대해서 알아두면 도움이 된다.

#### ● 유제 03-03-2 : 비트연산자를 이용하여 양수인지 음수인지를 판별하여 출력하라.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=-15, sign;
5 	sign = x & 0x80000000;
6 	sign ? printf("%d=음수\n", x) :  printf("%d=양수\n", x);
7 }
```

#### ● 해설
* 5: 2의 보수 체계에서 최상위 비트(MSB)가 1이면 음수, 0이면 양수이다. 따라서 최상위 비트가 무엇인지 안다면 쉽게 양수와 음수를 찾을 수 있다. 최상위비트를 x와 비트 AND하면 그 결과에 따라 판정할 수 있다. 최상위비트를 1로 지정하는 방법은 0x80000000이다. 8은 2진수로 1000이므로 32비트 중에서 최상위비트가 1로 지정된다.
* 6: sign이 0이면 거짓, 그 이외의 값이면 참이다. 이 연산에서는 x가 –15이므로 최상위비트가 1이며, 따라서 결과는 0이 아닌 값이므로 참이다. 결과는 “음수”가 출력된다.
> [!NOTE]
> 2진수 정수는 32비트(4바이트)이다.

```C
13 		   ( root = (-b + sqrt(d))/(2.*a), root2 = (-b - sqrt(d))/(2.*a));

```

#### ● 해설
* 8: 2차 방정식의 3개의 계수 a, b, c 입력

* 12,13: 판별식에 따라 근을 계산, 0일 때와 그렇지 않을 때 계산

* 14: 계산 결과 출력, 중근일 때 한 개의 근 출력, 그렇지 않을 때 두 개의 근 출력

#### ● 유제 03-06-3 : 내일의 날짜 구하기연월일을 입력하면 그 다음 날을 출력하는 프로그램을 작성하라. 예를 들어 2010년 4월 30일 다음날은 2010년 5월 1일이다. 또 2012년 2월 28일 다음날은 2012년 2월 29일이다. 모든 경우의 수를 생각하여 프로그램 하라.

```C
46 	}

53 		} else
54 			date++;		// 그렇지 않으면 날짜만 1 더한다.
```

#### ● 유제 03-08-1 : 어떤 정수의 약수를 구하는 프로그램을 goto를 이용하여 작성하라.
```C
1 #include <stdio.h>

3 {
4 	int x, d=1;
5 	printf("정수 입력 : ");
6 	scanf("%d", &x);
7 	printf("%d의 약수 = ", x);
8 
9 loop:	// 레이블 설정 
10 	if(x % d == 0)
11 		printf("%d ", d);
12 	d++;
13 	if(d <= x/2)		// x/2까지 즉, 입력한 수의 1/2까지만 계산
```

* 10: x를 d로 나누어떨어지면 출력한다.
* 12: d를 1증가시킨다.
* 13: d가 x의 절반보다 적으면 다시 loop란 레이블이 있는 곳으로 간다.
* 15: 위의 과정을 반복하다가 더 이상 조건이 만족하지 않으면 이 문장을 실행한다. 자기 자신을 출력한다.
> [!NOTE]
> 자기 자신의 절반까지만 계산해보면 약수가 무엇인지 출력할 수 있으며, 마지막에 자기 자신을 출력한다.

#### ● 유제 03-08-2 : goto를 이용하여 패스워드를 세 번까지만 검사하는 프로그램을 작성하라. 단, 패스워드가 일치하면 빠져나간다. 단, 패스워드는 정수값이라고 간주한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int password=12345, in_pass, cnt=1;
5 
6 start:				// 레이블 start 설정 
7 	printf("Input password: ");
8 	scanf("%d", &in_pass);
9 
10 	if( in_pass == password)
11 		goto end;		// 레이블 end로 이동 
12 	else if(cnt >= 3)
13 		goto error;		// 레이블 error로 이동 
14 	else {
15 		printf("%d: your password is incorrect!\n", cnt);
16 		cnt++;
17 	}
18 	goto start;		// 레이블 start로 이동 
19 
20 error:				// 레이블 error 설정 
21 	printf("You were wrong in 3 times!!! Please contact your manager.\n");
22 	goto stop;
23 end:				// 레이블 end 설정 
24 	printf("Congratulation!!! Your password is correct!!\n");
25 stop:
26 	return;
27 }
```

#### ● 해설
* 4: 변수 password를 선언하고 초깃값 12345를 지정하고 cnt를 선언하여 초깃값을 1로 지정한다.
* 6: 시작 지점 start레이블을 지정한다.
* 8: 패스워드를 입력받는다.
* 10: 입력 패스워드와 저장된 패스워드가 같은지 비교한다. 같으면 end레이블이 있는 곳으로 간다.
* 12: 세 번 입력해서도 패스워드가 맞지 않으면 에러 메시지를 출력하는 error레이블로 간다.
* 14~18: 그렇지 않고 세 번 이하로 틀렸을 경우에는 cnt를 하나 증가하고 다시 받아들이기 위해 start로 간다.
* 20: 에러 메시지를 출력하기 위해 이곳으로 온다.
* 23: 패스워드가 맞았을 때 이곳으로 온다.
* 25: 최종적으로 모든 과정이 이곳으로 도착해서 끝난다.
* 26: 마지막 stop:에 해당하는 문장이 있어야 하므로 아무런 동작도 하지 않는 return을 넣었다. return은 원래 함수에서 복귀할 때 사용하는 것이지만 main도 함수이므로 이 프로그램을 호출한 곳으로 복귀한다.

#### ● 유제 03-08-3 : 2의 0승에서 30승까지를 goto를 이용하여 계산하고 출력하는 프로그램을 작성하라. 단, 비트 연산자를 사용한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=0, power=1;
5 loop:	// 레이블 설정 
6 	printf("2**%d = %d \n", x, power);
7 	power <<= 1;
8 	x++;
9 	if(x <= 30)
10 		goto loop;
11 }
```

#### ● 해설
* 4: 변수 x와 power를 선언하고 각각 초깃값으로 0과 1을 지정한다. x는 지수, power는 계산 값이다.
* 5: 9, 10번에서 조건이 만족하면 여기로 온다. 
* 6: x의 누승의 값을 출력한다. 처음에는 2의 0승을 출력한다. 마지막은 2의 30승을 출력한다.
* 7: x를 좌로 한 비트 이동(shift)한다.
* 8: x를 하나 증가한다. 즉 30까지 하나씩 증가하게 된다.
* 9~10: x가 30보다 작을 때는 5번 줄 loop 레이블이 있는 곳으로 이동한다.
