### 03.02 조건연산자
이 절에서는 조건연산자를 알아보고 몇 가지 예제를 통해서 어떻게 사용하는지 살펴본다. 조건연산자는 다음 표와 같이 사용하며 먼저 식을 평가하고 식이 참(0 이외의 값)이면 두 번째 항을 실행하고 거짓(0)이면 세 번째 항을 실행한다.

#### ● 예제 03-02-1 : 조건연산자를 이용한 절댓값 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수를 입력하시오: "); 
6 	scanf("%d", &x);
7 	x = x<0 ? -x : x;
8 	printf("%d\n", x);
9 }
```

#### ● 해설
* 6: 정수를 입력한다.
* 7: `x = x<0 ? -x : x;` `x`가 0보다 작으면 `–x`를, 그렇지 않으면 `x`를, `x`에 저장한다.

#### ● 예제 03-02-2 : 조건연산자를 이용하여 소문자를 대문자로 바꾸기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char x;
5 	printf("문자를 입력하시오: "); 
6 	scanf("%c", &x);
7 	x = (x >= 'a' && x <='z') ? x - 32 : x;
8 	printf("%c\n", x);
9 }
```

#### ● 해설
* 6: 문자를 입력한다.
* 7: `x`가 소문자이면 대문자(`x-32`)로 그렇지 않으면 그대로 `x`에 저장한다. ASCII코드에서 대문자가 소문자보다 앞에 있으며, 그 간격이 32이다. ASCII코드 표를 참고하기 바란다.

#### ● 예제 03-02-3 : 조건연산자를 이용하여 양수⋅음수 구별하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수를 입력하시오: ");
6 	scanf("%d", &x);
7 	x<0 ? printf("%d:음수\n",x) : printf("%d:양수\n",x);
8 }
```

#### ● 해설
* 6: 정수를 입력한다.
* 7: `x`가 음수이면 "음수"를 양수이면 "양수"를 출력한다. 여기서는 변수에 값을 대입하는 것이 아니라, `printf` 문장 자체를 실행한다.
