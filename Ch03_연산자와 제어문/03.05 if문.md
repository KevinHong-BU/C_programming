### 03.05 if문

이 절에서는 if문을 이용하여 조건을 판단하는 기본적인 예를 살펴본다.

조건 판단은 키워드 if()를 이용하여 작성한다. 괄호 속에는 일반 수식, 비교연자를 이용한 식, 논리연산자를 이용한 식 등을 조합하여 사용할 수 있다.

하나의 조건만 있을 경우에는 if(조건)만 사용하며, 조건이 2개 일 때는 if(조건) ~ else ~를 사용한다.

조건이 여러 개 일 때는 if(조건) ~ else if(조건)...과 같이 사용하며 다음 절에서 살펴보도록 하자.

if ~ else 조건문

---
#### ● 예제 03-05-1 : if문을 이용한 절댓값 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수를 입력하시오: "); 
6 	scanf("%d", &x);
7 	if(x<0) {
8 		x = -x;
9 	}
10 	printf("%d\n", x);
11 }
```

#### ● 해설
* 7: if문을 이용하여 조건을 비교한다. 조건이 참이면 if블록을 실행하고 그렇지 않으면 if블록을 실행하지 않는다. if 블록은 { }이다. 이 블록 안의 문장이 한 개이면 {}은 생략할 수 있다. 이 예에서는 생략하지 않았다. x가 0보다 작으면 x=-x; 문을 실행하고 그렇지 않으면 그냥 지나간다.
> [!NOTE]
> 블록 안의 문장이 하나일 때도 {}을 쓰면 프로그램을 이해하는데 도움이 될 것이다.

---
#### ● 예제 03-05-2 : if문을 이용한 소문자를 대문자로 바꾸기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char x;
5 	printf("문자를 입력하시오: "); 
6 	scanf("%c", &x);
7 	if(x >= 'a' && x <='z')
8 		 x -= 32;
9 	printf("%c\n", x);
10 }
```

#### ● 해설
* 7,8: 비교연산자와 논리연산자를 이용하여 x의 범위를 비교한다. x가 ‘a’와 ‘z’ 사이에 있으면 즉, 소문자이면 대문자로 바꾼다.
> [!NOTE]
> 이 예에서는 if 블록에 해당하는 문장이 하나(x -= 32;)이므로 {}를 생략할 수 있다.

#### ● 예제 03-05-3 : if ~ else 문을 이용한 “양수” 또는 “음수”를 출력한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수를 입력하시오: "); 
6 	scanf("%d", &x);
7 	if(x<0)
8 		printf("%d:음수",x);
9 	else
10 		printf("%d:양수\n",x);
11 }
```

#### ● 해설
* 7: if 조건을 비교하여 x가 0보다 작으면 10번 문장을
* 8: x가 0보다 작을 때 실행되는 문장
* 9: 그렇지 않으면(7번 if문이 거짓일 때 연결되는 문장)
* 10: 7번 문장에서 조건이 거짓일 때 else 블록이 실행된다.
> [!NOTE]
> else에 해당하는 문장도 여러 개이면 블록 {}을 지정하여 사용할 수 있다. 여기서는 하나이므로 생략한다.

#### ● 예제 03-05-4 : if ~ else 문을 이용한 판단, 블록에 한 문장이상 일 때
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int height;
5 	printf("키: ");
6 	scanf("%d", &height);
7 	if(height < 120)
8 	{
9 		printf("키가 작습니다.\n");
10 		printf("이 놀이 기구는 탈 수 없습니다.\n");
11 		printf("키가 %dcm 더 커야 합니다.\n", 120-height);
12 	}
13 	else 
14 	{
15 		printf("환영합니다. 즐거운 시간되시기 바랍니다.\n");
16 		printf("놀이 기구를 탑승하시고 반드시 안전벨트를 매십시오.\n");
17 	}
18 }
```

#### ● 해설
* 7: if 조건을 비교하여 x가 120보다 작으면 9번~11번 문장을
* 8~12: x가 120보다 작을 때 실행되는 문장
* 13: 그렇지 않으면(7번 if문이 거짓일 때 연결되는 문장)
* 14~17: 7번 문장에서 조건이 거짓일 때 실행되는 문장
> [!NOTE]
> if 또는 else에 해당되는 문장이 여럿일 때는 블록 {}을 써야 한다.
