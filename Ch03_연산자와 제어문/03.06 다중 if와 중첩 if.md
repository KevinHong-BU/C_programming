### 03.06 다중 if와 중첩 if
앞 절에서는 조건이 1개 또는 2개일 때를 살펴보았다. 실제 응용에서는 여러 개의 조건이 있는 경우가 많이 있다. 조건이 여러 개일 때는 다양한 방법으로 사용할 수 있다. 동일한 수준의 조건일 때와 조건 속의 조건인 하위 수준의 조건 등 다양한 방법을 쓸 수 있다. 다중 if문은 동일한 수준의 조건일 때 사용하고, 중첩 if문은 조건 속의 하위 조건 판단이 필요할 때 사용한다. 마지막의 else는 있을 수도 있고 없을 수도 있다.

#### ● 예제 03-06-1 : if ~ else 문을 이용한 다중 조건문
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수: ");
6 	scanf("%d", &x);
7 	if(x < 0)
8 		printf("음수\n");
9 	else if(x > 0)
10 		printf("양수\n");
11 	else
12 		printf("Zero\n"); 
13 }
```

#### ● 해설
* 7: x가 0보다 작으면 8번을 실행한다.
* 9: x가 0보다 크면 10번을 실행한다.
* 11: x가 0보다 작지도 크지도 않으면 12번을 실행한다. 즉, 위의 두 조건을 모두 만족하지 않을 때.
> [!NOTE]
> 첫 번째 조건에서 바로 만족하면 다른 조건은 검사하지 않는다. 어떠한 조건 중 하나라도 만족하면 그 조건에 해당하는 문장을 실행하며, 다른 문장은 검사조차 하지 않는다.

#### ● 예제 03-06-2 : 점수의 평점 매기기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("점수: ");
6 	scanf("%d", &x);
7 	if(x >= 90)			// 90점 이상일 때
8 		printf("수\n");
9 	else if (x >= 80)		// 90점 미만, 80점 이상일 때
10 		printf("우\n");
11 	else if (x >= 70)		// 80점 미만, 70점 이상일 때
12 		printf("미\n");
13 	else if (x >= 60)		// 70점 미만, 60점 이상일 때
14 		printf("양\n");
15 	else					// 60점 미만일 때
16 		printf("가\n");
17 }
```

#### ● 해설
* 7: x가 90보다 크면 8번을 실행한다.
* 9: x가 80보다 크면 10번을 실행한다.
* 11: x가 70보다 크면 12번을 실행한다.
* 13: x가 60보다 크면 14번을 실행한다.
* 15: 위의 모든 조건이 만족하지 않으면 16번을 실행한다.
* **Try**: 5점 간격으로 A+에서 F까지 출력하도록 다시 작성하여 보자.

#### ● 예제 03-06-3 : 삼각형의 세 변수를 입력하여 다음과 같은 출력을 작성하라.
	1. 삼각형이 아님,	2. 정삼각형		3. 이등변삼각형,
	4. 직각 삼각형,		5. 일반삼각형
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int i, j, k;
5 	printf("세 변의 값을 입력하시오 : ");
6 	scanf("%d%*c%d%*c%d", &i, &j, &k);
7 	if( i+j < k || i+k < j || j+k < i )
8 		printf("삼각형이 아님\n");
9 	else if(i == j && j == k && k == i)
10 		printf("정삼각형\n");
11 	else if( i == j || j == k || k == i)
12 		printf("이등변삼각형\n");
13 	else if(i*i + j*j == k*k || j*j + k*k == i*i || k*k + i*i == j*j)
14 		printf("직각삼각형\n");
15 	else
16 		printf("일반삼각형\n");
17 }
```

#### ● 해설
* 6: 세 변의 길이를 입력한다.
* 7,8: 두 변의 합이 한 변의 길이보다 작으면 삼각형이 될 수 없다.
* 9,10: 세 변의 길이가 모두 같으면 정삼각형이다.
* 11,12: 두 변의 길이가 같으면 이등변 삼각형이다.
* 13,14: 두 변의 제곱의 합이 한 변의 제곱의 합과 같으면 직각삼각형이다.
* 15,16: 위의 모든 조건이 만족하지 않으면 일반삼각형이다.
* **Try**: 다중 조건에서는 프로그램의 순서에 따라 하나씩 걸러나가는 것이다. 따라서 이 프로그램에서 이등변삼각형은 정삼각형이 아닌 삼각형 중에서 두 변이 같은 삼각형이다. 그러므로 순서를 잘 결정해서 프로그래밍하여야 한다.

#### ● 예제 03-06-4 : 자동차를 새로 구입했을 세금계산하기
```C
1 #include <stdio.h>
2 int main()
3 { 
4 	unsigned int a, b, c, d, e, f, g; 
5 	/* a:출고가, b:특별소비세 및 교육세, c:부가가치세, d:취득세, 
6 	e:등록세, f:세금합계, g:차량 총 구입액 */ 
7 	unsigned int cc; /* 배기량 */ 
8 	unsigned int tax_ratio; /*세금 요율 */ 
9 
10 	printf("자동차 출고가를 입력하시오 : "); 
11 	scanf("%d", &a); 
12 	printf("자동차 배기량을 입력하시오 : "); 
13 	scanf("%d", &cc); 
14 	if(cc>2000) // 배기량별 세금 요율 적용 
15 		tax_ratio = 130;	// 13% * 1000 (정수로 계산하기 위함) 
16 	else if(cc>800)
17 		tax_ratio = 65;	// 6.5% * 1000 (정수로 계산하기 위함) 
18 	else 
19 		tax_ratio = 0;	// 0% 
20 
21 	// 특별 소비세 및 교육세 
22 	b = (a/1000) * tax_ratio;	 // 앞에서 요율에 1000을 곱한 값을 다시
23 						// 1000으로 나눠 줌 
24 	// 부가가치세 
25 	c = a/10 + b/10; 				// 각 10% 
26 	// 취득세 
27 	d = (a/100 + b/100) * 2; 		// 각 2% 
28 	// 등록세 계산 
29 	if(cc>800)
30 		e = (a/100 + b/100) * 5;	// 각 5% 
31 	else 
32 		e = (a/100 + b/100) * 2;	// 각 2% 
33 	f = b + c + d + e;		// 전체 세금 
34 	g = a + f;			// 차량판매가격 + 세금 
35 	printf("\n자동차 가격 : \t\t%10d원\n", a);
36 	printf("특별소비세 및 교육세: \t%10d원\n", b);
37 	printf("부가가치세 : \t\t%10d원\n", c);
38 	printf("취득세 : \t\t%10d원\n", d);
39 	printf("등록세 : \t\t%10d원\n", e);
40 	printf("세금 합계 : \t\t%10d원\n", f);
41 	printf("차량 총 구입액 : \t%10d원\n", g);
42 } 
```

#### ● 해설
* 14~19: 배기량에 따른 세금 요율 적용
* 21~32: 각종 세금을 계산한다.
* 33: 신차 구입시 부과되는 세금
* 34: 세금 + 차 가격

#### ● 예제 03-06-6 : 년, 월을 입력받아 해당 월의 마지막 날 찾기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int year, month;
5 	printf("년: ");
6 	scanf("%d", &year);
7 	printf("월: ");
8 	scanf("%d", &month);
9 	if(month >= 1 && month <= 12) {	// 1월과 12월 사이이면 
10 		if(month == 2) {		// 2월이면 
11 			// 윤년이면 
12 			if( year%400 == 0 || (year%100) && (year%4 == 0) )
13 				printf("%d년 %d월은 29일까지 있습니다.\n", year, month); 
14 			else // 윤년이 아니면 
15 				printf("%d년 %d월은 28일까지 있습니다.\n", year, month); 
16 		}
17 		// 4, 6, 9 또는 11월이면 
18 		else if(month==4 || month==6 || month==9 || month==11)
19 			printf("%d년 %d월은 30일까지 있습니다.\n", year, month); 
20 		else 	// 그 이외의 달(1, 3, 5, 7, 8, 10, 12월)인 경우 
21 			printf("%d년 %d월은 31일까지 있습니다.\n", year, month);
22 	}
23 	else {		// 1월과 12월 사이가 아니면 
24 		printf("잘못된 입력입니다.\n"); 
25 	} 
26 }
```

#### ● 해설
* 9: 월이 1월과 12월 사이이면 10~22 사이의 문장을 실행하고 그렇지 않으면 23~24의 문장을 실행한다.
* 10: 2월이면 11~16사이의 문장을, 4, 6, 9, 11월이면 19번 문장을, 그렇지 않으면 21번 문장을 실행한다.
> [!NOTE]
> 3중 중첩된 if문을 사용하였다.
