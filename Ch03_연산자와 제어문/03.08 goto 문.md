### 03.08 goto 문

조건 제어문의 마지막으로 `goto`를 살펴보자.

`goto`는 지정된 레이블 이름을 가진 곳으로 실행의 순서를 옮기는 것이다. 보통 `if`조건문과 함께 사용한다.

초기 컴퓨터 프로그램에서는 많이 사용하였으나 현재와 같이 컴파일러와 프로그램 자체의 수준이 많이 향상된 경우에는 거의 사용하지 않는 명령이다. 따라서 이 goto명령은 자주 사용하지 않는 것이 좋겠다.

하지만 꼭 필요한 곳에 쓰면 프로그램의 코딩을 줄일 수 있는 경우도 간혹 존재한다. 실제로 `goto` 만 있으면 모든 프로그램의 작성이 가능하다.

#### ● 예제 03-08-1 : goto를 이용한 1부터 100까지 합하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=0, sum=0;
5 
6 loop:		// 레이블 설정
7 	x++;		// x를 1증가 
8 	sum += x;	// sum = sum + x;와 같은 식
9 
10 	if(x<100)	// x가 100보다 작으면 
11 		goto loop;	// 지정된 레이블(loop)이 있는 곳으로 이동 
12 
13 	printf("sum = %d\n", sum);
14 }
```

#### ● 해설
* 4: 초깃값 `x`와 `sum`을 0으로 설정
* 6: 레이블 지정, 이 레이블은 변수명과 같은 수준으로 지정한다. `main` 함수 내의 어느 위치에 있던지 간에 `goto loop;`라고 하면 바로 여기로 제어를 옮긴다.
* 7,8: 계산식을 수행한다.
* 10: `x`가 100보다 작으면
* 11: `loop`라는 레이블이 있는 곳으로 이동한다. 즉, `x`가 100보다 크면 바로 다음 문장으로 내려간다. 레이블로 가서 다시 7,8번 문장을 실행한다. 반복 실행하다가 어느 순간 x가 100이 되면 loop로 더 이상 제어가 넘어가지 않는다. 즉, 100바퀴를 돌고나면 x가 100이 된다.
> [!NOTE]
> `goto`는 함수 밖으로, 또는 다른 함수 안으로는 갈 수 없다. 또 뒤에 나올 순환문(`for`, `while`, `do~while`)의 밖에서 순환문 속으로도 들어갈 수 없다. 단 순환문 안에서 밖으로 나오는 것은 가능하다.

#### ● 예제 03-08-2 : goto를 이용한 구구단 출력하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=1, y;
5 outer_loop: 	// 바깥쪽 레이블 설정 
6 	x++;
7 	printf("    %d단\n", x);
8 	y=0;
9 inner_loop:		// 안쪽 레이블 설정 
10 	y++;
11 	printf("%2d * %2d = %2d\n", x, y, x*y); 
12 	if(y<9)
13 		goto inner_loop; 
14 	printf("\n");
15 	if(x<9)
16 		goto outer_loop; 
17 }
```

#### ● 해설
* 4: 변수 `x`와 `y`를 선언하고 `x`에는 초깃값 1을 지정한다.
* 5: 바깥쪽 루프이다. 16번에서 이쪽으로 제어가 이동한다.
* 6: `x`를 1증가한다. `x`는 각 단의 수가 된다.
* 7: 단 이름을 출력한다.
* 8: 단 내의 변수 `y`를 0으로 설정한다.
* 9: 안쪽 루프이다. 13번에서 이쪽으로 제어가 이동한다.
* 10: `y`를 1증가한다.
* 11: 각 단의 한 줄을 출력한다. 예를 들면 맨 처음 출력되는 값은 2 * 1 = 2이다.
* 12,13: 단 내의 변화값인 `y`가 9보다 작으면 10으로 가서 1증가시킨다.
* 14: 단과 단 사이를 한 줄 비운다.
* 15,16: 단이 9보다 작으면 6으로 가서 `x`를 하나 증가시킨다.
> [!NOTE]
> 루프를 벗어나는 조건이 반드시 있어야 한다. 이 조건이 없으면 무한루프에 빠지게 된다. 여기서는 `x`의 값과 `y`의 값이 변화되고, `if`문에서 조건을 비교하여 빠져나가도록 한다.

#### ● 예제 03-08-3 : goto를 이용하여 알파벳 ‘A’부터 ‘Z’까지 출력하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char ch='A';	// 초깃값 설정
5 
6 loop:		// 레이블 설정
7 	printf("%c", ch);
8 	ch++;		// ch를 1증가 
9 	if(ch<='Z')	// ch가 ’Z’보다 작거나 같으면 
10 		goto loop;	// 지정된 레이블(loop)이 있는 곳으로 이동 
11 }
```

#### ● 해설
* 4: 초깃값 `ch`를 'A'로 설정
* 6: 레이블 지정
* 7: 문자 출력
* 8: 문자 1 증가, 문자형도 정수형이므로 증가, 감소가 가능하다.
* 9: `'Z'`보다 작거나 같으면 loop로 제어를 이동한다.
