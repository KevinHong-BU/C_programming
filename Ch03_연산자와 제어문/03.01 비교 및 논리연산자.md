### 03.01 비교 및 논리연산자

비교연산자는 좌측의 식과 우측의 식을 비교하여 참(1)과 거짓(0)을 판단한다. 여기서 식이란 상수, 변수, 산술식, 함수식 등 모든 식을 의미한다.

논리연산자는 좌측의 식과 우측의 식을 논리적으로 판단하여 참(1)과 거짓(0)을 결정한다. C언어에서는 0은 거짓이며, 0이외의 모든 값은 참으로 평가한다. 참을 나타내는 대표적인 값은 주로 1을 사용한다.

#### ● 예제 03-01-1 : 비교연산자
```C
1 #include <stdio.h>
2 int main()
3 {
4 	printf("10 == 15 => %d\n", 10 == 15);
5 	printf("10 != 15 => %d\n", 10 != 15);
1 	printf("10 < 15  => %d\n", 10 < 15);
6 	printf("10 <= 15 => %d\n", 10 <= 15);
7 	printf("10 > 15  => %d\n", 10 > 15);
8 	printf("10 >= 15 => %d\n", 10 >= 15);
9 }
```

#### ● 해설
* 4: `10 == 15;` 10은 15와 같은가? 같지 않으므로 거짓(0)이다.
* 6: `10 < 15;` 10은 15보다 작은가? 작으므로 참(1)이다.
> [!NOTE]
> 비교연산자는 좌측과 우측의 조건을 비교하여 참, 거짓을 판단한다.
> `!=`, `<=`, `>=` 연산자의 위치는 바뀌면 안 된다. `=!`, `=<`, `=>`처럼 쓸 수는 없다.

#### ● 예제 03-01-2 : 논리연산자
```C
1 #include <stdio.h>
2 int main()
3 {
4 	printf("  1 && 1 => %d\n", 1 && 1);
5 	printf("  0 && 1 => %d\n", 0 && 1);
6 	printf("10 && 15 => %d\n", 10 && 15);
7 	printf("  0 || 1 => %d\n", 0 || 1);
8 	printf("  0 || 0 => %d\n", 0 || 0);
9 	printf("10 || 15 => %d\n", 10 || 15);
10 	printf("     !10 => %d\n", !10);
11 }
```

#### ● 해설
* 4: `1 && 1;` 1은 참이므로 양쪽의 식이 모두 참이다. 따라서 결과는 참(1)이다.
* 7: `0 || 1;` 0은 거짓, 1은 참이므로, 둘 중 하나가 참이므로 결과는 참(1)이다.
* 10: `!10;` 10은 참이므로 이를 부정하면 거짓(0)이 된다.
> [!NOTE]
> 0은 거짓이며, 0 이외의 모든 값은 참이다. 이 예에서는 단순히 상수로 비교하였지만 실제로는 좌우에 수식이 일반적으로 사용된다. 다음 <예제 03-01-3>을 보라.

#### ● 예제 03-01-3 : 비교연산자와 논리연산자를 모두 사용
```C
1 #include <stdio.h>
2 int main()
3 {
4  	int a=10, b=5, c=-9, d=25; 
5 	printf("a=%d, b=%d, c=%d, d=%d일 때\n", a, b, c, d); 
6 	printf("a>b && a>c      => %d\n", a>b && a>c);
7 	printf("a>b && b>c      => %d\n", a>b && b>c);
8 	printf("a>b && c>=d     => %d\n", a>b && c>=d);
9 	printf("a>b || a>c      => %d\n", a>b || a>c);
10 	printf("a>=b || !(b>c)  => %d\n", a>=b || !(b>c));
11 	printf("a>=b && !(c>=d) => %d\n", a>=b && !(c>=d));
12 }
```

#### ● 해설
* 6: `10>5 && 10>-9;` `&&`의 좌우축의 식이 둘 다 참이므로 결과는 참(1)이다.
* 7: `10>5 && 5>-9;` `&&`의 좌우축의 식이 둘 다 참이므로 결과는 참(1)이다.
* 8: `10>5 && -9>=25;` `&&`의 우측의 식이 거짓이므로 거짓(0)이다.
* 10: `10>=5 || !(5>-9);` `||`의 좌측은 참, 우측은 거짓이므로 결과는 참(1)이다.
* 11: `10>=5 && !(-9>=25);` `&&`의 좌측은 참, 우측도 참이므로 결과는 참(1)이다.
* Try: 비교연산자와 논리연산자를 이용하여 여러 조건들을 검사할 수 있다. 많은 연습을 해 보라.
