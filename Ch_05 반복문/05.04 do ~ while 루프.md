### 05.04 do ~ while 루프

do ~ while은 while과 하나의 차이점만 있다. 그 차이는 while은 루프를 시작할 때 조건을 검사하고, do ~ while은 루프의 마지막에 조건을 검사한다는 점만 다르다. 따라서 최악의 경우 while은 처음부터 조건이 맞지 않으면 루프 내부를 한 번도 실행하지 않을 수 있으며, do ~ while은 조건을 루프의 끝에서 비교하기 때문에 최소 한번은 루프를 실행하게 된다.

#### ● 예제 05-04-1 : do ~ while 루프를 이용하여 양수가 입력될 때까지 데이터를 받아들인다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, y;
5 	do {
6 		printf("양수를 입력하시오 : "); 
7 		scanf("%d", &x);
8 	} while (x<=0);
9 	printf("x = %d\n", x);
10 }
```

#### ● 해설
* 5: 루프의 시작
* 7: 값을 입력받는다.
* 8: 값이 음수이면 다시 입력 받는다. 양수가 입력될 때까지 루프를 돌면서 받아들인다. 즉, 원하는 데이터가 입력될 때까지 루프를 빠져나가지 않는다.
> [!NOTE]
> do ~ while은 이와 같이 원하는 값이 되지 않으면 다시 처음부터 시작할 수 있게 할 수 있다.

#### ● 예제 05-04-2 : do ~ while을 이용하여 0 ~ 100사이의 값을 입력하고 평점을 출력한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int score;
5 	do {		// 0에서 100사이의 값이 아니면 다시 입력받는다.
6 		printf("0에서 100사이의 정수를 입력하시오 : ");
7 		scanf("%d", &score);
8 	} while (score < 0 || score > 100);
9 	printf("%d : ", score);
10 	switch(score/10) {	// 평점을 출력한다.
11 		case 10:
12 		case 9: printf("A"); break;
13 		case 8: printf("B"); break;
14 		case 7: printf("C"); break;
15 		case 6: printf("D"); break;
16 		default : printf("F");
17 	}
18 
19 	if( (score > 60 && score%10 >= 5) || score == 100 )
20 		printf("+");
21 	printf("\n");
22 }
```

#### ● 해설
* 5~8: 0에서 100사이의 값이 입력될 때까지 루프를 수행한다. 아니면 다시 입력한다.
* 10~17: 구간별로 평점을 출력한다.
* 19~20: 점수가 60점 이상이고 1의 자리 값이 5이상이거나 점수가 100점이면 ‘+’를 추가한다.

#### ● 예제 05-04-3 : do ~ while을 이용하여 Ctrl-Z가 입력될 때까지 데이터 입력하기
```C
1 #include <stdio.h>
2 int main()
3 {
4  	int x, sum=0;
5  	printf("Press Ctrl-Z to exit.\n");
6  	do {
7  		printf("input data : ");
8  		if(scanf("%d", &x) == EOF)	// Ctrl-Z가 입력되면 루프를 벗어난다.
9  			break;
10  		sum += x;
11  	} while(1);
12  	printf("sum = %d\n", sum);
13 }
```

#### ● 해설
* 6~11: Ctrl-Z(EOF)가 입력될 때까지 루프를 반복하면서 데이터를 입력한다.
* 8: 키보드에서 Ctrl-Z가 입력되었는지 검사한다. 마지막 데이터 입력 시 Ctrl-Z를 입력하고 엔터를 누른다.
* 12: 결과를 출력한다.

#### ● 예제 05-04-4 : Zeller의 공식을 이용하여 년, 월을 입력하면 1개월 간의 달력을 출력한다.Zeller의 공식 : , a:년도의 앞 두 자리, b:년도의 뒤 두 자리, c:월, d:일, []는 정수연산을 의미한다. 계산 결과를 7로 나누어 나머지가 0, 1, 2, 3, 4, 5, 6이면, 일, 월, 화, 수, 목, 금, 토가 된다. 단, 1월과 2월은 전년도의 13월과 14월이 된다.

* 39: 달력의 제목(년, 월)을 출력한다.

#### ● 예제 05-04-5 : do ~ while을 이용하여 키가 눌려지면 키 값 출력하고 ESC키가 눌려지면 끝내기
```C
1 #include <stdio.h>
2 #include <conio.h>
3 int main()
4 {
5 	int chr;
6 	do
7 	{
8 		chr = getch();
9 		printf("%d ", chr); 
10 		if(chr==0 || chr==0xe0)
11 		{
12 			chr = getch();
13 			printf("확장키code=%d\n", chr);
14 		}
15 		else
16 			printf("아스키code=%d\n", chr);
17 		if(chr == 27)		// ESC Key
18 			break;
19 	}while(1);
20 }
```

#### ● 해설
* 6~18: 키 입력에 대해서 키 값을 출력한다.
* 8: 키보드가 눌려지면 입력받는다.
* 10~14: 확장키일 때 키 값을 출력한다. Ctrl+Alt를 누르고 키를 누른다.
* 15~16: 아스키 키일 때 키 값을 출력한다.
* 17~18: ESC키가 눌려지면 루프를 끝낸다.
