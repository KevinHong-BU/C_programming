### 05.08 난수 발생 함수

C언어의 표준함수중에 난수를 발생하는 함수가 있다. 난수는 게임이나 기타 응용에서 많이 사용하는 함수이므로 여러분이 잘 알아두면 편리하게 사용할 수 있다. 난수 발생함수 rand()는 stdlib.h헤더 파일에 정의 되어있으며, 사용할 때 약간의 주의가 필요하다. 난수를 발생하기 전에 난수를 초기화해 주어야 한다. 초기화해 주지 않으면 항상 같은 순서로 난수가 발생하는 의사난수(pseudo random number)이기 때문에 의미가 없다. 난수를 초기화하는 함수는 srand()함수이며, 매개변수의 값을 변화시켜줌으로서 난수를 언제나 다르게 발생시킬 수가 있다. 매개변수로는 time(0)함수를 주로 사용한다. 즉, 시간은 항상 변하므로 매번 다른 순서로 난수를 만들어 낼 수 있다. time(0)는 현재 시간을 의미하며, time()함수는 time.h헤더파일에 정의되어 있다. 난수의 발생범위는 0부터 RAND_MAX까지이며 RAND_MAX 매크로는 stdlib.h에 정의되어 있으며, 보통 RAND_MAX는 32767로 지정되어 있다. 난수의 발생범위는 적절하게 조절할 수 있다. 난수 발생범위의 식은 “rand()%난수범위의_개수 + 시작_수”이다. 예를 들어 1부터 10까지 난수를 발생하고 싶다면 난수범위의 개수가 10개이고 시작 수가 1이므로 “rand()%10 + 1”로 하면 된다.

---
#### ● 예제 05-08-1 : 10개의 난수를 발생해 보자
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 int main()
5 {
6 	unsigned int i, seed;
7 	seed = time(0); 	// 난수 초기화 함수의 매개변수를 현재의 시간으로 설정
8 	srand(seed);	// 난수 초기화
9 
10 	for( i=0; i<10; i++)
11 		printf("%d, ", rand());		// 난수발생
12 }
```

#### ● 해설
* 7: 난수 초기화 함수에 사용할 매개변수로 seed란 변수를 사용하였고 현재 시간 time(0)을 저장한다.
* 8: 난수 초기화 함수 srand를 이용하여 난수를 초기화 한다. 매개변수로 현재시간을 쓴다.
* 10,11: 난수 10개를 발생하여 출력한다.
* 11: rand() 함수를 이용하여 난수 발생하며, 난수의 범위는 기본적으로 0~32767까지이다.
* Try: 8,9번 줄을 생략하고 여러 번 실행해 보면 항상 같은 수가 발생하는 것을 알 수 있다.

#### ● 예제 05-08-2 : 주사위를 1000번 굴려 발생하는 주사위 눈의 평균을 구하라. 난수의 발생범위(1~6)를 지정한다.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 int main()
5 {
6 	unsigned int i, seed, sum=0;
7 	double average=0;
8 	seed = time(0);
9 	srand(seed);
10 	
11 	for ( i=0; i<1000; i++)
12 		sum += rand()%6+1;		// 1부터 6까지 난수 발생
13 	printf("눈의 평균 = %f\n", sum/1000.0);
14 }
```

#### ● 해설
* 7: 평균을 저장할 실수 변수 선언
* 8: 난수 초기화 함수에 사용할 매개변수로 seed란 변수를 사용하면 현재 시간 time(0)을 저장한다.
* 9: 난수 초기화 함수 srand()를 이용하여 난수를 초기화 한다. 매개변수로 현재시간을 쓴다.
* 11,12: 주사위를 1000번 던진다. 난수의 범위를 1부터 6까지 설정한다.
* 13: 평균을 구하여 출력한다. 이론값이 3.5이며, 실제 평균값이 3.5정도 됨을 알 수 있다.
