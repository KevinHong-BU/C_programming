### 05.01 while 루프

while 루프는 반복문의 기본형이라 할 수 있다. while문을 시작하기 전에 조건에 사용할 값을 초기화해 두는 것이 기본이다. 이 값을 기준으로 조건을 판단하고 조건이 만족하면 while 반복문에 포함된 문장들(블록)을 실행한다. 블록이 실행되는 동안 조건에 사용할 값을 변경해 주어야 한다. 조건에 사용될 값의 변경은 블록의 끝에서 이루지는 것이 일반적이다.
일반적인 while 반복문의 구조는 다음과 같다.

① 초깃값;<br>
① 루프를 시작하기 전에 초깃값 설정<br>
② while(조건식) {<br>
② 시작 할 때 조건 검사: 조건이 맞으면 ③으로, 맞지 않으면 ⑥으로 이동<br>
③	루프 내에서 실행할 문장들;<br>
③ 조건이 성립하면 루프 내의 문장들 실행<br>
④	조건값 변경;<br>
④ 루프 내에서 조건 값 변경<br>
⑤ }<br>
⑤ 루프의 시작 지점 ②로 이동<br>
⑥ 루프 다음 문장;<br>
⑥ ②에서 조건이 맞지 않으면 여기로 이동<br>

#### ● 예제 05-01-1 : while을 사용하여 1부터 10까지 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int counter=1;		// 초깃값 
5 	while(counter<=10) {	// 반복할 조건 검사
6 		printf("%d ", counter);
7 		counter++;		// 루프를 벗어날 조건을 변화시킴 
8 	}
9 }
```

#### ● 해설
* 4: 변수 counter를 선언하고 초깃값을 1로 설정한다.
* 5: 조건을 비교하여 참이면 while 블록을 실행하고 그렇지 않으면 바로 while블록을 빠져나간다.처음에는 조건이 만족하므로 while블록을 실행한다.
* 6: 블록 내에서 counter의 값을 출력한다.
* 7: counter의 값을 1증가한다.
* 8: while 블록의 끝이다.
* 9: while문의 조건이 맞지 않으면 여기로 이동하여 더 이상의 실행문이 없으므로 프로그램을 끝낸다.
> [!NOTE]
> 블록 문을 벗어나는 조건이 반드시 있어야 한다. 그렇지 않으면 무한루프에 빠진다.

#### ● 예제 05-01-2 : while을 사용하여 1부터 10까지 합 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int counter=1, sum=0;
5 	// counter의 초깃값 1, sum은 초깃값 0을 가지고 있어야 한다. 
6 	while (counter<=10)
7 		sum += counter++;
8 	// { sum = sum + counter;
9 	// counter++; } 두 문장을 한 문장으로 만든 축약형, 두 문장으로 쓸 때는 {}으로 둘러싼다.
10 	printf("sum = %d\n", sum);
11 }
```

#### ● 해설
* 4: 초깃값을 각각 1, 0으로 하는 변수 counter, sum을 선언한다.
* 6: counter가 10보다 작거나 같으면 블록을 실행한다. 이 while 블록에는 하나의 문장(sum += counter++)만 있다. 따라서 {}를 사용하지 않아도 된다.
* 7: sum에 counter를 더하고, counter의 값을 1 증가시킨다. 조건값 변경이 이 식에 포함되어 있다. 이 문장이 실행되고 6번으로 이동하여 조건을 검사한다.
* 8,9: 두 문장을 7번과 같이 한 문장으로 쓸 수 있다.
* 10: 6번의 조건이 만족하지 않으면 여기로 이동하여 결과를 출력한다.

#### ● 예제 05-01-3 : while을 사용하여 약수 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int number, d=1;
5 	printf("정수 입력 : ");
6 	scanf("%d", &number);
7 	printf("%d의 약수 = ", number);
8 	while(number/2 >= d) { // number/2까지 즉, 입력한 수의 반(1/2)까지만 계산
9 		if (number % d == 0)
10 			printf("%d ", d);
11 		d++;
12 	}
13 	printf("%d \n", number); // 자기 자신 출력 
14 }
```

#### ● 해설
* 4: 변수 number, d선언하고 d의 초깃값으로 1을 지정. d는 1부터 number/2까지 1씩 증가하면서 나눠지는지 보기 위한 변수
* 6: number 값을 키보드에서 입력받는다.
* 8: number/2가 d보다 크면 while 블록 내의 문장들을 실행한다.
* 9: number가 d로 나눠지는지 확인하고, 나눠지면 10행을 실행한다.
* 10: if 문이 참이면 이 문장에서 약수를 출력한다.
* 11: d를 1 증가한다.
* 12: 블록의 끝, 8번의 while문으로 이동한다.
* 13: while의 조건이 만족하지 않으면 여기로 이동하여 마지막으로 자기 자신을 출력한다.

#### ● 예제 05-01-4 : 다음 식의 합이 0.69보다 커지면 끝낸다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int number = 1; 
5 	double sum= 0.0, sign = 1.0;
6 		//sign은 +, - 부호를 바꿔주기 위해 사용된 변수다. 
7 	while (sum < 0.69) { 
8 		sum += sign * (double)number/(double)(number+1);
9 		sign = -sign;
10 		number++; 
11 	}
12 	printf("x = %d, sum = %f\n", number, sum);
13 }
```

#### ● 해설
* 5: 합을 저장할 sum=0.0, 부호를 저장할 sign=-1.0을 선언한다. sign은 매 루프를 실행할 때마다 부호를 바꾸기 위한 방법이다.
* 7: sum이 0.69보다 작으면 루프를 반복한다.
* 8: sum에 누적값을 계산한다. 계산할 때 number를 캐스트 연산을 하여 실수로 변환한 다음에 계산한다. 그렇지 않으면 제대로 된 결과를 얻을 수 없다. 정수/정수는 정수이므로 실수로 바꾸지 않고 한다면 모든 항이0이 된다.
* 9: 부호를 바꾼다.
* 10: number를 1 증가한다.
* 12: 결과를 출력한다.

#### ● 예제 05-01-5 : 정수 x가 주어질 때, x가 짝수이면 2로 나누고(x=x/2), 홀수이면 3*x+1로 바꾸어 준다(x=3*x+1). 이 과정을 반복하다가 x가 1이 되면 루프의 반복을 멈추는 프로그램을 작성하라.* 이러한 반복을 수행하면 모든 양의 정수는 1로 수렴한다고 한다. 아직 모든 정수에 대해 성립한다는 것이 증명되지는 않았지만 성립하지 않는 경우도 발견되지 않았다고 한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수를 입력하시오 : ");
6 	scanf("%d", &x);
7 	printf("%d ", x);
8 	while(x > 1) {
9 		if(x % 2 == 0)
10 			x /= 2;
11 		else 
12 			x = x*3 + 1;
13 		printf("%d  ", x);
14 	}
15 }
```

#### ● 해설
* 6: 정수를 입력받는다.
* 8: 그 값이 1보다 크면 루프를 수행한다.
* 9: x가 2의 배수이면, 즉, 짝수이면 2로 나눈다.
* 11: 그렇지 않으면, 즉, 홀수이면 3을 곱하고 1을 더한다.
* 13: 하나씩 결과를 출력한다.
