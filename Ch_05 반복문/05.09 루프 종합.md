### 05.09 루프 종합

루프는 프로그램에서 아주 중요한 알고리즘에 속한다. 따라서 여러분이 다양한 예제를 통해 완전히 이해할 수 있을 정도로 공부하여야 할 부분이다. 루프를 이용하는 몇 가지 예를 더 살펴보도록 하자.

---
#### ● 예제 05-09-1 : 1부터 1539까지 종이에 숫자를 쓸 때 3이라는 수는 몇 번을 쓰게 되는가?
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int num=1539, count=0, x, y;
5 	for (x=1; x<= num; x++) {	// 1부터 1539까지 반복
6 		y = x;
7 		while (y > 0) { 		// 각 수에 3이 들어 있는지 확인 
8 			if (y % 10 == 3) // 끝자리가 3이면 
9 				count ++; 	// count를 증가 
10 			y /= 10;	 	// 10진수 한 자리 오른쪽 이동 
11 		}
12 	}
13 	printf("count = %d\n", count);
14 }
```

#### ● 해설
* 5: 1부터 1539까지 루프를 반복 실행한다.
* 7~11: 어떤 수의 각 자리에 3이 포함되어 있는지 검사하고 있으면 count를 1 증가한다.
* 8: 끝자리에 3이 있는지 확인한다.
* 9: 있으면 count를 증가한다.
* 10: 10진수를 한자리씩 오른쪽으로 이동한다.

#### ● 예제 05-09-2 : 어떤 자연수에서 첫 번째 자리의 수를 마지막 자리로 옮겼더니 원래 수의 2/3가 되었다. 이러한 자연수 중에서 가장 작은 수는 무엇인지 찾아내는 프로그램을 for와 while을 적절하게 이용하여 작성하라.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a, b, x;
5 	for (a=12; a<1000000; a+=3) { // 12부터 백만까지 찾기
6 		// 3으로 나누어지기 위해서는 3의 배수이어야 하므로 12부터 시작한다.
7 		b = a;
8 		x = 1;
9 		while(b/10) {	// 맨 앞자리를 찾기 위한 루프 
10 			b /= 10;  // 한자리씩 맨 왼쪽으로 이동:10진
11 			x *= 10;  // 맨 앞자리의 단위를 계산하기 위함
12 		}
13 		if( (a - b*x) * 10 + b == (a/3)*2 ) {
14 		// 맨 앞자리를 맨 뒤로 보냈을 때 원래 수의 2/3인가? 
15 			printf("%d * 2 / 3 = %d\n", a, (a - b*x) * 10 + b);
16 			break;
17 		}
18 	}
19 }
```

#### ● 해설
* 5: 12부터 백만까지 3씩 증가하면서 루프를 수행한다. 3씩 증가하는 이유는 3의 배수이므로.
* 8: 맨 앞자리의 값이 얼마인지 계산하기 위한 변수
* 9~12: 맨 앞자리를 맨 끝으로 이동하기 위한 루프.
* 13~17: 맨 앞자리를 끝으로 보냈을 때  2/3가 되는가? 되면 출력하고 끝낸다.

#### ● 예제 05-09-3 : 디지털 근은 구하는 프로그램을 while과 for를 혼용하여 작성
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int number, digital_root, dr, temp;
5 	printf("정수 입력: "); 
6 	scanf("%d", &number);
7 	digital_root = number;
8 	while(digital_root>=10) {
9 		for( temp = 0; digital_root>0; digital_root/=10 )
10 			temp += digital_root%10;
11 		digital_root = temp;
12 	}
13 	printf("%d의 디지털 근 : %d\n", number, digital_root);
14 }
```

#### ● 해설
* 7: drigital_root에 dr을 복사한다.
* 8~12: 디지털 근이 10보다 작은 경우 계속 반복한다.
* 9,10: 값을 한자리씩 더한다.
* 11: 계산된 값을 drigital_root에 저장하고, while루프를 다시 수행한다.
* 13: 디지털 근을 출력한다.

#### ● 예제 05-09-4 : 화면에 문자로 사인 그래프 그리기

```C
3 int main()

5 	int i, j;
6 	double radian, pi=3.141593;
7 	for(i=0; i<=360; i+=15) {
8 		radian = i * pi / 180;
9 		printf("%3d", i);
10 		for(j=0; j<= (int)(sin(radian)*30+30); j++)
11 			printf(" ");
12 		printf("*\n");
13 	}
14 }
```

#### ● 해설
* 6: radian 변수와 pi값을 저장하는 변수 선언
* 7~13: 0도부터 360도까지 15도씩 변화하면서 계산하고 출력한다.
* 8: 라디안 값을 계산한다.
* 10,11: 화면에 공백을 출력할 부분을 계산한다.
* 12: *을 출력한다.
