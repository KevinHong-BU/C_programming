### 05.06 for 무한 루프

for 무한 루프는 초깃값, 조건, 변화값을 모두 비워두고 세미콜론만 두 개 남겨서 사용한다. while(1) 또는 do ~ while(1)과 같은 의미이다.

---
#### ● 예제 05-06-1 : for 무한 루프를 이용하여 1부터 10까지 출력한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int count=1;		// 초깃값 
5 	for(;;) {
6 		printf("%d ", count);
7 		count++;		// 루프를 벗어날 조건을 변화시킴 
8 		if(count>10)	// 루프를 벗어나기 위한 검사
9 			break;
10 	}
11 }
```

● 해설
* 4: for에 사용될 초깃값 count를 미리 지정한다.
* 5: for에 대한 무한 루프이다.
* 7: 조건을 검사할 변수의 변화값이다.
* 8,9: 조건을 검사하여 만족하면 루프를 벗어난다.

---
#### ● 예제 05-06-2 : 개수가 정해지지 않은 데이터 값을 입력받아 합과 평균을 구한다. 단, 루프를 벗어난 조건은 음수가 입력될 때이며, 값은 0에서 100사이의 값이며 100을 초과한 값은 계산에 포함하지 않는다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, sum=0, count=0;
5 	double ave;
6 	for(;;) {
7 		printf("양의 정수를 입력하시오(음수 : 끝) : ");
8 		scanf("%d", &x);
9 		if( x < 0 ) // 음수이면 루프를 빠져나감. 
10 			break;
11 		else if( x > 100 )
12 			// 100 이상이면 계산하지 않고 for로 가서 다음 루프 실행 
13 			continue;
14 		sum += x;
15 		count++;
16 	}
17 	ave= (double) sum/ (double) count;
18 	printf(" 합계 : %d, 평균 : %f, 개수 : %d\n", sum, ave, count);
19 }
```

#### ● 해설
* 4,5: 변수 선언 및 sum과 count를 0으로 초깃값 설정.
* 6: for의 무한 루프
* 8: 키보드에서 데이터 입력
* 9,10: 입력 값이 0보다 작으면 루프 종료
* 11,13; 입력 값이 100보다 크면 아무것도 하지 않고 루프 시작부분으로 점프
* 14: 정상적인 값이 입력되었을 때 sum에 x를 누적
* 15: 정상적인 값이 입력되었을 때 count증가
* 17: 루프를 벗어난 뒤 평균 계산, sum과 count가 정수이므로 double로 형 변환
* 18: 결과 출력
* Try: continue를 쓰지 않고 이 프로그램을 작성해 보라.

---
#### ● 예제 05-06-3 : 피보나치 수열을 for 무한 루프를 이용하여 출력하고 황금비 계산
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=1, y=1, z=0;
5 	printf("1, 1, ");	// 처음 두 수인 "1, 1, "을 출력한다. 
6 	for(;;) {
7 		z = x + y;
8 		if(z < 0)
9 			// 가장 큰 정수보다 커지게 되면 음수로 바뀌는 성질을 이용한다. 
10 			break;
11 		printf("%d, ", z);
12 		x = y;
13 		y = z;
14 	}
15 	printf("\n");
16 	printf("황금비 = %f\n", (double)y / (double)x);
17 }
```

#### ● 해설
* 4: 변수 선언 및 초깃값 설정. 첫 번째, 두 번째 피보나치 항 설정
* 6: for의 무한 루프
* 7: 이어지는 피보나치 항 계산
* 8: 계산 결과가 음수이면 끝냄. 이것은 2의 보수 성질을 이용한 것임. 가장 큰 값보다 값이 커지면 음수가 됨.
* 11: 이어지는 항 출력
* 12,13: 다음 항을 계산하기 위해 바로 앞의 두 항 설정
* 16: 황금비 계산 출력, y와 x가 정수이므로 double로 형 변환
> [!NOTE]
> 굳이 무한 루프를 사용하지 않아도 되지만 예를 들기 위해 만든 것임.
