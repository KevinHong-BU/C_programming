### 05.07 for 중첩 루프

for 루프블록에는 어떠한 문장도 포함될 수 있다. 또 다른 루프를 포함할 수도 있다. for 뿐만 아니라  while, do ~ while등도 포함할 수 있다.

---
#### ● 예제 05-07-1 : for 중첩 루프를 이용한 구구단 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int dan, y;
5 	for(dan=2; dan < 10; dan++) {
6 		printf("%8d단\n", dan);
7 		for(y=1; y < 10; y++)
8 			printf(" %2d * %2d = %2d\n", dan, y, dan*y);
9 		printf("\n\n");
10 	}
11 }
```

#### ● 해설
* 4: 변수 선언
* 5: 단을 변경하기 위한 바깥쪽 루프
* 6: 단 제목 출력
* 7: 단 내의 값 1에서 9까지 변화를 위한 루프
* 8: 단에서 1부터 9까지 변화를 주면서 구구단 출력
* 9: 한 단이 끝나면 한 줄 건너 뜀.

---
#### ● 예제 05-07-2 : for 중첩 루프를 이용한 디지털 근 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int number, sum;
5 	printf(" 숫자를 입력하시오: ");
6 	scanf("%d", &number);
7 	for (;number>=10;) {
8 		sum = 0;
9 		for (;number != 0; number /= 10)
10 			sum += number % 10;
11 		number = sum;
12 		printf("==> %d ", number);
13 	}
14 	printf("결과:%d\n", number);
15 }
```

#### ● 해설
* 6: number의 초깃값은 키보드에서 받는다.
* 7: 바깥 for 루프에서 초깃값, 변화값은 설정하지 않았다. 조건은 한자리 값일 때까지 반복한다.
* 8: 각 자리의 합을 저장할 변수 sum을 0으로 초기화한다.
* 9: 내부 for 루프의 초깃값은 없고, 조건은 number의 값이 0이 아닐 때까지 반복하고, 변화값은 10진수로 한 자리씩 이동하면서 계산한다.
* 10: sum에 10진수의 각 자리의 합을 누적한다.
* 11: 다시 number를 sum으로 설정한다.
* 12: 중간 결과를 저장한다.

#### ● 예제 05-07-3 : *의 개수를 하나씩 늘려가면서 10줄을 출력하기.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, y;
5 	for (x=1; x<=10; x++ ) {
6 		for (y=1; y<=x; y++ )
7 			printf("*");
8 		printf("\n");		// 한 줄 출력하고 난 뒤 줄 바꿈 
9 	}
10 }
```

#### ● 해설
* 5: 바깥쪽 루프는 10줄을 반복한다.
* 6: 안쪽 루프는 각 줄에 *을 x개 만큼 출력한다.
* 8: 각 줄을 출력한 다음 줄을 바꾼다.

#### ● 예제 05-07-4 : abc + cac = bdd가 되는 a, b, c, d를 구하기.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a=1, b, c, d, abc, cac, bdd;
5 	for (a=1;a<10;a++) {
6 		for (b=1;b<10;b++) {
7 			for (c=1;c<10;c++) {
8 				for (d=1;d<10;d++) {
9 					abc = 100*a+10*b+c;	// abc값 계산 
10 					cac = 100*c+10*a+c;	// cac값 계산 
11 					bdd = 100*b+10*d+d;	// bdd값 계산
12 					if (abc + cac == bdd)
13 						printf(" %d\n"
14 							 "+%d\n"
15 							 "-----\n"
16 							 " %d\n\n", abc, cac, bdd);
17 				}
18 			}
19 		}
20 	}
21 }
```

#### ● 해설
* 5: a 변수는 1부터 시작하여 9까지 검사한다.
* 6: b 변수도 1부터 시작하여 9까지 검사한다.
* 7: c 변수도 1부터 시작하여 9까지 검사한다.
* 8: d 변수도 1부터 시작하여 9까지 검사한다. 0부터 해야 되지만 사실 0의 결과는 없다.
* 12: 문제의 조건을 만족하는 a, b, c, d가 있으면 출력한다.
* 13~16: 한 줄로 써도 되지만 보기 좋게 하기위해 분리하였다.
> [!NOTE]
> 변수가 4개이므로 4중 루프를 사용한다.

#### ● 예제 05-07-5 : 알파벳 ‘A’부터 ‘Z’까지 하나씩 늘려가면서 출력하기
```C
1 #include <stdio.h>
2 int main()
3 {
4  	char i, j;
5  	for(i='A'; i<='Z'; i++) {		// 문자형은 정수형의 한 종류이다.
6 		printf(" ");
7 		for(j='A'; j<=i; j++)
8 			printf("%c", j);
9 		printf("\n");
10 	}
11 }
```

#### ● 해설
* 5: i 변수는 ‘A’부터 시작하여 ‘Z’까지 반복한다.
* 6: 출력하기 전에 한 칸 띄운다.
* 7: j 변수는 ‘A’부터 I까지 반복한다.
* 8: 해당하는 문자를 출력한다.
* 9: 한줄 출력하고 줄을 바꾼다.

#### ● 예제 05-07-6 : 2!부터 10!까지 순서대로 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4  	int fact, s, e;
5  	for(s=2; s<=10; s++) {
6 		fact = 1;
7 		printf("  %d!=", s);
8 		for(e=1; e<=s-1; e++) {
9 			fact *= e;
10 			printf("%d*", e);
11 		}
12 		fact *= s;
13 		printf("%d = %d\n", s, fact);
14 	}
15 }
```

#### ● 해설
* 5: 팩토리얼을 구할 순서
* 6: 1부터 시작한다.
* 8~11: 각 팩토리얼을 구한다.
* 12: 마지막 수를 곱한다.
* 13: 출력한다.

#### ● 예제 05-07-7 : 1부터 100사이의 수 중에서 피타고라스 수 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4  	int x, y, z;
5  	for(x=1; x<=98; x++) {
6 		for(y=x+1; y<=99; y++) {
7 			for(z=y+1; z<=100; z++) {
8 				if(x*x + y*y == z*z)
9 					printf(" %d, %d, %d\n", x, y, z);
10 			}
11 		}
12 	}
13 }
```

#### ● 해설
* 8~9: 피타고라스 수이면 출력한다.
