1) 뺄셈을 이용하는 방법<br>
x = 252, y = 105 <br>
① x = x - y = 252 - 105 = 147 (결과 x = 147, y = 105)<br>
② x = x - y = 147 - 105 = 42 (결과 x = 42, y = 105)<br>
③ y = y - x = 105 - 42 = 63 (결과 x = 42, y = 63)<br>
④ y = y - x = 63 - 42 = 21 (결과 x = 42, y = 21)<br>
⑤ x = x - y = 42 - 21 = 21 (결과 x = 21, y = 21 -> 두 수가 같아지면 그 값이 최대공약수이다.)<br>
<br>
2) 나머지 연산을 이용하는 방법 : 더 빠르다.<br>
x = 252, y = 105<br>
① z = x % y = 252 % 105 = 42 (x = 105, y = 42)<br>
② z = x % y = 105 % 42 = 21 (x = 42, y = 21)<br>
③ z = x % y = 42 % 21 = 0 (x = 21, y = 0 -> 나머지가 0이면 제수의 값이 최대공약수이다.)<br>
