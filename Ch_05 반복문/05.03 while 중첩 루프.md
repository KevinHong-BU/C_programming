### 05.03 while 중첩 루프

반복문 속에 또 다른 반복문이 있을 수 있다. 이를 중첩 루프(nested loop)라 한다. 자주 사용되므로 잘 익혀두기 바란다.

---
#### ● 예제 05-03-1 : 2중 while 루프를 이용한 구구단 작성
```C
1 #include <stdio.h>
2 #include <conio.h>
3 int main()
4 {
5 	int dan, y;
6 	dan = 2;		// 2단부터 시작한다.
7 	while( dan < 10) {	// 9단까지
8 		printf("%8d단\n", dan);
9 		y = 1;		// 각 단에 곱해지는 수는 1부터 시작한다.
10 		while(y < 10) { // 각 단에 곱해지는 수는 9까지
11 			printf(" %2d * %2d = %2d\n", dan, y, dan*y); // 화면에 출력
12 			y++;	// 곱해지는 수 증가
13 		}
14 		printf("\n\n");
15 		getch(); 
16 		dan++;		// 단 증가
17 	}
18 }
```

#### ● 해설
* 7~17: 바깥쪽 루프
* 10~13: 안쪽 루프
* 8: 단의 머리글을 출력한다.
* 9: 단 내의 수의 변화는 1부터 시작한다.
* 10: 안쪽 루프는 y가 9까지 반복한다.
* 11: 안쪽 루프를 돌면서 구구단 표를 한 줄씩 출력한다.
* 12: y를 1 증가한다.

---
#### ● 예제 05-03-2 : 2중 루프를 이용하여 디지털 근을 구한다.디지털 근은 어떤 정수의 각 자리의 수를 합하여 한 자리 수가 될 때까지 반복한다. 한 자리수가 나오면 그것이 디지털 근이다. 예: 3829 => 3+8+2+9 = 22 => 2+2 = 4(디지털 근)
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int number, sum;
5 	printf(" 숫자를 입력하시오: ");
6 	scanf("%d", &number);
7 	while(number>10) {		// 한자리수가 될 때까지 반복하는 루프
8 		sum = 0;			// 각 자리수의 합을 저장할 변수
9 		while(number != 0) {	// 각 자리수를 하나씩 누적하는 루프
10 			sum += number % 10;	// 맨 뒷자리를 더한다.
11 			number /= 10;		// 10진수 값을 한자리 오른쪽으로 이동한다.
12 		}
13 		number = sum;
14 		printf("==> %d ", number);
15 	}
16 	printf("결과:%d\n", number);		// 한자리수가 되면 여기로 와서 출력한다.
17 }
```

#### ● 해설
* 7~15: 바깥쪽 루프
* 9~12: 안쪽 루프
* 7: 수가 10보다 크면 계속 반복한다. 즉, 한 자리 수가 될 때까지 반복한다.
* 8: 각 자리의 합을 저장할 변수 sum을 0으로 둔다.
* 9: 입력된 수 또는 합을 구한 수가 0이 될 때까지 한자리씩 잘라서 sum에 저장한다.
* 10: sum에 한자리씩 누적한다. 즉, 끝자리를 나머지 연산으로 계산해서 더한다.
* 11: 그리고 그 수를 10진수로 한자리씩 오른쪽으로 이동하기 위해 10으로 나눈다.
* 13: 합을 다시 number로 지정한다.
* 14: 중간 결과를 출력하고 바깥 루프의 처음으로 가서 조건을 비교한다.
* 16: 최종 결과를 출력한다.

#### ● 예제 05-03-3 : *을 하나씩 증가하면서 10줄을 출력하여 삼각형을 만든다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, y;
5 	x = 1;
6 	while ( x<=10 ) { // 10줄 출력하기 위한 루프 
7 		y=1;
8 		while ( y<=x ) { // 한 줄에 x개의 별 출력
9 			printf("*");
10 			y++;
11 		}
12 		printf("\n"); // 한 줄 출력하고 난 뒤 줄을 바꾼다.
13 		x++;
14 	}
15 }
```
#### ● 해설
* 5: x에 첫 줄에 출력할 *의 개수를 지정한다. 바깥쪽 루프 내에서 1씩 증가한다(14행). 최대 10까지 증가한다.
* 6: 10줄까지 출력한다. x는 1부터 10보다 작거나 같을 때까지 반복한다.
* 7: 한 줄에 출력할 *의 개수를 1부터 시작하여 x보다 작거나 같을 때까지 출력한다.
* 8~11: 한 줄에 *의 수를 1부터 x개까지 출력한다.
* 12: 줄을 바꾼다.
* 13: 한 줄에 출력할 *의 개수 x를 1증가한다.

#### ● 예제 05-03-4 : abc + cac = bdd가 되도록 하는 a, b, c, d를 구한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a=1, b, c, d, abc, cac, bdd;
5 	while(a<10) {
6 		b=1;
7 		while(b<10) {
8 			c=1;
9 			while(c<10) {
10 				d=0;
11 				while(d<10) {
12 					abc = 100*a+10*b+c;	// abc값 계산
13 					cac = 100*c+10*a+c;	// cac값 계산
14 					bdd = 100*b+10*d+d;	// bdd값 계산
15 					if (abc + cac == bdd)
16 						printf(" %d\n"
17 							"+%d\n"
18 							"-----\n"
19 							" %d\n\n", abc, cac, bdd);
20 					d++;
21 				}
22 				c++;
23 			}
24 			b++;
25 		}
26 		a++;
27 	}
28 }
```

#### ● 해설
* 4: a가 0이 되면 안 되므로 1부터 시작한다. b, c도 마찬가지이고, d는 0부터 시작해도 된다.
* 5~27: 미지수가 4개이므로 4개의 중첩 루프가 필요하다.
* 12~14: 세 자리 수 abc, cac, bdd를 계산한다.
* 15: abc+cac=bdd가 되는 수가 있으면 출력한다.
