### 05.02 while 무한 루프

반복문의 조건을 항상 참이 되도록 하면 무한루프를 수행하게 된다. 보통 조건을 1로 두고 사용한다. 0 이외의 어떤 값을 써도 무관하지만 1을 가장 보편적으로 많이 사용한다. 한 가지 주의할 점은 반드시 루프를 벗어나는 지점이 있어야 한다는 것이다. 즉, 루프를 수행하다가 어떤 조건을 만족하면 루프를 벗어나게 하는 명령이 반드시 필요하다. 물론 일부러 무한 루프를 벗어나지 않게 하는 경우도 있다. 무한 루프는 조건이 여러 개이거나 정해진 회수가 없는 경우 등 다양하게 사용할 수 있다. 조건을 이용하여 벗어날 때는 if문을 사용하여 조건을 검사하고 조건이 만족할 때 break를 이용하면 루프를 벗어난다. 단, 벗어날 때는 자기가 직접 속해 있는 루프 하나만 벗어난다. 또 다른 제어문으로는 continue가 있으며, 이는 루프의 시작부분으로 제어를 이동하며, continue 아랫부분은 실행하지 않는다. break와 continue는 무한 루프 뿐만 아니라 어떠한 루프 속에서도 사용할 수 있으나, continue는 가급적 사용하지 않는 것이 좋다. continue를 자주 사용하거나 많이 사용하면 프로그램의 가독성(readability)을 저해하고 혼란을 초래하여 디버깅을 어렵게 만들 수 있다. continue를 사용하지 않고도 프로그램을 얼마든지 작성할 수 있으므로 반드시 필요한 경우에만 사용하도록 한다.

---
#### ● 예제 05-02-1 : while의 무한 루프를 이용하여 1부터 10까지 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int count =1;		// 초깃값 
5 	while(1) {		// 반복할 조건 검사, 여기서는 조건이 1이므로 무한 루프이다.
6 		printf("%d ", count);
7 		count++;		// 루프를 벗어날 조건을 변화시킴 
8 		if(count>10)	// 루프를 벗어나기 위한 검사
9 			break;
10 	}
11 }
```

#### ● 해설
* 5: 무한루프 설정, while의 조건이 1이므로 항상 참이다.
* 7: 조건 변수를 변화한다.
* 8: 조건을 루프 내에서 검사한다. 조건이 맞으면
* 9: break를 만나서 빠져나간다.
> [!NOTE]
> 반드시 빠져나가는 조건이 필요하다. 그렇지 않으면 무한루프가 된다.


#### ● 예제 05-02-2 : 0에서 100사이의 정수를 여러 개 입력받아 합, 평균, 개수를 출력하라. 단, 0보다 작으면 루프를 끝내고, 100보다 큰 수는 개수, 합, 평균에서 제외하며 계속 다음 데이터를 입력받는다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, sum=0, count=0;
5 	double ave;
6 	while(1) {
7 		printf("양의 정수를 입력하시오(음수 : 끝) : ");
8 		scanf("%d", &x);
9 		if( x < 0 ) // 음수이면 루프를 빠져나감.
10 			break;
11 		else if( x > 100 )
12 			continue; // 100 이상이면 계산하지 않고 while로 가서 다음 루프 실행
13 		sum += x;
14 		count++;
15 	}
16 	ave= (double) sum/ (double) count;
17 	printf(" 합계 : %d, 평균 : %f, 개수 : %d\n", sum, ave, count);
18 }
```

#### ● 해설
* 4: 변수를 선언하고 필요한 변수에 초깃값을 설정한다. sum은 합, count는 데이터의 개수를 저장.
* 5: 평균을 계산하여 저장할 double변수를 선언한다.
* 6: 무한 루프 설정
* 8: 변수를 입력한다.
* 9,10: 음수이면 break에 의해서 루프를 벗어난다.
* 11,12: 그렇지 않고 100보다 크면 아무것도 수행하지 않고 while로 바로 간다. continue를 만나면 다음에 있는 문장을 수행하지 않고 다음 순서의 루프를 수행하기 위해서 루프의 처음(6번)으로 간다.
* 13: 0보다 작거나 100보다 크지 않으면 합을 누적하고
* 14: 입력 데이터의 개수를 하나 증가한다.
* 16: 음수가 입력되었을 때 while 루프를 빠져나와서 평균을 계산한다. 평균을 계산할 때 sum과 count를 실수로 변환한다.
* Try: continue를 사용하지 않고 다시 작성해 보고, 어느 프로그램이 더 이해하기가 쉬운지 비교해 보라.
> [!NOTE]
> continue는 가급적이면 사용하지 않도록 한다. 사실 continue없이도 프로그래밍이 가능하다. 간혹 사용하면 코드가 간단해 지는 경우는 있다. 꼭 필요할 때만 사용한다. 프로그램을 이해하는데 도움이 되지 않을 경우가 많기 때문이다.



#### ● 예제 05-02-3 : while 무한루프를 이용하여 피보나치수열을 출력하고 황금비(golden ratio)를 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=1, y=1, z=0;
5 	printf("%d, %d, ", x, y);	// 처음 두 수 1, 1을 출력한다.
6 	while(1) {
7 		z = x + y;
8 		if(z < 0) // 가장 큰 정수보다 커지게 되면 음수로 바뀌는 성질을 이용한다.
9 			break;
10 		printf("%d, ", z);
11 		x = y;
12 		y = z;
13 	}
14 	printf("\n");
15 	printf("황금비 = %f\n", (double)y / (double)x);
16 }
```

#### ● 해설
* 4: 처음 두 항을 1로 설정한다.
* 5: 처음 두 항을 출력한다.
* 6: 무한루프 설정, while의 조건이 1이므로 항상 참이다.
* 7: 앞의 두 수를 더하여 그 다음 수를 만든다.
* 8: 만들어진 수가 0보다 작으면 루프를 끝낸다. 2의 보수체계에서 가장 큰 수보다 더 커지게 되면 음수가 된다.
* 10: 그렇지 않으면 만들어진 피보나치 수를 출력하고
* 11,12: 다음 수를 만들기 위해 새로운 수로 앞의 수를 설정한다.
* 15: 피보나치의 연속하는 두 수의 비가 황금비이다. 즉, 뒤의 수 나누기 앞의 수를 하면 된다.
   황금비는 1:1.618이다.
* Try: 무한 루프를 사용하지 않고 이 프로그램을 다시 작성해 보라.


#### ● 예제 05-02-4 : while 무한루프를 이용하여 정수의 자릿수 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=5820703, count=0;
5 	
6 	x = x<0 ? -x: x;
7 	while(1){
8 		x /= 10;
9 		count++;
10 		if(!x)
11 			break;
12 	}
13 	printf("%d자리\n", count);
14 }
```

#### ● 해설
* 4: 변수를 선언하고 초깃값을 설정한다.
* 6: 음수이면 양수로 바꾼다.
* 7~12: 무한루프를 반복하면서 자릿수를 계산한다.
* 8: 정수 값을 10으로 나눈다.
* 9: 카운터를 1증가한다.
* 10~11: x가 0이면 루프를 끝낸다.
