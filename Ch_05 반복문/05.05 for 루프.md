### 05.05 for 루프

for 루프는 while 루프를 변형한 형태라 볼 수 있다. while은 초깃값을 루프를 시작하기 전에 주고 while에서는 조건만 검사하며, 루프 속에서 조건에 사용되는 값을 변화시키는 것이 일반적이다. 그러나 for는 이 세 가지를 모두 for()의 괄호 속에서 처리하도록 한 것이다. 즉, 초깃값, 조건, 조건 값의 변화식을 모두 괄호 속에 나열한다. 따라서 좀 더 편리하게 사용할 수 있도록 만들어졌다. 특히 for는 반복 횟수가 정해져 있는 경우 더 편리하게 사용할 수 있다. 한 가지 주의할 점은 초깃값은 루프를 시작할 때 한번만 적용되고, 조건은 매번 검사하며, 조건 값의 변화식은 매번 루프가 끝나는 시점에서 이루어진다. 이를 잘 알고 사용할 수 있도록 한다.

#### ● 예제 05-05-1 : for를 이용하여 1부터 10까지 출력하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int count;
5 	for(count=1; count<=10; count++)
6 		printf("%d ", count);
7 	printf("\n");
8 }
```

#### ● 해설
* 4: 변수 count선언
* 5~6: for 루프
* 5: 초깃값(count=1)은 루프를 시작할 때 한번만 설정된다. 조건(count<=10)은 매번 루프를 회전할 때마다 검사한다. 조건 값의 변화식 count++는 한번 루프를 회전할 때마다 변경된다. 다만 루프의 끝에서 이루어진다. 따라서 count++는 printf를 실행하고 난 뒤 증가한다.
> [!NOTE]
> for 루프 블록도 해당되는 문장이 하나만 있을 때는 {}를 생략할 수 있다.

#### ● 예제 05-05-2 : for를 이용하여 1부터 10까지 합 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int count, sum;
5 	for(count=1, sum=0; count<=10; count++)
6 		sum += count;
7 	printf("sum = %d\n", sum);
8 }
```

#### ● 해설
* 4: 변수 count, sum을 선언한다. 초깃값을 지정하지 않았다.
* 5: for루프를 시작한다. 초깃값 count, sum을 각각 1, 0으로 콤마로 구분하여 지정한다. 초깃값은 시작할 때 한번만 설정된다. 조건은 count가 10보다 작거나 같을 동안에 계속 루프를 실행하기 위한 것이며, 증분은 루프가 한 번 반복할 때마다 변경된다.
* 6: 매 루프를 반복할 때마다 sum을 누적한다.
* 7: 조건이 만족하지 않으면 즉, 10보다 커지게 되면 이 문장으로 제어가 넘어와서 결과를 출력한다.
> [!NOTE]
> 초깃값을 두 개를 지정할 수도 있다. 물론 조건 변수의 변화값도 두 개가 올 수도 있다.

#### ● 예제 05-05-3 : for를 이용하여 임의의 정수 x의 약수 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, d=1;
5 	printf("정수 입력 : ");
6 	scanf("%d", &x);
7 	printf("%d의 약수 = ", x);
8 	for ( ; x/2 >= d; d++)
9 		if (x % d == 0)
10 			printf("%d ", d);
11 	printf("%d \n", x); // 자기 자신 출력 
12 }
```

#### ● 해설
* 4: 변수 x, d를 선언하고 d의 초깃값을 1로 설정한다.
* 6: x의 값을 키보드에서 입력한다. 초깃값으로 사용한다.
* 8: for의 초깃값을 설정하지 않았다. 조건은 x/2가 d보다 크거나 같은 동안에 루프를 반복 실행한다. 조건값의 변경은 d를 1증가 시킨다.
* 9,10: x가 d로 나누어지면(d가 x의 약수이면) 그 값을 출력한다.
> [!NOTE]
> 초깃값을 생략할 수 있다. 조건 및 증분값도 생략가능하다. 다른 예를 통해서 알아보자. 또 여기서 for에 해당하는 문장이 2개 인 것처럼 보이지만 실제는 하나이다. for에 속한 것은 if이고 printf는 다시 if에 속해 있으므로 실제로 for에 속한 문장은 하나이다. 따라서 {}을 생략할 수 있다.

#### ● 예제 05-05-4 : 다음 식의 합이 0.69보다 커질 때까지 계산한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int number; 
5 	double sum= 0.0, sign = 1.0;
6 		//sign은 +, - 부호를 바꿔주기 위해 사용된 변수다. 
7 	for (number=1; sum < 0.69; number++, sign=-sign)
8 		sum += sign * (double)number/(double)(number+1);
9 	printf("number = %d, sum = %f\n", number, sum);
10 }
```

#### ● 해설
* 4: 정수 변수 number선언
* 5: 실수 변수 sum과 sign을 선언하고 초깃값 지정
* 7: for루프의 초깃값 number=1 지정, 조건은 sum이 0.69보다 작은 동안에 루프 반복실행, 조건값의 변경은 number를 1증가하고, sign은 매번 루프를 회전할 때마다 부호를 바꿔준다.
* 8: sum에 각 항을 누적한다.
* 9: 결과를 출력한다. 실제로 number는 for에서 한 번 더 증가한 다음에 출력된 것이다. 루프의 마지막에 증가하고 조건을 비교하기 때문이다. 따라서 실제 결과 number는 하나 더 증가한 값이다.
> [!NOTE]
> 초깃값과 변화값은 여러 개가 올 수도 있다. 너무 많이 쓰게 되면 혼동을 가져올 수 있으므로 많이 쓰지 않는다.

#### ● 예제 05-05-5 : <예제 05-01-5>를 for 루프를 이용하여 다시 작성하라.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x;
5 	printf("정수를 입력하시오 : ");
6 	scanf("%d", &x);
7 	printf("%d ", x);
8 	for (; x > 1; ) {
9 		if (x % 2 == 0)
10 			x /= 2;
11 		else 
12 			x = x*3 + 1;
13 		printf("%d  ", x);
14 	}
15 }
```

#### ● 해설
* 4: 정수 변수 x선언
* 6: x를 입력 받는다.
* 8: for 루프에서 3가지 값 중에서 조건만 사용한다. 초깃값은 키보드에서 받고, 값의 변화는 때에 따라 다르므로 루프 내에서 변화를 하도록 한다.
* 9,10: 짝수이면 그 값을 2로 나눈다.
* 11,12: 홀수이면 3을 곱하고 1을 더한다.
* 13: 중간 과정을 계속 출력한다.
> [!NOTE]
> for루프에서 3가지 요소 중에 어느 것도 생략가능하다. 적절히 잘 사용하면 편리하게 사용할 수 있다.
