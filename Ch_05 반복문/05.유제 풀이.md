#### ● 유제 05-01-1 : 양의 정수를 입력받아 소인수분해하는 프로그램을 while loop를 이용하여 작성하라.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, d=2; // d는 처음 2부터 시작함
5 	printf("정수 입력 : ");
6 	scanf("%d", &x);
7 	while(x != 1) {
8 		if(x % d == 0) {
9 			printf("%d ", d);
10 			x /= d;
11 		}
12 		else
13 			d++;
14 	}
15 	printf("\n");
16 }
```

#### ● 해설
* 6: 양의 정수를 입력한다.
* 7: x가 1이 아니면 루프를 반복 수행한다.
* 8,9: x가 d로 나누어지는지 확인한다. 나누어지면 d(소인수)를 출력한다
* 10: 그리고, x를 d로 나눈다.
* 13: 나누어지지 않으면 d를 하나 증가한다. 다시 루프의 처음으로 가서 조건을 비교하고 반복한다.
* 15: 마지막에 줄을 바꾼다.
> [!NOTE]
> d는 나누어지지 않을 때만 증가한다. 왜냐하면 예를 들어 8과 같은 경우 소인수는 2, 2, 2이다. 따라서 같은 수로 계속 나누어지는지 보기 위하여 한 번 나누어져도 계속 그 값으로 안 나누어 질 때까지 계속 나누어 본다.
