### 08.02 함수에 포인터 전달
C언어에서 함수는 원칙적으로 결과 값 하나를 반환하거나 반환하지 않을 수 있다. 따라서 두 개 이상의 결과 값을 반환할 수 없다. 두 개 이상의 결과 값을 반환 받기 위해서는 포인터를 이용한 간접 반환 방법을 사용하여야 한다. 포인터를 이용하여 반환받기 위해서는 미리 해당하는 포인터를 함수에 넘겨주고 그 포인터를 이용하여 데이터를 처리한다. 또 다른 방법은 전역 변수를 사용하는 방법도 있다. 매개변수로 포인터를 넘겨줄 때는 포인터 변수를 쓰거나 일반 변수의 주소를 넘겨주면 된다. 주소를 넘겨주면 받을 때 포인터로 받는다. 포인터를 주고받을 때는 실제 데이터를 주고받는 것이 아니므로 이런 경우 대부분은 함수가 void형이다.
포인터(주소)를 이용하여 데이터를 주고받는 것을 주소에 의한 호출(call by reference, 참조에 의한 호출)이라 한다. 반대로 값을 직접 주고받는 경우를 값에 의한 호출(call by value)라고 한다.

---
#### ● 예제 08-02-1 : 매개변수로 포인터를 사용할 때와 그렇지 않을 때
```C
1 #include <stdio.h>
2 void add_xy(int, int, int);
3 void add_xy_p(int, int, int *);
4 
5 int main()
6 {
7 	int x, y, total;
8 	x = 10;
9 	y = 5;
10 	total = 0;
11 	printf("main 함수에서의 total = %d\n\n", total);
12 	add_xy(x, y, total);
13 	printf("main 함수에서의 total = %d\n\n", total);
14 	add_xy_p(x, y, &total);
15 	printf("main 함수에서의 total = %d\n\n", total);
16 }
17 
18 void add_xy(int x, int y, int total) {
19 	total = x + y;
20 	printf("add_xy 함수에서의 total = %d\n", total);
21 }
22 
23 void add_xy_p(int x, int y, int *total) {
24 	*total = x + y;
25 	printf("add_xy_p 함수에서의 total = %d\n", *total);
26 }
```

#### ● 해설
* 11: 현재 main함수에서 total값을 출력한다. 결과 0이다.
* 12: 값을 함수 add_xy에 전달한다.
* 13: 현재 main함수에서 total값을 출력한다. 결과 0이다.
* 14: 값의 주소를 add_xy_p에 전달한다.
* 15: 현재 main함수에서 total값을 출력한다. 결과 15이다.
* 18~\21: 값을 받아 계산하는 함수이다. 이 때 total은 함수 내에서의 지역 변수이므로 이 함수 내에서만 사용하고 없어지게 된다. 따라서 호출한 main함수로 돌아갔을 때 main함수의 total의 값에는 영향을 미치지 못한다. 그러므로 14번 출력에서 0이 출력됨을 알 수 있다.
* 20: 함수 내에서 total의 값을 출력한다. 결과 15이다.
* 23\~26: x와 y는 값을 받고 total은 주소(포인터)를 받는다. 여기서 total은 main함수의 주소를 받아서 그 주소에 x와 y의 값을 저장하므로 main함수에서 total값이 바뀜을 알 수 있다. 여기서 total은 main 함수에서의 total과는 달리 지역 포인터 변수이다. 즉 변수 명은 다르지만 주소로 연결되어 결국은 같은 값을 의미한다.
* 25: 포인터의 값을 출력한다. 결과 15이다.

#### ● 예제 08-02-2 : 포인터를 이용한 두 값의 교환
```C
1 #include <stdio.h> 
2 void swap(int *, int *);    // 함수의 원형 
3 
4 int main()
5 {
6 	int x = 15, y = 37;
7 	printf("함수 실행 전 값 x = %d, y = %d\n", x, y);
8 	swap(&x, &y); 		// 변수가 가지고 있는 값의 주소만 넘겨준다.
9 	printf("함수 실행 후 값 x = %d, y = %d\n", x, y);
10 }
11 
12 void swap(int *a, int *b)
13 {
14 	printf("함수 내 실행 전 값 a = %d, b = %d\n", *a, *b);
15 	*a ^= *b, *b ^= *a, *a ^= *b;	// 두 변수의 값 교환
16 	printf("함수 내 실행 후 값 a = %d, b = %d\n", *a, *b);
17 }
```

#### ● 해설
* 2: swap 함수 원형 선언, 이 함수의 반환값은 없고, 매개변수는 정수 포인터 변수 2개이다.
* 8: swap함수를 호출할 때 x, y의 주소를 넘겨준다.
* 12\~17: swap 함수 정의, 이 함수는 매개변수로 포인터를 받아서 함수내부에서 포인터로 연산한다. 포인터 연산은 간접연산이므로 매개변수의 주소에 저장된 값을 간접적으로 계산한다. 따라서 호출한 쪽의 주소를 이용하므로 원래 저장되어 있던 값이 변하게 된다.

#### ● 예제 08-02-3 : 배열을 포인터로 함수에 전달하여 버블 정렬하는 프로그램.
```C
1 #include <stdio.h>
2 
3 void bubble(int *p, int N);
4 int compare(int *m, int *n);
5 
6 int main(void)
7 {
8 	int arr[10] = { 43, 86, 31, 52, 13, 98, 48, 18, 75, 23 };
9 	int i;
10 	printf("\n");
11 	for(i = 0; i < 10; i++)			// 정렬하기 전에 출력한다.
12 		printf("%d ", arr[i]);
13 
14 	bubble(arr,10);				// 정렬함수 호출
15 	printf("\n");
16 
17 	for(i = 0; i < 10; i++)			// 정렬하고 나서 출력한다.
18 		printf("%d ", arr[i]);
19 	printf("\n");
20 }
21 
22 void bubble(int *p, int N) {		// 버블 정렬한다.
23 	int i, j, t;
24 	for(i = N-1; i >= 0; i--) {
25 		for(j = 1; j <= i; j++) {
26 			if(compare(p+j-1, t+j)) {	// 두 수를 비교하는 함수 호출
27 				t = *(t+j-1);		// 데이터 교환
28 				*(t+j-1) = *(t+j);
29 				*(t+j) = t;
30 			}
31 		}
32 	}
33 }
34 
35 int compare(int *m, int *n) {	// 크기를 비교하여 앞의 값이 크면 1을 그렇지 않으면
36 	return (*m > *n);		// 0을 반환한다.
37 }
```

#### ● 해설
* 3: 버블 정렬 함수 원형
* 4: 두 수 비교하는 함수 원형
* 8: 배열 선언
* 14: 정렬 함수 호출, 배열명과 배열의 요수의 개수를 넘겨준다. 배열명은 자체가 포인터이다.
* 22\~33: 포인터를 이용한 버블 정렬 함수
* 26: 포인터를 비교함수에 전달한다. 함수의 반환 결과가 1이면 if문 내부를 수행한다.
* 27-29: 두 수를 교환한다.
* 35\~37: 두 수를 비교하는 함수
* 36: 비교하여 m이 가리키는 곳의 값이 n이 가리키는 곳의 값보다 크면 1을 반환하고, 그렇지 않으면 0을 반환한다.
> [!NOTE]
> 이 예제는 배열을 함수로 전달하여 포인터로 처리하는 방법을 보여준다. 다음 절에서는 배열을 전달하여 배열 그 자체를 처리하는 방법을 설명한다.

#### ● 예제 08-02-4 : 배열을 포인터로 함수에 전달하여 최댓값과 최솟값을 동시에 계산한다. 단, 최댓값과 최솟값에 해당하는 포인터를 함수에 전달한다.
```C
1 #include <stdio.h>
2 
3 void MaxMin(int *array, int Number, int *maximum, int *minimum);
4 
5 int main(void)
6 {
7 	int arr[10] = { 43, 86, 31, 52, 13, 98, 48, 18, 75, 23 };
8 	int max, min;
9 	
10 	MaxMin(arr, 10, &max, &min);
11 	
12 	printf("Max = %d, Min = %d\n", max, min);
13 }
14 
15 void MaxMin(int *dim, int N, int *max, int *min) {
16 	int x;
17 	*max = *min = *dim;		// 최대, 최소에 대한 초깃값을 배열의 첫 요소로 지정
18 	for(x = 1; x < N; x++) {	// 배열의 두 번째 요소부터 비교
19 		dim++;			// 포인터 이동
20 		if( *max < *dim)		// 최댓값 구하기
21 			*max = *dim;
22 		if( *min > *dim)		// 최솟값 구하기
23 			*min = *dim;
24 	}
25 }
```

#### ● 해설
* 10: 배열 이름, 개수, 최댓값 변수의 주소, 최솟값 변수의 주소를 함수로 전달한다.
* 12: 계산된 두 값을 출력한다.
* 15\~25: 배열에서 최댓값과 최솟값을 계산하는 함수
* 17: 배열의 첫 번째 요소를 최대, 최솟값의 초깃값으로 지정한다.
* 18: 두 번째 요소부터 비교하여 최댓값과 최솟값을 구하기 위한 루프
* 20\~21: 최댓값을 구하는 if문
* 22\~23: 최솟값을 구하는 if문

#### ● 예제 08-02-5 : 배열 포인터로 2차원 배열 전달
```C
1 #include <stdio.h>
2 int sum(int (*x)[], int, int) ;
3 
4 int main(void)
5 {
6 	int p, a[2][3]={{1,2,3}, {4,5,6}};
7 	int b[3][3] = { {9,8,7}, {6,5,4}, {3,2,1}};
8 	int (*pt)[3];	// 배열 포인터 
9 
10 	pt = a ;	// pt는 배열 a[0], a[1]을 가리키는 포인터 
11 	p = sum(pt, 2, 3) ;	// sum(a, 2, 3); 
12 	printf("%d\n", p) ;
13 	pt = b ;	// pt는 배열 b[0], b[1], b[2]를 가리키는 포인터 
14 	p = sum(pt, 3, 3) ;	// sum(b, 3, 3); 
15 	printf("%d\n", p) ;
16 }
17 
18 int sum(int (*x)[3], int y, int z)
19 {
20 	int tot=0 ;
21 	int i, j ;
22 
23 	for(i=0 ; i < y ; i++) {
24 		for(j=0 ; j < z ; j++)
25 			tot+= (*x)[j] ;
26 		x++;		// x의 포인터 증가 
27 	}
28 	return tot ;
29 }
```

#### ● 해설
* 6\~7: 배열과 변수 선언
* 8: 배열의 포인터 선언, 3열을 가지는 배열에 대한 포인터
* 10: 배열 a의 주소를 pt에 대입
* 11: 함수 호출, 배열 포인터와 행(2), 열(3)을 전달
* 14: 함수 호출, 배열 포인터와 행(3), 열(3)을 전달
* 18: 배열 포인터 및 행,열을 받아 처리하는 함수
* 23\~27: 행, 열의 값을 모두 더한다.
* 28: 더한 결과를 반환한다.
