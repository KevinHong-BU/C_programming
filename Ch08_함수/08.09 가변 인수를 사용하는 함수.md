### 08.09 가변 인수를 사용하는 함수
가변 인수(매개변수)는 함수의 매개변수의 개수가 지정되지 않은 경우에 사용하는 방법이다.
가변 인수를 사용하려면 표준 헤더 파일인 stdarg.h를 포함하여야 한다.
이 헤더 파일에 가변 인수와 관련된 여러 가지 매크로가 정의 되어 있다
.
va_list는 데이터 형을 재정의한 것이며, va_start(), va_arg(), va_end()는 매크로로 정의 되어 있다.
va_start()는 가변 인수의 시작을 지시하며, va_arg()는 인수를 하나씩 읽어 내는 매크로이고, va_end()는 가변 인수의 처리가 끝났음을 알려준다.
이러한 가변 인수를 사용하는 표준함수로 printf, scanf, sprintf, sscanf, fprintf, fscanf등의 입출력함수에서 찾아 볼 수 있다.

표준 헤더 파일인 stdio.h에 보면 이러한 형식으로 함수 원형이 정의되어 있는 것을 찾을 수 있다.
그러나 실제로는 가변 인수를 사용하는 경우는 많이 찾아볼 수 없다.
이러한 형태를 쓰지 않더라도 프로그램이 가능하므로 크게 신경 쓸 필요는 없을 것으로 생각된다.

---
#### ● 예제 08-09-1 : 개수에 관계없이 덧셈하는 프로그램(정수)
```C
1 #include <stdio.h>
2 #include <stdarg.h>
3 
4 int sum(int nargs, ...);		// 가변 인수를 사용하는 함수 원형 선언
5 
6 int main(void)
7 {
8 	int s1, s2;
9 
10 	/* 1부터 5까지 더한 값을 s1에 저장. */ 
11 	s1 = sum(5, 1, 2, 3, 4, 5);
12 
13 	printf("s1 = %d\n", s1);
14 	/* 100, 200, 300을 더한 값을 s2에 저장. */ 
15 	s2 = sum(3, 100, 200, 300);
16 	printf("s2 = %d\n", s2);
17 }
18 
19 int sum(int nargs, ...)
20 {
21 	va_list argptr;
22 	int i, total = 0;
23 
24 	va_start(argptr, nargs);        /* 가변 인자 처리 시작 */ 
25 
26 	for (i = 0; i < nargs; i++)
27 		total += va_arg(argptr, int); /* 하나씩 가변 인자 얻기 */ 
28 	va_end(argptr);                 /* 가변 인자 처리 끝 */ 
29 	return total;
30 }
```

#### ● 해설
* 4: 가변 인수를 사용하는 함수 원형 선언
* 11, 15: 가변 인수를 가진 함수 호출
* 19: 가변 인수 함수의 시작
* 21: 가변 인수의 시작 포인터 변수 선언
* 24: va_start는 가변 인수의 시작을 알리는 매크로, 첫 번째 인수를 얻어 온다.
* 27: 두 번째 인수부터 하나씩 가져온다. 가져올 인수의 데이터 형이 int임을 나타낸다.
* 28: 가변 인수의 처리가 끝났음을 나타낸다.
> [!NOTE]
> 첫 번째 매개변수는 반드시 지정되어야 한다.


#### ● 예제 08-09-2 : 개수에 관계없이 출력하는 프로그램
```C
1 #include <stdio.h>
2 #include <stdarg.h>
3  
4 void printargs(int arg1, ...) /* 모든 매개변수를 출력한다. 끝은 –1이다. */ 
5 {
6 	va_list ap;
7 	int i;
8 
9 	va_start(ap, arg1); 
10 	for (i = arg1; i != -1; i = va_arg(ap, int))
11 		printf("%d ", i);
12 	va_end(ap);
13 	putchar('\n');
14 }
15  
16 int main(void)
17 {
18 	printargs(5, 2, 14, 84, 97, 15, 24, 48, -1);
19 	printargs(84, 51, -1);
20 	printargs(-1);
21 	printargs(1, -1);
22 }
```

#### ● 해설
* 4: 가변 인수를 사용하는 함수 선언, 함수가 사용되기 전에 선언되면 함수원형을 쓰지 않아도 된다.
* 5: 가변 인수의 시작 포인터 변수 선언
* 9: va_start는 가변 인수의 시작을 알리는 매크로, 첫 번째 인수를 얻어 온다.
* 10: 두 번째 인수부터 하나씩 가져온다. 가져올 인수의 데이터 형이 int임을 나타낸다. 읽어온 값이 –1이 될 때까지 계속한다.
* 12: 가변 인수의 처리가 끝났음을 나타낸다.


#### ● 예제 08-09-3 : 개수에 관계없이 최댓값을 구하여 출력하는 프로그램
```C
1 #include <stdlib.h>
2 #include <stdarg.h>
3 #include <stdio.h>
4 
5 int maxof(int, ...) ;
6 void f(void);
7 
8 int main() {
9 	f();
10 }
11 
12 int maxof(int n_args, ...) {
13 	register int i;
14 	int max, a;
15 	va_list ap;
16 
17 	va_start(ap, n_args);
18 	max = va_arg(ap, int);
19 	for(i = 2; i <= n_args; i++) {
20 		if((a = va_arg(ap, int)) > max)
21 			max = a;
22 	}
23 
24 	va_end(ap);
25 	return max;
26 }
27 
28 void f(void) {
29 	int i = 5;
30 	int j = 24;
31 	printf("%d\n", maxof(3, i, j, 55));
32 }
```

#### ● 해설
* 5: 가변 인수를 사용하는 함수 원형 선언
* 31: 가변 인수를 가진 함수 호출
* 12: 가변 인수 함수의 시작
* 15: 가변 인수의 시작 포인터 변수 선언
* 17: va_start는 가변 인수의 시작을 알리는 매크로, 첫 번째 인수를 얻어 온다.
* 18: 두 번째 인수를 가져와서 max의 초깃값으로 한다.
* 20: 그 다음부터 하나씩 가져와서 max와 비교하여 최댓값을 구한다.
* 24: 가변 인수의 처리가 끝났음을 나타낸다.
* 25: max값을 반환한다.


#### ● 예제 08-09-4 : 개수에 관계없이 덧셈하는 프로그램(실수)
```C
1 #include <stdio.h>
2 #include <stdarg.h>
3 
4 double sum_series(int num, ...);
5 
6 /* 가변 인수처리 함수 예: 실수 값 모두 더하기. */ 
7 int main(void)
8 {
9 	double d;
10 
11 	d = sum_series(5, 0.5, 0.25, 0.125, 0.0625, 0.03125);
12 
13 	printf("Sum of series is %f.\n", d);
14 }
15 
16 double sum_series(int num, ...)
17 {
18 	double sum=0.0, t;
19 	va_list argptr;
20 
21 	/* 첫 번째 매개변수의 포인터를 가져온다. */ 
22 	va_start(argptr, num);
23 
24 	/* 두 번째 매개변수부터 모든 값을 가져와 더한다. */ 
25 	for( ; num; num--) {
26 		t = va_arg(argptr, double); /* 차례대로 하나씩 매개변수를 가져온다. */ 
27 		sum += t;
28 	}
29 
30 	/* 매개변수의 처리를 끝낸다. */ 
31 	va_end(argptr);
32 	return sum;
33 }
```

#### ● 해설
* 4: 가변 인수를 사용하는 함수 원형 선언
* 11: 가변 인수를 가진 함수 호출
* 16: 가변 인수 함수의 시작
* 19: 가변 인수의 시작 포인터 변수 선언
* 22: va_start는 가변 인수의 시작을 알리는 매크로, 첫 번째 인수를 얻어 온다.
* 26: 두 번째 인수부터 하나씩 가져온다. 가져올 인수의 데이터 형이 double임을 나타낸다.
* 31: 가변 인수의 처리가 끝났음을 나타낸다.
