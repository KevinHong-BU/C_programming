### 08.04 함수에 문자열 전달

매개변수로 문자열을 전달하는 방법을 알아보자. 정수나 실수의 전달과 문자열의 전달은 약간의 차이가 있다. 어떤 차이가 있는지 살펴보고 문자열을 전달하는 방법과 전달받아서 취급하는 방법을 살펴보자.

#### ● 문자열은 배열 또는 포인터로 전달할 수 있다.
* 문자배열이나 포인터나 사실은 같은 것이라 볼 수 있다.
* 문자열의 크기는 전달할 필요가 없다.
* 문자열의 끝이 null이라는 것을 이용한다.
```C
function(char []);
function(char);
```

---
#### ● 예제 08-04-1 : 문자열의 길이를 구하는 함수 작성(배열로 이용)
```C
1 #include <stdio.h>
2 int StrLen(char[]);		// int StrLen(char *)
3 
4 int main()
5 { 
6   	int len;
7   	char Str[50] = "This is test.";
8   	printf("%d\n", StrLen(Str));
9 }
10 
11 int StrLen(char s[])		// int StrLen(char *s)
12 { 
13 	int length=0, i=0;
14 	while(s[i]) {		// 널 문자를 만날 때까지 반복한다.
15 		length++;
16 		i++;
17 	}
18 	return length; 
19 }
```

#### ● 해설
* 2: 함수 원형 선언
* 8: 함수 StrLen 호출
* 11\~19: 문자열의 길이를 구하는 함수, 함수는 설명문에 작성한 것처럼 해도 된다.
* 14\~17: 문자열의 길이를 구하는 루프
> [!NOTE]
> 문자열을 전달하는 함수는 문자열 포인터를 전달하는 것과 동일하다. main함수에서는 Str이 포인터 상수로 사용되지만, StrLen함수로 포인터가 넘어가면 포인터 변수로 사용할 수 있다. 굳이 배열로 쓸 필요는 없다. 다음 예는 모두 포인터로 바꾸어 작성하였다.

#### ● 예제 08-04-3 : 문자열의 앞, 뒤에 제거할 문자를 찾아 제거한다.
```C
1 #include <string.h>
2 #include <stdio.h>
3 #include <stdlib.h>
4 
5 void rtrim(char *, char *);
6 void ltrim(char *, char *);
7 
8 int main() {
9 	char *str, *str2;
10 	str = (char *)malloc(100);	// 100바이트 메모리 할당
11 	str2 = (char *)malloc(100);	// 100바이트 메모리 할당
12 	strcpy(str, "  This is example  ");
13 
14 	printf( "%s, %d\n", str, strlen(str));
15 	ltrim(str, " ");		// 문자열의 왼쪽에서 공백 삭제
16 	printf("%s<\n", str);
17 	rtrim(str, " ");		// 문자열의 오른쪽에서 공백 삭제
18 	printf("%s<\n", str);
19  	free(str);
20  	free(str2);
21 }
22 
23 // 문자열의 오른쪽에서 지정된 문자를 만나면 자른다.
24 void rtrim( char * string, char * trim )
25 {
26 	int i;
27 	i = strlen(string) - 1;
28 	for( ; i >= 0 && strchr( trim, string[i] ) != NULL; i--)
29 		string[i] = '\0';		// 문자열의 끝에 NULL 삽입
30 }
31 
32 // 문자열의 왼쪽에서 지정된 문자를 만나면 문자열을 한 칸씩 왼쪽으로 밀어 넣는다.
33 void ltrim( char * string, char * trim )
34 {
35 	while ( string[0] != '\0' && strchr ( trim, string[0] ) != NULL )
36 		memmove( &string[0], &string[1], strlen(string) );
37 }
```

#### ● 해설
* 5,6: 세 개의 함수 원형 선언
* 10,11: 메모리 공간을 할당하여 문자 포인터 변수에 대입
* 15: 문자열의 왼쪽 공백을 자르기 위한 함수 호출
* 17: 문자열의 오른쪽 공백을 자르기 위한 함수 호출
* 19,20: 할당받은 메모리 공간 반납
* 24\~30: 문자열의 오른쪽에 지정된 문자가 있으면 잘라버린다. 여기서는 main함수에서 공백을 넘겨받았으므로 공백이 있으면 자른다.
* 28\~29: 문자열의 끝(오른쪽)에서 지정된 문자열에 해당되는 문자가 있으면 하나씩 널을 삽입한다.
* 33\~37: 문자열의 왼쪽에 지정된 문자가 있으면 문자열을 memmove함수를 이용하여 한 칸씩 왼쪽으로 이동한다.
* Try: 양쪽 모두의 공백을 잘라내는 함수를 작성해 보라.

#### ● 예제 08-04-4 : 문자열을 거꾸로 만드는 함수
```C
1 #include <string.h>
2 #include <stdio.h>
3 #include <stdlib.h>
4 void *strrev(char *);
5 
6 int main() {
7 	char *str, i, len, temp;
8 	str = (char *)malloc(100);
9 	strcpy(str, "This is example.");
10 	printf("%s\n", str);
11 	strrev(str);
12 	printf("%s\n", str);
13  	free(str);
14 }
15 
16 void *strrev(char *s) {
17 	unsigned char len, temp, i;
18 	len = strlen(s);
19 	for(i=0; i<len/2; i++) {
20 		temp = *(s+i);
21 		*(s+i) = *(s+len-i-1);
22 		*(s+len-i-1) = temp;
23 	}
24 }
```

#### ● 해설
* 4: 함수 원형 선언
* 8: 메모리 공간 100바이트 할당
* 9: 문자열 복사
* 11: 문자열 역전함수 호출
* 16\~24: 문자열 역전 함수 정의
* 19\~23: 처음부터 문자열 길이의 반까지 반복하면서 끝의 문자열과 바꾼다.
> [!NOTE]
>  문자열 역전함수 strrev는 표준함수는 아니지만 대부분의 컴파일러에서 제공하고 있다. 여기서는 제공되는 함수가 아닌 직접 만들어 사용한다.

#### ● 예제 08-04-7 : 2진수를 32자리이하로 입력받아 10진수로 변환한다.
```C
1 #include <stdio.h>
2 #include <string.h>
3 
4 int bin2dec(char *);
5 int is32digit_binary(char *);
6 
7 int main()
8 {
9 	char a[100]; 
10 	printf("2진수 32자리 이하를 입력하시오 : ");
11 	scanf("%[01]", a);		// 0과 1만 입력으로 받아들임.
12 
13 	if(is32digit_binary(a))
14 		printf("2진수 %s의 10진수 값은 %d\n", a, bin2dec(a));
15 }
16 
17 /* 2진수 배열을 10진수로 변경하는 함수 */ 
18 int bin2dec(char *x)
19 {
20 	int i, dec=0, exp=1, size;
21 	size = strlen(x);
22 	for (i=size-1; i>=0; i--) {	// 2진수의 뒷자리에서부터 카운트 해준다.
23 		// 해당하는 자릿수가 1일 때만 2의 해당 제곱 승을 더해준다.
24 		if(x[i] == '1')
25 			dec |= exp;
26 		exp <<= 1; // 루프를 한 번씩 돌 때마다 2를 곱해준다.
27 	}
28 	return dec;
29 }
30 
31 /* 문자열 x가 제대로 입력되었는지를 검사하는 함수 */
32 int is32digit_binary(char *x) {
33 	if( strlen(x) > 32 ) {
34 		printf("32자리를 초과하였습니다!!!\n");
35 		return 0;
36 	}
37 	return 1;
38 }
```

#### ● 해설
* 4,5: 함수 원형 선언
* 11: 문자열 입력, 0과 1만 입력받을 수 있게 한다. 다른 문자가 입력되면 직전까지만 받는다.
* 13: 32자리인지 확인하고,
* 14: 2진수 문자열을 10진수로 변환하는 함수 bin2dec를 호출한다.
* 18\~29: 2진수로 된 문자열을 10진수로 변환하는 함수
* 20: exp는 1을 저장하고 루프를 회전할 때만다 왼쪽으로 한자리씩 시프트한다.
* 21: 문자열의 길이를 계산
* 22\~27: 한자리씩 뒷자리부터 1이 있는지 찾아서 계산한다.
* 24: 2진수 문자열이 ‘1’이면 exp와 dec를 비트 OR연산을 하여 그 자리에 값을 넣는다.
* 26: 왼쪽으로 한자리 시프트한다. 2를 곱해주는 역할을 한다. 컴퓨터 내부의 자리를 생각해보라.
* 32\~38: 입력된 문자열이 32자이 이하인지 검사한다.

2 #include <limits.h>

5 void dec2bin(int, char *);

17 	dec2bin(x, binary);

22 void dec2bin(int x, char *a) {
23 	int i;
24 	strcpy(a, "");	// 빈 문자열을 복사한다.
25 	// MSB부터 한자리씩 1인지 0인지 검사한다.
26 	for(i=0; i < 32; i++, x<<=1)

28 }

* 17: 10진수를 문자열로 바꿀 함수 호출, 10진수와 문자열 포인터를 넘겨준다.
* 22\~28: 20진수를 2진수 문자열로 변환하는 함수
* 24: 빈 문자열을 a에 복사한다.
* 26,27: 문자열을 하나씩 최상위 비트부터 검사하여 1이면 ‘1’을 0이면 ‘0’을 문자열에 추가한다.

#### ● 예제 08-04-9 : itoa함수 구현, 표준함수에는 없지만 대부분의 컴파일러에서 제공한다. 이 함수는 10진수 정수를 어떠한 진수로도 바꿀 수 있게 만들었다. 최대 36진법까지 가능하다.
```C
1 #include <stdio.h>
2 
3 void itoa(int , char *, int );
4 
5 int main()
6 {
7 	int dec;
8 	char bin[33];
9 	
10 	printf("%5s|%5s |%5s |%9s\n", "10진", "16진", "8진", "2진");
11 	for(dec = 0; dec < 256; dec++) { // 0부터 255까지 각 진법으로 변환
12 		itoa(dec, bin, 2);		// dec를 2진수로 바꾸어 bin에 저장 
13 		printf("%4d | 0x%02X | %04o | %08s\n", dec, dec, dec, bin);
14 	}
15 }
16 
17 void itoa(int x, char *b, int r)
18 {
19 	char *radix = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";
20 	char *sp, *ep;
21 	if( r < 2 || r > 36)
22 		printf("Error, itoa()'s 3rd argument is greater than 2 "
23 			"and less than 36");
24 	if( x == 0) {
25 		*b = '0';
26 		*++b = '\0';
27 		return;
28 	}
29 	ep = b;
30 	while(x) {		// 나머지 계산하여 문자열 b에 저장하기 
31 		*ep = *(radix + x%r); // 나머지 값에 해당되는 문자를 찾아서 저장 
32 		x /= r;		// 몫 계산 
33 		ep++;			// 포인터 하나 증가 
34 	}
35 	*ep = '\0';		// 끝에 널 문자 삽입 
36 	ep--;				// NULL 앞으로 포인터 이동 
37 	sp = b;			// sp를 b의 첫 번째 주소로 지정 
38 	while(sp < ep) {		// 문자열 거꾸로 만들기
39 		*sp ^= *ep;	// 문자열 위치 바꾸기 
40 		*ep ^= *sp;
41 		*sp ^= *ep;
42 		sp++;			// 첫 포인터 증가
43 		ep--;			// 마지막 포인터 감소
44 	}
45 }
```

#### ● 해설
