### 08.06 main 함수의 매개변수

main함수도 함수이므로 매개변수를 받을 수 있다. main함수의 매개변수는 2개이며, 첫 번째 매개변수는 매개변수의 개수이고, 두 번째 매개변수는 입력된 매개변수의 2차원 문자배열이 된다. 실제 입력되는 매개변수는 모두 문자열로 입력된다. 매개변수는 명령어 입력 줄에서 실행할 때 입력되는 값들이다. 대부분의 통합 컴파일러 환경에서는 실행 시 통합 환경에서 매개변수를 지정할 수 있도록 되어 있다.
```C
int main(int argc, char *argv[]) {
	...
}
```

여기서 argc는 입력된 매개변수의 개수를 나타내며, argv는 입력된 매개변수가 하나씩 문자열로 저장된 문자열 배열이다. argv[0]에는 자기 자신의 실행파일이름이 저장되며, argc[1]부터 실제 입력된 문자열을 저장한다.

---
#### ● 예제 08-06-1 : argc, argv값의 출력 예
```C
1 #include <stdio.h>
2 int main(int argc, char* argv[])
3 { 
4 	int i; 
5 	printf("argc = %d\n", argc); 
6 	for (i=0; i< argc; ++i) 
7 		printf("argv[%d] = %s\n",i, argv[i]); 
8 }
```

#### ● 해설
* 2: main함수의 매개변수를 사용한다.
* 5: argc의 값을 출력한다.
* 6,7: argv의 값을 출력한다.
> [!NOTE]
> 실행은 명령 프롬프트에서 결과와 같이 입력하는 방법과 통합 개발환경에서 입력하는 방법이 있다. 입력되는 데이터들의 구분은 공백을 기준으로 한다. 따라서 결과에서 보는 바와 같다. argv[0]에는 자기 자신의 함수이름이 들어가고, 나머지 argv[1]부터 실제 입력된 값이 저장된다. 단 모든 값은 문자열로 저장된다.

---
#### ● 예제 08-06-2 : main 함수의 매개변수로 16진수를 읽어 10진수로 나타낸다.
```C
1 #include <stdio.h>
2 int main(int argc, char* argv[]) 
3 { 
4 	unsigned int i, dec=0; 
5 	if (argc !=2) {
6 		printf("사용방법 : \n\t mainargs 16진수\n"); 
7 		return 0; 
8 	} 
9 	sscanf(argv[1], "%x", &dec); //문자열 argv[1]에서 16진수를 읽음
10 	
11 	printf("16진수 %s의 10진수 값은 %d이다.\n", argv[1], dec); 
12 }
```

#### ● 해설
* 2: main함수의 매개변수를 사용한다.
* 5\~8: argc가 2가 아니면 사용법을 출력하고 프로그램을 끝낸다.
* 9: argv[1]에서 16진수로 값을 읽어 dec에 저장한다.
> [!NOTE]
> sscanf를 이용하여 바로 값을 변수에 읽어 들일 수 있다. 다른 방법을 사용하게 되면 프로그램이 길어지게 된다. 즉, 일일이 한 글자씩 문자열을 읽어 10진수로 바꾸려면 힘들다.

---
#### ● 예제 08-06-3 : main 함수의 매개변수로 2진수를 읽어 10진수로 나타낸다.
```C
1 #include <stdio.h>
2 #include <string.h>
3 int bin2dec(char *);		// 2진수 문자열을 10진수로 변환
4 int is32digit_binary(char *);	// 32자리인지 확인하는 함수
5 
6 int main(int argc, char* argv[])
7 { 
8 	char a[100];
9 	if (argc !=2) {
10 		printf("사용방법 : \n\t mainargs 2진수\n"); 
11 		return 0; 
12 	} 
13 	sscanf(argv[1], "%[01]", a); // 0과 1만 입력으로 받아들임. 
14 	if (is32digit_binary(a))
15 	    printf("2진수 %s의 10진수 값은 %d\n", a, bin2dec(a));
16 }
17 
18 /* 2진수 배열을 10진수로 변경시키는 함수 */ 
19 int bin2dec(char *x)
20 {
21 	int i, dec=0, exp=1, size;
22 	size = strlen(x);
23 	for (i=size-1; i>=0; i--) { // 2진수 뒤에서부터 카운트 한다.
24 		// 해당하는 자릿수가 1일 때만 2의 해당 누승을 더해준다. 
25 		if(x[i] == '1')
26 			dec |= exp;
27 		exp <<= 1; // 루프를 한 번씩 돌 때마다 2를 곱해준다. 
28 	}
29 	return dec;
30 }
31 // 전에 작성했던 함수를 다시 사용
32 
33 /* 문자열 x가 제대로 입력되었는지를 검사하는 함수 */ 
34 int is32digit_binary(char *x)
35 {
36 	if( strlen(x) > 32 ) {
37 		printf("32자리를 초과하였습니다!!!\n");
38 		return 0;
39 	}
40 	return 1;
41 } // 전에 작성했던 함수를 다시 사용
```

#### ● 해설
* 6: main함수의 매개변수를 사용한다.
* 9\~12: argc가 2가 아니면 사용법을 출력하고 프로그램을 끝낸다.
* 13: argv[1]에서 sscanf 함수를 이용하여 0과 1이 있는 곳까지만 읽어 들인다.
> [!NOTE]
> 나머지 함수에 대한 설명은 앞에서 찾아보기 바란다.
