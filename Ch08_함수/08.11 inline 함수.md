### 08.11 inline 함수

inline함수는 C99에 도입된 새로운 기능으로 간단한 기능을 하는 함수를 함수 내에 포함하는 것이다.

키워드 inline을 사용하여 함수를 정의하며 일반적인 함수와 같은 기능을 하지만 포함된 함수 내에서만 사용할 수 있다.

---
#### ● 예제 08-11-1 : inline 함수를 이용하여 두 수 중에서 큰 값 구하기
```C
1 #include <stdio.h>
2 
3 int main()
4 {
5 	int x=56, y=90, m;
6 	inline int max(int a, int b)		// inline 함수의 정의
7 	{
8   		return (a > b) ? a : b;
9 	}
10 
11 	m = max(x,y);
12 	printf("%d\n", m);
13 
14 	int z=100;			// C99부터는 변수 선언을 어디서나 할 수 있다.
15 	m = max(m, z);
16 	printf("%d\n", m);
17 }
```

#### ● 해설
* 6: inline 함수 max선언
* 11: inline 함수 사용
* 14: C99부터는 변수 선언을 어디서나 할 수 있다.
* 15: inline 함수의 사용

---
#### ● 예제 08-11-2 : inline 함수를 이용하여 두 수 교환하기
```C
1 #include <stdio.h>
2 
3 int main()
4 {
5 	int x=56, y=90;
6 	inline void swap(int *a, int *b) {
7 		*a = *a + *b - (*b = *a);
8 	}
9 
10 	swap(&x, &y);
11 	printf("%d, %d\n", x, y);
12 }
```

#### ● 해설
* 6: inline 함수 max선언
* 11: inline 함수 사용

```C
13 			s += i;

17 	if ( x > y )
18 		swap(&x, &y);
19

21 }
```

#### ● 해설
* 6,10: inline 함수 max선언
