### 08.01 사용자정의 함수
사용자정의 함수는 표준함수 또는 컴파일러가 제공하는 함수 이외에 프로그래머가 직접 만들어 쓰는 함수이다.
C언어에서 함수의 기본은 값을 직접 주고받는다. 값을 주고받는다는 의미는 원래의 값을 그대로 주고 값을 복사해서 함수로 전달하고,
그 결과 값도 마찬가지로 복사해서 전달한다.

따라서 호출하는 쪽의 원래 값에는 아무런 영향을 미치지 않는다. 그리고 함수 내부에서 정의되고,
계산된 값은 특별히 지정하지 않으면 함수가 끝나면 없어지는 것이 기본이다.
이와 관련된 내용을 08-07절 스토리지 클래스에서 다시 명확하게 설명할 것이다.

#### ● 예제 08-01-1 : 약수의 개수를 구하는 함수
```C
1 #include <stdio.h>
2 int NoOfDivisor(int);	// 함수 원형
3 
4 int main()
5 {
6  	int i;		// main함수의 지역변수 선언
7  	for(i=1; i<=20; i++)
8  		printf("%d의 약수의 개수 = %d\n", i, NoOfDivisor(i)); // i: 실인수
9 }
10 
11 // 함수 정의, 함수의 데이터 형:int, 함수이름:NoOfDivisor, 매개변수: 하나의 int형
12 // 약수의 개수를 세어 그 수를 반환한다. n: 가인수
13 int NoOfDivisor(int n)
14 {
15  	int count=0, i;		// NoOfDivisor함수의 지역변수 선언
16  	for(i=1; i<=n/2; i++)	// 1부터 n/2까지 검사하여 약수이면 카운트
17  		if(n%i == 0)
18  			count++;
19 	count++;		// 자기 자신 카운트
20 	return count;	// 결과반환, 반환 값의 데이터 형과 함수의 데이터 형은 같아야 한다.
21 }
```

#### ● 해설
* 2: 함수 원형 선언. 함수 원형은 이 함수가 뒤에 정의되어 있다는 것을 나타낸다. 이 함수의 반환 값은 int형이며, 함수 이름은 NoOfDivisor, 매개변수는 하나 있으며 매개변수의 데이터 형은 int이다.
* 6: main함수의 지역 변수 선언.
* 8: main함수에서 사용자 정의 함수 NoOfDivisor호출, 매개변수는 i이다.
* 13\~21: 사용자 정의 함수 NoOfDivisor 정의, 함수 원형과 형식이 모두 같아야 한다. 단 매개변수의 이름은 달라도 관계없다.
* 15: 함수 NoOfDivisor의 지역변수 선언, 같은 변수 이름을 사용해도 함수가 다르면 다른 변수로 취급한다.
* 20: 결과 값 반환. 함수의 데이터 형과 반환하는 값의 데이터 형이 일치하여야 한다. 이 결과 값은 어떤 정수의	약수의 개수이다.
> [!NOTE]
> 호출하는 쪽의 매개변수를 실인수(실 매개변수)라고 하고, 호출 받는 쪽의 매개변수를 가인수라 하며, 이들은 실제 서로 다른 메모리 장소를 차지한다. 그 값만 복사하여 사용한다. 즉, 실인수에서 가인수로 값을 복사한다. 따라서 가인수의 값이 바뀌어도 실인수의 값은 바뀌지 않는다.

#### ● 함수 원형(prototype)
* 앞의 예에서 : int NoOfDivisor(int);이 함수 원형이다.
* 함수 원형은 프로그램의 맨 처음에 정의하는 것이 보통이다.
* 함수 원형은 이러한 함수가 있다는 것을 미리 알려준다.
* 함수 원형은 그 함수의 반환 값, 함수이름, 매개변수의 데이터 형 및 개수를 표시한다.
* 함수 원형과 실제 함수가 완전히 일치하여야 한다.
* 단, 함수 원형과 실제 함수에서 매개 변수 이름은 일치하지 않아도 되며, 함수 원형에서는 매개 변수 이름을 쓰지 않아도 된다.
* 함수가 사용되기 전에 정의가 되면 함수 원형을 쓰지 않아도 된다. 그러나 가급적이면 함수 원형을 프로그램의 시작 부분에 나열하는 것이 좋다.

#### ● 함수 반환 값(return value)
* 함수의 반환 값은 한 개 또는 없을 수도 있다. 
* 2개 이상의 값을 반환하고 싶을 때는 특별한 방법을 써야한다. 포인터를 쓰거나 배열, 전역변수를 통해 2개 이상의 값을 반환할 수 있다.
* 함수의 반환 값이 없을 경우에는 함수의 데이터 형을 void로 선언한다.
* 함수의 반환 값의 데이터 형은 어떤 타입이든지 가능하다.
* 함수 원형의 데이터 형과 함수 프로그램 부분에서의 반환 데이터 형이 일치하여야 한다.
* 여러 개의 반환 값이 필요할 때는 배열이나 포인터를 사용한다.

#### ● 지역변수(local variables)
* main함수를 포함한 모든 함수에서 지역변수를 선언할 수 있다.
* 지역변수가 없는 경우에는 선언하지 않아도 된다.
* 지역변수는 함수 내에서만 통용되는 변수이다.
* 따라서 지역변수와 다른 함수의 지역변수와 이름이 같아도 서로 다른 변수로 취급한다.(앞의 예에서 i변수)
* 지역변수와 전역변수의 이름이 같을 때는 지역변수가 우선된다.
* 지역 변수는 스택 영역에 생성된다.
* 지역변수는 함수가 호출될 때 만들어지고 함수의 실행이 끝나면 없어진다.
* 단, 지역변수의 값을 유지하고자 할 때는 static으로 정의한다.
* variable scope에서 설명

#### ● 전역변수(global variables)
* 전역변수는 보통 main함수가 시작하기 전에 선언한다.
* 전역변수는 선언된 이후부터 어디에서나 사용할 수 있다.
* 전역변수는 함수 밖에서 선언한다.

#### 예제 08-01-2 : 밑변과 높이를 받아서 삼각형의 넓이를 구하는 함수
```C
1 #include <stdio.h>
2 double CircleArea(double, double);
3

9 
10 double CircleArea(double b, double h)
11 {	// 지역변수가 선언되지 않았다.

13 }
```

#### ● 해설
* 2: CircleArea의 함수 원형 선언
* 7: main함수에서 CircleArea함수 호출
* 10\~13: 함수 CircleArea 정의
* 12: 계산 값 반환, 지역변수 선언 없이 바로 계산하여 반환한다.
> [!NOTE]
> 지역 변수는 없을 수도 있으며, 간단한 경우에는 바로 값을 계산하여 반환할 수 있다.

#### ● 예제 08-01-3 : 최대공약수 구하는 함수 작성
```C
1 #include <stdio.h> 
2 int gcd(int, int);  // 함수 원형(prototype) 
3 
4 int main()
5 {
6 	int i, j; 
7 	printf("최대공약수를 구할 두 수를 입력하시오. "); 
8 	scanf("%d%*c%d", &i, &j); 
9 	printf("두 수의 최대 공약수는 %d이다.\n", gcd(i, j)); 
10 	printf("48과 56의 최대 공약수는 %d이다.\n", gcd(48, 56)); 
11 }
12 
13 int gcd(int a, int b)	// greatest common divisor
14 { 
15 	int z=1;
16 	while(z != 0) { // 나머지를 이용한 유클리드 호제법 
17 		z = a % b;	// a를 b로 나눈 나머지를 z에 저장
18 		a = b;		// 다시 b를 a에 저장, 나머지를 계산한 뒤 큰 값을 a에
19 		b = z;		// 작은 값을 b에 저장한다.
20 	}
21 	return a;
22 }
```

#### ● 해설
* 2: gcd의 함수 원형 선언
* 9,10: main함수에서 gcd함수 호출
* 13\~22: gcd함수의 정의
* 15: 내부 변수 선언
* 16~20: 두 수의 gcd를 유클리드 호제법으로 계산. 여기서는 나머지 연산을 이용하여 더 빨리 계산할 수 있게 작성되었다.
> [!NOTE]
> 지역 변수는 없을 수도 있으며, 간단한 경우에는 바로 값을 계산하여 반환할 수 있다.

#### ● 예제 08-01-4 : 친화수(friendly number) 구하기. 친화수란 어떤 두 정수에서 한 정수의 약수의 합이 다른 정수와 같아지는 수이다.
```C
1 #include <stdio.h>
2 int sum_of_exact_divisor(int);
2 int is_friendly_no(int, int);
3 
4 int main()
5 {
6 	int i, j;
7 	for(i=1; i<1000; i++)
8 		for(j=i+1; j<1000; j++)
9 			if(is_friendly_no(i, j))
10 				printf("%d, %d는 친화수이다.\n", i, j); 
11 } 
12 
13 int sum_of_exact_divisor(int x)
14 {
15 	int i, sum=0;
16 	for(i=1; i<=x/2; i++)
17 		if(x % i == 0)
18 			sum += i;
19 	return sum;
20 }
21 
22 int is_friendly_no(int x, int y)
23 {
24 	int sum1, sum2;
25 	sum1 = sum_of_exact_divisor(x);
26 	sum2 = sum_of_exact_divisor(y);
27 	if( sum1 == y && sum2 == x)
28		return 1;
29 	else 
30 		return 0;
31 }
```

#### ● 해설
* 3: sum_of_exact_divisor의 함수 원형 선언
* 4: is_friendly_no의 함수 원형 선언
* 11: 두 정수를 매개변수로 is_friendly_no 함수로 전달
* 15\~22: sum_of_exact_divisor함수 정의, 이 함수는 약수의 합을 구해주는 함수이다.
* 24\~33: 두 정수의 약수의 합을 구하고, 자기 자신과 상대 수의 약수의 합이 모두 일치하면 참(1)을 반환하고, 그렇지 않으면 거짓(0)을 반환한다.
> [!NOTE]
> 가능하면 한 개의 함수를 너무 길지 않게 작성하는 것이 좋고, 작은 단위로 만들 수 있으면 그렇게 하는 것이 프로그램 작성에 도움이 된다.

#### ● 예제 08-01-5 : 삼각형의 세 변의 길이 a, b, c를 인수로 전달받아 삼각형이 가능한지를 판단하는 함수를 작성하고, 이것이 삼각형을 될 때 삼각형의 면적을 구하는 함수를 헤론의 공식(Heron's formula)을 이용하여 작성하라.heron의 공식 : , , 단, a, b, c는 삼각형의 세변의 길이. S가 삼각형의 면적이다.
```
1 #include <stdio.h>
2 #include <math.h>
3 double triangle_area(int, int, int); /* 함수 원형(prototype) */ 
4 int is_triangle(int, int, int);
5 
6 int main()
7 {
8 	int i, j, k;
9 	printf("첫 번째 수를 입력하시오 : ");
10 	scanf("%d", &i);
11 	printf("두 번째 수를 입력하시오 : ");
12 	scanf("%d", &j);
13 	printf("세 번째 수를 입력하시오 : ");
14 	scanf("%d", &k);
15 	if(is_triangle(i, j, k))
16 		printf("삼각형의 면적은 %f이다.\n", triangle_area(i, j, k));
17 }
18 
19 double triangle_area(int x, int y, int z) {
20 	double s;
21 	s = (x+y+z)/2.0;
22 	return sqrt(s*(s-x)*(s-y)*(s-z));
23 }
24 
25 int is_triangle(int i, int j, int k) {
26 	if( i <= 0 || j <= 0 || k <= 0) // 한 변이라도 0이하이면 삼각형이 아님
27 		return 0;
28 	// 두 변의 길이의 합이 한 변의 길이의 합보다 작으면 
29 	if( i+j < k || i+k < j || j+k < i )
30 		return 0;		// 삼각형이 아니면,
31 	else 
32 		return 1;		// 그렇지 않으면 삼각형이다.
33 }

#### ● 해설
* 3: triangle_area의 함수 원형 선언
* 4: is_triangle의 함수 원형 선언
* 16: 세 개의 정수 매개변수를 triangle_area 함수로 전달
* 19\~23: 세 변수를 넘겨받아 면적을 계산하여 반환한다.
* 25\~33: is_triangle는 삼각형 되는지 안 되는지 검사하는 함수이다. 세 변수 중 하나라도 음수이면 삼각형이 아니며, 또한 두 변수의 합이 다른 한 변수의 합보다 작으면 또한 삼각형이 아니다.
> [!NOTE]
> 함수의 원형만 제대로 정의되어 있다면, 프로그램 파일 내에서 실제 함수의 위치는 중요하지 않다. 다만, 함수 원형을 선언하지 않을 경우 반드시 사용하기 전에 정의되어야 한다.
```
