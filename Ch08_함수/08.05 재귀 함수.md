### 08.05 재귀 함수
재귀함수(recursive function)란 자기 자신을 직간접적으로 호출하는 함수를 의미한다. 즉, 하나의 함수 속에서 자기 자신을 호출하는 것이다.

재귀함수를 사용할 때 주의할 점은 반드시 끝나는 시점이 있어야 한다. 그렇지 않으면 시스템이 다운될 수도 있으므로 끝나는 시점을 확실하게 정의하여야 한다. 또 너무 많이 호출하지 않도록 한다. 

그렇지 않으면 속도가 느려질 뿐 아니라 문제가 발생할 소지가 있다. 따라서 이점이 없을 때는 쓰지 않는 것이 좋다.

---
#### ● 예제 08-05-1 : 재귀함수를 이용하여 1부터 지정된 수까지 출력하기
```C
1 #include <stdio.h>
2 void print_series(int);
3 
4 int main() {
5 	int x;
6 	printf("정수를 입력하시오 : ");
7 	scanf("%d", &x);
8 	print_series(x); 
9 }
10 
11 void print_series(int x) {
12 	if(x==0)	// 끝나는 시점 표시
13 		return;
14 	print_series(x-1);
15 	printf("%d ", x);
16 }
```

#### ● 해설
* 2: print_series 함수 원형 선언
* 8: 재귀함수 호출
* 11\~16: 재귀함수 정의
* 12: 재귀함수의 종료시점 정의
* 14: 자기 자신을 재귀 호출한다.
* 15: 재귀함수에서 반환된 뒤의 작업
Notice: 재귀(再歸, recursive)란 자기 자신으로 다시 돌아간다. 즉, 자기 자신을 다시 호출한다는 의미이다.

#### ● 예제 08-05-2 : 재귀함수를 이용하여 1부터 지정된 수까지 합 구하기
```C
1 #include <stdio.h>
2 
3 int sum_series(int);
4 
5 int main()
6 {
7 	int x;
8 	printf("정수를 입력하시오 : ");
9 	scanf("%d", &x);
10 	printf("%d\n", sum_series(x)); 
11 }
12 
13 int sum_series(int x) {
14 	if(x==0)	// 끝나는 시점 표시
15 		return 0;
16 	return (sum_series(x-1)+x);
17 }
```

#### ● 해설
* 3: sum_series 함수 원형 선언
* 10: 재귀함수 호출
* 13\~17: 재귀함수 정의
* 14: 종료 시점 표시
* 16: 0이 될 때까지 자신을 다시 호출한다. 예를 들어 100이라면 sum_series(99)+100, 즉, 99까지의 합에 100을 더한다. 마찬가지로 99는 sum_series(98)+99이다. 이런 식으로 계속하여 자기 자신을 호출하게 되면 1부터 정해진 수까지의 합을 구할 수 있다. ((...((((1) + 2) + 3) + 4) + ...) + 99) + 100이 된다.

#### ● 예제 08-05-3 : 재귀함수로 피보나치 수열을 구한다.
```C
1 #include <stdio.h>
2 int fib(int); 
3 
4 int main()
5 { 
6 	int ite, l; 
7 	printf("Enter limit : "); 
8 	scanf("%d",&l); 
9 	for(ite=0; ite<l; ite++)
10 		printf("%d ", fib(ite));
11 } 
12 
13 int fib(int n) {
14 	if(n <= 0)
15 		return 0;
16 	else if(n == 1)
17 		return 1;
18 	return fib(n-2)+fib(n-1);
19 } 
```

#### ● 해설
* 2: fib 함수 원형 선언
* 10: 재귀함수 호출, 정해진 항을 한 번에 구할 수 있다. 여기서는 처음부터 정해진 항까지 모두 구하는 것을 
	보기 위해 루프를 사용한다. 즉, ite가 10이라면 바로 10번째 항을 구할 수 있다.
* 13\~19: 피보나치 수열을 구하는 재귀함수 정의
* 14\~17: 종료 시점 표시, 여기서는 종료 시점이 두 가지이다.
* 18: 피보나치 수열을 앞의 두 항을 더해서 현재 항을 구한다. 따라서 항의 수가 늘어날 수로 기하급수적으로 계산할 내용이 많아지게 되므로 너무 많은 항을 입력하게 되면 아주 느려지게 된다. 40이하의 수를 입력하기 바란다. 40을 입력해도 많은 시간이 걸리는 것을 확인할 수 있다.

#### ● 예제 08-05-4 : 재귀함수를 이용하여 정렬되어 있는 정수 데이터에서 값을 찾는다.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 // 정렬되어 있는 배열에서 2진 검색으로 자료를 찾음
4 int Bsearch(int v[], int x, int left, int right)
5 {
6 	int mid;	// bsearch함수는 stdlib.h에 있음.
7 	if(left > right)
8 		return -1;
9 	mid = (left+right)/2;
10 	if(v[mid] == x)
11 		return mid;
12 	else if(v[mid] < x)
13 		return Bsearch(v, x, mid+1, right);
14 	else 
15 		return Bsearch(v, x, left, mid-1);
16 }
17 
18 int main()
19 {
20 	int a[] = {2, 5, 6, 7, 9, 11, 12, 15, 19, 34};
21 	int x, n, num;
22 	num = sizeof(a)/sizeof(a[0]);
23 	printf("찾을 데이터를 입력하시오 : ");
24 	scanf("%d", &x);
25 	n = Bsearch(a, x, 0, num-1);
26 	if( n < 0)
27 		printf("찾으시는 데이터 %d는 없습니다.\n", x);
28 	else 
29 		printf("%d은(는) %d번째에 있습니다. \n", x, n);
30 }
```

#### ● 해설
* 4\~16: Bsearch 함수 원형 선언, 이 함수를 사용하기 전에 미리 정의하게 되면 함수원형을 선언하지 않아도 된다. 여기서는 main함수에서 사용하며, 그 이전에 미리 정의되어 있다.
* 13: 자기 자신을 다시 호출. 찾고자 하는 값이 현재 위치보다 왼쪽에 있을 때, 즉, 값이 작을 때.
* 15: 자기 자신을 다시 호출. 찾고자 하는 값이 현재 위치보다 오른쪽에 있을 때, 즉, 값이 클 때.
* 18: main함수 정의, main 함수는 프로그램 내에 어디에 있든지 상관없다. 다만 사용되는 다른 함수들은 함수 원형을 앞에 선언하거나 사용하기 전에 함수를 정의하여야 한다.
* 20: 정렬된 데이터
* 22: 배열의 크기 계산
* 25: Bsearch함수 호출, a는 배열, x는 찾고자하는 데이터, 0은 시작 위치, num-1은 끝 위치.
Notice: bsearch함수는 stdlib.h에 정의되어 있는 표준함수이다. 여기서는 그 함수와는 달리 만들어 본 것이다. bsearch함수에 대해서는 좀 뒤에 알아보자. bsearch는 binary search이며 2진 검색이라 한다. 2진 검색은 데이터를 반씩 나누어서 빠르게 검색하는 방법으로서 재귀적으로 사용하기에 적당하다.

#### ● 예제 08-05-5 : 재귀함수를 이용하여 10진수를 2진수로 바꾸어 출력하는 프로그램
```C
1 #include <stdio.h>
2 void dec2bin(int);
3 
4 int main()
5 {
6 	int data;
7 	printf("정수 입력: ");
8 	scanf("%d", &data);
9 	dec2bin(data);
10 }
11 
12 void dec2bin(int x)
13 {
14 	if( !x )
15 		return;
16 	dec2bin(x/2);
17 	printf("%d", x%2);
18 }
```

#### ● 해설
* 9: 재귀함수 호출
* 14: x가 0이면 재귀 호출을 끝낸다.
* 16: 몫을 다시 재귀함수로 호출한다.
* 17: 2로 나눈 나머지를 출력한다.
