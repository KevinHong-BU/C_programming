### 08.08 표준함수 qsort와 bsearch 사용하는 방법

qsort() 함수는 퀵 소트(quick sort)를 수행하는 표준함수이며, bsearch() 함수는 2진 검색(binary search)을 수행하는 표준함수로 stdlib.h 헤더파일에 정의되어 있다. 이 함수들은 유용한 함수이지만 사용하는 방법이 조금 까다롭다. 두 함수 모두 어떠한 데이터에 대해서도 모두 사용할 수 있으나 사용되는 데이터형에 따라 비교하는 함수를 따로 정의해야 하는 문제가 있다. 다음의 예제들을 통해서 이들 함수를 사용하는 방법을 익혀보자.

---
#### ● 예제 08-08-1 : qsort함수를 이용한 정수 배열 정렬
```C
1 /* qsort example */ 
2 #include <stdio.h>
3 #include <stdlib.h>
4 int values[] = { 40, 10, 100, 90, 20, 25 };
5 
6 int compare (const void * a, const void * b) {
7 	return ( *(int*)a - *(int*)b );
8 }
9 
10 int main ()
11 {
12 	int n;
13 	qsort (values, 6, sizeof(int), compare);
14 	for (n=0; n<6; n++)
15 		printf ("%d ",values[n]);
16 }
```

#### ● 해설
* 3: qsort함수는 stdlib.h 헤더 파일에 정의되어 있다.
* 4: 정렬되지 않은 배열 선언
* 6\~8: qsort함수는 모든 데이터에 대해서 정렬이 가능하다. qsort함수를 사용하기 전에 데이터 비교함수를 만들어야 한다. 이 함수는 정수데이터를 비교하는 함수이며, 반환 값은 두 데이터를 비교하여 대소를 나타내는 정수 값이다. 앞의 데이터가 크면 양수, 두 개가 같으면 0, 뒤의 데이터가 크면 음수를 반환하도록 만들어야 한다. 다른 데이터 형을 비교할 때도 마찬가지이다.
* 13: qsort함수를 호출한다. 첫 번째 매개변수는 정렬할 배열, 두 번째 매개변수는 배열에 포함된 데이터의 개수, 세 번째 매개변수는 한 데이터가 메모리에서 차지하는 크기, 네 번째 매개변수는 비교함수이다.
* 14,15: 결과를 출력해 보면 데이터가 정렬되어 있음을 알 수 있다.
> [!NOTE]
> qsort와 뒤에 설명할 bsearch함수를 잘 사용하면 아주 편리하게 데이터 정렬 및 검색을 할 수 있다.


#### ● 예제 08-08-2 : qsort함수를 이용한 문자열 배열 정렬
```C
1 #include <stdio.h>
2 #include <string.h>
3 #include <stdlib.h>
4 static int compare(const void *a, const void *b);
5 
6 int main()
7 { 
8 	char line[1024];
9 	char *line_array[] = {"grape", "pear", "banana", "apple",
10 				"orange", "grapefruit"};
11 	int j = 0;
12 	qsort(line_array, 6, sizeof(char *), compare);
13 	while(j < 6)
14 		printf("%s\n", line_array[j++]);
15 }
16 
17 static int compare(const void *a, const void *b) { 
18 	return strcmp(*(char **)a, *(char **)b);
19 }
```

#### ● 해설
* 4: 문자열 비교함수 원형 선언
* 9,10: 정렬되지 않은 문자열 배열 선언
* 12: qsort함수를 호출한다. 첫 번째 매개변수는 정렬할 배열, 두 번째 매개변수는 배열에 포함된 데이터의 개수, 세 번째 매개변수는 한 데이터가 메모리에서 차지하는 크기, 네 번째 매개변수는 비교함수이다.
* 13,14: 결과를 출력해 보면 데이터가 정렬되어 있음을 알 수 있다.
* 17: qsort에 사용할 문자열 비교함수 선언. 문자열은 단순 비교가 안 되므로 strcmp함수를 활용한다.


#### ● 예제 08-08-3 : qsort함수를 이용한 실수 배열 정렬
```C
1 #include <stdlib.h>
2 #include <stdio.h>
3 // 실수를 비교하는 함수! 
4 int compare_doubles (const double * x, const double * y)
5 {
6 	if (*x > *y)
7 		return 1;
8 	else if (*x < *y)
9 		return -1;
10 	else 
11 		return 0;
12 }
13 
14 int main ()
15 {
16 	int i=0;
17 	double a[] = {29.3, 493.2, 9.89, 8393.4, 2939.4, 0.002,
18 			1929.3, 18.1};
19 	qsort(a, 8, sizeof(double), compare_doubles);
20 	printf("after sorting:\n");
21 	for (i=0;i<8;i++)
22 		printf("%f\n",a[i]);
23 }
```

#### ● 해설
* 4\~12: qsort에 사용할 실수 비교함수 선언.
* 17,18: 정렬되지 않은 실수 배열 선언
* 19: qsort함수를 호출한다. 첫 번째 매개변수는 정렬할 배열, 두 번째 매개변수는 배열에 포함된 데이터의 개수, 세 번째 매개변수는 한 데이터가 메모리에서 차지하는 크기, 네 번째 매개변수는 비교함수이다.
* 21,22: 결과를 출력해 보면 데이터가 정렬되어 있음을 알 수 있다.


#### ● 예제 08-08-4 : bsearch 함수를 이용한 정수 데이터 검색
```C
1 /* bsearch example */ 
2 #include <stdio.h>
3 #include <stdlib.h>
4 
5 int compare_ints (const void * a, const void * b)
6 {
7 	return ( *(int*)a - *(int*)b );
8 }
9 
10 int values[] = { 10, 20, 25, 40, 90, 100 }; // 미리 정렬이 되어 있어야 한다.
11 // 그렇지 않으면 정렬한 다음에 사용하여야 한다.
12 
13 int main ()
14 {
15 	int * pItem;
16 	int key = 40;
17 	pItem = (int*) bsearch (&key, values, 6, sizeof (int), compare_ints);
18 	if (pItem!=NULL)
19 		printf ("%d is in the array.\n", *pItem);
20 	else 
21 		printf ("%d is not in the array.\n", key);
22 }
```

#### ● 해설
* 5\~8: bsearch 함수에 사용할 정수 비교함수 선언.
* 10: 정렬되어 있는 정수 배열
* 17: bsearch 함수를 호출한다. 첫 번째 매개변수는 찾을 데이터, 두 번째 매개변수는 정렬되어 있는 배열, 세 번째 매개변수는 배열에 포함된 데이터의 개수, 네 번째 매개변수는 한 데이터가 메모리에서 차지하는 크기, 다섯 번째 매개변수는 비교함수이다.
* 18,19: 검색이 되었다면 검색된 데이터 출력.
* 20,21: 검색이 되지 않았을 때 출력할 메시지
> [!NOTE]
> bsearch는 qsort와 사용방법이 유사하다. 다만 맨 앞에 검색 키가 하나 더 들어간다.


#### ● 예제 08-08-5 : bsearch 함수를 이용한 문자열 데이터 검색
```C
1 /* bsearch example with strings */ 
2 #include <stdio.h>
3 #include <stdlib.h>
4 #include <string.h>
5 
6 char strvalues[][20] = {"some", "example", "strings", "here"};
7 // 정렬되어 있지 않으므로 정렬한 후 검색한다.
8 
9 int main ()
10 {
11 	char * pItem;
12 	char key[20] = "example";
13 	/* 문자열 배열이 정렬되어 있지 않으므로 정렬한다. */ 
14 	qsort (strvalues, 4, 20, (int(*)(const void*, const void*)) strcmp);
15 	/* 정렬된 문자열에 대해서 2진 검색을 한다. */ 
16 	pItem = (char*) bsearch (key, strvalues, 4, 20,
17 			(int(*)(const void*, const void*)) strcmp);
18 
19 	if (pItem!=NULL)
20 		printf ("%s is in the array.\n", pItem);
21 	else 
22 		printf ("%s is not in the array.\n", key);
23 }
```

#### ● 해설
* 6: 문자열 배열 선언, 정렬되어 있지 않다. bsearch를 사용하기 위해서는 먼저 정렬하여야 한다.
* 14: 문자열을 qsort함수를 이용하여 정렬한다. 이 때 비교함수를 따로 쓰지 않고 strcmp를 바로 사용한다.
* 16,17: bsearch 함수를 호출한다. 비교함수를 쓰지 않고 바로 strcmp를 이용한다.
* 19,20: 검색이 되었다면 검색된 데이터 출력.
* 21,22: 검색이 되지 않았을 때 출력할 메시지


#### ● 예제 08-08-6 : qsort를 모든 데이터를 비교하기 위해 하나로 통합하여 쓴다.
```C
1 /* qsort all in 1*/ 
2 #include <stdio.h>
3 #include <string.h>
4 #include <stdlib.h>
5 
6 int (*compare) (const void *, const void *);	// 함수의 포인터
7 // 모든 함수를 동일함 형태로 다시 만든다.
8 int compare_ints (const void * a, const void * b) {
9 	return ( *(int*)a - *(int*)b );
10 }
11 // 모든 함수를 동일함 형태로 다시 만든다.
12 int compare_strings (const void *a, const void *b) { 
13 	return strcmp(*(char **)a, *(char **)b);
14 }
15 
16 // 모든 함수를 동일함 형태로 다시 만든다.
17 int compare_doubles (const void * x, const void * y) {
18 	if (*(double *)x > *(double *)y)
19 		return 1;
20 	else if (*(double *)x < *(double *)y)
21 		return -1;
22 	else 
23 		return 0;
24 }
25 // 초기값, 정렬되지 않은 값을 미리 배열에 저장해 둔다.
26 int values[] = { 40, 10, 100, 90, 20, 25 };
27 char *line_array[] = {"grape", "pear", "banana", "apple", "orange", "grapefruit"};
28 double a[] = {29.3, 493.2, 9.89, 8393.4, 2939.4, 0.002, 1929.3, 18.1};
29 
30 int main ()
31 {
32 	int i = 0;
33 	compare = compare_ints;	   // 함수의 포인터에 정수 비교함수 대입
34 	qsort (values, 6, sizeof(int), compare);	// 정수 정렬
35 	printf("정수 정렬 :\n"); 
36 	for (i=0; i<6; i++)
37 		printf ("%d ",values[i]);
38 	printf("\n\n");
39 
40 	compare = compare_strings;  // 함수의 포인터에 문자열 비교함수 대입
41 	qsort(line_array, 6, sizeof(char *), compare);
42 	printf("문자열 정렬 :\n"); 
43 	for (i=0; i<6; i++)
44 		printf("%s  ", line_array[i]);
45 	printf("\n\n");
46 
47 	compare = compare_doubles;  // 함수의 포인터에 실수 비교함수 대입
48 	qsort(a, 8, sizeof(double), compare);
49 	printf("실수 정렬 :\n"); 
50 	for (i=0;i<8;i++)
51 		printf("%7.1f  ",a[i]);
52 	printf("\n\n");
53 }
```

#### ● 해설
* 6: 함수 포인터를 정의, 이 함수 포인터를 이용하여 서로 다른 비교함수를 필요에 따라 이름을 바꿔 쓴다.
* 8~10: 정수 비교함수 정의
* 12~14: 문자열 비교함수 정의
* 17~24: 실수 비교함수 정의
* 26~28: 초깃값을 가진 정수, 문자열, 실수 배열 선언
* 33: 함수 포인터 compare에 정수 비교함수인 compare_ints 대입
* 34: 정수 배열 정렬
* 36,37: 정렬된 정수 배열 출력
* 40: 함수 포인터 compare에 문자열 비교함수인 compare_strings 대입
* 41: 문자열 배열 정렬
* 43,44: 정렬된 문자열 배열 출력
* 47: 함수 포인터 compare에 실수 비교함수인 compare_doubles 대입
* 48: 실수 배열 정렬
* 50,51: 정렬된 실수 배열 출력
