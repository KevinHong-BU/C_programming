### 08.03 함수에 배열 전달
매개변수로 배열을 전달하는 방법을 알아보자. 정수나 실수의 전달과 문자열의 전달은 약간의 차이가 있다. 이 절에서는 수의 전달에 대해서 살펴보고 다음 절에서는 문자열의 전달에 대해서 알아본다.

---
#### ● 예제 08-03-1 : 배열을 넘겨받아 최댓값을 반환하는 함수를 작성하라.
```C
1 #include <stdio.h>
2 int findMaxNum(int[], int);
3 
4 int main()
5 {
6 	int num1[] = {5, 34, 56, -12, 3, 19};
7 	int num2[] = {1, -2, 34, 207, 93, -12};
8 
9 	printf("numb1[]의 최댓값은 %d이다.\n", findMaxNum(num1, 6));
10 	printf("numb2[]의 최댓값은 %d이다.\n", findMaxNum(num2, 6));
11 }
12 
13 int findMaxNum(int number[], int elements)
14 { 
15 	int largest_value, i;
16 	largest_value = number[0];	// 배열 첫 번째 값을 largest_value 변수에 넣는다.
17 	for (i=0; i < elements; i++)
18 		if(number[i] > largest_value)
19 			// 배열의 값을 순차적으로 largest_value와 비교한다.
20 			largest_value = number[i];
21 	return largest_value;
22 }
```

#### ● 해설
* 2: findMaxNum 함수 원형 선언, 첫 번째 매개변수가 배열임을 나타낸다.
* 9,10: 매개변수로 배열과 그 배열이 가진 요소의 개수를 넘겨준다. 
* 13\~22: 배열을 받아서 최댓값을 찾아서 최댓값을 반환한다.
* 17\~20: 배열의 처음부터 일일이 비교하여 최댓값을 찾는다.
> [!NOTE]
> 배열의 개수를 모르면 어디까지 반복해야 하는지 알 수 없다. 따라서 숫자 배열인 경우 개수도 넘겨주어야 한다.


#### ● 예제 08-03-2 : 어떤 경기의 심판이 7명이다. 이중 최댓값과 최솟값을 제거하고 평균을 계산하여 출력하는 함수를 작성하라. 최댓값과 최솟값을 구하는 함수를 따로 작성하라.
```C
1 #include <stdio.h>
2 double average(int[], int);
3 int findMaxNum(int[], int);
4 int findMinNum(int[], int);
5 
6 int main()
7 {
8 	int num[7] = { 9, 8, 9, 7, 9, 10, 6}, i;
9 	printf("심판들의 점수 ");
10 	for(i=0; i<7; i++)
11 		printf(" %d  ", num[i]);
12 	printf("\n 이 선수의 점수 : %f\n", average(num, 7));
13 }
14 
15 double average(int array[], int x)
16 {
17 	int i, sum=0;
18 	for(i=0; i<x; i++)
19 		sum += array[i];
20 	sum -= ( findMaxNum(array, 7)+ findMinNum(array, 7)); // 최댓값과 최솟값을 뺀다.
21 	return ((double)sum/(double)(x - 2));
22 }
23 
24 int findMaxNum(int number[], int elements) // 최댓값 구하는 함수 
25 { 
26 	int largest_value, i;
27 	largest_value = number[0];	// 배열 첫 번째 값을 largest_value 변수에 넣는다.
28 	for (i=0; i < elements; i++)
29 		if(number[i] > largest_value) // 배열의 값을 순차적으로 largest_value와 비교한다.
30 			largest_value = number[i];
31 	return largest_value;
32 }
33 
34 int findMinNum(int number[], int elements) // 최솟값 구하는 함수 
35 {
36 	int i, smallest_value;
37 	smallest_value = number[0]; // 배열 첫 번째 값을 smallest_value 변수에 넣는다.
38 	for(i=1; i<elements; i++)
39 		if(smallest_value > number[i]) // 배열의 값을 순차적으로 smallest_value와 비교한다.
40 			smallest_value = number[i];
41 	return smallest_value;
42 }
```

#### ● 해설
* 2\~4: 세 함수의 함수 원형을 선언한다.
* 8: 심판들의 점수를 배열에 저장해 둔다.
* 12: 평균함수를 호출한다.
* 15\~22: 평균을 구하는 함수이다.
* 18\~19: 모든 심판들의 점수합을 구한다.
* 20: 총합에서 최댓값과 최솟값을 뺀다.
* 21: 최댓값과 최솟값을 뺀 평균을 구하여 반환한다.
* 24\~32: 최댓값을 찾아주는 함수
* 34\~42: 최솟값을 찾아주는 함수


#### ● 예제 08-03-3 : 배열을 넘겨받아 정렬하는 함수
```C
1 #include <stdio.h> 
2 void sort(int[], int, int); //배열의 이름, 개수, 정렬방법
3 int main() { 
4 	int number[] = {5,34,56,-12,3,19}, loop; 
5 	printf("소팅하기 전의 배열 : "); 
6 	for(loop=0; loop < 6; loop++) 
7 		printf("%d ", number[loop]); 
8 
9 	sort(number, sizeof(number)/sizeof(number[0]), 1);
10 
11 	printf("\n소팅한 후의 배열 : "); 
12 	for(loop=0; loop < 6; loop++) 
13 		printf("%d ", number[loop]); 
14 	printf("\n"); 
15 }
16 
17 void sort(int number[], int elements, int up_down) {
18 	int i, j, temp;
19 	switch(up_down) {
20 	case 0:	// 오름차순 정렬 
21 		for (i=0; i < elements; i++) 
22 			for (j=0; j <= elements; j++) 
23 				if(number[i] < number[j]) {
24 					temp = number[i];
25 					number[i] = number[j];
26 					number[j] = temp;
27 				}
28 		break;
29 	default:	// 내림차순 정렬 
30 		for (i=0; i < elements; i++) 
31 			for (j=0; j <= elements; j++) 
32 				if(number[i] > number[j]) {
33 					temp = number[i];
34 					number[i] = number[j];
35 					number[j] = temp;
36 				}
37 	}
38 }
```

#### ● 해설
* 3: sort 함수의 함수 원형을 선언한다.
* 4: 정렬할 배열 선언
* 6,7: 정렬하기 전의 배열 값 출력
* 9: 정렬함수 호출, 세 번째 매개변수는 오름차순 또는 내림차순으로 정렬하기 위한 정렬방법 지정변수로써 0이면 오름차순, 다른 값이면 내림차순이다.
* 12,13: 정렬한 후의 배열 값 출력
* 17\~38: 정렬 함수 sort정의
* 19: 정렬 방법에 따른 선택
* 20\~28: 오름차순 정렬
* 29\~37: 내림차순 정렬
> [!NOTE]
> 배열은 앞장에서 포인터라고 했다. 따라서 포인터를 주고받는다. 그러므로 여러 개의 값을 주고받는다고 볼 수 있다. 실제로는 시작 포인터만 주고받는 것이다. 그러므로 원래 값이 저장되어 있는 곳의 주소를 주고받으므로 원본 값이 변화가 된다. 이 예에서 보다시피 sort함수의 데이터 형은 void이다. 즉, 실제 값은 주고받지 않지만 포인터로 인하여 원본 값의 변화가 발생하는 것이다.


#### ● 유제 08-03-1 : 배열을 넘겨주고 산술, 조화, 기하평균을 구하는 함수를 각각 작성하라.

산술평균 :
$$\frac 1 n \sum_{n=1}^{n} x+i = \frac 1 n (x_1 + x_2 +x_3 +...+x_n)$$

기하평균 : 
$$\bigl ( \prod_{i=1}^{n} ^{1/n} \bigr ) = n \sqrt{x_1 x_2 x_3 ... x_n} = e^{\frac 1 n \sum_{i=1}^{n}ln x_i} $$

조화평균 : 
$$\Bigl ( \frac {1} {x_1} + \frac {1} {x_1} + \frac {1} {x_1} + ... + \frac {1} {x_1} \Bigr ) = n \Bigl ( \sum_i=1^n \frac 1 {x_1}  \Bigr )^-1$$
