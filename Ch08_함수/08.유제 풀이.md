#### ● 유제 08-04-4 : 문자열에서 white space(space, tab, carriage return, vertical tab, new line 등)가 여러 개 연속으로 출현할 때 하나의 공백으로 만드는 프로그램을 함수를 사용하여 작성하라.

```C
23 
24 	src = string;
25 	dest = string++;
26 
27 	while( (ch = *src++) != 0 ){
28 		if( isWhite( ch ) ) {	// white space일 때 처리
```

```C
10 		return (__int64)GetTickCount();

```
