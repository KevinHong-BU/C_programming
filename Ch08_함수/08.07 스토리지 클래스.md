#### ● 예제 08-07-1 : 자동변수의 선언 및 통용범위
```C
1 #include <stdio.h>
2 int main(void)
3 {
4 	auto int a=5, b=10;	// 함수 내에서의 지역변수 선언 
5 	{ // 블록 시작
6 		int hap;	// 블록 내에서의 지역변수 hap 선언 
7 		hap = a + b;
8 		printf("%3d\n", hap);
9 	} // 블록 끝
10 	printf("a=%d, b=%d\n", a, b);
11 	printf("hap=%d\n", hap); // 블록 밖에서는 hap변수를 사용할 수 없음:컴파일 시 error 발생
12 } // 자동 변수는 그 변수가 속한 블록이나 함수를 벗어나면 사라짐
```

#### ● 해설
* 3: main함수의 매개변수에 void라고 하는 것은 매개변수가 없다는 의미이다. 여기서 void는 생략할 수 있다.
* 4: 자동 변수 a, b 선언, auto는 보통 생략한다.
* 5: 블록의 시작, 이렇게 쓰는 경우는 드물지만 가능하다.
* 6: 블록 내에서 자동 변수 선언. auto는 생략하였다.
* 8: 블록 내에서는 사용 가능하다.
* 11: 블록 밖에서 hap변수는 사용 불가능이다. 따라서 컴파일 시 변수가 선언되지 않았다고 에러가 난다.
> [!NOTE]
> while, for, do\~while, switch~case등에서 {}가 블록이다. 함수도 물론 블록에 해당한다.<br>
> ●  컴파일 시 에러 발생

#### ● 예제 08-07-2 : 레지스터 변수 사용
```C
1 #include <stdio.h>
2 
3 int main(void)
4 {
5 	register int sum=0, i;
6 	for(i=0; i<1000; i++)
7 		sum += i;
8 	printf("sum=%d\n", sum);
9 }
```

#### ● 해설
* 5: 레지스터 변수 sum, i선언 
* 6,7: 레지스터 변수를 이용하여 0부터 999까지의 합을 구한다.
> [!NOTE]
> 레지스터 변수는 잘 사용되지 않는다. 레지스터의 개수가 한정되어 있고, 변수의 개수가 많으면 별 의미가 없어지게 된다.

#### ● 예제 08-07-3 : 지역변수로써 자동변수와 정적변수의 차이점
```C
1 #include <stdio.h>
2 void sub(void)
3 {
4 	int count=1;	// 자동(auto local) 변수 
5 	static int s_count=1;
6 	// 지역변수로 선언된 정적(static local)변수
7 	printf("count=%d s_count=%d\n", count, s_count);
8 	count++;		// count 값은 함수가 끝나면 없어짐
9 	s_count++;		// s_count값은 그대로 유지됨
10 } 
11 
12 int main(void)
13 {
14 	int a;			// 지역(auto local) 변수
15 	for(a=1; a<=5; a++)
16 		sub();
17 }
```

#### ● 해설
* 2\~10: sub함수 정의
* 4: 자동 변수 count 선언. 자동변수는 초깃값을 주지 않으면 쓰레기 값이 지정된다. 자동 변수는 함수가 호출될 때 매번 새로 생성되고 함수가 끝나면 없어진다.
* 5: 정적 변수 s_count선언. 정적 변수는 초깃값을 주지 않으면 0으로 지정된다. 정적 변수는 함수가 처음 호출될 때 한번만 생성하고 그 값을 계속 유지한다.
* 7: 따라서 count는 매번 호출될 때마다 1을 초깃값으로 하므로 항상 1이 된다. 그러나 정적 변수는 처음에만 1이고, 계속 그 값을 유지하면서 출력한다.
* 8: 자동 변수 count는 값이 변해도 이 함수가 끝나면 없어지고, 다시 호출되면 초깃값이 1로 설정된다.
* 9: 정적 변수 s_count는 값이 변하면 그 값을 계속 유지한다.
* 16: main함수에서 sub함수 다섯 번 호출.

#### ● 예제 08-07-4 : 전역변수 선언
```C
1 #include <stdio.h>
2 void change_var();
3 
4 int x;	// 전역 변수 
5 
6 int main()
7 {
8  	x = 10;	// 전역 변수 사용
9  	printf("x=%d\n", x);
10  	change_var();
11  	printf("x=%d\n", x);
12 }
13 
14 void change_var()
15 {
16 	x = 20;	// 전역 변수 사용
17 }
```

#### ● 해설
* 4: 전역 변수 선언, 전역 변수는 모든 함수보다 위에 선언하는 것이 일반적이다.
* 8: main함수에서 전역 변수 사용
* 9: 전역 변수 출력
* 10: 함수 호출
* 11: 전역 변수 출력
* 16: 함수에서 전역 변수 값 수정
> [!NOTE]
> 전역 변수는 그 변수가 선언된 이후부터는 어디서든지 사용할 수 있다.

● 전역 변수의 선언 위치에 따른 문제
프로젝트에 여러 파일을 사용할 때 다른 파일에 있는 전역변수를 끌어다 사용하고자 할 때도 “exterm”이란 키워드를 사용하여야 한다. 선언하지 않고 그냥 다른 파일의 전역 변수를 쓸 수 없으며, 이때는 컴파일이 되지 않고 에러가 발생한다. 전역변수를 “static”으로 선언하였을 경우에는 “extern”으로도 끌어 쓸 수 없다. 전역변수를 static로 선언하면 그 파일 안에서만 사용 가능하다.

#### ● 예제 08-07-6 : 전역 변수의 선언 위치에 따른 문제
```C
1 #include <stdio.h>
2 void change_var();
3 
4 int main()
5 {
6  	extern int x;
7 	// 전역 변수가 아래쪽, 또는 다른 파일에 선언되었을 때, 공유하는 법
8  	x = 30;
9  	printf("x=%d\n", x);
10  	change_var();
11  	printf("x=%d\n", x);
12 }
13 
14 int x;	// 전역 변수 선언 
15 
16 void change_var()
17 {
18 	x = 20;
19 }
```

#### ● 해설
* 6: 전역 변수가 아래쪽에 선언된 경우 그 변수를 사용하려면 extern이란 키워드를 사용한다.
* 14: 전역 변수 선언, 선언된 시점 이후부터 사용가능하다. 그러나 위쪽에서도 extern이란 키워드를 사용하면 사용할 수 있다.
* 17: 물론 아래쪽에서는 당연히 사용가능하다.
> [!NOTE]
> 전역 변수는 그 변수가 선언된 이후부터는 어디서든지 사용할 수 있다.

● 정적 함수(static function)
정적 함수는 같은 파일내의 다른 함수에서만 사용할 수 있으며, 프로젝트내의 다른 파일에서는 사용할 수 없다.

#### ● 예제 08-07-7 : 정적 함수의 사용 예
```C
1 // 파일 이름: main.c 
2 #include <stdio.h>
3 static void Func1(void);
4 void Func2(void);
5 
6 int main()
7 {
8 	Func1();	// 이 함수는 정적 함수이므로 파일이 다르면 사용할 수 없다. 
9 	Func2();	// 이 함수는 정적 함수가 아니므로 사용할 수 있다. 
10 }

1 // 파일 이름: funcs.c 
2 #include <stdio.h>
3 
4 static void Func1(void) {
5 	puts("Func1 called");
6 }
7 
8 void Func2(void) {
9 	puts("Func2 called");
10 }
```

#### ● 해설
* 3,4 : 두 함수의 원형 선언
* 8: 파일 이름이 다르다. 그리고 Func1은 static로 함수가 선언되었으므로, 같은 프로젝트 내라고 하더라도 파일이 다르면 사용할 수 없다. static을 제거하고 실행하면 결과와 같이 된다.
* 9: static이 아닌 함수로 선언된 Func2는 사용할 수 있다.
> [!NOTE]
> 이 예에서는 두 개의 파일로 나누어 프로젝트로 생성하였다. 두 개의 파일 모두에서 static을 제거하고 컴파일하면 실행된다. 파일이 다를 때는 프로젝트에 파일을 추가하여 사용하여야 한다.
