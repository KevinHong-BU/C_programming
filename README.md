# C_programming

**C 프로그래밍 개요**
- **정의:** C는 시스템 프로그래밍과 범용 소프트웨어 개발에 널리 쓰이는 절차적 프로그래밍 언어입니다.
- **특징:** 빠른 실행 속도, 저수준 메모리 제어(`포인터`), 풍부한 표준 라이브러리(`stdio.h`, `stdlib.h` 등).
- **용도:** 운영체제, 임베디드 시스템, 컴파일러, 고성능 애플리케이션 구현에 적합.
- **장점:** 성능과 유연성, 광범위한 플랫폼 지원, 다른 언어(예: C++, Python 확장)의 기반.
- **단점:** 수동 메모리 관리로 인한 버그(메모리 누수, 댕글링 포인터), 비교적 낮은 추상화 수준.

**기본 구조 (핵심 요소)**
- **파일 포함:** `#include <stdio.h>`
- **메인 함수:** `int main(void)` — 프로그램 진입점
- **표준 입출력:** `printf`, `scanf`
- **변수/타입:** `int`, `char`, `float`, `double`, 배열, 구조체
- **제어문:** `if`, `for`, `while`, `switch`
- **함수:** 재사용 가능한 코드 블록, 선언과 정의로 구성
- **포인터:** 메모리 주소 직접 조작 — 핵심적이지만 주의 필요

**간단한 예제:** Hello World
```c
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

**컴파일 · 실행**
```bash
gcc hello.c -o hello
./hello
```
