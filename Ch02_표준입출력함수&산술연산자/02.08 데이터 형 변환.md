### 02.08 데이터 형 변환

앞에서도 언급하였다시피 같은 형의 데이터끼리 계산하여야 하는 것이 원칙이다. 특히 정수형을 나눗셈하면 정수형이 된다. 이 때 정수형을 나누어서 실수형으로 만들고 싶을 때 어떻게 하는지 또는 실수형을 정수형으로 어떻게 바꾸는지 이번 절에서 살펴보도록 하자.

형을 변환하는 방법에는 두 가지가 있다. 묵시적(implicit) 형 변환과 명시적(explicit) 형 변환 방법이 있다. 묵시적 형 변환은 서로 다른 데이터 형끼리 연산하거나 서로 다른 데이터를 대입하는 경우이고, 명시적 형 변환은 캐스트(cast) 연산자를 이용하여 프로그래머가 인위적으로 데이터 형을 변경하는 방법이다.

묵시적인 형 변환은 자동적으로 알아서 서로 다른 데이터 형끼리 계산될 때 데이터의 표현 범위가 작은 것이 큰 쪽으로 변환된다. 명시적인 형 변환은 프로그래머가 직접 바꾸고자 하는 데이터 형으로 지정하는 방법이다. 묵시적 형 변환의 경우 원하는 결과를 얻지 못하는 경우도 발생할 수 있다. 가능하다면 명시적 형 변환 방법을 사용하도록 한다.

---
#### ● 예제 02-08-1 : 묵시적 형 변환 및 결과 비교
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a=29, b=7, c;
5 	double x=5.3, y;
6 	c = (a/b) * x;
7 	y = (a/b) * x;
8 	printf("c=%d, x=%f\n", c, y);
9 	c = (a/x) * b;
10 	y = (a/x) * b;
11 	printf("c=%d, x=%f\n", c, y);
12 }
```

#### ● 해설
* 6: a, b, c는 정수형이고 x는 실수형이다. 먼저 a를 b로 나누면 29/7은 정수 4가 되며 여기에 실수 5.3을 곱하면 실수 21.2가 된다. 이것을 c에 대입하면 다시 정수로 바뀐다. 따라서 c에는 21이 저장된다.
* 7: y는 실수 변수이므로 21.2가 y에 저장됨을 알 수 있다.
* 8: 결과를 확인 해 보자.
* 9: 정수 a를 실수 x로 나누면 결과는 실수 5.4716이 된다. 여기에 정수 b를 곱하면 결과는 실수 38.301887이다. 이 값을 정수형 변수 c에 저장하면 38이 된다.
* 10: 같은 결과를 실수 변수 y에 저장하면 38.301887이 된다. 결과를 보자.

> [!NOTE]
> 결과에서 보다시피 연산을 어떻게 하느냐에 따라 또는 결과를 어디에 저장하느냐에 따라 다른 값이 된다는 것을 알 수 있다. 프로그램을 작성할 때 이러한 점을 잘 고려하여야 한다.

#### ● 예제 02-08-2 : 명시적 형 변환 및 결과 비교
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a=29, b=7, c;
5 	double x=5.3, y, z;
6 	y = (a/b) * x;					// 묵시적 형 변환
7 	z = ((double)a/(double)b) * x;		// 명시적 형 변환
8 	printf("y=%f, z=%f\n", y, z);
9 	y = (a/x) * b;					// 묵시적 형 변환
10 	z = (a/(int)x) * b;				// 명시적 형 변환
11 	printf("y=%f, z=%f\n", y, z);
12 }
```

#### ● 해설
* 6: a, b, c는 정수형이고 x는 실수형이다. 먼저 a를 b로 나누면 29/7은 정수 4가 되며 여기에 실수 5.3을 곱하면 실수 21.2가 된다. 이것을 y에 대입한다.
* 7: a와 b를 모두 캐스트 연산자를 사용하여 실수로 바꾼 다음에 계산한 결과를 z에 저장한다.
* 8: 결과를 확인 해 보자. 둘이 많은 차이를 나타냄을 볼 수 있다.
* 9,10: 정수 연산에서 명시적 형 변환의 결과를 비교해 보라.

> [!NOTE]
> 결과에서 보다시피 연산을 어떻게 하느냐에 따라 또는 결과를 어디에 저장하느냐에 따라 다른 값이 된다는 것을 알 수 있다.

#### ● 예제 02-08-3 : 긴 정수형에서 짧은 정수형으로 변환할 때의 현상
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char ch;
5 	int dec = 365;	// 365 = 101101101
6 	
7 	ch = (char)dec;		// 01101101 = 109
8 	printf("%d\n", ch);
9 }
```

#### ● 해설
* 4: 문자형 변수 선언, char은 1바이트 정수형이다.
* 5: 정수형에 초깃값 365저장, int형은 4바이트 정수형이다.
* 7: 문자형을 정수형으로 바꾸어 문자형 변수에 저장한다. 뒤의 1바이트만 문자형에 저장된다.
* 8: 따라서 109만 저장되었음을 알 수 있다.

> [!NOTE]
> 큰 double형의 값을 int형 변수에 저장하면 어떻게 되는지도 확인해 보자.

#### ● 유제 02-08-1 : 다음 프로그램의 결과를 예측해 보라.
```C
1 #include <stdio.h>
2 int main()
3 {
4  	char c = 'A';
5 	int a=29, b=7;
6 	double x=5.3, y, z;
7 	c = 'A' + (char)32;		// 32를 문자형으로 형 변환
8 	printf("c=%c\n", c);
9 	y = (b/a) * x;
10 	z = ((double)b/a) * x;		// b를 실수형으로 변환
11 	printf("y=%f, z=%f\n", y, z);
12 	y = (x/b) * a;
13 	z = ((int)x/b) * a;		// x를 정수형으로 변환
14 	printf("y=%f, z=%f\n", y, z);
15 }
```

#### ● 유제 02-08-2 : 다음 프로그램의 결과를 예측해 보라.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a=3, b=7;

8 	printf("%d\n\n", (double)a/b);

10 	printf("%f\n", y/x);
11 	printf("%f\n", (int)y/x);
12 	printf("%d\n", (int)y/(int)x);
13 	printf("%d\n", (int)y/x);
14 }
```
