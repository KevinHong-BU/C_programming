# 제2장 표준입출력함수와 산술연산자
2장에서는 C 언어를 시작하는데 가장 기초적인 표준입출력함수, 상수, 변수 및 데이터 형에 관한 내용을 학습한다. C 언어는 입출력을 실행하는 키워드(명령)가 없으며, 입출력을 수행하는 표준함수를 제공하여 이 기능을 대신한다. C 언어에서 사용하는 기본 데이터 형은 크게 정수와 실수로 구분할 수 있다. 기본 데이터 형을 이해하고, 이들 데이터 형을 컴퓨터 프로그램에서 표현하는 방법을 알아보고, 수치 데이터를 처리하는 산술연산과 수식계산을 위한 수학함수를 C 프로그램으로 표현하는 방법을 살펴본다.
- 상수(constant) : 미리 정해져서 고정된 불변의 수
- 변수(variable) : 수시로 값이 변하는 수
- 입출력함수(I/O function) : 데이터를 입력하고 출력하는 기능을 수행하는 함수
- 데이터 형(data type) : 컴퓨터에서 데이터를 표시하기 위하여 정해진 일정한 형태
