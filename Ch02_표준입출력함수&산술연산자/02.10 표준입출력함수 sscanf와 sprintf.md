### 02.10 표준입력함수 `sscanf`와 `sprintf`함수
함수 앞의 접두어 `s`는 string을 의미한다. 즉, 이 두 함수는 문자열을 매개로 입력과 출력을 한다.

`sscanf()`는 문자열 변수에서 정해진 형식의 데이터를 받아들이고, `sprintf()`함수는 형식지정된 출력을 문자열 변수에 저장한다. 즉, 키보드와 모니터가 아닌 문자열 변수를 매개체로 데이터를 입출력한다. 이 함수도 상당히 유용하게 사용할 수 있으므로 알아두는 편이 좋다.

---
#### ● 예제 02-10-1 : `sscanf` 와 `sprintf` 함수를 이용한 데이터 입출력1
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char in_data[20]="50, 129";
5 	char out_data[80];
6 	char out_format[40] = "x의 값은 = %d, y의 값은 = %d\n";
7 	int x, y;
8 	sscanf(in_data, "%d%*c%d", &x, &y);
9 	sprintf(out_data, out_format, x, y);
10 	puts(out_data);
11 }
```

#### ● 해설
* 4: 입력할 데이터를 저장해 놓은 문자열 변수
* 5: 출력할 데이터가 저장될 문자열 변수
* 6: 출력형식지정 문자열
* 8: `"%d%*c%d"`도 문자열로 지정해 두고 써도 된다. 문자열 `in_data`에서 형식지정 데이터 `x`와 `y`를 입력받는다.
* 9: 형식지정문자열로 `x`와 `y`를 `out_data` 문자열에 저장한다.
* 10: 출력된 문자열 `out_data`를 모니터에 출력한다.


---
#### ● 예제 02-10-2 : `sscanf` 와 `sprintf` 함수를 이용한 데이터 입출력2
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char in_data[20]="3.141592, 15.5", in_format[20] = "%lf%*c%lf";
5 	char out_data[80];
6 	char out_format[40] = "반지름이 %f인 원의 넓이는 %f이다.\n";
7 	double pi, radius;
8 	sscanf(in_data, in_format, &pi, &radius);
9 	sprintf(out_data, out_format, radius, pi*radius*radius);
10 	puts(out_data);
11 }
```

#### ● 해설
* 4: 입력할 데이터 및 출력할 형식을 저장해 놓은 문자열 변수
* 5: 출력할 데이터가 저장될 문자열 변수
* 6: 출력형식지정 문자열
* 8: 문자열 `in_data`에서 형식지정 데이터 `pi`와 `radius`를 입력받는다.
* 9: 형식지정문자열로 `radius` 및 원의 면적을 계산하여 `out_data` 문자열에 저장한다.
* 10: 출력된 문자열 `out_data`를 모니터에 출력한다.
> [!NOTE]
> `scanf`, `printf`에서도 입출력할 데이터의 형식을 지정해서 사용할 수 있다.


---
#### ● 예제 02-10-3 : `sscanf` 와 `sprintf` 함수를 이용한 데이터 입출력3
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char Email[40]="kdhong@korea.org";
5 	char in_format[30]="%[^@]%[@]%[^\n]";
6 	char ID[20], at[2], server[30];
7 	sscanf(Email, in_format, ID, at, server);
8 	printf("%s\t%s\n", ID, server);
9 }
```

#### ● 해설
* 4: 입력할 데이터(e-메일 주소)를 저장해 놓은 문자열 변수
* 5: 입력 받을 형식을 지정해 놓은 문자열 변수
* 6: 데이터를 저장할 변수
* 7: Email문자열에서 지정된 형식인 `in_format`으로 `ID`, `at`, `server`를 읽는다.
* 8: 데이터를 화면에 출력한다.

#### ● 유제 02-10-2 : `sscanf()` 함수와 `sprintf()` 함수를 이용하여 문자열에서 10진 정수를 읽어서 16진 정수로 문자열에 출력하고 화면에도 출력하라.
