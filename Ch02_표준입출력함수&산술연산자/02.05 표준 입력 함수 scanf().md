### 02-05 표준 입력 함수 scanf()
C언어는 입력 명령이 없으며, 표준 입력 함수로 입력을 처리하도록 하고 있다. 표준입력장치는 키보드이다. 표준 입력 함수로 많이 쓰이는 `scanf()` 함수에 대해서 살펴본다. `scanf()` 함수  원형은 다음과 같다. 이 함수의 원형은 헤더파일 `<stdio.h>`에 정의되어 있다.

⇨ `scanf() 함수의 원형: int scanf  ( const char * format, ... );`

| int | scanf | ( | const char * | format | , | ... | ) | ; |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:
| ① | ② | ③ | ④ | ⑤ | ⑥ | ⑦ | ⑧ | ⑨ |


| ① | int | 함수를 실행한 뒤 반환하는 값의 데이터 형 |
|:---|:---|:---|
| ② | scanf | 함수명 |
| ③ | ( | 함수에 전달할 매개변수의 시작 |
| ④ | const char * | 첫 번째 매개변수의 데이터 형 |
| ⑤ | format | 첫 번째 매개변수의 이름, 이 이름은 큰 의미는 없음 |
| ⑥ | , | 매개변수 구분 문자 |
| ⑦ | ... | 여러 개의 매개변수가 가능하다는 것을 표시(가변 매개변수) |
| ⑧ | ) | 매개변수의 끝 |
| ⑨ | ; | 문장의 끝을 표시 |

※ 이 함수의 실행 결과는 실제 입력된 데이터의 개수를 가져온다.

`scanf()` 함수에서 숫자 데이터를 입력할 때 공백, 탭, 줄 바꿈 등으로 구분하여야 하며, 콤마는 해당하지 않는다. `scanf()` 함수에서 형식지정자들 사이의 문자들은 반드시 입력하여야 한다. 예를 들어 `scanf("%d, %d", &x, &y)`와 같이 `%d`뒤에 콤마가 있으면 입력할 때 반드시 콤마를 넣어야 한다. 따라서 `scanf()` 함수를 쓸 때 이러한 점에 주의해야 한다. <예제 02-05-4>는 콤마가 있을 때와 없을 때 모두 처리하는 방법을 보여준다. 또 02-06절에서는 더 다양한 `scanf()` 함수의 사용법도 알아본다.

---
#### ● 예제 02-05-1 : scanf를 이용한 10진 정수의 입력
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 int main()
4 {
5  	int a, b, *c;		// a, b는 일반 변수이고, c는 포인터 변수이다.
6  	c = (int *)malloc(4);	// 포인터 변수는 사용하기 전에 공간을 확보한다.
7 	// c = (int *)malloc(sizeof(int));로 해도 됨
8  	scanf("%d %d %d", &a, &b, c);
9  	printf("a=%d, b=%d, c=%d\n", a, b, *c);
10 	free(c);
11 }
```

#### ● 해설
* 5: 변수 선언, *c는 포인터 변수이다. 포인터 변수는 7장에서 배우는데 여기서는 `scanf`함수의 사용법을 배우기 위한 일환으로 쓴 것이므로 `scanf`의 개념을 배운다는 의미에서 알아두자.
* 6: 포인터변수는 주소가 지정되어야 하므로 주소를 할당하기 위해 `malloc()` 함수를 썼다. 이 함수도 7장에서 배우도록 하자. `malloc`의 매개변수 4는 정수가 메모리에서 차지하는 공간의 크기를 의미하며, 할당된 메모리 공간의 시작 주소를 c에 저장한다.
* 7: 정수가 메모리에서 차지하는 공간은 `sizeof(int)` = 4바이트이다.
* 8: `&a`, `&b`에서 `&`는 주소연산자로써 변수 `a`, `b`의 주소를 의미한다. 즉, `a`, `b`의 주소(포인터)를 나타낸다. 그러나 c는 그 자체가 포인터(정수포인터)이므로 앞에 주소연산자를 붙일 필요가 없다. `%d`는 10진 정수를 입력하겠다는 의미이다. 또, `scanf`에서 `%d`와 `%d`사이에 공백을 넣었는데 이는 입력할 때 공백을 기준으로 데이터를 구분하겠다는 의미이다.
* 9: 각 변수의 값들을 출력한다. `a`, `b`는 변수이름을 쓰고, `c`는 포인터 변수이므로 포인터의 값인 `*c`로 쓴다.
* 10: `free`함수를 이용하여 `malloc`로 할당된 메모리를 해제한다. 나중에 설명한다.

> [!NOTE]
> `scanf`함수는 매개변수로 포인터를 쓴다. 일반 변수일 경우에는 `&`를 붙여 쓰고, 포인터 변수는 그냥 쓴다. 출력 함수에서 형식지정자 이외에는 단순히 모니터에 출력된다.

> [!NOTE]
> `scanf`에서 데이터 구분은 형식지정문자열에 지정된 문자로 구분한다. 위에서는 공백으로 하였으나 콤마나 다른 기호를 쓸 수도 있다. 그 기호를 중심으로 데이터를 구분한다.

#### ● 예제 02-05-2 : scanf를 이용하여 실수(float, double) 입력
```C
1 #include <stdio.h>
2 int main()
3 {
4  	float x;
5 	double y;
6 	printf("float, double 입력 : ");
7 	scanf("%f, %lf", &x, &y);	// 입력 데이터 구분자로 ,를 사용하였다.
8  	printf("x=%f, y=%f\n", x, y);
9 }
```

#### ● 해설
* 6: 무엇을 입력할 것인지를 화면에 출력해 준다. `scanf`를 하기 전에 이와 같이 출력해 줌으로써 명확하게 사용자가 무엇을 입력하여야 하는지 알 수 있다.
* 7: `float`에 대한 입력형식지정자는 `%f`이며, `double`에 대한 입력형식지정자는 `%lf`이다. 실수 변수에 데이터를 입력할 때에도 주소 연산자인 `&`를 변수 앞에 사용한다.
* 8: 출력형식지정자는 `float`, `double` 모두 `%f`이다.

> [!NOTE]
> `double`형에 `%f`를 쓰면 제대로 된 데이터가 입력되지 않는다. `scanf`에서는 형식지정자 이외에 다른 문자가 입력되면 문법적으로는 문제가 없지만 데이터에 오류를 일으킬 수 있다.

#### ● 예제 02-05-3 : scanf를 이용하여 문자와 문자열의 읽기
```C
1 #include <stdio.h>
2 int main()
3 {
4  	char c, str1[20], str2[20];
5 	printf("문자, 문자열1, 문자열2 입력 : ");
6 	scanf("%c %s %s", &c, str1, str2);
7  	printf("c=%c, str1=%s, str2=%s\n", c, str1, str2);
8 }
```

#### ● 해설
* 4: 문자 및 문자열에 대한 변수 선언
* 6: 문자 및 문자열을 키보드에서 입력한다. 문자에 대해서는 주소연산자를 써야하고 문자열에 대해서는 주소연산자를 쓰지 않는다. 문자열 자체가 포인터이기 때문이다.

> [!NOTE]
> 문자열은 시작되는 곳의 주소를 가지는 포인터이다. 문자열은 `NULL`로 끝난다. 실행결과에서 보는 바와 같이 문자열을 입력하면 공백을 기준으로 구분되어 입력된다. 즉 한 단어만 입력된다는 점에 주의하자.

#### ● 예제 02-05-4 : scanf()에서 %d와 %i의 차이점
```C
1 #include <stdio.h>
2 
3 int main()
4 {
5 	int x, y;
6 	printf("input : ");
7 	scanf("%d%*c%d", &x, &y);
8 	printf(" %%d => x:%d, y:%d\n", x, y);
9 	fflush(stdin);			// 키보드에 남아있는 찌꺼기를 제거한다.
10 	printf("input : ");
11 	scanf("%i%*c%i", &x, &y);
12 	printf(" %%i => x:%d, y:%d\n", x, y);
13 }
```

#### ● 해설
* 7: `%d`로 `031` 및 `0x31`로 입력받을 때 10진수로 인식한다. `%*c`는 `x`와 `y`변수 사이에 어떤 구분자가 오더라도 인식을 하게 하려는 방법이다.
* 8: `x`는 31을 입력받고 `y`는 0을 입력받는다, 입력데이터 `0x31`에서 `x`는 문자로 인식되어 앞의 0만 받아들인다.
* 9: `fflush()`함수는 키보드처럼 입출력 장치의 버퍼를 비우는 역할을 한다.
* 11: `%i`를 형식 지정자로 사용하면 `031`은 8진수로 `0x31`은 16진수로 자동 인식한다.
* 12: 따라서 출력은 8진수 `031`의 10진수 변환 값인 `25`, 16진수 `0x31`은 10진수 변환 값인 `49`가 된다. 즉, `%i`는 10진수, 8진수, 16진수를 구별하여 정확한 데이터 입력을 지원한다.
