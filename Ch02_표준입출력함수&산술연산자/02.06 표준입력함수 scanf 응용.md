### 02-06 표준입력함수 scanf 응용
앞 절에서 scanf() 함수에 대한 기본적인 사항들을 살펴보았다. 이번 절에서는 scanf() 함수를 편리하게 사용하는 방법들을 살펴보자.

---
#### ● 예제 02-06-1 : scanf를 이용하여 구분자를 마음대로 하기
```C
1| #include <stdio.h>
2| int main()
3| {
4|   	double a, b, c;
5| 	printf("3변의 길이 : ");
6|	scanf("%lf%*c%lf%*c%lf", &a, &b, &c);
7| 	printf("a = %f, b = %f, c = %f\n", a, b, c);
8| }
```

#### ● 해설
* 6: 형식지정문자에서 %*c는 데이터와 데이터 사이에 숫자와 .을 제외한 다른 문자가 오면 제거하고 받아들인다. 물론 완벽하지는 않지만 편리하게 사용할 수 있다.
* 7: 아래의 출력 결과들을 살펴보라. 공백이나, 콤마, 세미콜론 등이 있어도 문제없이 읽어 들인다.

> [!NOTE]
> 좀 더 세밀하게 제어할 필요가 있을 때는 다른 방법이 있다. 몇 가지 예를 이어서 보도록 하자.

---
#### ● 예제 02-06-2 : scanf를 이용하여 문자열을 받아들일 때의 문제 및 해결방법
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char string[100];
5 	printf("문자열을 입력하시오 : ");
6 	scanf("%s", string);
7  	printf("입력된 문자열 = %s\n", string);
8 	fflush(stdin); 
9 	printf("문자열을 입력하시오 : ");
10 	scanf("%[^\n]", string);
11 	printf("입력된 문자열 = %s\n", string);
12 }
```

#### ● 해설
* 6: %s형식지정자로 문자를 받아들이면 결과가 어떻게 되는지 결과를 보라. 문자열로 받아들일 때 구분자는 콤마나 공백을 만나면 끝나게 된다. 따라서 한 단어밖에 읽어 들이지 못한다.
* 8: fflush는 키보드에 남아 있는 값들을 지운다. 어떤 컴파일러에서는 안 될 수도 있다. stdin: 표준입력장치 즉, 키보드를 의미한다.
* 10: [^...]은 ...에 포함된 문자 직전까지 읽어 문자열에 저장한다. 따라서 여기서는 \n, 즉, 엔터키 바로 직전까지 읽어 들인다. 이 형식지정자는 반드시 문자열로 읽어 들인다. 한 줄의 내용을 모두 읽을 수 있다.

> [!NOTE]
> ^문자 뒤에는 가급적이면 ASCII코드 순서대로 쓰기 바란다. 순서에 따라 가끔 오동작을 일으키기도 한다.


---
#### ● 예제 02-06-3 : 우편번호와 주소를 입력하여 우편번호와 주소 분리하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char address[50], zipcode[20];
5 	printf("우편번호와 주소 입력 : ");
6 	scanf("%[0123456789-]%*c%[^\n]", zipcode, address);
7 	printf("우편번호 : %s\n주소 : %s\n", zipcode, address);
8 }
```

#### ● 해설
* 6: [...]은 ...에 포함된 문자까지 읽어 문자열에 저장하고, [^...]은 ...에 포함되지 않은 문자까지 읽는다. %[0123456789-]는 숫자와 –가 있을 때까지만 읽고, %[^\n]은 엔터 키 직전까지 읽는다. %*c는 사이에 포한된 불필요한 구분 문자를 제거한다.실행 결과에서와 같이 330-180은 숫자와 –만 가지고 있어서 이것만 zipcode에 입력받고, 가운데 콤마와 공백을 제거하고 다음 문자열을 엔터키가 입력될 때까지 읽어서 address에 저장한다.
