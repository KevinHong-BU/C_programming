## 02-04 실수의 산술 연산 및 수학함수

C언어의 실수 연산과 수학함수에 대해서 살펴보자. 실수 연산자는 정수에서 사용하는 4칙 연산자(`+`, `-`, `*`, `/`)와 부호(`+`, `-`)가 있다. 이들은 정수에서와 같은 방법으로 사용한다. 정수와 실수를 혼합하여 계산할 수 있다. 정수와 실수가 혼합하여 계산이 이루어 질 때는 정수 상수 및 정수 변수는 실수로 자동 변환되어 계산되며, 그 결과는 실수이다. 따라서 이러한 점을 잘 고려하여 계산식을 써야 한다.

실수 데이터를 처리하기 위한 다양한 수학기호 또는 함수를 표준함수로 제공하고 있으므로 이들을 사용하여 다양한 연산을 처리할 수 있다. 수학 함수를 사용하려면 표준 헤더파일인 `<math.h>`를 포함하여 컴파일하여야 한다. 수학 함수도 일반 함수처럼 사용하면 되므로 크게 어려운 점은 없다. 다만 삼각함수의 경우 각도를 사용하는 것이 아니라 라디안(radian)을 사용하므로 주의를 요하며, 매개변수의 개수와 용도를 정확하게 익혀둘 필요가 있다. <표 2-7>에 함수명과 사용하는 방법을 표시하여 두었으며, 필요한 곳에 적절하게 쓰기만 하면 된다.

프로그래밍에서는 수학 함수를 사용하는 방법만 알면 된다. 수학에서처럼 일일이 계산할 필요가 없으므로 수학에 대해서 잘 몰라도 함수의 사용방법만 알아도 프로그래밍할 수 있다.

#### ● 예제 02-04-1 : 원의 면적과 둘레의 길이 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	double pi=3.141592, radius=5.0;	// 실수 변수 선언
5 	double area, circumference;
6 	area = pi*radius*radius;		// 실수 연산, 원의 면적= 
7 	circumference = 2.0*pi*radius;	// 원의 둘레=
8 	printf("원의 넓이 = %f\n", area);	// 실수 출력
9 	printf("원의 둘레 = %f\n", circumference);
10 }
```

#### ● 해설
* 4: 실수 변수 선언 및 초깃값 지정
* 5: 원의 면적과 둘레의 길이를 저장할 변수 선언
* 6: 원의 면적 구하기
* 7: 원의 둘레 구하기
* 8,9: 실수 값을 출력할 때는 %f 형식지정자를 사용한다.

#### ● 예제 02-04-2 : 화씨를 섭씨온도로 바꾸기
$$ \degree C = ( \degree F - 32 ) * \frac {5} {9} $$
```C
1 #include <stdio.h>
2 int main()
3 {
4 	double fahrenheit=80.0, celsius;
5 	celsius = (fahrenheit – 32.)* 5./9.;	// 실수 연산
6 	printf("화씨 %f = 섭씨 %f\n", fahrenheit, celsius);
7 }
```

#### ● 해설
* 4: 실수 변수 선언
* 5: 온도 변환 공식에 대입, 괄호가 우선순위가 높다. 순서대로 좌에서 우로 계산한다. 프로그램에서는 *를 생략할 수 없다.

> [!NOTE]
> 소수점 이하의 값이 0일 때 0은 생략할 수 있다. 실수 상수는 소수점을 포함하여 나타낸다.실수 값의 기본형은 double이며, float으로 나타낼 때는 뒤에 f를 붙인다. 5.0f, 9.0f 등.

#### ● 예제 02-04-5 : 삼각형의 세 변의 길이 , , 를 알고 있을 때, Heron의 공식을 이용한 삼각형의 넓이 구하기,  : sqrt()
```C
1 #include <stdio.h>
2 #include <math.h>
3 int main()
4 {
5 	double a=3., b=4., c=5., s;
6 	s = (a + b + c)/2.;
7 	s = sqrt(s*(s-a)*(s-b)*(s-c));		// 함수 sqrt사용
8 	printf("삼각형의 면적 = %f\n", s);
9 }
```

#### ● 해설
* 2: math.h는 수학함수들의 함수 원형이 정의되어 있는 헤더파일이다.
* 5: 삼각형의 세 변을 실수 변수로 선언하고 넓이가 저장될 변수 s도 선언한다.
* 6: 세 변의 길이의 합을 2로 나눈 결과를 s에 저장한다.
* 7: sqrt는 square root로 제곱근을 구하는 수학함수이다. 이 함수는 math.h에 정의되어 있으며 계산할 값을 괄호 속에 넣어주기만 하면 그 결과를 계산하여 준다. 괄호속의 값을 매개변수라 하며 이 함수는 매개변수의 데이터 형이 double이며, 계산 결과도 double이다. 따라서 여기서 매개변수는 계산식 s*(s-a)*(s-b)*(s-c)이며, 그 결과를 계산하여 돌려준다.

> [!NOTE]
> 수학함수를 사용할 때 math.h헤더 파일을 include하여야 하며, 수학식을 계산할 때 편리하게 사용할 수 있다. 여러분은 단순히 함수만 호출하여 쓰기만 하면 된다. 우리가 계산할 필요가 전혀 없다. 필요한 수학함수가 무엇인지 알기만 하면, 필요할 때 적절히 가져다 쓰기만 하면 된다.

#### ● 예제 02-04-6 : 2차 방정식 ()의 근 구하기근의 공식 : , 
```C
1 #include <stdio.h>

3 int main()
```

* 5: 2차 방적식의 세 개의 계수와, 두 개의 근을 저장할 변수를 선언한다.
* 6,7: 함수를 이용하여 제곱근과 근을 계산을 한다. 일반적으로 근이 두 개이므로 두 번 계산한다.

#### ● 예제 02-04-7 : sin, cos, tan, asin, acos, atan 값 구하기, 
```C
1 #include <stdio.h>
2 #include <math.h>
3 int main()
4 {
5 	const double pi=3.141592;
6 	double degree=30., s, c, t;
7 	printf("계산 결과는 다음과 같습니다.\n");
8 	s = sin(degree*pi/180.0); 		// 사인 30° 값을 구한다.
9 	c = cos(degree*pi/180.0);		// 코사인 30° 값을 구한다.
10 	t = tan(degree*pi/180.0);		// 탄젠트 30° 값을 구한다.
11 	printf("sin(%f) = %f\n", degree, s);
12 	printf("cos(%f) = %f\n", degree, c);
13 	printf("tan(%f) = %f\n", degree, t);
14 
15 	// 다시 각도로 만들기 위해 180/3.141592를 곱한다.
16 	printf("asin(%f) = %f\n", s, asin(s)*180/pi); // 아크사인 값을 구한다.
17 	printf("acos(%f) = %f\n", c, acos(c)*180/pi); // 아크코사인 값을 구한다.
18 	printf("atan(%f) = %f\n", t, atan(t)*180/pi); // 아크탄젠트 값을 구한다.
19 }
```

#### ● 해설
* 5: 정해진 값을 선언할 때는 이와 같이 상수 기호로 선언하는 것이 좋다.
* 8,9,10: sin, cos, tan함수를 이용하여 그 결과를 구한다. 단 주의할 사항은 삼각함수의 매개변수는 radian 값을 사용한다는 것이다. 각도를 라디안을 변경하여 매개변수로 사용하여야 한다.
* 16,17,18: asin, acos, atan는 삼각함수의 역함수로써 그 결과는 라디안이다. 따라서 여기서 다시 각도로 바꾸어 준다.

> [!NOTE]
> 삼각함수의 매개변수는 radian이다. 역삼각함수의 결과도 radian이다. radian과 각도를 변환하는 방법을 알아두자.
