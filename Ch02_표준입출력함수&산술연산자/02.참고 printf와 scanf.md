### 참고 : printf() 함수: 형식 변환 출력

#### 1. 함수 원형 (<stdio.h>에 정의)

```C
int printf(const char *format, ...)
```

이 함수의 반환(return) 값은 출력된 문자의 개수(여백<whitespace> 포함)이며, 오류가 발생하였을 경우에는 음수 값이 반환된다.

#### 2. 형식지정문자열

형식지정문자열의 구성은 다음과 같다.

```C
% [플래그] [필드 폭] [.소수이하 자리 수] [길이 변환형식] 변환문자
```

여기서 []은 생략할 수 있다. 최소한 %와 변환문자는 있어야 한다(예, %i).

#### 3. 플래그

이들은 순서에 관계없이 사용할 수 있다.

| 플래그 | 의미 |
|:---|:---|
| - | 출력은 필드 내에서 왼쪽 정렬된다. 기본은 오른쪽 정렬이다. |
| + | 부호가 있는 수인 경우 항상 부호를 표시한다(+ 또는 -). |
| space | 양수인 경우 공백이 앞에 추가된다(음수는 -부호가 추가된다). |
| 0 | 숫자 형식인 경우 필드의 폭 만큼 0을 앞에 채운다. |
| # | 다양한 형식으로 출력한다. o(8진수)형에서는 첫 번째 자리는 '0'. x 또는 X(16진수)형일 때 0이 아닌 결과 값에 대해서 "0x" 또는 "0X"을 앞에 출력한다. e, E, f, F, g 및 G형에 대해서는 항상 소수점을 출력하고; g와 G형에 대해서는 뒤에 이어지는 0들을 제거하지 않는다. |

#### 4. 필드 폭

출력될 매개 변수의 폭은 최소한 이 값보다는 크거나 같아야 한다. 출력할 변수의 폭이 지정된 길이 보다 작을 때에는 왼쪽(또는 오른쪽: 왼쪽정렬일 때)에 다른 문자가 채워진다. 채워질 문자는 일반적으로 ' ' (공백, space)이며, 플래그가 0일 때는 '0'으로 채워진다.
필드 폭이 숫자가 아닌 *으로 지정될 때에 필드 폭은 변수로 지정되며 정수이어야 한다. 예를 들어 printf("%*d\n", 5, x);는 x변수를 출력하되 필드 폭은 5칸으로 한다.

#### 5. 소수 이하 자리 수

점 '.'뒤에는 소수 이하 자리 수를 지정한다. 이것도 *로 지정할 수 있으며 마찬가지로 변수로 지정하며, 정수이어야 한다. 예를 들어 printf("%*.*f\n", 10, 5, pi);는 전체 자리수가 10자리이며, 소수이하 자리 수를 5자리로 지정하고, pi를 출력한다.

| 변환문자 | 의미 |
|:---|:---|
| s | 문자열을 출력할 때 소수이하 자리가 지정되면 지정된 자리만큼의 문자만 출력한다. |
| e, E, f | 소수점 뒤에 출력될 자리수를 지정한다. |
| g, G | 소수이하 가수의 자리수를 지정한다. |
| d, i, o, u, x, X | 최소한의 자리 수만 출력한다. 필드 폭만큼 빈자리는 0으로 채워진다. |


#### 6. 길이변환 형식(Length modifier)

| 문자 | 의미 |
|:---|:---|
| h | 값은 short 또는 unsigned short로 출력된다. |
| l | d, i, o, u, x 또는 X 변환: 매개 변수는 int가 아니라 long이다. |
| L | e, f, g 또는 G 변화: 매개변수는 long double이다.|


#### 7. 변환 문자(Conversion character)

| 문자 | 의미 |
|:---|:---|
| d, i | 부호가 있는 10진수 형식으로 정수를 출력한다. |
| o | 부호가 없는 8진 형식으로 정수를 출력한다(0을 앞에 출력하지 않는다). |
| u | 부호가 없는 10진수 형식으로 정수를 출력한다. |
| x, X | 부호가 없는 16진 형식으로 정수를 출력한다(앞에 0x 또는 0X을 출력하지 않는다). x는 소문자 형식으로, X는 대문자 형식으로 출력한다. |
| c | 하나의 문자를 출력한다(부호가 없는 정수형으로 변환한 후에). |
| e, E | double 또는 float(double로 변환한 후에) 과학적 표기방법으로 출력한다. e는 소문자 형식으로, E는 대문자 형식으로 출력한다. |
| f | 10진 형식으로 double 또는 float(double로 변화한 후에)을 출력한다. |
| g, G | g는 e 또는 f 형식으로 출력한다. 값의 크기와 지정된 소수 이하 자리 수에 따라 자동적으로 선택된다. G도 마찬가지이며 E 또는 F형식으로 출력한다. |
| n | 아무것도 출력되지 않는다. 해당되는 매개변수는 반드시 정수변수에 대한 포인터이어야 하며, 변환된(출력된) 문자의 개수가 이 변수에 저장된다. |
| s | 문자열을 출력한다. 매개변수는 문자 포인터이다. 문자는 '\0'을 만날 때까지 출력하거나 지정된 소수이하 자리 수까지만 출력한다(단말 문자인 '\0'는 출력되지 않는다). |
| p | 포인터를 출력한다(to any type). |
| % | %를 출력한다. |


### 참고 : scanf() 함수: 형식지정 입력

#### 1. 함수원형(<stdio.h>에 정의)
int scanf(const char *format, ...)

이 함수의 반환 값은 입력된 값의 개수이거나 'EOF'이다. 'EOF'는 파일의 끝에 도달했거나 변환이 일어나기 전에 오류가 발행한 경우이다.

#### 2. 형식지정 문자열
형식지정 문자열은 입력을 어떻게 변환할 지를 알려준다. 다음과 같은 것들이 있다:

공백과 탭은 무시되지만 가독성을 위해서 삽입할 수 있다.
입력에서 일반문자들은 여백이 아닌 문자들에 대하여 연결된다. (여백 문자는 공백(' '), 탭('\t'), 새줄 문자('\n', newline), 캐리지 리턴('\r', carriage return), 수직 탭('\v') 및 종이 넘김('\f')등이다). 나머지 모든 문자는 여백이 아니다. 문자 %는 특별한 문자이다(아래 참조).
하나이상의 변환 지정문자가 있어야 하며, 각각은 다음형식과 같다.

% [*] [필드 폭] [길이 변환형식] 변환문자

여기서 []는 생략할 수 있다. 최소한 %와 변환문자는 있어야 한다(예, %i). 일반적으로 하나의 변수에 대해 하나의 변환지정자가 있어야 한다. 변환과정은 형식지정문자가 끝나거나 입력에 형식변환에 문제가 발생했을 때 멈춘다.

입력 필드는 여백이 아닌 문자들로 정의된다.

#### 3. 대입 억제 플래그 `*`
`*` 플래그가 포함되면 대입이 되지 않는다. 즉, 입력 필드는 지정된 형식에 따라 변환이 되지만 변수에 대입은 되지 않는다. 그 이외에는 해당되는 매개변수에 대입되며, 이 때 해당되는 매개변수는 포인터이어야 한다.

#### 4. 필드의 폭
입력 필드(여백이 포함되지 않은 문자열)는 이후의 여백 문자나 지정된 필드 폭(지정되어 있다면)이 될 때까지 간다. 따라서 scanf는 입력받기 위해서 라인의 끝까지 검색할 것이다. 왜냐하면 newline이 여백이기 때문이다.

#### 5. 길이변환 형식(Length modifier)

문자
의미
h
d, i, o, u 또는 x 변환에 대해서 입력 값은 int가 아니라 short이다.
l
d, i, o, u 또는 x 변환에 대해서 입력 값은 int가 아닌 long이 된다.
e, f 또는 g 변환에 대해서 입력 값은 float가 아니라 double이 된다.
L
e, f 또는 g 변환에 대해서 입력 값은 long double이 된다.


6. 변환 문자(Conversion character)

| 문자 | 의미 |
|:---|:---|
| d | 10진 정수 입력; 매개변수는 반드시 int *이어야 한다. |
| i | 정수 입력; 매개변수는 반드시 int *이어야 한다. 정수는 8진수(0으로 시작) 또는 16진수(0x 또는 0X로 시작) 일수도 있다. |
| o | 8진수 정수 입력(0이 앞에 있어도 되고 없어도 된다); 매개변수는 반드시 int *이어야 한다. |
| u | 부호 없는 10진 정수; 매개변수는 반드시 int *이어야 한다. |
| x | 16진 정수 입력(0x 또는 0X가 앞에 있어도 되고 없어도 된다); 매개변수는 반드시 int *이어야 한다. |
| c | 하나 이상의 문자 입력; 매개변수는 반드시 char *이어야 한다. 문자는 배열에 저장되며, 주어진 필드 폭까지 저장된다(단말 문자인 '\0'이 추가되지 않는다). 필드 폭이 주어져 있으면, 여백을 건너뛰지 않는다. 필드 폭이 주어져 있지 않으면, 하나의 문자만 입력된다(바로 다음에 여백이 아닌 문자를 읽으려면 %1s를 사용한다). |
| s | 여백이 아닌 문자열 입력; 매개변수는 반드시 char *이어야 한다. 단말 문자인 '\0'이 추가된다. |
| e, f, g | 부동소수점 수 입력; 매개변수는 반드시 float *이어야 한다. 입력 형식은 숫자열(소수점을 포함할 수 있음) 이며 부호는 옵션이고, 지수 필드는 옵션으로서 E나 e뒤에 필요한 자리수를 정수(부호가 있어도 됨)로 지정한다. 따라서 입력으로 가능한 수는 10진수나 과학적 표기 방법으로 표현된 수이다. |
| p | 포인터 값을 입력받는다. 매개변수는 반드시 void *이어야 한다. |
| n | 입력이 없다. 대신, 이 입력이 읽혀지기 전까지 받아들인 문자의 수를 매개변수에 저장한다; 매개변수는 반드시 int *이어야 한다. |
| [...] | 문자열 입력; 매개변수는 반드시 char *이어야 한다. 단말 문자인 '\0'이 추가된다. 괄호 [] 속에 나열된 문자들만 받아들인다. 괄호 속에 해당되지 않는 문자를 만나면 정지한다. ']'를 입력으로 받기 위해서는 []...]로 시작하면 된다. |
| [^...] | 문자열 입력; 매개변수는 반드시 char *이어야 한다. 단말 문자인 '\0'이 추가된다. 괄호 속에 나열되지 않는 문자까지만 입력으로 받아들이다. 입력은 일치하지 않는 문자까지만 받아들이거나 필드 폭까지만 받아들인다. ']'문자를 받아들이지 않으려면 [^]...]로 시작한다. |
| % | 문자 %; '%'문자를 읽어 들인다. 할당은 되지 않는다. |
