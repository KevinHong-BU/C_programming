## 02-02 C언어 데이터 형 및 변수
C언어의 기본 데이터 형은 크게 정수형(고정소수점)과 실수형(부동소수점)으로 나눌 수 있다. 기본 데이터 형은 컴퓨터 내부에서 표현하는 데이터 형을 그대로 사용하여 데이터를 처리할 수 있도록 지정하는 것을 말한다. 따라서 기본 데이터 형의 표현 범위는 프로세서(processor)에 의존적이며, 때로는 컴파일러(compiler)에 따라서도 다를 수 있다.

변수(變數:variables)는 상수와 달리 프로그램이 실행 중에 수시로 값이 변경될 수 있는 수를 말한다. 컴퓨터에서 데이터는 주기억장치(DRAM)에 저장된다. 데이터는 모두 2진수로 저장되며, 정수와 실수는 컴퓨터 내부에서 표시하는 방법이 전혀 다르다. 이들 데이터의 표현 방법은 부록을 참고하기 바란다. 그리고 실제 저장된 데이터를 2진수로 확인하는 방법에 대해서는 이후 여러 군데서 하나씩 설명한다.

### 02-02-1 기본 데이터 형의 종류
데이터 형은 크게 정수와 실수로 구분할 수 있다. 정수형에는 `char`(문자형), `int`(정수형), `enum`(열거형:8장)이 있으며, 이들은 메모리에 저장되는 크기에 따라서 다양한 형태가 존재한다. 정수는 양수와 음수를 모두 표현할 수도 있고, 양수만으로 나타낼 수도 있다.

`char`은 문자를 표시하기 위한 방법을 제공하며, 이를 ASCII코드라 한다. 실제 문자는 2진수로 저장되며 표시하는 방법에 따라 문자로 바꾸어 표시하거나, 숫자로 표시하기도 한다. ASCII코드 표는 부록을 참고한다. 또한 `unsigned` 접두어를 앞에 붙여서 양수만 표시할 수도 있다. `char`은 문자 표시뿐만 아니라 작은 범위(-128~+127, or 0~255)의 정수를 나타낼 때도 유용하게 사용할 수 있으며, 계산에 사용할 수 있다. 문자열(string)은 기본 데이터 형이 아니라 문자를 여러 개 정해진 순서에 따라 나열한 것으로서 문자배열이라고도 하며 6장에서 자세히 살펴본다.

정수를 나타내는 기본 데이터 형은 4바이트(32비트) `int`이다. `int`형도 마찬가지로 양수와 음수를 모두 나타낼 수 있으며, 양수만 나타낼 수도 있다. 또한 접두어 `short`를 쓰면 2바이트 정수로 나타내고, `long`을 쓰면 8바이트로 나타낼 수 있게 한다. 그러나 `long`의 경우는 프로세서나 컴파일러에 따라 4바이트로 쓰는 경우도 있다. C99 버전에서는 `long long`(8바이트: 64비트 정수형)이 새로 추가되어 사용할 수 있다.

enum(열거형)은 어떤 데이터를 정해진 순서에 따라 기호로 나열하고 그 기호에 번호를 붙여서 나타내며 기본적으로 정수형이다. 열거형에 대해서는 9장에서 설명한다.

실수형은 IEEE754표준에 따라서 표시하며, 반정밀도(16비트), 단일정밀도(32비트), 복수정밀도(64비트), 4배정밀도(128비트)가 제시되어 있으나, 일반적으로 PC환경에서는 2가지(32비트, 64비트) 또는 3가지(32비트, 64비트, 80비트)가 쓰이기도 한다. 32비트 실수는 float, 64비트 실수는 double로 정의하여 사용하며, 기본형은 double이다.

문자의 기본형은 char, 정수의 기본형은 int, 실수의 기본형은 double이다. 기본형이란 상수를 지정할 때 ‘a’, 123, 343.345와 같이 지정하면 기본형으로 취급한다는 의미이다. 즉, 문자는 부호를 가진 8비트 2의 보수로 -128~127까지 지정되며, 정수는 부호가 있는 32비트 2의 보수 데이터, 실수는 64비트 IEEE754 표준 데이터로 지정된다는 것을 의미한다.

### 02-02-2 변수
변수(變數:variable number)란 변하는 값을 의미한다. 즉, 프로그램 실행 중에 값을 필요에 따라 변경할 수 있다는 의미이다. 변수는 특정한 메모리 위치에 이름(변수명)을 붙여 데이터가 저장될 장소(메모리의 위치)를 지정한 다음 그곳에 값을 저장한다. 따라서 변수를 사용하려면 먼저 그 위치를 나타내는 이름을 선언한 뒤 사용하여야 한다.

변수명은 식별자(identifier)라고 하며, 32글자 이내의 알파벳, 숫자, 밑줄(_, underscore)로 구성되며, 이외의 문자는 사용할 수 없다. 또한 첫 글자에 숫자는 사용할 수 없으며, 키워드(keyword)도 변수명으로 사용할 수 없다. 변수는 상수와 달리 초깃값을 정해주지 않아도 된다. 즉, 데이터가 저장될 공간(위치)만 지정하고 나중에 값을 저장해도 된다. 모든 데이터 형은 변수로 선언할 수 있다.

변수 선언은 프로그램의 시작 전에 선언하는 것이 일반적이며, 블록({ }) 내에서도 선언 가능하다. 다만 블록 내에서 선언한 경우에는 그 블록 안에서만 유효하다. C99 표준에서는 변수의 선언을 프로그램의 시작이 아닌 어디서나 선언할 수 있게 하였다. 같은 종류의 데이터는 콤마로 구분하여 여러 개를 나열할 수도 있으며, 따로 따로 하나씩 개별적으로 선언할 수도 있다.

### 02-02-3 예제 프로그램
#### ● 예제 02-02-1 : 변수 선언과 초깃값 지정 및 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int price = 35000;		//정수변수 선언 및 초깃값 설정
5 	double pi = 3.141592;		//실수변수 선언 및 초깃값 설정
6 	char c = 'x';			//문자변수 선언 및 초깃값 설정
7 	char str[20] = "C language";	//문자열변수 선언 및 초깃값 설정
8 	printf("%d, %f, %c, %s\n", price, pi, c, str);	// 변수의 값 출력
9 }
```

#### ● 해설
* 4: 정수(int) 변수 price를 선언하고 초깃값 35000을 대입(=)한다. 대입연산자 =은  변수에 값을 저장한다. 변수에 값을 저장하면 기존의 값이 새로운 값으로 대체한다.
* 5: 실수(duble) 변수 pi를 선언하고 초깃값 3.141592를 대입한다.
* 6: 문자(char) 변수 c를 선언하고 초깃값 ‘x’를 대입한다.
* 7: 문자열(char []) 변수 str을 선언하고 초깃값 “C language”를 대입한다. 문자열은 문자의 배열로서 뒤에서 배운다.
* 8: 변수에 저장된 값을 출력한다.
> [!Note]1
>  문자열의 끝에는 항상 NULL 문자가 들어간다. 출력할 때 이 널 문자 앞까지만 출력한다. 지정된 길이보다 문자열을 더 길게 대입하면 그 데이터를 보장할 수 없다. 문자열의 크기는 넉넉히 지정한다.

> [!Note]1
> Notice2: 정수의 기본형은 int, 실수의 기본형은 double이다. 위에서 실수 상수 3.141592는 double형이다. 만약 float로 값을 지정하고자 한다면 3.141592f로 하여야 한다. 즉, 상수 값 뒤에 f를 붙인다.

> [!Note]3
> Notice3: 변수에 값을 저장할 때는 대입연산자 =을 사용한다. 변수 = 식, 여기서 식에는 상수, 변수 또는 다양한 식이 올 수 있다. 이 예에서는 변수를 선언하면서 상수를 변수에 대입하였다.

#### ● 예제 02-02-2 : 다양한 데이터를 정수 변수에 대입
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a, b=94, c=0112, d=0x9a, e='t';		// 변수 선언 
5 	printf("%d, ", a);
6 	printf("%d, ", b);
7 	printf("%d, ", c);
8 	printf("%d, ", d);
9 	printf("%d\n", e);
10 }
```

#### ● 해설
* 4: a는 선언만 하고 값은 대입하지 않았고, b에는 10진수 94를, c에는 8진수 0112를, d에는 16진수 0x9a를 e에는 ASCII 문자 ‘t’를 대입하였다.
* 5: 아무 값도 대입하지 않는 a는 어떤 값이 출력될지 알 수 없다.
* 6: b에 저장된 94가 출력된다.
* 7: 8진수 0112가 10진수(%d) 74로 출력된다.
* 8: 16진수 0x9a가 10진수(%d) 154로 출력된다.
* 9: 문자 ‘t’의 ASCII코드 값인 116으로 출력된다.
* Notice: 문자는 정수형 데이터이다. 작은 범위(-128~127)의 숫자를 쓸 때는 정수변수로도 사용할 수 있다.


#### ● 예제 02-02-3 : 다양한 진법으로 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int a=30;			// 변수 선언 
5 	printf("%d, ", a);	// 10진수로 출력
6 	printf("%o, ", a);	// 8진수로 출력
7 	printf("%x, ", a);	// 16진수 소문자로 출력
8 	printf("%X\n", a);	// 16진수 대문자로 출력
9 }
```
#### ● 해설
* 4: 정수 변수 a를 선언하고 초깃값 30 대입
* 5: 정수 변수 a의 값을 10진 정수(%d)로 출력
* 6: 정수 변수 a의 값을 8진 정수(%o)로 출력
* 7: 정수 변수 a의 값을 16진 정수 소문자(%x)로 출력
* 8: 정수 변수 a의 값을 16진 정수 대문자(%X)로 출력


#### ● 예제 02-02-4 : 실수 값의 다양한 표현방법
```C
1 #include <stdio.h>
2 int main()
3 {
4 	double a=123.456, b=12.3e4, c=0., 
5 		d=.0, e=.2345, f=0.23e-4; // 변수 선언 
6 	printf("%f, %f, %f\n", a, b, c);
7 	printf("%f, %f, %f\n", d, e, f);
8 }
```

#### ● 해설
* 4,5: 실수를 다양한 형태로 값을 지정할 수 있다. 일반형, 지수형 등. 0.0일 때는 앞이나 뒤의 0을 하나 생략할 수 있다. 지수는 양수 또는 음수가 가능하다. 변수를 선언하고 초깃값을 지정할 때는 대입연산자인 “=”를 사용한다. 대입이란 변수에 값을 지정하는 것을 의미한다. 대입연산은 기존의 값을 새로운 값으로 대체한다. 세미콜론을 만나기까지는 한 문장이 끝난 것이 아니다. 즉, 이 두 줄은 한 문장이다.
* 6,7: 저장된 변수의 값을 실수(%f)형태로 출력한다.


#### ● 예제 02-02-5 : 문자 변수의 다양한 표현과 연산
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char x='a';
5 	printf("%c, ", x-32);	// 32는 ASCII 코드에서 ‘a’와 ‘A’사이의 거리이다.
6 	printf("%c, ", x);
7 	printf("%c, ", 65);	// 대문자 A는 ASCII코드 값 65이다.
8 	printf("%d%c", x-'A', '\n');
9 }
```

#### ● 해설
* 4: 문자 변수 x에 소문자 ‘a’를 저장한다.
* 5: 소문자 ‘a’에서 32를 빼면 대문자 ‘A’가 된다. ASCII 코드로 ‘a’는 97이며, ‘A’는 65이다.
* 6: 소문자 ‘a’를 출력한다.
* 7: 정수 65를 문자로 출력하면 ‘A’가 된다.
* 8: 소문자 ‘a’와 대문자 ‘A’사이의 거리는 32이다.
* Notice: 문자는 정수형의 한 종류이므로 연산을 정수처럼 할 수 있다. 정수형이라고 취급하면 된다.단, 표현 범위가 작을 뿐이다.


#### ● 예제 02-02-6 : 문자열 변수의 선언
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char x[20] = "This is string!";	// 문자열 배열로 선언
5 	char *y = "This is string too!";	// 문자열 포인터로 선언
6 	printf("%s\n", x);
7 	printf("%s\n", y);
8 }
```

#### ● 해설
* 4: 문자열 변수를 선언하는 방법은 두 가지이다. 첫 번째 문자의 배열로 선언한다.
* 5: 문자열 변수를 포인터 변수로 선언한다. 문자열 변수는 5장, 포인터는 6장에서 다시 다룬다.
* 6,7: 문자열을 출력할 때에는 형식지정자 %s를 사용한다.
* Notice: 특히 포인터 변수는 배열 변수와 약간의 차이가 있다.여기서는 문자열 배열과 문자열 포인터 변수를 선언함과 동시에 값을 저장했지만 프로그램이 실행 중일 때는 이와 같이 직접 대입할 수 없다. 또한 포인터 변수는 더욱 사용할 때 신중을 기해야 한다. 포인터 변수를 선언만 하여서는 사용할 수 없다.


#### ● 예제 02-02-7 : 각 데이터 형이 메모리에서 차지하는 크기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	printf("%d : char\n", sizeof(char));
5 	printf("%d : short int\n", sizeof(short int));
6 	printf("%d : int\n", sizeof(int));
7 	printf("%d : long int\n", sizeof(long int));
8 	printf("%d : long long int\n", sizeof(long long int));
9 	printf("%d : float\n", sizeof(float));
10 	printf("%d : double\n", sizeof(double));
11 	printf("%d : long double\n", sizeof(long double));
12 }
```

#### ● 해설
* 4: sizeof는 함수처럼 보이지만 함수는 아니며 연산자 키워드이다. 이 키워드는 데이터 형의 크기를 계산한다.
* 4~10: 각 데이터 형의 크기를 바이트 단위로 계산하여 출력한다. 데이터 형에 대한 크기를 구할 때는괄호를 생략할 수 없다. 변수나 상수에 대한 크기는 괄호를 생략할 수 있다. <예제 * 02-02-8> 참조.
* Notice: sizeof는 기본 데이터 형뿐만 아니라 사용자 정의 데이터 형(응용데이터 형)에 대한 크기도 알려준다. 데이터 형의 크기(길이)는 컴파일러마다 약간 다를 수 있다.


#### ● 예제 02-02-8 : 변수와 상수가 차지하는 메모리의 크기를 알아보기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=5;
5 	double y=10.0;
6 	printf("%d, %d\n", sizeof x, sizeof y);
7 	printf("size of constant: %d\n", sizeof 2);
8 }
```

#### ● 해설
* 4,5: 변수 선언
* 6: 변수가 차지하는 메모리의 크기를 출력한다. 여기서 변수일 경우 괄호를 생략할 수 있다. sizeof가 연산자이기 때문이다. 그러나 데이터 형인 경우 괄호는 써야한다. <예제 02-02-7>참조.
* 7: sizeof에 상수를 사용할 수 있다. 2는 기본 정수형인 int이다.


#### ● 예제 02-02-9 : float, long long int, long double 데이터 형의 상수 표현 및 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4      float x = 3.14159f;
5      long long y = 54684569654653215LL;
6      long double z = 98765431212.341E+1293L;
7      printf("float : %f\n", x);
8      printf("long long : %I64d\n", y);
9      printf("long double : %Le\n", z);
10 }
```

#### ● 해설
* 4: float 형의 실수 상수를 쓸 때는 상수 뒤에 ‘f’를 적는다.
* 5: long long int 형의 정수 상수를 쓸 때는 상수 뒤에 ‘LL’을 적는다.
* 6: long double 형의 실수 상수를 쓸 때는 상수 뒤에 ‘L’을 적는다.
* 7: float 값을 출력할 때는 형식지정자 %f를 쓴다.
* 8: long long int 값을 출력할 때는 형식지정자 %I64d를 쓴다.
* 9: long double 값을 출력할 때는 형식지정자 %Lf, %Le 또는 %Lg 등을 쓴다.* 
* Notice: scanf로 float을 입력할 때는 %f, double을 입력할 때는 %lf, long double을 입력할 때는 %Lf를, long int를 입력할 때는 %ld, long long int를 입력할 때는 %I64d를 사용한다.이 프로그램을 실행할 때는 C99표준으로 설정하여 컴파일하여야 한다. C99가 정상적으로 적용되지 않을 수도 있다.
