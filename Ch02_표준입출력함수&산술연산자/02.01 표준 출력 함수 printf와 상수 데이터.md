## 02-01 표준 출력 함수 `printf`와 상수 데이터

**`printf`** 는 프로그래머가 지정하는 형식에 맞추어 화면에 데이터를 출력하는 함수이다. 데이터 형에 따라 출력형식을 지정해 주어야 한다. 출력 형식을 지정할 때는 `%`와 함께 출력할 데이터 형에 맞게 다양한 형식을 지정할 수 있다.

상수 데이터는 기호로 정의할 수도 있고 단순히 숫자나 문자(열)로 나타낼 수도 있다. 기호로 정의하는 상수는 여러 번 사용할 상수를 지정하고, 단순 숫자나 문자(열)는 한번 사용하는 경우에 주로 사용한다.

### 02-01-01 `printf` 함수

**`printf`** 함수는 원하는 결과를 기본 출력장치인 모니터로 보여주는 표준함수이다. 표준 함수의 사용방법은 단순히 함수의 이름과 함수에 전달될 데이터가 어떻게 구성되는지만 알면 상세한 처리과정은 알 필요가 없다. 
먼저 **`printf`** 함수를 사용하여 상수를 출력하는 간단한 프로그램을 하나 보자.

- 프로그램 예

```C
#include <stdio.h>
int main()
{
	printf("printf 함수 예\n");	// %가 없으면 단순 내용 출력
	printf("%d\n", 455);		// 정수(int) 상수 455 출력
	printf("%f\n", 3.14159);	// 실수(double) 상수 3.14159출력
	printf("%s\n", "C program language");	// 문자열 상수 출력
}
```

**`printf`** 함수를 사용하려면 먼저 **`printf`** 함수의 원형(prototype)이 정의되어 있는 헤더파일 `#include <stdio.h>`을 포함하여야 하며, 프로그램을 시작하기 전에 지정한다. 표준함수 또는 컴파일러가 제공하는 헤더파일은 `<xxx.h>`와 같이 삼각괄호(< ... >)를 사용한다.

데이터를 화면에 출력할 때는 표준출력함수인 **`printf`** 함수를 주로 이용한다. **`printf`** 함수의 괄호() 속에 출력하고자 하는 형식과 데이터를 나열한다. 괄호 속의 데이터를 매개변수(argument)라고 하며, **`printf`** 함수의 매개변수는 두 영역으로 구분한다. 위의 예에서 보는 바와 같이 앞부분(“”속)에는 출력 형식(format)을 지정하고, 뒷부분에는 출력할 데이터(변수, 상수, 식 등)를 나열한다. 출력형식은 큰따옴표(“”) 속에 문자열로 나타내고, 출력할 데이터는 뒤에 순서대로 콤마로 구분하여 나열한다. 첫 번째 printf에서처럼 단순한 문자열을 쓰면 그대로 화면에 출력되며, 데이터를 지정하지 않아도 된다. 나머지 5, 6, 7번 줄은 “”속에 형식(`%d`, `%f`, `%s`)을 뒷부분에 출력할 데이터를 지정한 것이다. `‘\n’`은 제어 문자로 줄 바꿈(new line)을 의미한다.

**`printf`** 함수의 원형은 다음과 같다. 

```C
printf 함수의 원형: int printf(const char* format, ... );
```

| int | printf | ( | const char * | format | , | ... | ) | ; |
| :--- | :--- | :---: | ---: | :---: | ---: | :---: | ---: | :---: |
| ① | ② | ③ | ④ | ⑤ | ⑥ | ⑦ | ⑧ | ⑨ |

| ① | int | 함수를 실행한 뒤 반환하는 값의 데이터 형 |
| :---: | :---: | :--- |
| ② | printf | 함수명
| ③ | ( | 함수에 전달할 매개변수의 시작 |
| ④ | const char * | 첫 번째 매개변수의 데이터 형 |
| ⑤ | format | 첫 번째 매개변수의 이름, 이 이름은 큰 의미는 없다. |
| ⑥ | , | 매개변수 구분 문자 |
| ⑦ | ... | 여러 개의 매개변수가 가능하다는 것을 표시(가변 매개변수) |
| ⑧ | ) | 매개변수의 끝|
| ⑨ | ; | 문장의 끝 표시|

※ **`printf`** 함수의 실행 결과 반환 값은 출력된 문자의 개수, 출력을 실패하면 음수 값


다음 <표 2-1>은 형식지정문자 중 일부를 나타낸 것이며, 2장 후반부에 나머지 더 상세한 내용을 살펴본다. 형식지정문자는 `%` 와 함께 사용하며, 변수 또는 상수를 지정된 형식으로 출력한다. 형식문자와 데이터가 일치하지 않을 때는 원하는 결과를 출력하지 못한다. 단, 형식지정문자가 아닌 문자는 그대로 출력된다.

### 02-01-02 상수
프로그램에서 데이터를 표시하는 방법은 상수(常數:constants)와 변수(變數, variables) 두 가지가 있다. 상수는 고정된 데이터를 나타낼 때 사용한다. 상수를 표현하는 방법은 직접 값을 나타내는 방법과, 상수 기호를 이용하여 나타내는 방법이 있다.

<표 2-2> 상수
| 방법 | 예 |
|:---|:---|
| 직접 값을 지정 | 정수상수 : 0, 128, 65535, -10, 01736, 0x248ff, 0XAB09 <br> 실수상수 : 3.141592, 0.0f, -273.0, 2.39E5 <br> 문자상수 : ‘a’, ‘\n’, ‘\007’ <br> 문자열상수 : “This is string.”, “독도는 우리 땅” |
| 상수 기호를 이용하여 값 지정 | const int x=3; <br> const double pi=3.141592; |

정수상수는 10진수, 8진수, 16진수의 세 가지로 나타낼 수 있다. 컴퓨터에서 모든 데이터는 내부적으로 2진수로 나타내지만, 화면에 보이기 위해서는 10, 8, 16진수로 바꾸어 나타내며, 2진수로는 보여주지 않는다. 2진수로 보기 위해서는 다른 방법을 사용하여야 한다. 10, 8, 16진수 상수를 구별하기 위해서 시작하는 숫자를 다르게 표시한다. 

* 8진수
 
  * `0123`처럼 `0`으로 시작

  * `0~7`사이의 수로 표시
­
  * `8`이나 `9`가 포함되면 안 됨 : `0139`과 같이 `8`진수가 아닌 `9`가 포함되어 있으므로 에러 발생

* 16진수

  * `0x` 또는 `0X`로 시작

  * `0~9` 및 `a(A)~f(F)`의 16개의 기호 사용

* 10진수

  * `0~9`까지의 수로 표시
­
  * `0`을 제외한 수로 표시 : `0`으로 시작하면 8진수로 취급하기 때문

실수상수는 항상 10진수로만 표시되며 반드시 소수점을 포함하여야 한다. 지수(exponent)를 나타내고자 할 때는 e나 E를 뒤에 추가한다. 실수값 0을 표시할 때는 `0.0`, `0.`, `.0`과 같이 소수점을 포함하여야 하고 0이 최소한 한 개는 있어야 한다. 실수 상수는 일반적으로 `float`(32비트)형과 `double`(64비트)형이 있으며, `double`형이 기본형이다. `float`형은 `0.0f`와 같이 끝에 `f`를 추가하여 표시하며, 끝에 아무 표시 없이 `0.0`으로 실수를 표시하면 기본형인 `double`형으로 인식한다.

문자 상수는 작은따옴표를 문자 앞뒤에 쓴다. 특수한 문자인 경우에는 <표 2-3>과 같이 역슬래시(`\`)를 따옴표 안에 표시하고 특수문자를 쓴다.

기호로 상수를 표시할 때는 사용하기 전에 기호상수를 선언하고 값을 저장한 다음 사용하여야 한다. 이 기호상수는 프로그램 실행 중에 값을 변경할 수 없으며, 값의 변경을 시도하면 에러가 발생한다. 따라서 기호상수는 값이 변하지 않는 데이터를 정의할 때 사용한다. 예를 들면 원주율(π=3.141592)과 같이 절대적으로 변하지 않는 값을 지정할 때 사용한다. 상수들은 과학이나 수학에 자주 등장하는데 이들을 프로그램에서 사용할 때 기호상수를 쓰면 유용하다.

<표 2-3> 특수문자
| 특수문자 | 의미 |
|:---|:---|
| \a | 경고음, bell, ^G |
| \b | 백 스페이스(Backspace), ^H |
| \f | 종이 넘김, 새 페이지, ^L |
| \n | 줄 넘김, 캐리지 리턴과 줄 바꿈. ^M^J |
| \o | 8진 상수, \oddd or \ddd |
| \r | 캐리지 리턴, 줄 바꿈 하지 않음, ^M |
| \t | 수평 탭, tab, ^I |
| \v | 수직 탭, ^K |
| \x | 16진 상수, \xdd 0 <= d <= F |
| \0 | 널, 아무것도 없는 문자, ^@ |
| \" | 큰따옴표 |
| \' | 작은따옴표 |
| \\ | 역 슬래시 |
| \? | 물음표 |
| \ddd | 8진수로 표현한 문자 값, 0 <= d <= 7 |

모든 데이터 형은 상수로 지정할 수 있으며 필요한 경우에만 사용하고, 특별한 값이 아니라면 굳이 상수로 선언할 필요는 없다.

● 예제 02-01-1 : 문자열 상수의 표현과 표준출력함수를 이용한 문자열 상수 출력

```C
#include <stdio.h>	// 표준입출력 함수원형을 포함하고 있는 헤더파일
int main()
{
	// 문자열 상수 출력 
	printf("C programming Language!");

	// 형식지정자 %s를 이용한 문자열 상수 출력
	printf("%s\n", "C programming Language!");
}
```

● 해설
* 1: 표준입출력 함수 원형이 지정되어 있는 헤더파일이다. 이 문장을 쓰면 표준입출력함수를 사용할 수 있다.	이 문장은 C언어에서는 무조건 쓴다고 생각하면 된다.
* 5: 형식지정문자 없이 단순히 문자들만 출력한다.
* 8: 형식지정 문자 %s를 사용하여 데이터(“” 속)를 출력한다. 형식지정은 %로 시작하고 %뒤에 출력형식을 지정한다. \n은 줄 바꿈(new line)을 실행한다.
* 1,4,7: 설명(comment)문으로써 프로그램의 설명을 달아준다. //으로 시작되는 줄은 한 줄 설명이고 여러 줄에 걸쳐 설명을 달고자 할 때는 <예제 02-01-2>와 같이 /* ... */ 사이에 설명을 쓴다.간단한 매뉴얼을 만든다고 생각하자. 설명문은 프로그램 실행에 영향을 주지 않는다.
* Notice: 들여쓰기(indentation)를 함으로써 프로그램의 가독성(readability)를 향상시키고 프로그램의 구성을 쉽게 파악할 수 있게 한다.
  
● 예제 02-01-2 : 정수의 표현과 표준출력함수를 이용한 정수 상수 출력
```C
#include <stdio.h>		/* 표준입출력 함수원형을 포함하고 있는 헤더파일 */
int main()
{
 	printf("%d ", 75);	//10진수 정수 상수 
 	printf("%d ", 0113);	//8진수 정수 상수(0으로 시작)를 10진수로 출력
 	printf("%d ", 0x4b);	//16진수 정수 상수(소문자 0x로 시작)를 10진수로 출력
 	printf("%d ", 0X4B);	//16진수 정수 상수(대문자 0X로 시작)를 10진수로 출력
 	printf("%u\n", -1);
 	/* unsigned %u형식(부호 없는 즉, 0과 양수)으로 정수 상수 출력 
 	   %x(%X), %o로 unsigned 정수를 한번 출력해 보자. */
}
```

● 해설
* 4: 75라는 10진수 상수를 %d, 즉 10진수로 모니터에 출력한다. 정수 상수는 소수점이 없다.
* 5: 0113이라는 8진수 상수를 %d, 즉 10진수로 모니터에 출력한다. 숫자가 0으로 시작되면 8진수라는 것을 의미한다. 8진수는 0~7까지의 수만 사용하여야 한다.
* 6: 0x4b라는 16진수 상수를 10진수로 출력한다. 0x로 시작하면 소문자 16진수를 의미한다. 16진수는 0~9, A-F까지 16개의 기호로 나타낸다.
* 7: 0x4B라는 16진수 상수를 10진수로 출력한다. 0X로 시작하면 대문자 16진수를 의미하며, x와 X의 차이는 없음.
* 8: %u는 음수 값이지만 양수로 출력한다. \n(￦n)은 new line이란 의미로 줄을 바꾸라는 의미이다.
* 9,10: 여러 줄에 걸친 설명문, /*로 시작해서 */로 끝낸다. 설명문을 프로그램에 영향을 주지 않는다.
* Try: %d를 %x, %X, %o 등으로 바꾸어 출력해 보자.

● 예제 02-01-3 : 실수의 표현과 표준출력함수를 이용한 실수 상수 출력
```C
#include <stdio.h>
int main()
{
 	const double pi=3.141592;	// 기호상수 선언
 	printf("%f ", 123.456);	// 실수 상수 출력
 	printf("%f ", 12.E4);		// 실수 상수 출력
 	printf("%f ", .0);		// 소수점 앞의 0은 생략 가능 
 	printf("%f ", 0.);		// 소수점 뒤의 0은 생략 가능
 	printf("%f ", .2345); 	// 소수점 앞의 0은 생략 가능
 	printf("%f\n", 0.23E-4);	// 실수 상수 출력
 	printf("%f\n", pi);		// 기호상수에 저장된 값 출력
}
```

● 해설
* 4: 상수는 단순히 값을 쓸 수도 있지만 상수기호에 저장할 수도 있다. 상수기호는 프로그램 도중에 값이 바뀔 수 없다. = 기호는 대입연산자로써 우측의 값, 변수 또는 식을 좌측의 변수에 저장하라는 의미이다.
* 5: 실수 값(상수 또는 변수)을 출력할 때는 형식지정문자로 %f를 사용한다. 실수 상수를 나타내는 방법은 다양하다. 실수 상수를 타나낼 때는 소수점을 포함하여야 한다.
* 6: 실수 상수를 10의 지수형태로 나타낼 때는 e 또는 E를 쓰고 뒤에 지수를 적는다. 지수는 정수이다.
* 7: 0.0일 때 앞의 0을 생략할 수 있다.
* 8: 0.0일 때 뒤의 0을 생략할 수 있다. 그러나 0을 모두 생략하여 소수점만으로는 나타낼 수 없다.
* 9: 0.2345에서 앞의 0을 생략할 수 있다.
* 10: E-4는 10의 음의 지수를 나타낸 것으로서, 을 의미한다.
* 11: 선언된 상수 pi를 출력한다.
* Notice: 상수와 변수는 다르며, 상수는 고정된 값, 변수는 변하는 값을 뜻한다. 변수는 02-02절 참조

● 예제 02-01-4 : 문자의 표현과 printf를 이용한 문자 상수 출력
```C
#include <stdio.h>
int main()
{
 	const char x='a';
 	printf("%c ", 'A');	//문자 상수 'A'
 	printf("%c ", x);	//문자 상수 'a' 출력
 	printf("%c ", 74);	//문자 상수 'J' 출력
 	// ASCII 코드 'J' = 74 = 0X4A = 0112
}
```

● 해설
* 4: 문자 상수 x 선언하고 초깃값 ‘a’저장, 상수이므로 변경이 불가능하다.
* 5: 문자 상수 ‘A’ 출력
* 6: 문자 상수 x에 저장된 ‘a’ 출력
* 7: ASCII값 74는 문자 ‘J’이다.
* 8: ASCII코드를 다양한 형태로 표현할 수 있다. 여기에 나타낸 값들은 모두 같은 의미이다.
* Notice: 문자(character)와 문자열(string)은 다르다. 문자는 하나의 ASCII문자를 ‘’ 속에 나타내고 문자열은 “”속에 나타낸다. 문자열에 대해서는 5장에서 자세히 다룰 것이다.

● 유제 02-01-1 : 다음 출력 문장을 이해하고 잘못된 문장은 정상적인 문장으로 고쳐라.
```C
#include <stdio.h>
int main()
{
 	printf("%c ", A);			//무엇이 문제인가?
 	printf("%s ", This is string);	//무엇이 문제인가?
 	printf("%d ", 84.4);			//실행결과는?
 	printf("%x ", 0123);			//무슨 의미인가?
}
```

● 유제 02-01-2 : 다음 프로그램을 실행해 보고 그 결과를 분석하라.
```C
/* 문자 상수 출력 예 */ 
#include <stdio.h>

int main()
{
 	const char w='A', x=65, y='\x41', z='\101'; // 기호상수(const) 선언
 	printf("%c\n", w); /* printf는 표준출력함수이다. */ 
 	/* %c는 문자상수 또는 문자변수의 출력형식을 의미한다.
 	    w를 문자형식으로 출력하라는 의미이다.
 	    \n은 줄을 바꾸라는 의미로 w의 내용을 문자로 출력하고 줄을 바꾼다. */
 	printf("%c\n", x);
 	printf("%c", '\n'); /* printf("\n");와 같다. ‘\n’이라는 문자를 출력 */
 	printf("%c\n", y);
 	printf("%c\n", z);
 	printf("%c\n", 'A'); /* 변수로 지정하지 않고 상수 ‘A’와 같이 사용할 수 있다. */ 
 	printf("A\n"); /* 직접 문자 ‘A’를 출력한다. */ 
}
```
