### 02.09 문자와 문자열 입출력 함수
지금까지 `scanf()` 함수와 `printf()` 함수를 이용하여 데이터를 입출력하는 방법을 살펴보았다. 이 절에서는 문자 및 문자열 입출력 함수를 살펴본다.

`getc()`, `getchar()`, `putc()`, `putchar()`은 표준함수이고, `getch()`, `getche()`, `putch()`, `ungetch()`, `kbhit()`는 비표준 함수이다. 비표준함수도 콘솔에서 작업할 때 가끔 사용되며, 대부분의 컴파일러에서 제공하고 있다. `getc(stdin)`와 `getchar()`은 같으며, 둘 다 엔터키를 눌러야 입력되고, `getch()`와 `getche()`는 엔터키를 누르지 않아도 된다.

`getch()`와 `getche()`의 차이는 표에 표시하였다시피 화면에 `echo`가 있느냐 없느냐의 차이이다. `getche()`의 끝의 e가 `echo`를 의미한다. `putc(c, stdout)`, `putchar(c)`, `putch(c)`는 모두 같은 기능을 한다.

`ungetch()` 함수는 받아들인 문자를 다시 원래 위치로 돌려준다.

`kbhit()` 함수는 키보드의 아무키나 눌려졌는지를 검사하는 함수이다. 눌려졌을 경우 어떤 일을 처리하도록 만들 수 있다.

<표 2-10> 문자열 입출력 함수
| 함수명 | 함수원형 | 헤더파일 | 예 |
|:---|:---|:---|:---|
| gets | char * gets ( char * str ); | stdio.h | gets (string); |
| puts | int puts ( const char * str ); | stdio.h| puts (string); |

`gets()`는 문자열을 키보드로 입력받는다.

`scanf("%s", string)`와 달리 완전한 문자열 즉, 엔터키를 누를 때까지 입력된 모든 문자열을 문자열 변수로 입력받는다. 즉, `gets(string)`는 `scanf("%[^\n]", string)`와 같다고 볼 수 있는데, 잠시 뒤에 살펴보자.

`puts()`는 문자열을 모니터로 출력한다.

---
#### ● 예제 02-09-1 : `getchar()`과 `putchar()`을 이용한 문자 입출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char character;
5 	printf("문자를 입력하시오 : ");
6 	character = getchar();
7 	printf("입력된 문자 = ");
8 	putchar(character);
9 	printf("\n");
10 }
```

#### ● 해설
* 6: 문자를 `getchar()` 함수를 이용하여 키보드에서 입력받아 변수 `character`에 저장한다.
* 8: 입력된 문자를 모니터에 출력한다.

> [!NOTE]
> 문자를 입력하고 엔터를 눌러야 한다.


---
#### ● 예제 02-09-2 : `getc()`와 `putc()`를 이용한 문자 입출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char character;
5 	printf("문자를 입력하시오 : ");
6 	character = getc(stdin);
7 	printf("입력된 문자 = ");
8 	putc(character, stdout);
9 	putc('\n', stdout);	// printf(“\n“);과 같다.
10 }
```

#### ● 해설
* 6: `getc(stdin)`함수를 이용하여 키보드에서 문자를 입력받아 변수 `character`에 저장한다. `stdin`은 표준입력장치(키보드)이다.
* 8: 입력된 문자를 `putc()`함수를 이용하여 모니터에 출력한다. `stdout`은 표준출력장치(모니터)이다.
> [!NOTE]
> `getc()`와 `putc()`함수는 파일에서도 사용할 수 있다. `stdin` 및 `stdout` 대신에 파일 포인터를 쓴다. 문자를 입력하고 엔터를 눌러야 한다.

#### ● 예제 02-09-3 : `getch()`와 `putch()`를 이용한 문자 입출력
```
1 #include <stdio.h>
2 #include <conio.h>
3 int main()
4 {
5  	char character;
6  	printf("문자를 입력하시오 : ");
7  	character = getch();
8  	putch('\n');	// printf("\n");과 같다.
9 	printf("입력된 문자 = ");
10  	putch(character);
11  	putch('\n');	// printf("\n");과 같다.
12 }
```

#### ● 해설
* 7: `getch()`함수를 이용하여 키보드에서 문자를 입력받아 `character` 변수에 저장한다.여기서 화면에 입력한 문자는 보이지 않는다. 즉, echo가 없다.
* 8: `putch('\n')`는 줄바꿈 문자 하나를 화면에 출력한다.
* 10: 입력된 문자를 화면에 출력한다.
* 12: 키보드에서 한 문자를 입력할 때까지 기다리다가 아무키나 누르면 프로그램이 끝난다.
> [!NOTE]
> `getc()`와 `getche()`함수는 동일하게 문자를 입력받지만 `getche`는 echo가 있다.

#### ● 예제 02-09-4 : `gets`와 `puts`를 이용한 문자열 입출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char string[100];
5 	printf("문자열을 입력하시오 : ");
6 	gets(string);
7 	printf("입력된 문자열 = ");
8 	puts(string);
9 	fflush(stdout);
10 }
```

#### ● 해설
* 6: gets 함수는 '\n'직전까지 키보드에서 입력받아 문자열 변수에 저장한다.
* 8: 입력된 문자열을 화면에 출력한다.
* 9: 표준출력장치를 원 위치로 되돌린다. 즉 버퍼를 비운다.
> [!NOTE]
> `gets`함수는 보안상 위험하다고 한다. 문자열을 몇 개를 받아들일 것인가가 정해져 있지 않고 엔터키를 받아들이지 않기 때문인 것으로 보인다. 대신에 `fgets(string, 문자개수, stdin)`으로 쓰는 것을 권장한다.
