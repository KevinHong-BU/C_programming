### 02-03 정수의 산술 연산
C언어의 정수 연산에 대해서 살펴보자. C언어의 산술 연산자는 `+`, `-`, `*`, `/`의 연산자가 있으며, 이들 4칙 연산은 정수와 실수 모두에 사용할 수 있다. 4칙 연산은 수학에서와 같다. 단, 한 가지 주의할 점은 정수 데이터끼리 연산을 하면 결과도 정수가 된다는 것이다. 특히 나눗셈의 경우 계산결과는 소수점 이하가 절삭되어 정수가 된다는 것에 주의하여야 한다. 정수 연산의 결과는 실수가 아니므로 원하는 결과를 얻으려면 실수연산을 하여야 한다. 물론 정수연산만 필요한 경우에는 편리할 수도 있다. 한 가지 프로그래밍 언어를 처음 배우는 학생들은 `*`를 생략하는 경우가 간혹 발생하는데 수학과 달리 곱셈기호를 생략할 수는 없다.

C언어에서는 정수에만 사용할 수 있는 산술연산자는 `%`, `++`, `--`의 3가지 연산자가 있다. `%`은 나머지 연산자로서 정수를 정수로 나누었을 때 나머지를 구해준다. `%` 연산은 특히 유용하게 사용되므로 잘 알아두면 편리하게 사용할 곳이 아주 많다. 예를 들어 짝수와 홀수를 구분할 때 `x%2`의 결과를 이용하면 된다. 즉, `x%2`의 결과가 0이면 짝수, 1이면 음수이다.

`++`는 증가연산자로서 정수 값을 1 증가 시키는데 사용하며, `--`는 감소연산자로서 정수 값을 1 감소시키는데 사용한다. 특히 4장에서 배울 반복문에 유용하게 사용할 수 있다. 증가⋅감소연산자는 피연산자(operand)의 앞이나 뒤에 올 수 있으며, 앞에 오면 정수 값을 먼저 증가⋅감소한 다음 다른 연산을 하고, 뒤에 올 경우에는 다른 연산을 다 처리한 다음 증가⋅감소한다. 그렇지만 독립적으로 사용할 경우에는 그 결과는 동일하다.

`+`, `-`는 부호로 사용할 수 있으며 이때는 부호를 먼저 적용하고 다른 연산을 처리한다.

연산자에는 2항(binary) 연산자와 단항(unary) 연산자가 있다. 2항 연산자는 피연산자(operand)가 2개 있는 연산자를 의미한다. 4칙 연산자와 나머지(`%`) 연산자는 2항 연산자이며, 부호와 증가⋅감소연산자는 단항 연산자이다. 그 이외에도 단항연산자가 몇 개 더 있는데 뒤에서 설명한다.

연산자의 우선순위는 수학에서와 같으며, <표 2-6>에 나열하였다. 우선순위를 모두 알고 적용하기는 좀 까다로운 점이 있다. 따라서 수식을 쓸 때는 프로그래머가 적절하게 괄호를 사용하여 우선순위를 정해주는 것이 프로그램 작성에도 좋으며, 보기에도 좋다. 그리고 가급적이면 복잡한 식을 사용하지 않는 것도 프로그램의 버그를 줄이는 좋은 방법이다.

표에서 결합 순서 LR 또는 RL은 결합의 방향을 나타낸다. 결합 순서란 같은 우선순위의 연산자들이 나열될 때 어느 쪽에서부터 결합을 할 것인가를 나타내는 것이다.

LR은 왼쪽에서 오른쪽으로 결합한다. 예를 들면 `a - b + c + d;`란 식에서 우선순위가 같으므로 왼쪽부터 하나씩 결합하여 연산한다는 의미이다. RL은 오른쪽에서 왼쪽으로 결합한다. 예를 들면 `a = b = c = 5;`에서는 모두 대입연산자로써 같은 우선순위를 가지므로 오른쪽에서 왼쪽으로 결합이 이루어진다. 즉, 먼저 5를 `c`에 대입하고 다시 `c`를 `b`에, `b`를 `a`에 대입하는 순서대로 결합이 이루어진다.

#### ● 예제 02-03-1 : 정수의 4칙 연산
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=19, y=5;		// 변수 선언 및 초깃값 설정
5 	printf("%d + %d = %d\n", x, y, x+y);
6 	printf("%d - %d = %d\n", x, y, x-y);
7 	printf("%d * %d = %d\n", x, y, x*y);
8 	printf("%d / %d = %d\n", x, y, x/y);
9 }
```

#### ● 해설
* 4: 변수 x와 y를 선언하고 초깃값을 저장한다.
* 5: 변수 x, y 그리고 x+y의 결과를 출력한다. %d가 3개 이고, 뒤의 식(x, y, x+y)이 3개이다.
* 6,7,8: 뺄셈 연산, 곱셈 연산, 나눗셈 연산을 한다.
* 8: 나눗셈 결과를 확인하기 바란다. 수학적으로 보면 결과가 정확하지 않지만, 컴퓨터에서는 정수끼리 연산하면 결과는 정수이다. 따라서 19/5는 3.8이 아니라 3이다. 즉, 정수에서 소수이하는 취급하지 않는다. 그러므로 몫을 구하고자 할 때 나누기 연산자를 쓴다.

> [!NOTE]
> 정수끼리의 연산 결과는 항상 정수이다. 정수와 실수를 연산하면 실수이다. 뒤에서 확인하자.

#### ● 예제 02-03-2 : 증가⋅감소 연산자
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=10;
5 	printf("x = %d\n", ++x);	// 먼저 증가
6 	x=10;					// 다시 x에 10 대입
7 	printf("x = %d\n", x++);	// 나중 증가
8 	x=10;					// 다시 x에 10 대입
9 	printf("x = %d\n", --x);	// 먼저 감소
10 	x=10;					// 다시 x에 10 대입
11 	printf("x = %d\n", x--);	// 나중 감소
12 }
```

#### ● 해설
* 5: x를 먼저 1증가 한 후에 출력한다.
* 7: x를 먼저 출력한 후에 1증가한다.
* 9: x를 먼저 1감소 한 후에 출력한다.
* 11: x를 먼저 출력한 후에 1감소한다.
* 6,7,10: x의 값이 변경되었으므로 다시 10을 대입한 후 다음 연산을 한다.

> [!NOTE]
> 연산 식에서도 같은 방법으로 적용된다. 증가⋅감소연산자의 위치에 따라 계산 결과가 달라질 수 있다.

#### ● 예제 02-03-3 : 나머지 연산자
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=10, y=3;
5 	printf("x를 y로 나눈 몫 = %d\n", x/y);
6 	printf("x를 y로 나눈 나머지 = %d\n", x%y);
7 }
```

#### ● 해설
* 4: x에 10, y에 3을 저장한다.
* 5: x/y의 연산 결과는 x를 y로 나눈 몫이 된다. 앞에서도 말했지만 정수끼리의 연산 결과는 정수이다.
* 6: x%y의 연산 결과는 x를 y로 나눈 나머지가 된다.

> [!NOTE]
> 나머지 연산자는 정수형의 데이터에서만 가능하며, 정수 나눗셈 결과는 몫이다.

#### ● 예제 02-03-4 : 연산자의 결합 순서와 대입, 증가⋅감소연산자의 예
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=10, y=3, z;
5 	x = x + y - (z = y);
6 	printf("x = %d\n", x);
7 	printf("z = %d\n", z);
8 	z = x++ + --y;
9 	printf("x++ + --y = %d\n", z);
10 }
```

#### ● 해설
* 5: 연산자의 우선순위가 같을 경우에 결합의 순서는 좌에서 우로 이루어진다. 여기서는 x와 y를 더하고 뺄셈을 한다. 뺄셈을 하기 전에 y를 z에 먼저 대입하고 뺄셈을 한다. 따라서 x+y는 13이고 여기에서 z의 값을 뺀다. 따라서 결과는 10이 되며 계산이 모두 이루어진 후에 x에 10을 저장한다.
* 6: x를 출력하면 10이다.
* 7: z를 출력하면 3이다.
* 8: x와 y를 1 감소하여 더한 후 z에 대입한다. 그 다음에 x가 1 증가한다. 따라서 이 식이 계산된 후의각 변수의 값은	x는 11, y는 2, z는 12가 된다.

> [!NOTE]
> 결합의 우선순위는 왼쪽에서 오른쪽으로, 지금까지 배운 연산자의 우선순위는 *, /, %가 높고, +,-가 그 다음이다. 물론 괄호가 있으면 괄호 안부터 먼저 계산한다. 또 피연산자와 결합된 부호는 가장 먼저 평가가 이루어진다. 수학에서의 우선순서와 같다.
