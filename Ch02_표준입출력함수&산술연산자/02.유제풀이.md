● 유제 02-01-1 : 다음 출력 문장을 이해하고 잘못된 문장은 정상적인 문장으로 고쳐라.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	printf("%c ", A);			//무엇이 문제인가?
5 	printf("%s ", This is string);	//무엇이 문제인가?
6 	printf("%d ", 84.4);			//실행결과는?
7 	printf("%x ", 0123);			//무슨 의미인가?
8 }
```

● 해설
* 4: A는 변수 또는 상수로 취급된다. 따라서 컴파일 할 때 에러가 난다. 문자를 출력하고자 한다면 ‘A’로 하여야 한다.
* 5: “This is string”로 바꾸어야 한다. 문자열은 “”로 앞뒤에 표시한다.
* 6: 84.4는 실수인데 출력지정형식은 %d로 정수로 출력하도록 하고 있다. 결과는 이상한 값이 나온다.
* 7: 0123은 정수(8진)이다. 이를 16진 정수 %x로 출력한다. 따라서 결과는 16진수 53이 된다.
> [!NOTE]
> 8진수 0123은 2진수로는 001010011, 10진수로는 83, 16진수로는 53이다. 이들 상수 값을 구별하기 위해서 8진수는 0으로 시작하고 16진수는 0x 또는 0X로 시작하고 10진수는 0이 아닌 수로 시작한다. 실제로 컴퓨터 내부에서는 2진수로 표시되지만 C언어에서는 2진수로 표시하는 수단을 제공하지 않는다. 8진수 1자리는 2진수 3자리, 16진수 1자리는 2진수 4자리로 표시한다.

● 유제 02-01-2 : 다음 프로그램의 실행결과를 분석하라.
```C
1 /* 문자 상수 출력 예 */ 
2 #include <stdio.h>
3 int main()
4 {
5 	const char w='A', x=65, y='\x41', z='\101'; // 상수(const) 선언

7 	/* %c는 문자상수 또는 문자변수의 출력형식을 의미한다.  
8 	    w를 문자형식으로 출력하라는 의미이다. 
9 	    \n은 줄을 바꾸라는 의미로 w의 내용을 문자로 출력하고 줄을 바꾼다. */ 
10 	printf("%c\n", x); 
11 	printf("%c", '\n'); /* printf("\n");와 같다. ‘\n’이라는 문자를 출력 */ 
12 	printf("%c\n", y);
13 	printf("%c\n", z);
14 	printf("%c\n", 'A'); /* 기호로 지정하지 않고 바로 'A'와 같이 사용할 수 있다. */
```

● 유제 02-03-2 : 5928초는 몇 시,  몇 분,  몇 초인가?
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x=39209, hour, minute, second;
5 	printf("%d초 = ", x); 
6 	hour = x / 3600;		// 1시간은 3600초이므로 x를 3600으로 나눈 몫
7 	x = x % 3600;		// 시간을 계산하였으므로 나머지 계산
8 	minute = x / 60;		// 1분은 60초이므로 x를 60으로 나눈 몫
9 	second = x % 60;		// 남은 값에서 60으로 나눈 나머지가 초이다.
10 	printf("%d시 %d분 %d초\n",  hour, minute, second);
11 }
```

● 해설
* 7~10: 나눗셈연산과 나머지연산을 적절히 사용하여 간단히 시, 분, 초를 계산할 수 있다.
> [!NOTE]
> 정수 나눗셈은 몫이라는 것과 정수 나머지 연산을 이용하면 계산할 수 있다. 이 방법은 아주 많이 사용되므로 꼭 기억해 두기 바란다.
