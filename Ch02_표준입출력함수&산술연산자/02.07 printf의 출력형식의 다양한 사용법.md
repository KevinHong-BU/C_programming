### 02-07 printf의 출력형식의 다양한 사용법
이 절에서는 형식지정문자열의 다양한 용도에 대해서 살펴보도록 한다.

출력형식지정자 : `%[flags][width][.precision][length]specifier`

출력할 공간이 충분하지 않을 때는 자동으로 확장된다.

● 예제 02-07-1 : 형식지정문자열을 다양한 방법으로 사용하기
1 #include <stdio.h>
2 int main()
3 {
4 	int i=100, *j;
5 	j = &i;	// i의 주소를 j에 저장 
6 	printf("    %%d--->>%d<<\n", 123);
7 	printf("  %%+5d--->>%+5d<<\n", 345);
8 	printf("  %%05d--->>%05d<<\n", 345);
9 	printf("  %%13u--->>%13u<<\n", -1);
10 	printf("   %%0d--->>%0d<<\n", 567);
11 	printf("  %%05x--->>%05x<<\n", 507);
12 	printf("   %%5x--->>%5x<<\n", 507);
13 	printf("  %%#5x--->>%#5x<<\n", 507);
14 	printf("   %%6o--->>%6o<<\n", 507);
15 	printf("  %%#6o--->>%#6o<<\n", 507);
16 	printf("   %%0x--->>%0x<<\n", 507);
17 	printf("  %%-5d--->>%-5d<<\n", 678);
18 	printf(" %%6.1f--->>%6.1f<<\n", 789.38);
19 	printf("%%10.1e--->>%10.1e<<\n", 789.38);
20 	printf(" %%7.1f--->>%7.1f<<\n", 789.45);
21 	printf("    %%p--->>%p<<\n", j);
22 	printf("  %%20s--->>%20s<<\n", "abcdefg789.45");
23 	printf("  %%25s--->>%25s<<\n", "Korean team fighting!!!");
24 	printf(" %%6.1x--->>%6.1x<<\n", 345);
25 }
● 해설
7: +부호 출력한다.
8: 빈자리에 0을 채운다.
9: -1을 부호가 없는 양수로 출력한다.
13: #은 8진수, 16진수를 10진수와 구별할 수 있게 0 또는 0x를 숫자 앞에 붙여준다.
17: -는 왼쪽 정렬하여 출력한다.
19: e는 지수형태로 출력한다.
21: 변수의 주소를 출력한다.
Notice: 출력할 때 >>와 <<를 사용한 것은 출력되는 자릿수를 정확하게 보여주기 위함이다.

● 예제 02-07-2 : 가변 길이로 출력하기
1 #include <stdio.h>
2 int main()
3 {
4 	int x=2939;
5 	printf(">>%*d<<\n", 5, x);
6 	printf(">>%*d<<\n", 6, x);
7 	printf(">>%*d<<\n", 7, x);
8 	printf(">>%*d<<\n", 8, x);
9 	printf(">>%*d<<\n", 9, x);
10 }
● 해설
6: 출력할 길이를 가변으로 할 수 있다. 이 문장에서는 5이므로 5개의 공간에 데이터 x를 출력한다.가변길이 출력을 할 때는 *를 사용하며 %와 형식지정문자 사이에 넣는다. 실수형일 때는 정수 부분과소수 부분에 대해서 모두 지정할 수 있다.
9: 9자리로 변수 x를 출력한다.
Notice: 출력할 때 >>와 <<를 사용한 것은 출력되는 자릿수를 정확하게 보여주기 위함이다.

● 예제 02-07-3 : 제어문자 출력하기
1 #include <stdio.h>
2 int main()
3 {
4 	printf("I love Korea!");
5 	printf("");		// 아무것도 출력하지 않는다.
6 	printf(" ");		// 공백을 하나 출력한다.
7 	printf("\"I love Korea!\"");	// 따옴표를 출력한다.
8 	printf("대~\t한민국\n");		// 탭을 출력한다.
9 	printf("\\I love C!!!\\");
10 	printf("%%d는 정수를 출력한다.\n"); 	// %자체를 출력한다.
11 	printf("This is a very, very, very, \
12 very, very, very long line.\n");
13 	printf("This is a very, very, very, "		// 이렇게 하면 두 문장을 하나로
14 		  "very, very, very long line.\n");	// 연결한다.
15 }
● 해설
7: “를 출력할 때는 \”라고 쓴다.
8: 탭을 출력할 때는 \t를 쓴다.
9: \를 출력하고자 할 때는 \\를 쓴다.
11,12: \ 뒤에는 아무것도 오면 안 된다. 이 문장과 다음 줄의 문장이 바로 연결된다. 따라서 다음 문장의 앞에 무엇이 있으면 그대로 출력된다. 따옴표 속의 문장이 길 때 사용할 수 있다.
13,14: 아주 긴 문장을 출력할 때 12번 줄과 같이 하여도 되지만 좀 불편하다. 따라서 이와 같이 “”으로 여러 문장으로 분리하여 순서대로 나열하면 된다, 콤마는 쓰지 않는다.
Notice: 14번 줄에서 콤마를 사용하면 뒤의 “”속의 문장은 출력할 데이터인 매개변수로 인식한다.

● 유제 02-07-1 : 다음 printf 함수의 실행 결과를 예측하라.
1 #include <stdio.h>
2 int main()
3 {
4 	int a=8357, cc = 0234512, xx=0x3cb49; // a는 10진 정수, cc는 8진 정수, xx는 16진 정수 
5 	unsigned b = 3495823734; 
6 	float y = 293.29384983e+32;
7 	double z = 23.4958697; // double이 float보다 더 큰 수를 표시할 수 있다.
8 	printf("%%d = %d \n", a); //%자체를 출력하고자 할때는 %%를 쓴다. 
9 	printf("%%o = %o \n", a); //%o : 정수 a의 값을 8진수(Octal)로 출력  
10 	printf("%%x = %x \n", a); //%x : 정수 a의 값을 16진수(Hexadecimal) 소문자로 출력 
11 	printf("%%X = %X \n", a); //%X : 정수 a의 값을 16진수(Hexadecimal) 대문자로 출력 
12 	printf("%%f = %f \n", y); //%f : 실수 y의 값 출력 
13 	printf("%%e = %e \n", y); //%e : 실수 y의 값을 지수(exponential)형식으로 출력 
14 	printf("%%u = %u \n", b); //%u : 정수 b의 값을 부호가 없는(양의 정수)로 출력 
15 	printf("%%d = %d \n", b); //%d : b는 unsigned로 선언되었지만 부호있는 정수로 출력,
16 					  // 여기서는 음수값 출력 
17 	printf("%%o = %o \n", cc); //%o : 정수 cc의 값을 8진수(Octal)로 출력 
18 	printf("%%x = %x, %%d = %d \n\n", xx, xx); //%x : xx를 10진수와 16진수로 출력 
19 	printf("%%d = %10d \n", a); //%10d : a를 10자리로 맞추어 출력 
20 	printf("%%f = %10.3f \n", y); //%10.3f : y를 전체 10자리로 출력하며, 소수이하 자리 수는 3자리로 
21 	printf("%%f = %10.3f \n", z); //%10.3f : z를 전체 10자리로 출력하며, 소수이하 자리 수는 3자리로 
22 	printf("%%e = %.3e \n", y); //y를 지수형식으로 출력하되 소수이하는 3자리로 
23 }

15     printf("부동소수점수\n");
