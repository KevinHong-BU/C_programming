### 06-01. 1차원 배열

1차원 배열의 예로 int Array[5] = {5, 3, 4, 9, 1};를 살펴보자.

그림에서 깃발에 쓰여 진 번호가 인덱스이고, 통속에는 해당되는 배열의 순서에 따라 데이터가 저장되어 있다. 이와 같이 C언어에서 배열을 선언하면 메모리에 순서대로 저장된다. C언어에서 인덱스는 항상 0부터 시작한다.

메모리에 저장된 배열의 값은 배열을 선언할 때 지정하거나 값을 넣어준 것이며, 이를 사용할 때는 배열명[인덱스]로 호출한다.
위 그림에서 메모리번지는 예를 든 것이며, 정수는 4 바이트를 자치하므로 메모리 번지는 4씩 증가한다. 물론 바이트단위 저장방식일 경우이다.
바이트단위 저장방식은 주소 하나에 한 바이트 데이터가 저장되며, 워드단위 저장방식은 주소 하나에 2바이트 또는 4바이트 단위로 데이터가 저장되며, 시스템에 따라 다를 수 있다.

#### ● 예제 06-01-1 : 배열을 하나 선언하고 초깃값을 저장한 뒤, 출력하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int Array[5] = {5, 3, 4, 9, 1}, i;
5 	for (i=0; i<5; i++)	// 배열은 0번부터 저장된다.
6 		printf("Array[%d] = %d\n", i, Array[i]);	// 배열에 저장된 값 출력
7 }
```

#### ● 해설
* 4: 초깃값 5개를 가진 배열 Array, 인덱스변수 i를 선언한다.
* 5,6: 배열에 저장된 값을 인덱스로 구분하여 출력한다.
> [!NOTE]
> 배열의 저장순서는 반드시 0번부터이며, 배열의 모든 요소는 메모리에 순서대로 저장된다.

* 10~11: 배열에 저장된 값을 출력한다.

#### ● 예제 06-01-3 : 실수 배열을 선언하고 그 배열에 사인값을 15도 간격으로 계산
```C
1 #include <stdio.h>
2 #include <math.h>
3 int main()
4 {
5 	int i;
6 	double sine[25];
7 	for (i=0; i<=24; i++)
8 		sine[i] = sin((i*15)*3.141592/180.);
9 
10 	for (i=0; i<=24; i++)
11 		printf("sine(%3d) = %5.2f\n", i*15, sine[i]);
12 }
```

#### ● 해설
* 6: 25개의 요소를 가지는 실수 배열 sine을 선언한다.
* 7~8: 배열의 요소에 sin값을 15도 각도로 증가하면서 계산하여 저장한다.
* 10~11: 배열에 저장된 값을 출력한다.
* Notice: 사인 함수의 매개변수는 라디안이다. 8번 줄은 각도를 라디안을 계산하는 방법이다.

#### ● 예제 06-01-4 : 정수 배열에 초깃값을 저장하고, 저장된 값에 따른 수 만큼 ‘*’ 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int i, j, x[] = {5, 5, 3, 4, 6, 8, 10, 1, 2, 3}; 
5 	int array_count = sizeof(x)/sizeof(x[0]); //개수 파악
6 
7 	for (i = 0; i < array_count; i++) { 
8 		printf("%2d : ", x[i]); 
9 		for (j=0; j < x[i]; j++) 
10 			printf("%c", '*'); 
11 		printf("\n"); 
12 	}
13 }
```

#### ● 해설
* 4: 배열 x를 선언하고 초깃값을 지정, 초깃값이 있는 배열의 경우 요소의 개수는 생략할 수 있다. 생략되면 자동으로 초깃값의 개수만큼 지정된다.
* 5: 크기를 알지 못하므로 식으로 계산할 수 있다. 전체 배열의 이름을 하나의 크기로 나눈다. 정수 한 개의 크기는 4바이트이다.
* 7~12: 전체 배열을 하나씩 순환한다.
* 9,10: 각 요소에 저장된 수만큼 *을 출력한다.

#### ● 예제 06-01-5 : 배열에 정수 데이터를 저장한 다음, 각 값의 석차 구하기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int i, j, rank; 
5 	int stud[10] = {45, 92, 88, 34, 88, 95, 82, 59, 70, 76}; 
6 
7 	for (i=0; i<10; i++) {
8 		rank=0;
9 		for (j=0; j<10; j++)
10 			if(stud[i] <= stud[j])	// 자기보다 큰 값일 때 1씩 증가
11 				rank ++;
12 		printf("%d : %d위\n", stud[i], rank);
13 	}
14 }
```

#### ● 해설
* 5: 배열을 선언하고 초깃값을 지정한다.
* 7~13: 배열의 요소를 자기 자신과 나머지 데이터를 검사한다.
* 9~11: 자기보다 큰 값을 가진 데이터가 몇 개인지 카운트한다. 배열의 처음부터 끝까지 세어본다.
* 12: 점수와 등수를 출력한다.

#### ● 예제 06-01-6 : 배열의 데이터를 크기순으로 정렬
```C
1 #include <stdio.h> 
2 int main()
3 {
4 	int i, j, size, temp; 
5 	int stud[] = {45, 92, 88, 34, 88, 95, 82, 59, 20, 45, 60, 76}; // 배열 초기화
6 
7 	//배열의 개수를 모를 때 개수를 알아내는 방법
8 	size = sizeof(stud)/sizeof(stud[0]);
9 	printf("Before sort : ");
10 	for(i=0; i<size; i++)
11 		printf("%d ", stud[i]);
12 	printf("\n");
13 	// 각 수를 반복 비교하여 크기순으로 정렬한다. 
14 	for (i=0; i<size-1; i++) {
15 		for (j=i+1; j<size; j++) { 
16 			if(stud[i] < stud[j]) {	// 배열을 비교하여 순서를 바꾼다.
17 				stud[i] ^= stud[j];	// 정수이므로 XOR연산으로 하였다.
18 				stud[j] ^= stud[i];	// 임시 변수를 써서 교환하여도 된다.
19 				stud[i] ^= stud[j];
20 			}
21 		}
22 	}
23 	printf("Sorted : "); // 정렬된 결과 출력
24 	for (i=0; i<size; i++)
25 		printf("%d ", stud[i]); 
26 }
```

#### ● 해설
* 5: 배열에 초기 데이터를 지정한다.
* 10~11: 정렬하기 전의 초기 데이터를 출력한다.
* 14~22: 데이터를 비교하여 큰 값을 앞으로 가져온다. i는 0부터 크기보다 1작은 곳까지, j는 i부터 배열의 개수까지 비교한다. 비교의 횟수를 조금 줄일 수 있다. 가장 단순한 정렬방법이다. 데이터가 많을 때는 좋지 않다.
* 16~20: 두 데이터를 비교하여 앞의 값이 작으면 뒤의 값과 교환한다.
* 24~25: 정렬된 데이터를 출력한다.

#### ● 예제 06-01-7 : 로또 번호 발생 프로그램
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 int main()
5 {
6 	int lotto[45], shuffle, x, y;
7 	srand(time(0));  // 난수 초기화
8 	for (x=0; x<45; x++)
9  		lotto[x] = x+1; // 1~45까지 배열에 저장
10 	// 배열에 저장된 로또 번호를 5000번 섞는다.
11 	for (shuffle = 0; shuffle <5000; shuffle++ ) {
12 		x = rand()%45; // 교환할 첫 번째 숫자 위치
13 		y = rand()%45; // 교환할 두 번째 숫자 위치
14 		lotto[x] = lotto[x] + lotto[y] - (lotto[y] = lotto[x]);
15 		// 선택된 두 개의 위치 교환 
16 	}
17 	// 처음 6개(맨 앞쪽으로 이동된 번호)의 번호만 정렬
18 	for (x=0; x<5; x++)
19 		for (y=x+1; y<6; y++) //비교하여 위치 교환
20 			if (lotto[x] > lotto[y])
21 				lotto[x] = lotto[x] + lotto[y] - (lotto[y] = lotto[x]);
22 
23 	// 처음 6개의 선택된 번호만 출력 
24 	for (x=0; x<6; x++)
25 		printf(" %d  ", lotto[x]);
26 	printf("\n");
27 }
```

#### ● 해설
* 6: 로또 번호를 저장할 배열과 다른 변수 선언.
* 8,9: 1부터 45까지 번호를 배열에 저장, 즉, 45개의 공을 만든다고 생각하면 된다.
* 11~16: 1~45까지 저장된 로또 배열을 5,000번 섞는다. 즉, 공 2개를 선택하여 서로 바꾸는 과정을 5,000번 반복한다.
* 18~21: 5000번 섞은 로또를 앞에서부터 6개를 정렬한 뒤 출력한다. 즉, 5000번 섞은 뒤 맨 앞의 6개를 선택된 데이터로 취급한다.
* 24~25: 선택된 6개의 데이터를 출력한다.

#### ● 예제 06-01-8 : 바코드의 검사 수 생성: 바코드는 체계는 여러 가지가 있지만 세계 상품 코드(Universal Product Code; UPC) 체계가 가장 널리 사용되고 있다. 그 중에서 13자리 상품 바코드에서 검사 수를 생성하는 방법을 프로그램으로 작성해 보자. 검사 수를 만드는 방법은 다음과 같다. 바코드의 맨 뒷자리가 검사 수이다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 //	int barcode[13] = {8,8,0,1,0,9,4,0,1,3,0,0,}; // 8801094013004
5 //	int barcode[13] = {8,8,0,9,0,1,0,6,6,5,0,0,}; // 8809010665009
6 	int barcode[13] = {5,3,9,1,5,1,9,8,9,0,6,0,}; // 5391519890608
7 //	int barcode[13] = {4,9,8,2,8,3,3,5,9,2,9,2,}; // 4982833592920
8 	int i, sum, tobe, x=0, y=0;
9 	
10 	printf("12자리 코드: ");
11 	for(i=0; i<12; i+=2){	// 0부터 2씩 증가하면서 루프 수행 
12 		printf("%d %d ", barcode[i], barcode[i+1]);
13 		x += barcode[i];			// 홀수 번째 수의 합 
14 		y += barcode[i+1];		// 짝수 번째 수의 합 
15 	}
16 	y *= 3;	// 짝수 번째 합의 3배 
17 	tobe = sum = x + y;		// 두 수의 합을 변수에 저장 
18 	printf("\n\n%d+%d = %d\n", x, y, sum);
19 	if (sum%10)		// 1의 자리 값이 0이 아니면
20 		tobe = (sum/10 + 1)*10; // 올림하여 10의 배수가 되도록 한다. 
21 	barcode[12] = tobe - sum;	   // 10의 배수가 되기 위한 검사 수 생성 
22 	printf("%d+%d=%d --> ", sum, barcode[12], tobe);
23 	printf("check number is %d.\n\n", barcode[12]);
24 	printf("12자리+검사 수: ");
25 	for(i=0; i<13; i++)				// 전체 바코드 출력
```

* 14: 짝수 번째 코드의 합을 구한다.
* 16: 홀수 번째 수의 합의 3배를 한다.
* 17: 두 수의 합을 tobe와 check_sum에 저장한다. tobe는 10의 배수가 될 값, sum은 x와 y의 합이 저장되는 변수이다.

* 21: 바코드의 맨 끝에 검사 수를 저장한다.
* 25~26: 전체 바코드 수를 출력한다.
* Try: 코드에 주석으로 처리되어 있는 다른 바코드도 검사해 보라.

#### ● 예제 06-01-9 : 배열의 경계를 벗어나서 데이터를 저장한 경우
```C
1 #include <stdio.h>
2 #define SIZE 4
3 int main(void)
4 {
5 	int value1 = 44;
6 	int array[SIZE];
7 	int value2 = 88;
8 	int i;
9 
10 	printf("value1 = %d, value2 = %d\n", value1, value2);
11 	for (i = -1; i <= SIZE; i++)	// -1 인덱스는 배열 공간이 아니다.
12 		array[i] = 2 * i + 1;
13 
14 	for (i = -1; i < 7; i++)	// 4,5,6 인덱스도 배열 공간이 아니다.
15 		printf("%2d  %d\n", i , array[i]);
16 	printf("value1 = %d, value2 = %d\n", value1, value2);
17 }
```

#### ● 해설
* 11: 배열의 경계인 0보다 작은 인덱스를 사용하였다. 그 결과 에러는 없지만 다행이 제대로 저장되었다. 0,1,2,3 인덱스까지만 배열공간이다.
* 14~15: 경계를 벗어난 –1과, 4,5,6 인덱스의 값을 출력하여 본다. 마찬가지로 에러 없이 정상적으로 출력되었다. 하지만 경계를 벗어난 곳의 데이터는 안전성을 보장할 수 없다. 또한 경계를 벗어나 곳에 데이터를 저장하면 다른 데이터에도 영향을 미칠 수 있다. 이 프로그램의 결과 value1의 값이 88로 저장되었지만 실제로 출력되는 값은 88이 아님을 알 수 있다. 즉, 배열 공간이 아닌 곳을 배열로 간주하여 연속적으로 데이터를 저장하면 다른 데이터에 영향을 줄 수 있다.
