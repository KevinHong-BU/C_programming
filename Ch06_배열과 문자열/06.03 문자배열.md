### 06.03 문자배열
문자배열(문자열)은 문자를 순서대로 저장하는 것을 의미한다. 
그러나 앞에서 살펴본 숫자 배열과는 다른 점이 있다. 통상적으로 문자배열의 끝에는 널(NULL) 문자를 삽입한다. 
출력할 때는 %s로 출력형식을 지정하는데 문자열의 끝(NULL)까지만 출력한다. 
따라서 문자열을 다룰 때는 NULL을 항상 신경 써야 한다. NULL의 값은 0이다.

#### ● 문자열(문자배열)?
* 문자열은 문자를 순서대로 메모리에 저장한다.

#### ● 문자열 선언 및 초기화 방법
```C
① char s1[20] = "String!";
② char s2[20] = {'S', 't', 'r', 'i', 'n', 'g', '!', '\0'};
③ char *s3 = "String!";
```

#### ● 위 예에서 ①,③은 문자열의 끝에 보이지 않는 NULL문자가 자동으로 삽입된다. ②와 같이 배열에 문자열을 하나씩 따로 지정할 경우에는 NULL문자를 끝에 넣어 주어야 한다. ③은 선언할 때에는 값이 저장되지만 실제로 s3는 프로그램 중에 다른 용도로 사용할 수 없다. 이 방법은 사용하면 안 된다.

---
#### ● 예제 06-03-1 : 문자열 변수의 초깃값 지정 방법
```C
1 #include <stdio.h>
2 int main()
3 {
4  	char s1[20] = "String!";
5  	char s2[20] = {'S', 't', 'r', 'i', 'n', 'g', '!', 0}; // 0 = '\0'
6  	char *s3 = "String!";	//포인터 선언
7  	printf("%s\n", s1);
8  	printf("%s\n", s2);
9  	printf("%s\n", s3);
10 }
```

#### ● 해설
* 4: 문자열 상수를 문자열 변수에 저장
* 5: 문자 상수를 하나씩 문자열 변수에 저장
* 6: 문자열 상수를 문자 포인터 변수에 저장
* 7~9: 각각의 문자열 변수를 출력
> [!NOTE]
> 문자열의 끝에는 NULL문자가 저장된다. 포인터 변수는 초깃값을 지정할 수 있으나 향후 다른 값으로 	지정될 때는 먼저 포인터(주소)를 지정해 주어야 한다. 그렇지 않으면 에러가 발생한다. 7장에서 설명. 4번 줄과 같이 지정하는 것이 보통이다.

#### ● 예제 06-03-2 : 문자열에서 각 문자의 위치에 해당하는 주소를 지정해 주면 그 위치에서부터 끝(NULL)까지 출력한다.
```C
1 #include <stdio.h>
2 #include <string.h>
3 int main()
4 {
5 	int i, j;
6 	char c[30] = "Programming is imagination!";
7 	for (i=0; i<strlen(c); i++)
8 		printf("%s\n", &c[i]);
9 }
```

#### ● 해설
* 6: 문자열 상수를 문자열 변수에 저장
* 7\~8: i를 1씩 증가하면서 배열 c[i]의 주소로부터 끝(NULL)까지 출력한다. 출력 결과를 보는 바와 같다.
* 7: strlen() 함수는 문자열의 길이를 구해주는 함수이다. 반환 값은 정수.

3 {

* 6: 문자열의 11번째 NULL을 삽입하여 문자열의 뒷부분을 잘라 버림. 실제로 잘리는 것이 아니라 NULL까지만 문자열을 출력한다. 즉, 자르고 싶은 위치에 NULL을 삽입하면 된다.
* 7: 문자열 출력

#### ● 예제 06-03-4 : 문자 배열을 이용하여 점수에 대한 평점을 매기기
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int score;
5 	char grade[] = { 'F', 'F', 'F', 'F', 'F', 'F', 'D', 'C', 'B', 'A', 'A'};
6 
7 	printf("성적을 입력하시오 : ");
8 	scanf("%d", &score);
9 
10 	if (score < 0 || score > 100) {
11 		printf("점수가 잘못되었습니다.\n");
12 		return 0;
13 	}
14 
15 	printf(" %d : %c", score, grade[score/10]);
16 	if((score > 60 && score%10 >=5 ) || score == 100 )
17 		printf("+\n");
18 }
```

#### ● 해설
* 5: 평점을 매길 문자를 정의하는 문자열 배열
* 10\~13: 점수가 정상범위 내에 있는지 검사한다.
* 15: 등급에 맞는 평점에 해당하는 문자를 출력한다.
* 16,17: ‘+’를 출력할 것인지 검사하고 필요하면 출력한다. 60점 이상이고 끝자리가 5이상이면 ‘+’를 출력하고 또는 100점일 때도 ‘+’를 출력한다.

#### ● 예제 06-03-5 : 문자열을 이용하여 길이가 긴 두 정수의 합 구하기
```C
1 #include <stdio.h>
2 #include <string.h>
3 #define SIZE 51
4 int main()
5 {
6 	int x[SIZE]={0,}, y[SIZE]={0,}, z[SIZE]={0,}, i, j, carry;
7 	char a[SIZE], b[SIZE], c[SIZE];
8 	printf("첫 번째 수 입력:");
9 	scanf("%[0123456789]", a);		// 숫자만 입력받기
10 	fflush(stdin);			// 입력장치 버퍼 비우기
11 	printf("두 번째 수 입력:");
12 	scanf("%[0123456789]", b);
13 	fflush(stdin);
14 	for(i=strlen(a)-1, j=SIZE-1; i>=0; i--, j--)
15 		x[j] = a[i] - '0';		// a의 끝에서부터 문자를 숫자로 변환 
16 	for(i=strlen(b)-1, j=SIZE-1; i>=0; i--, j--)
17 		y[j] = b[i] - '0';		// b의 끝에서부터 문자를 숫자로 변환 
18 
19 	carry = 0;		// 캐리를 0으로 초기화 
20 	for(i=SIZE-1; i>=0; i--) {		// 덧셈 시작 
21 		z[i] = x[i] + y[i] + carry;		// 덧셈 
22 		carry = z[i] / 10;				// 캐리 계산 
23 		z[i] = z[i] % 10;				// 현재 자리 값 계산 
24 	}
25 
26 	for(i=0; i<SIZE; i++)	// 계산 결과 처음 0이 아닌 곳으로 이동 
27 		if(z[i] != 0)		// 처음 0이 아닌 값을 만나면 루프를 벗어난다. 
28 			break;
29 	// 처음 0이 아닌 곳에서부터 문자로 변환하여 문자열 c에 저장한다. 
30 	for(j=0; i<SIZE; i++, j++)
31 		c[j] = (char)z[i] + '0';	// 계산 결과를 문자로 변환 
32 	c[j] = 0;		// NULL 삽입 
33 	printf("\n%52s\n+ %50s\n= %50s\n", a, b, c); // 결과 출력 
34 }
```

#### ● 해설
* 6: 정수 배열과 정부 변수 선언
* 7: 문자열 배열 선언, a, b: 입력 문자열, c: 출력 문자열
* 9,12: 숫자만 받아들인다.
* 10,13: 키보드에 남아 있는 찌꺼기를 제거한다.
* 14,15: 문자열 a를 끝에서부터 하나씩 숫자로 변환하여 x에 저장한다.
* 16,17: 문자열 b를 끝에서부터 하나씩 숫자로 변환하여 y에 저장한다.
* 19: 처음 캐리를 0으로 지정한다.
* 20\~24: 오른쪽 끝에서부터 숫자를 계산한다.
* 26\~28: 계산 결과 중에서 앞쪽의 0을 건너뛰어 숫자로 시작하는 부분으로 이동한다.
* 30\~31: 처음 숫자가 있는 곳에서부터 문자로 변환하여 문자열 c에 저장한다.
* 32: c 문자열 끝에 널을 삽입한다.
* 33: 결과를 출력한다.

#### ● 예제 06-03-7 : 저항색을 입력하여 저항값 계산하기
```C
1 // 저항색 : 다음 10가지 색 중에서 세 가지 색이 기본 지정됨 
2 // "검정", "갈색", "빨강", "오렌지", "노랑", "녹색", "파랑", "보라", "회색", "흰색"

7 // Brown, Red, Green, Blue, Violet, Gray, Gold, Silver, None
8 #include <stdio.h>
9 #include <math.h>
10 
11 int main()
12 {
13 	int x[4] = {0,}, i, j;
14 	double resistor;
15 	char *color[] = { "검정", "갈색", "빨강", "오렌지", "노랑", "녹색",
16 				  "파랑", "보라", "회색", "흰색" };

18 				  "회색", "금색", "은색", "없음"};
19 	double tolerance[] = { 1., 2., 0.5, 0.25, 0.1, 0.05, 5., 10., 20.};
20 
21 	for(j=0; j<10; j++)

23 
24 	printf("\n");
25 	for(i=1; i<4; i++) {
26 		printf("%d번째 색 선택 : ", i);
27 		scanf("%d", &x[i-1]);
28 	} 
29 
30 	// 오차 색 선택 
31 	printf("\n");
32 	for(j=0; j<9; j++)
33 		printf("%d:%s  ", j, tolerance_color[j]);

53 	}
54 	else {
55 		resistor = (x[0]*10 + x[1]) * pow(10, x[2]-9);
56 		printf("%.1f TΩ", resistor);
57 	}

```
