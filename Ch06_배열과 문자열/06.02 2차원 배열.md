### 06.02 2차원 배열
2차원 배열이 실제 메모리 공간의 할당은 1차원적으로 이루어진다. 
그러나 2차원적으로 배열된 것으로 간주하여 처리한다. 
2차원 배열은 포인터를 이용하여 1차원적으로도 액세스할 수 있다. C언어는 행우선 배열이다.

#### ● 2차원 배열의 선언 방법
```C
데이터형 배열명[행_인덱스][열_인덱스] = {{0행 초기값}, {1행 초기값},... {n행 초기값} };
```
* 초기값은 생략할 수 있다.
* 2중으로 {}를 사용하지 않아도 되지만 오류를 줄여준다.

#### ● 2차원 배열 선언 예:
```C
int dim[5][5]={{5,3,4,9,1}, {7,6,7,3,4}, {8,4,6,1,2}, {5,2,3,4,6}, {9,8,7,6,4} };
```
* C언어에서 2차원 이상의 배열은 행 우선으로 인덱스는 (0,0), (0,1), (0,2), ...의 순서로 메모리에 저장된다.

#### ● 2차원 배열의 초기화 방법: 일반 변수의 초기화와 같다.
* 배열을 선언할 때
* 선언 후 입력장치(키보드)를 통해서
* 계산에 의해서
* 파일로부터: 파일 입출력 부분에서 설명

#### ● 배열 선언 시 초깃값을 줄 때 행 인덱스를 생략할 수 있다.
```C
double arr[][3] = { {1.5, 2.3, 3.6}, {9.7, 8.6, 4.9}, {6.7, 8.1, 4.9}, {5.4, 6.5, 7.6} };
```
* 컴파일러가 알아서 행을 계산한다. 여기서 arr[4][3]이 된다.

#### ● 2차원 배열의 접근법
* r×c개의 요소를 가지는 배열의 인덱스는 행이 0부터 r-1까지, 열이 0부터 c-1까지 이다.
* 배열명[행_인덱스][열_인덱스]로 접근한다.
```C
array[2][3]; // array라는 배열의 3번째 행, 4번째 열 요소
```
* 배열의 한 요소는 변수와 동일한 역할을 한다.
```C
x = array[2][3] * 10; // array배열의 3행 4열 요소의 값과 10을 곱하여 x에 저장한다.
array[2][3] = i * 10; // i에 10을 곱하여 array배열의 3행 4열에 저장한다.
```
* 배열의 인덱스로 정수 변수를 사용할 수 있다.
```C
k = 7; array[k][2] = sum; // 8행 3열에 sum을 저장한다.
```

---
#### ● 예제 06-02-1 : 2차원 배열의 초기화 및 배열의 값 출력
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int Array[3][3] = { {5, 3, 4}, {9, 1, 5}, {9, 5, 4} }, row, col;
5 	for (row=0; row<3; row++) {
6 		for (col=0; col<3; col++)
7 			printf("Array[%d][%d] = %d, ", row, col, Array[row][col]);
8 		printf("\b\b \n");
9 	}
10 }
```

#### ● 해설
* 4: 2차원 배열을 선언하고 초깃값을 지정한다.
* 5\~9: 2차원 배열을 행, 열을 구별하여 출력한다. 바깥 for 루프는 행의 값 변화, 안쪽 for루프는 열의 값 변화
* 8: 한 행을 출력한 다음 줄을 바꾼다. 줄을 바꾸기 전에 콤마와 공백을 하나 지우고 다시 공백을 출력한 다음 줄을 바꾼다. 즉, 불필요한 문자를 지우고 다음 줄로 간다. \b는 커서 앞의 한 문자 삭제하는 제어 문자이다. \b\b 다음에 공백을 넣지 않으면 삭제되지 않고 바로 줄을 바꾼다. 한 번 시도해 보라. 컴파일러에 따라 다를 수도 있다.

#### ● 예제 06-02-2 : 2차원 배열은 메모리에서는 1차원적으로 저장된다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int matrix[][3] = { 5, 3, 4, 9, 1, 5, 9, 5, 4};
5 	int row, col, size, i, *p;
6 	size = sizeof(matrix) / sizeof(matrix[0][0]); // 배열의 전체 데이터의 개수 구함
7 	for(row=0; row<size/3; row++) {
8 		for(col=0; col<3; col++)
9 			printf("Array[%d][%d] = %d, ", row, 
10 				col, matrix[row][col]);
11 		printf("\b\b \n");
12 	}
13 	p = matrix[0];
14 	for(i=0; i<size; i++, p++) //포인터 이용 1차원적 액세스
15 		printf("%d ", *p);
16 	putch('\n');
17 }
```

#### ● 해설
* 4: 2차원 배열을 선언한다. 여기서 초깃값이 지정된 경우 행의 개수는 생략할 수 있다. 자동으로 카운트해 준다. 그러나 열의 개수는 생략할 수 없다.
* 5: 2차원 배열이 순서대로 저장되었다는 것을 보기 위해 포인터변수 p를 이용한다.
* 6: 전체 배열의 개수를 알아본다.
* 7\~12: 전체 배열의 요소를 배열의 행렬 인덱스를 이용하여 출력한다.
* 13: 배열의 주소를 p에 저장한다.
* 14\~15: 포인터를 이용하여 배열이 1차원 적으로 순서대로 저장되었다는 것을 확인한다.
> [!NOTE]
> 포인터는 7장에서 살펴보고, 여기서는 2차원 배열이 실제 메모리에 순서대로 저장된다는 것만 알도록 하자.

#### ● 예제 06-02-3 : 국어, 영어 점수를 배열에 저장하고, 총점과 평균을 저장할 수 있게 배열을 10행 4열로 만들고 총점과 평균은 계산에 의해서 저장한다.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int i, score[10][4] =  {{30,54,0,0}, {46,84,0,0},
5 	 {75,88,}, {65,87,}, {98,65,}, {99,94,},
6 	 {90,88,}, {95,78,}, {98,80,}, {95,81,}};
7 	// 국어, 영어 점수만 초깃값을 넣고, 합과 평균은 비워둔다.
8 	for (i=0; i<10; i++) { 
9 		score[i][2] = score[i][0] + score[i][1]; 
10 		score[i][3] = score[i][2] / 2; 
11 		printf("국어점수가 %d이고 ", score[i][0]);
12 		printf("영어점수가 %d인 학생의 ", score[i][1]);
13 		printf("총점은 %d이고 ", score[i][2]);
14 		printf("평균은 %d이다.\n", score[i][3]);
15 	}
16 }
```

#### ● 해설
* 4\~6: 배열에 두 점수만 저장하고 나머지는 계산의 의해서 저장하기 위해서 10행 4열로 배열 선언.
* 8\~15: 배열에 점수를 더해서 총점과 평균을 저장한다.
* 9: 총점 계산하여 각 행의 세 번째 요소에 저장
* 10: 평균을 계산하여 각 행의 네 번째 요소에 저장
* 11\~14: 저장된 값을 출력한다.
> [!NOTE]
> 배열의 인덱스는 0부터 시작한다.

#### ● 예제 06-02-4 : 3행 3열인 두 2차원 행렬의 덧셈
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int i, j, total[3][3];
5 	int arr1[3][3] = {{3,6,9},{6,8,4},{7,8,9}},
6 	    arr2[3][3] = {{5,7,3},{9,6,1},{9,4,6}};
7 	for (i=0; i<3; i++)	// 두 행렬의 덧셈 
8 		for (j=0; j<3; j++)
9 			total[i][j] = arr1[i][j] + arr2[i][j]; 
10 
11 	for (i=0; i<3; i++) { // 계산 결과 출력 
12 		for (j=0; j<3; j++)
13 			printf("%4d ", total[i][j]);
14 		printf("\n");
15 	}
16 }
```

#### ● 해설
* 4: 인덱스 변수와 합이 저장될 행렬 total선언
* 5,6: 두 행렬 선언 및 초기화
* 7\~9: 두 행렬 더하기
* 11\~15: 결과 행렬 출력하기

#### ● 예제 06-02-5 : 배열을 선언하고 특정 위치에만 초깃값을 저장하는 방법
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x[10] = { [4]=10, 15, [9]=50 };
5 	int y[5][5]= { [0][0]=1, [1][1]=1, [2][2]=1, [3][3]=1, [4][4]=1};
6 	int a, b;
7 	
8 	for(a=0; a<10; a++)
9 		printf("%d ", x[a]);
10 	printf("\n\n");
11 
12 	for(a=0; a<5; a++) {
13 		for(b=0; b<5; b++)
14 			printf("%d ", y[a][b]);
15 		printf("\n");
16 	}
17 }
```

#### ● 해설
* 4: 배열 x의 4번 인덱스에 10, 5번 인덱스에 15, 9번 인덱스에 50을 저장한다. 나머지에는 0이 저장된다.
* 5: 2차원 배열의 대각선 인덱스에만 1을 저장하는 단위행렬을 선언한다.
* 8\~9: 1차원 행렬 출력
* 12\~16: 2차원 행렬 출력
> [!NOTE]
> 각 행렬의 지정된 위치에만 값을 저장할 수 있으며, 지정되지 않은 곳에는 0을 초깃값으로 한다. 쓸 일이 많지는 않지만 간혹 필요한 곳만 초깃값을 지정하는 곳에 사용될 수 있을 것이다.

#### ● 예제 06-02-6 : $x^2 + y^2 = 10^2$의 그래프를 2차원 배열에 그려 넣고 출력하는 프로그램
```C
1 #include <stdio.h>
2 int main()
3 {
4 	int x, y, f, g;
5 	static short int circle[25][25];
6 
7 	for(x=0; x<25; x++)		// y축 그리기 
8 		circle[x][12] = '|';
9 	circle[0][12] = 'Y';
10 
11 	for(y=0; y<25; y++)		// x축 그리기 
12 		circle[12][y] = '-';
13 	circle[12][24] = 'X';
14 
15 	for(x=-12 ; x<=12; x++) {
16 		for(y=-12; y<=12; y++) {
17 			f = x*x + y*y - 100;
18 			if( f >= -5 && f <= 5 )		// 해당되는 위치에 ‘*’찍기, 허용오차 ±5 
19  			    circle[x+12][y+12] = '*';
20         }
21 	}
22 
23 	circle[12][12] = '0';		// 원점 찍기
24 	for(x=-12; x<=12; x++) { 
25 		for(y=-12; y<=12; y++) 
26 			printf("%2c", circle[x+12][y+12]);
27 		printf("\n");
28 	}
29 }
```

#### ● 해설
* 5: static으로 선언하면 초깃값이 모두 0으로 설정된다. static은 뒤에서 설명한다.
* 7\~13: x, y축을 그린다.
* 15\~21: 원 함수에 해당하는 위치에 ‘*’을 찍는다.
* 23: 가운데 원점을 찍는다.
* 24\~28: 2차원 배열에 그려 넣은 문자들을 화면에 출력한다. 0은 문자 NULL이므로 아무것도 출력되지 않는다.
