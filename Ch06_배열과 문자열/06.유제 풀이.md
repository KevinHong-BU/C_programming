```C
11 	}
12 
13 	for (i=0; i<32; i++) // 결과를 앞에서부터 출력한다.
14 		printf("%d", bin[i]);
15 	printf("\n");
16 }
```

#### ● 해설
* 4: 2진수를 저장할 배열 bin을 선언하고, 모든 값을 0으로 초기화한다.
* 8\~11: 10진수를 2로 나누어 나머지를 배열의 뒤에서부터 거꾸로 저장한다.
* 9: 각 배열의 요소에 2로 나눈 나머지를 저장한다. dec%2를 dec&1로 하여도 된다. 나머지 연산 대신 비트 AND연산을 이용하여 1의 자리가 1인지 아닌지 검사하는 것이다.
* 10: 10진 데이터를 한 비트 오른쪽으로 이동한다. 즉, 2로 나눈 것과 같다.
> [!NOTE]
> 모든 값을 0으로 초기화하고자 할 때 알아두기 바란다. 필요한 데이터만 지정하고 나머지는 그냥 두면 모두 0으로 초기화된다. 또는 static 키워드를 써도 된다. 뒤에서 다시 언급한다.

#### ● 유제 06-05-2 : 문자열 "The 16 men and 21 women ate 3 pigs"를 입력하여 다음과 같이 숫자만 찾아내는 프로그램을 작성하라. 즉, 16, 21, 3을 찾아서 출력하라.
```C
1 #include <string.h>
2 #include <stdio.h>
3 int main()
4 {
5 	char string[100] = "The 16 men and 21 women ate 3 pigs";
6 	char *result = string;
7 	int a[10], i=0, j;
8 	while( result = strpbrk( result, "0123456789" )) {
9 		printf("%s\n", result);
10 		sscanf(result, "%d", &j );
11 		a[i] = j;
12 		while(j){
13 			result++;
14 			j /= 10;
15 		}
16 		i++;
17 	}
18 	printf("\n");
19 	for(j=0; j<i; j++)
20 		printf("%d, ", a[j]);
21 }
```

#### ● 해설
* 8\~17: strpbrk를 이용하여 숫자가 있는 곳까지 이동하고, NULL이면 루프를 끝낸다.
* 10: 현재 위치에서 숫자를 읽어서 j에 넣는다.
* 11: j를 배열에 저장해 놓는다.
* 12~15: 출출해 낸 숫자의 크기만큼 포인터를 뒤로 이동한다.
* 19: 추출해서 저장해 놓은 데이터를 출력한다.
