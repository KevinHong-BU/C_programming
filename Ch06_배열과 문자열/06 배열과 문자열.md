## 06. 배열과 문자열

앞에서 하나의 변수가 하나의 값만 가지도록 선언하고 사용하였다. 이 장에서는 하나의 변수 이름에 같은 종류의 데이터를 여러 개 가질 수 있도록 하는 방법인 배열(array)에 대해서 살펴본다.
즉, 배열을 하나 선언하고 그 배열에 여러 개의 데이터를 저장할 수 있도록 하는 것이며, 이들 각각의 데이터들은 인덱스 번호로 구별한다.

인덱스란 배열에 저장된 데이터의 순서를 의미하며, 인덱스를 이용하여 저장된 순서대로 데이터를 액세스 할 수 있다.
모든 종류의 데이터 형에 대해서 배열을 선언하여 사용할 수 있다.
