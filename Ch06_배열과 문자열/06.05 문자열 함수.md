### 06.05 문자열 함수
문자열을 처리하기 위한 다양한 함수가 존재한다. 이 함수들은 자주 이용하게 되므로 반드시 익혀두기 바란다.
특히 이 문자열 함수를 잘 이용하면 프로그램을 편리하게 작성할 수 있다. 특히 문자열 복사함수는 자주 사용하게 될 것이다.
문자열은 선언할 때를 제외하고는 단순히 문자열 변수에 대입할 수 없다. 따라서 이 함수를 이용하여야만 문자열을 복사할 수 있다.
이 절에서는 문자열 함수들에 대해서 살펴보자.

함수명
설명
strchr
해당하는  첫 번째 문자가 있는 곳으로 이동
char * strchr (char * str, int character );
strrchr
뒤에서부터 처음 해당되는 문자가 있는 곳으로 이동
char * strrchr (char * str, int character );
strspn
str1 문자열에서 str2 문자열에 포함된 문자의 개수를 반환한다. str2에 없는 문자열을 만나면 멈춤.
size_t strspn ( const char * str1, const char * str2 );
strcspn
strspn과 반대로 str2에 없는 문자열의 개수를 반환한다. str2에 해당되는 문자를 만나면 멈춤.
size_t strcspn ( const char * str1, const char * str2 );
strstr
str1에서 str2문자를 찾아 그 위치를 반환한다.
char * strstr (char * str1, const char * str2 );
strpbrk
str1에서 str2의 어떤 문자라도 해당되는 것이 있으면 그 위치를 반환한다.
char * strpbrk (char * str1, const char * str2 );
strtok
문자열을 토큰으로 나눈다.
char * strtok ( char * str, const char * delimiters );
memchr
메모리 블록에서 해당되는 문자가 있는 위치를 찾는다.
void * memchr ( void * ptr, int value, size_t num );
