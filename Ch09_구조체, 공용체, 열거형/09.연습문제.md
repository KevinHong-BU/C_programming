### 09.연습문제

#### 1. double(64비트)을 2진수로 나타내는 프로그램을 union과 struct를 이용하여 프로그래밍 하라.

<예제 09-04-2>를 바꾸어 응용하면 된다. 주의할 점은 정수 2개(8바이트)를 하나의 double로 연계하여야 하며, little endian 또는 big endian에 따라 출력순서가 다르다.

2진수 문자열로 변환하는 itoa함수를 사용한다. 단, itoa함수가 지원되지 않는 컴파일러라면 16진수로 출력하라.

#### 2. 어떤 가게에서 다음과 같이 간단한 고객을 관리하는 프로그램을 작성하라. 고객정보는 고객ID, 구매등급, 전체 매출액, 지난 달 매출액, 이번 달 매출액으로 구성되며, 지난 달 매출액과 이번 달 매출액이 백만 원이 넘으면 구매등급을 한 단계 올려주며, 등급은 최고 10등급까지이고 등급은 매번 조정하는 것이 아니고 마지막에 끝낼 때 한다. 고객의 구조체를 만들고, 고객의 정보는 기본적으로 5명이 입력되어 있다고 가정한다. 단, 이번 달 매출액은 키보드에서 입력받는다. 이번 달 매출액이 입력되면 전체 매출액이 바뀌어야 한다. 여기서는 파일에 저장하지 않기 때문에 프로그램이 끝나면 모든 것이 원상 복귀된다. 10장에서 파일을 배우고 파일로 저장하는 방법을 생각해 보자.
```C
	struct customer {
		int ID, grade, Total, last_mon, this_mon;
	};
	struct customer c[5]  = {
		{ 12345, 0, 534000, 340500, },
		{ 12346, 1, 1023000, 500000, },
		{ 12347, 2, 4030290, 490500, },
		{ 12348, 1, 1203000, 786000, },
		{ 12349, 1, 1300000, 909300, }};
```

#### 3. 어떤 홈페이지에서 회원관리를 한다고 가정하자. 회원의 정보는 ID, password, 이름, 별명을 저장한다. 이를 구조체로 작성하고, 회원의 정보를 입력받아 저장하는 프로그램을 작성하라. 단, password는 간단하게 함호화하라. 암호화 방법은 ID의 모든 자리의 값을 ‘0xAA’와 XOR하고 입력된 패스워드는 “0x55”와 각 자리를 XOR한다. 회원정보는 5명을 먼저 암호화하여 저장한 후 ID를 검색한다.

#### 4. 도서관리 프로그램을 작성하자. 도서의 정보는 구조체에 저장한다. 구조체의 정의는 도서명, 보유권수, 대출권수, 도서위치로 한다. 도서정보는 5개가 저장되어 있다고 가정하고, 도서를 검색하도록 한다. 도서가 검색되었으면 대출가능 여부와 어디에 있는지 출력하고, 도서가 없으면 없다는 메시지를 출력하라.

#### 5. 비트필드 구조체(계급 표시), 공용체(union), 열거형(계급이름을 표시)를 이용하여 각 군인의 정보를 표시하는 프로그램을 작성하라. 단 구조체는 다음과 같다.
```C
	struct Rank {
		unsigned char Level:3; 	// 각 계층별 계급 (순서대로 0, 1, 2, 3, ..., 7) 
		unsigned char Class:5;	// 계층: 장군(16:10000), 영관(8:01000), 위관(4:00100), 
				 	// 부사관(2:00010), 병(1:00001) (각각 1비트씩 차지) 
	};

	union RNK {	// 공용체: 계급을 저장할 때, 단순히 숫자로 저장하거나 계급과 계급내 순서를 
		//저장하도록 할 수 있게 한다.
		unsigned char rank; // 하나의 정보로 저장, 예) 0x41이면 1000 0001이므로 앞의 5비트는
				// 계층(10000)이므로 장군급, 뒤의 3비트는 001은 준장을 의미한다.
		struct Rank rank_s;	// 또는 두 가지 정보로 나누어서 저장
	};

	struct Soldier {		// 군인의 정보를 저장한다.
		unsigned int SN;	// 군번
		union RNK rank;	// 계급
		char name[30];		// 이름
		char belonging[30];	// 소속
	};

	typedef struct Soldier soldier;
```

#### 6. 모두 다른 정수를 저장하는 2진 트리를 만들어라. 같은 수가 들어오면 데이터가 있다는 메시지를 출력한다. 데이터를 검색하는 함수도 만들고, 전체 트리의 leaf node(자식 노드가 없는 노드)가 몇 개인지 출력하는 함수도 만들어라.

#### 7. 어떤 회사에서 월급 계산을 한다고 하자. 먼저 호봉 테이블을 작성한다. 호봉은 1호봉부터 40호봉까지 있고, 직급 테이블이 있다고 하자. 직급 테이블은 사원/대리/과장/차장/부장/상무/전무/사장 등으로 하고 직급에 따른 직급 수당을 따로 정하라.직원 10명에 대해서 구조체를 만들고 구조체에서는 이름, 부서, 직급, 호봉을 저장한다. 각 직원의 월급을 계산하는 프로그램을 작성하라. 월급은 호봉에 따른 본봉과 직급수당을 더하여 지급한다고 가정하자.

#### 8. 복수소에 대한 구조체를 만들고 복소수의 덧셈, 뺄셈, 곱셈, 나눗셈을 계산하는 함수를 작성하라. C99에서는 복소수 데이터 형을 제공하고 있지만 없다고 생각하고 새로 만들기 바라며, 복소수 데이터 형을 이용해서도 한번 시도해 보고 결과가 같은지 확인하라.

#### 9. 어떤 좌표에 대한 구조체를 만들고 좌표간의 직선거리를 구하는 함수를 작성하라.
