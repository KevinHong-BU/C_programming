#### 09.02 구조체 포인터
구조체 포인터 변수의 멤버를 액세스 할 때는 “->”연산자를 사용한다.
즉, “구조체포인터변수명->멤버”로 쓴다. 포인터는 앞에서도 설명하였다시피 간접적으로 데이터를 액세스한다. 
구조체 포인터변수를 직접적인 방법을 이용해서 액세스 할 때는 (*구조체포인터변수명).멤버로 한다.

---
#### ● 예제 09-02-1 : 구조체 포인터 변수의 사용 예
```C
1 #include <stdio.h> 
2 
3 struct student {    // 구조체 정의 
4 	char *name; 	// 구조체 멤버로 포인터 선언
5 	int korean; 
6 	int english; 
7 }; 
8 
9 int main()
10 {
11 	struct student kdhong;		// 구조체 변수 kdhong 선언 
12 	struct student *sp = &kdhong;	// 구조체 포인터 변수 선언 
13 
14 	sp->name = "홍길동";	// 구조체 포인터 변수의 멤버 액세스
15 	sp->korean = 90;		// 데이터 대입 
16 	(*sp).english = 95;	// 직접 액세스는 (*구조체 포인터 변수).멤버 
17 
18 	printf("이름 : %s\n", sp->name); 
19 	printf("국어 : %d\n", sp->korean); 
20 	printf("영어 : %d\n", sp->english); 
21 }
```

#### ● 해설
* 3~7: 구조체 struct student 정의
* 11: 구조체 변수 선언
* 12: 구조체 포인터 변수 선언 및 주소대입, kdhong구조체 변수의 주소를 sp 구조체 포인터 변수에 대입
* 14: 구조체 포인터 변수의 멤버에 데이터 대입, name멤버가 포인터이기 때문에 바로 문자열 대입이 가능하다.
* 15: 구조체 포인터 변수의 멤버에 데이터 대입.
* 16: 구조체 포인터 변수에 직접적으로 데이터 대입. 굳이 이러한 방법을 쓰는 일은 드물지만 이것도 가능하다.
* 18\~20: 구조체 포인터 변수 멤버의 값들을 출력한다.

#### ● 예제 09-02-2 : 동적 메모리 할당을 통한 구조체 포인터 변수의 활용
```C
1 #include <stdio.h> 
2 #include <stdlib.h> 
3 
4 struct student {    // 구조체 정의 
5 	char *name; 
6 	int korean, english; 
7 }; 
8 
9 int main()
10 {
11 	struct student *sp;	// 구조체 포인터 변수 선언 
12 	sp = (struct student*)malloc(sizeof(struct student)); //메모리 할당
13 	sp->name = "홍길동";	// 구조체 포인터 변수의 포인터 멤버
14 	sp->korean = 90;		// 데이터 대입 
15 	sp->english = 95;
16 	printf("이름: %s, 국어: %d, 영어: %d\n", sp->name, sp->korean, sp->english);
17 	free(sp);
18 }
```

#### ● 해설
* 4\~7: 구조체 struct student 정의
* 11: 구조체 포인터 변수 선언
* 12: 구조체 포인터 변수에 동적 메모리 할당을 통하여 메모리 할당.
* 13\~15: 구조체 포인터 변수의 멤버에 데이터 대입, name은 포인터 멤버이기 때문에 문자열 대입 가능.
* 16: 구조체 포인터 변수 멤버의 값들을 출력한다.

#### ● 예제 09-02-3 : 함수에 구조체 변수 전달
```C
1 #include <stdio.h> 
2 
3 struct student { 
4 	char name[20]; 
5 	int korean, english; 
6 }; 
7 
8 void disp1(struct student aa) {		// 구조체 변수를 매개변수로 받음 
9 	printf("%s %d %d \n", aa.name, aa.korean, aa.english); 
10 }
11 
12 void disp2(struct student *bb) { 		// 구조체 포인터를 매개변수로 받음 
13 	printf("%s %d %d \n", bb->name, bb->korean, bb->english); 
14 } 
15 
16 int main()
17 { 
18 	struct student hong={"Hong Gil Dong", 90, 80}; // 구조체 변수 선언 
19 
20 	disp1(hong); 		// 구조체 변수를 함수의 매개변수로 사용 
21 	disp2(&hong); 		// 구조체 변수를 주소(포인터)로 전달 
22 }
```

#### ● 해설
* 3\~6: 구조체 struct student 정의
* 8\~10: 구조체를 매개변수로 가지는 함수
* 9: 구조체를 일반 변수로 전달 받았으므로 직접 데이터를 액세스한다.
* 12\~14: 구조체 포인터를 매개변수로 가지는 함수
* 14: 구조체를 포인터 변수로 전달 받았으므로 간접적으로 데이터를 액세스한다.
* 20: 구조체 변수를 매개변수로 전달
* 21: 구조체 주소(포인터)를 매개변수로 전달

#### ● 예제 09-02-4 : 함수에 구조체 변수 전달
```C

2 #include <string.h> 
3 
4 typedef struct {

10 	// 함수원형을 따로 선언하지 않아도 된다.
11 	int i; 
12 	if ( strstr(aa->name, name) )
13 		return aa->name; 
14  	else
15  		return NULL;
16 }
17 
18 int main()
19 { 
20 	stud students[3]={ {"Hong Gil Dong", 95, 83},
21 		 	{"Park Gi Dong", 90, 78}, {"Kim Gi Joong", 79, 77}}; 
22 	char *name = "Dong", i; 
23 	char *result = (char *)malloc(50);
24 
25 	for (i=0; i<3; i++)	// "Dong"이 포함된 이름을 모두 찾는다.

```

#### ● 해설
* 4\~7: 구조체를 선언하고 이 구조체를 새로운 데이터 타입인 stud로 정의한다.
* 9\~16: 구조체와 문자열을 매개변수로 하는 함수 search정의
* 12,13: 구조체 멤버인 name중에 입력데이터인 name을 가지고 있으면 그 이름을 반환하고
* 14,15: 그렇지 않으면 NULL을 반환
* 20,21: 구조체 stud의 배열을 선언하고 배열의 요소의 값을 선언한다.
* 22: 검색할 문자열 선언
* 23: 결과를 저장할 문자 포인터 변수 선언
* 25\~30: 배열 요소와 검색할 데이터를 함수로 전달
* 27: 문자열을 포함한 결과를 저장한다.
* 28,29: NULL이 아니면 출력한다.

#### ● 예제 09-02-5 : qsort 함수를 이용한 구조체 정렬
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <string.h>
4 struct score {
5 	unsigned char name[20];
6 	unsigned char Kor, Eng, Math, Sci;
7 	unsigned int Tot;
8 	float Avg;
9 };
10 
11 int cmpare(const void *a, const void *b) {
12 	struct score *ia = (struct score *)a;
13 	struct score *ib = (struct score *)b;
14 	return strcmp(ia->name, ib->name);
15 }
16 
17 int main()
18 {
19 	int i=0;
20 	struct score st[20] = {
21 	     {"강인한", 95, 89, 87, 90,}, {"장윤아", 90, 91, 93, 89, },
22 	     {"김수영", 76, 87, 89, 79,}, {"민효연", 92, 91, 87, 77, },
23 	     {"성유리", 87, 90, 78, 70,}, {"한태연", 88, 77, 68, 78, },
24 	     {"곽서현", 88, 99, 66, 77,}, {"박규리", 79, 87, 97, 99, },
25 	     {"강지영", 67, 69, 79, 78,}, {"한지영", 98, 78, 97, 88, },
26 	     {"한준수", 57, 67, 65, 76,}, {"홍준호", 78, 73, 69, 79, },
27 	     {"임찬성", 68, 78, 67, 65,}, {"강우영", 80, 90, 89, 77, },
28 	     {"박택훈", 47, 65, 55, 67,}, {"조성권", 76, 56, 67, 87, },
29 	     {"이창훈", 88, 89, 85, 92,}, {"정진석", 89, 90, 78, 80, },
30 	     {"김유정", 91, 93, 89, 90,}, {"홍민수", 89, 99, 87, 93, }};
31 
32 	for(i=0; i<20; i++) {
33 	     st[i].Tot = st[i].Kor + st[i].Eng + st[i].Math + st[i].Sci;
34  	     st[i].Avg = st[i].Tot / 4.0;
35 	}
36 
37 	qsort(st, 20, sizeof(struct score), cmpare);
38 
39 	for(i=0; i<20; i++) {
40 		printf("%s %d %d %d %d %d %f\n", st[i].name, 
41 			st[i].Kor, st[i].Eng, st[i].Math, st[i].Sci,
42 			st[i].Tot, st[i].Avg); 
43 	}
44 }
```

#### ● 해설
* 4\~9: 구조체 정의
* 11\~15: 구조체에서 이름을 비교하는 함수
* 20\~30: 구조체 데이터 초깃값 지정
* 32\~35: 합계와 평균을 구한다.
* 37: qsort함수를 이용하여 구조체 정렬
* 39\~43: 정렬된 구조체 출력, 이름 순서대로 정렬하여 출력한다.
