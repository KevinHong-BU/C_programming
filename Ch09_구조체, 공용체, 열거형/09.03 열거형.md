### 09.03 열거형
열거형(enumeration)이란 데이터를 순서대로 열거하여 그 순서대로 크기를 지정하는 것을 의미하며,
열거형은 기호로서 그 순서를 지정한다는 데 의미가 있다.

#### 열거형 사용 시 고려할 점
* 정해진 범위뿐만 아니라, 그 이상, 이하도 실제로 값이 존재하게 된다. 따라서 <예제 09-03-2>와 같이 정확한 범위를 표현할 수 있게 프로그램에서 제어를 해 주어야 한다.
* 이러한 내용은 C++에서는 보완되었다. 
* 열거형 자체는 정해진 수에 대한 기호라고 생각하면 편리하게 제어할 수 있다.
* 기호의 대소비교 및 연산이 가능하다는 장점이 있다.
* 가독성(readability)이 향상된다.

---
#### ● 예제 09-03-1 : 열거형 예, 요일을 순서대로 기호로 지정한다.
```C
1 #include <stdio.h> 
2 enum day {sunday, monday, tuesday, wednesday, thursday, friday, saturday};
3 typedef enum day wday; // 새로운 데이터 형 wday로 선언
4 
5 int main()
6 {
7 	wday cnt;
8 	char* d_name[] = {"일", "월", "화", "수", "목", "금", "토"}; 
9 	for(cnt=sunday; cnt<=saturday; cnt++) //enum변수와 상수를 이용한 값의 이용
10 		printf("%s요일\n", d_name[cnt]);
11 }
```

#### ● 해설
* 2: 열거형을 정의하고 그 순서대로 기호에 값을 부여한다. 즉, sunday라는 기호에는 0을, monday라는 기호에는 1, 등과 같이 값이 부여되며, 그 크기가 결정된다.
* 3: 열거형 day를 새로운 데이터 타입인 wday로 재 정의한다.
* 7: wday의 변수 cnt를 선언한다.
* 8: 문자열 배열 선언
* 9: 반복문을 sunday(0)로 부터 Saturday(6)까지 1씩 증가한다.
* 10: 열거형을 배열의 인덱스로 사용하여 문자열의 값을 출력한다.

#### ● 예제 09-03-2 : 열거형
```C
1 #include <stdio.h> 
2 
3 // 초깃값이 없으므로 0~6까지 변수 값이 대입된다. 
4 enum day {sun, mon, tue, wed, thu, fri, sat};
5 typedef enum day day; 
6 char* day_name[] = {"일", "월", "화", "수", "목", "금", "토"}; 
7 
8 day tomorrow(day d) { // 내일의 요일 찾기 
9 	return ((day)(((int)d+1)%7)); // 정확하게 제어를 해 주어야 한다.
10 } 
11 
12 day yesterday(day d) { // 어제의 요일 찾기 
13 	return ((day)(((int)d+6)%7)); // 정확하게 제어를 해 주어야 한다.
14 } 
15 
16 int main()
17 {
18 	day today=sun; // 오늘은 일요일 
19 	printf("어제는 %s요일\n", day_name[yesterday(today)] ); 
20 	printf("오늘은 %s요일\n", day_name[today] );
21 	printf("내일은 %s요일\n", day_name[tomorrow(today)] ); 
22 }
```

#### ● 해설
* 4: 열거형을 정의하고 그 순서대로 기호에 값을 부여한다. 즉, sun라는 기호에는 0을, mon라는 기호에는 1, 	등과 같이 값이 부여되며, 그 크기가 결정된다.
* 5: 열거형 day를 새로운 데이터 타입인 day로 재 정의한다.
* 6: 문자열 배열 day_name선언 및 초깃값 대입
* 8\~10: 내일의 요일을 계산하기 위한 함수.
* 9: 내일은 오늘+1이다. 그러나 7을 넘어가면 열거형의 값의 범위를 벗어나므로 다시 7로 나눈 나머지로 계산하여야 한다. 변수 d를 정수화 한 다음 계산하여 다시 day형으로 변환한다.
* 12\~14: 어제의 요일을 계산하기 위한 함수.
* 13: 어제는 오늘-1이어야 하지만 일요일의 어제는 –1을 하게 되면 계산 범위를 벗어나므로 +6을 하여 다시 7로 나눈 나머지로 계산하여야 한다. 변수 d를 정수화 한 다음 계산하여 다시 day형으로 변환한다.
* 19: today의 yesterday를 계산하기 위하여 함수를 호출하고 그 값을 인덱스로 가지는 문자열을 출력한다.
* 20: today의 인덱스에 대한 요일을 출력한다.
* 21: today의 tomorrow를 계산하기 위하여 함수를 호출하고 그 값을 인덱스로 가지는 문자열을 출력한다.

#### ● 예제 09-03-3 : 광역자치 단체의 영문과 한글이름을 맵핑하는 프로그래밍
```C
1 #include <stdio.h>
2 
3 typedef enum { Seoul, Busan, Daegu, Incheon, Gwangju, Daejeon, Ulsan,
4 	Gyeonggi, Gangwon, Chungbuk, Chungnam, Jeonbuk, Jeonnam,
5 	Gyeongbuk, Gyeongnam, Jeju, Etc
6 	} Regions;
7 
8 char * NameEng[] = { "Seoul", "Busan", "Daegu", "Incheon", "Gwangju", 
9 	"Daejeon", "Ulsan", "Gyeonggi", "Gangwon", "Chungbuk", "Chungnam",
10 	"Jeonbuk", "Jeonnam", "Gyeongbuk", "Gyeongnam", "Jeju", "Etc"};
11 
12 char * NameKor[] = {"서울특별시", "부산광역시", "대구광역시", "인천광역시",
13 	"광주광역시", "대전광역시", "울산광역시", "경기도", "강원도",  "충청북도",
14 	"충청남도", "전라북도", "전라남도", "경상북도", "경상남도", "제주특별자치도",
15 	"해외"}; 
16 
17 int main(void)
18 {
19 	Regions r;	// 열거형 변수 선언
20 
21 	for(r=Seoul; r<=Etc; r++)
22 		printf("%s: %s\n", NameKor[r], NameEng[r]);
23 }
```

#### ● 해설
* 3\~6: 열거형을 정의하고 그 순서대로 기호에 값을 부여한다. Seoul(0)에서 Etc(16)까지 정의. 동시에	열거형의 새로운 데이터 형 Regions선언.
* 8\~10: 열거형에 영문 이름을 맵핑하기 위한 문자열 배열 선언
* 12\~15: 열거형에 한글 이름을 맵핑하기 위한 문자열 배열 선언
* 21\~22: Seoul(0)에서부터 Etc(16)까지 1씩 증가하면서 한글이름과 영문이름의 지방자치단체 명칭을	출력한다.
