### 09.01 구조체

구조체(Structure)는 같은 형의 데이터를 여러 개 또는 서로 다른 형의 데이터를 묶어서 하나의 새로운 데이터 형식을 만드는 것이다. 
구조체에 포함된 데이터를 멤버라 하며, 어떠한 데이터 형도 멤버가 될 수 있다. 
즉, 기본 데이터 형, 사용자 정의 데이터 형 등 모든 데이터가 멤버가 될 수 있다. 
여러 가지 정보를 하나의 데이터 형식으로 사용할 수 있게 한다. 데이터베이스에서 사용되는 형식을 C언어에서 제공하는 방식이다. 
즉, 구조체는 레코드가 되고, 멤버는 필드가 된다.

● 구조체 변수의 멤버 액세스
1) 일반 구조체 변수 액세스 : 구조체변수명.멤버명
2) 포인터 구조체 변수 액세스 : 구조체변수명->멤버명

● 예제 09-01-1 : 구조체 book을 선언하고 사용하는 예
1 #include <stdio.h> 
2 
3 struct book { 			// 책의 정보를 저장할 구조체 선언 
4 	char title[50]; 		// 책 제목 
5 	char author[35]; 	// 저자 
6 	int pages; 		// 페이지 수 
7 	int price; 		// 가격 
8 }; 
9 
10 int main()
11 {
12 	struct book book1;		// 구조체 변수 선언 
13 	printf(" 도서 제목(string) : "); 
14 	gets(book1.title);	// 책 제목 입력, 구조체의 멤버변수 액세스 방법 
15 	printf(" 저자명(string) : "); 
16 	gets(book1.author);		// 저자 입력 
17 	printf(" 전체 페이지수(int) : "); 
18 	scanf("%d", &book1.pages);	// 책 페이지 수 입력 
19 	printf(" 가격(int) : "); 
20 	scanf("%d", &book1.price); 	// 책값 입력 
21 
22 	// 입력 받은 데이터 출력, 구조체의 멤버변수 액세스하여 출력.  
23 	printf("%s, %s, %dp, %d원\n", book1.title, book1.author,
24 		 book1.pages, book1.price); 
25 }
● 해설
4~9: 구조체 struct book 정의, 멤버로 title, author, pages, price를 가진다.
12: 구조체 struct book의 변수 book1선언
14: 구조체 변수 book1의 멤버인 title을 키보드에서 입력, 포인터 변수가 아닌 구조체 멤버의 액세스는 .을 
	사용한다. 즉, “구조체변수.멤버”이다.
16: 구조체 변수 book1의 멤버인 author를 키보드에서 입력
18: 구조체 변수 book1의 멤버인 pages를 키보드에서 입력
20: 구조체 변수 book1의 멤버인 price를 키보드에서 입력
23,24: 구조체 변수에 저장된 각 멤버를 출력한다.
Notice: 구조체 포인터 변수일 때의 액세스는 “구조체변수->멤버”를 사용한다.

● 예제 09-01-2 : 구조체 배열 및 초기화 방법
1 #include <stdio.h> 
2 
3 struct score {    // 구조체 정의. struct 구조체명 
4 	char name[30]; 
5 	int kor; 
6 	int eng; 
7 	int math; 
8 	int total; 
9 	float avr; 
10 }; 
11 
12 int main()
13 {
14 	struct score Class[3] = { {"강아지", 95, 87, 97, }, 
15 		   { "박기동", 99, 89, 80, }, 
16 		   { "홍길동", 88, 77, 66, }}; // 구조체 배열 선언 및 초기화
17 	int i; 
18 	for (i=0; i<3; i++) { 
19 		Class[i].total = Class[i].kor + Class[i].eng + Class[i].math; 
20 		Class[i].avr = (float)Class[i].total/3.; 
21 
22 		printf("이름 : %s, 국어 : %d, 영어 : %d, 수학: %d, ",
23 		    Class[i].name, Class[i].kor, Class[i].eng, Class[i].math);
24 		printf("총점 : %d, 평균 : %5.1f\n\n", Class[i].total,
25 			Class[i].avr);
26     }
27 }
● 해설
3~10: 구조체 struct score 정의, 멤버로 name, kor, eng, math, total, avr을 가진다.
14~16: 구조체 변수에 초깃값을 지정한다. 구조체에 초깃값을 지정할 때는 하나의 구조체 요소를 {}로 구분해서 나타내어야 한다. 또 초깃값이 없는 뒷부분은 비워두면 된다. 그러나 가운데 부분이 없을 때는 허용되지 않는다.
19: 구조체 변수의 멤버 중 total을 계산한다.
20: 평균값 계산하여 멤버 avr에 저장한다.
22~25: 구조체 배열의 내용을 출력한다.
Notice: 배열의 인덱스는 “배열명[인덱스].멤버”로 배열명 뒤에 붙인다.

● 예제 09-01-3 : 구조체에서 데이터 찾기
1 #include <stdio.h> 
2 #include <string.h> 
3 
4 struct member {	// 구조체 정의 
5 	char name[20];	// 이름 
6 	char gender;	// 성별 : 'M', 'F'로 지정 
7 	int age;		// 나이 
8 	char level[20];	// 직위 
9 }; 
10 
11 int main()
12 {
13 	struct member members[5]= { {"홍길동", 'M', 49, "사장"}, 
14 		{"이몽룡", 'M', 25, "사원"}, {"임꺽정", 'M', 39, "부장"}, 
15 		{"유관순", 'F', 28, "과장"}, {"김유신", 'M', 43, "이사"}};
16 		// 구조체 배열 선언 및 초기화 
17 	int i;
18 	char name[20];
19 
20 	for (i=0; i<5; i++) { // 40세 이상의 남자 모두 찾기 
21 		if (members[i].gender == 'M' && members[i].age >= 40) 
22 			printf("이름 : %s \n직위 : %s\n",
23 			    members[i].name, members[i].level);
24 	}
25 	printf("\n"); 
26 
27 	printf("검색할 직원 이름 입력 : ");	// 찾고자 하는 직원 이름 입력 
28 	scanf("%s", name); 
29 	for (i=0; i<5; i++) {		// 해당되는 직원 검색 및 출력 
30 		if (strcmp(members[i].name, name) == 0)
31 		     printf("이름 : %s \n직위 : %s\n나이 : %d\n",
32 		     members[i].name, members[i].level, members[i].age); 
33 	}
34 }
● 해설
4~9: 구조체 struct member 정의, 멤버로 name, gender, age, level을 가진다.
13~15: 구조체 배열에 초깃값을 지정한다. 
20~24: 구조체 멤버 중에서 데이터를 검색한다.
21: 성별이 남자(‘M’)이고, 나이(age)가 40 이상인 모든 구성원을 출력한다.
28: 직원을 검색하기 위한 이름을 입력받는다.
29~33: 직원 중에  해당하는 이름이 있는지 검색한다.
Try: 직원 중 “김”씨 성을 가진 사람을 찾아보라.
