### 07.03 문자/문자열 포인터
* 문자는 8비트이므로 하나의 문자는 하나의 주소에 저장된다. 
* 문자열은 메모리 주소 순서대로 한 글자씩 저장되며, 문자열의 마지막을 의미하는 NULL이 함께 저장된다. 
* 문자열의 시작주소를 문자열의 포인터로 사용할 수 있다.
* 문자열 포인터도 ++, --연산자를 사용하면 자동으로 주소(포인터)가 1씩 증감하게 된다.
* 문자열 포인터 변수 a에는 처음에 주소 0x200이므로 i를 가리키고 있으며, 이 주소는 다른 주소로 변경할 수 있다. 포인터 변수 a가 j의 주소로 바뀌면 내용이 0x220으로 바뀐다.

---
#### ● 예제 07-03-1 : 문자열과 문자 포인터의 개념
```C
1 #include <stdio.h>
2 int main()
3 {
4  	char x[]="string", y[]="test", *b;
5 	b = x;		 // 포인터 변경, 문자열은 이름 대입
6 	printf("%p, %p, ", &x, b);	// 주소 출력
7 	printf("%s\n", b);		// 문자열 출력
8 	b = y;		 // 포인터 변경, 주소연산자(&) 사용 안함.
9 	printf("%p, %p, ", &y, b);	// 주소 출력
10 	printf("%s\n", b);		// 문자열 출력
11 }
```

#### ● 해설
* 4: 문자열과 문자 포인터 변수 선언
* 5: x의 주소를 b에 대입
* 6: x의 주소와 b의 값 출력
* 7: b출력
* 8: y의 주소를 b에 대입
* 9: y의 주소와 b의 값 출력
* 10: b출력
> [!NOTE]
> 문자열도 배열이며, 그 자체가 포인터이다. 그러나 문자열은 포인터 상수이다. 따라서 그 포인터를 다른 포인터 변수에 대입하여 포인터 변수로 사용할 수 있다.

#### ● 예제 07-03-2 : 문자열과 문자 포인터의 개념
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char *str = "C language Programming.", *s;
5 	s=str; //str을 그냥 써도 되지만 포인터가 변하면 처음 위치를 모르게 된다.
6 	while(*s)
7 		printf("%s\n", s++);
8 }
9 // str[]="C language Programming."는 *str="C language Programming."
10 // 와는 다르다. str[]는 포인터 상수가 되고, *str은 포인터 변수가 된다.
```

#### ● 해설
* 5: 문자 포인터에 문자열 지정 및 포인터 변수 s선언, 문자배열이나, 문자 포인터나 같은 것이지만, 문자배열을 포인터 상수이고, 문자 포인터는 포인터 변수이다. 여기서는 문자 포인터로 선언하였다.
* 6: str의 주소를 s에 대입
* 7: s가 가리키는 곳의 문자(*s)가 널이 아닐 동안 루프 반복하며, 그 주소에서부터 문자열 끝까지 출력. *s는 문자 하나를 나타내며, s는 문자열의 시작 주소를 나타낸다. 문자 배열이나 문자 포인터나 같은 것이다.
> [!NOTE]
> 한 가지 주의할 점은 포인터 변수로 선언하면서 문자열을 대입하는 것은 가능하며, 이렇게 선언한 문자 포인터에는 프로그램 실행 중에도 문자 포인터 변수에 문자열을 strcpy함수를 사용하지 않고 직접 대입할 수 있다. 그러나 문자배열로 선언하면 프로그램 중에서 직접 대입은 불가능하다. 또 단순히 포인터 변수만 선언한 다음에 프로그램 중에 문자열 대입은 직간접적으로 불가능하다.


* 추가 설명: 프로그램 중에 str에 문자열을 직접대입 가능하다. 그러나 이렇게 쓰면 뒤에서 문제가 발생할 가능성이 있으므로 쓰지 않는 것이 좋다. strcpy함수를 쓰도록 하자.
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char *str = "C language Programming.", *s;
5 	s=str; //str을 그냥 써도 되지만 포인터가 변하면 처음 위치를 모르게 된다.
6 	while(*s)
7 		printf("%s\n", s++);
8 	str = "This is test.";	// 직접 대입가능하다. 그러나 문제가 발생할 가능성이 있다.
9 	printf("%s ", str);
10 }
```

#### ● 예제 07-03-3 : 포인터 상수와 인덱스를 이용한 직접 문자열 복사
```C
1 #include <stdio.h>
2 #include <string.h>
3 int main()
4 {
5 	char str[30] = "C programming language";
6 	char target[100], i;
7 	for(i=0; i<strlen(str);i++)	// 문자열 복사
8 		*(target+i) = *(str+i);		// 배열의 인덱스를 사용하는 것과 동일하다.
9 	*(target+i) = '\0';	// 문자열의 끝에 널 삽입
10 	printf("str: %s\n", str);
11 	printf("target: %s\n", target);
12 }
```

#### ● 해설
* 5,6: 문자열 배열을 선언한다.
* 7,8: 문자열을 하나씩 복사한다. 단, 포인터 변수를 쓴 것이 아니라 포인터 상수인 target와 str의 위치 정보를 사용하였다. 앞에서 설명하였다시피 target와 str은 포인터 상수이므로 직접 포인터 증감, 또는 연산이 안 되므로 간접적으로 사용하였다. *(target+i)는 target의 시작 주소에 포인터 변위를 이용하여 그 위치를 계산하고 그 값에 *(str+i)의 값을 저장하는 것이다. 즉, 배열의 인덱스를 사용하는 것과 동일하다. 즉, target[i] = str[i]의 다른 표현이다.
* 9: target의 마지막에 NULL을 넣는다.
* 10,11: 원 데이터와 복사한 데이터가 같음을 알 수 있다.

---
#### ● 예제 07-03-4 : 포인터 변수를 이용한 문자열의 복사
```C
1 #include <stdio.h>
2 int main()
3 {
4 	char str[30] = "C programming language";
5 	char target[100], *s, *t;
6 	s = str;
7 	t = target;
8 	while(*s)	// 문자열 복사
9 		*t++ = *s++;
10 	*t = '\0';	// 문자열의 끝에 널 삽입
11 	printf("str: %s\n", str);
12 	printf("target: %s\n", target);
13 }
```

#### ● 해설
* 6: 포인터 변수 s에 str의 주소가 저장된다.
* 7: 포인터 변수 t에 target의 주소가 저장된다.
* 8,9: 포인터 변수를 이용하여 그 포인터가 가리키는 곳의 데이터를 복사한다. ++가 변수의 뒤에 있으므로 먼저 s가 가리키는 곳의 값을 t가 가리키는 곳의 값으로 대치하고, s, t의 포인터가 증가한다.
* 10: t의 끝에 NULL을 삽입한다.
* 11,12: 원 데이터와 복사한 데이터가 같음을 알 수 있다.

#### ● 예제 07-03-5 : 포인터를 이용하여 문자열 거꾸로 만들기

```C
10 
11 	len = strlen(str);
12 	for(i=0; i<len/2; i++) {
13 		temp = *(str+i);

16 	}
17 	printf("%s\n", str);
18  	free(str);			// 메모리 공간 해제

```

#### ● 해설
* 6: 문자 포인터 및 변수 선언

* 8: 문자열 포인터에 문자열 복사.
* 11: 문자열의 길이 획득.
* 12\~16: 문자열의 절반까지 이동하면서 맨 앞과 맨 뒤를 바꾸면서 한 포인트씩 앞, 뒤로 이동하면서 작업한다.
* 17: 결과 출력

```C
1 #include <string.h>
2 #include <stdio.h>
3 #include <stdlib.h>
4 #include <ctype.h>
5 int main()

7 	char *str, temp, in_str=0;

10 	strcpy(str, "Char string_name[size];");
11 	len = strlen(str);
12 	printf("%s\n", str);
13 	for(i=0; i<len; i++) {
14 		if(!in_str && isalnum(*(str+i)) ) {
15 			if(islower(*(str+i)))
16 				*(str+i) = toupper(*(str+i));
17 			in_str = 1;
18 		}
19 		else if(isspace(*(str+i)) || ispunct(*(str+i)))
20 			in_str = 0;
21 	}
22 	printf("%s\n", str);
23  	free(str);
24 }
```

#### ● 해설
* 7: 문자 포인터 및 변수 선언
* 9: 문자열을 저장할 메모리 공간 400바이트 할당.
* 10: 문자열 포인터에 문자열 복사.
* 11: 문자열의 길이 획득.
* 13\~21: 단어의 첫 글자가 소문자이면 대문자로 바꾼다.
* 14: 단어 속에 있지 않고(!in_str), 알파벳 또는 숫자이면
* 15,16: 소문자이면 대문자로 바꾼다.
* 17: 단어 속으로 들어갔다는 표시를 한다.(in_str=1)
* 19,20: 공백이거나 구두문자이면 단어 밖으로 나왔다는 표시(in_str=0)를 한다.
* 22: 결과 출력
* 23: 할당된 메모리를 해제한다.

#### ● 예제 07-03-7 : 문자의 2차 포인터 개념
```C
1 #include <stdio.h>
2 #include <string.h>
3 int main()
4 {
5 	static char *str[] = {"C language", "Java", "Python", "C++", "HTML"};
6 	char **s, i;
7 	s=str;
8 	for(i=0; i<5; i++, s++)
9 		printf("%p:%p:%s\n", &*s, *s, *s);
10 }
```

#### ● 해설
* 5: 문자열 배열 선언 및 초기화, 문자열의 배열(2차원 배열)을 포인터로 선언한다. 
* 6: 문자열의 2차원 포인터 변수 s선언
* 8,10: 문자열 배열을 s의 포인터로 출력하기. 첫 번째 출력 값은 포인터의 배열의 주소를 출력하고, 두 번째 출력은 포인터의 포인터 주소를 출력하고, 세 번째는 포인터의 포인터 값, 즉 문자열을 출력한다.
> [!NOTE]
> 포인터 배열은 4만큼 증가하는 것을 볼 수 있다. 즉, 포인터를 저장하고 있는 배열이고 이 값은 주소의 크기(4바이트)만큼 증가한다. *str[0]의 주소가 00408000이고 이 주소에 저장된 값이 0040906e이며, *str[1]의 주소가 00408004이고, 이 주소에 저장된 값이 00409079 등이다.	두 번째 값은 각 문자열의 시작 주소이다. 즉, 실제 문자열이 저장된 시작주소이다.
