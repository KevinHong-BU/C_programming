#### ● 유제 07-01-1 : 배열을 동적으로 100개 할당하여 난수를 발생하여 저장한 다음, 최댓값과 최솟값이 저장된 위치를 찾아라.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 int main()
5 {
6 	int *array, *ap, i, *max, *min;
7 	array = (int *)malloc(400);	//정수 100개 
8 	srand(time(NULL));		// 난수 초기화
9 
10 	ap = array;
11 	for(i=0; i<100; i++, ap++) // 포인터를 증가하면서 
12 		printf("%d ", *ap = rand()); //100개의 난수 발생 
13 
14 	ap = array;
15 	max = min = ap;	// max, min초기값을 배열의 첫 요소로 설정
16 	ap++;			// 포인터 증가
17 	for(i=1; i<100; i++, ap++) { // 1번 요소부터 끝까지 비교
18 		if(*ap > *max)
19 			max = ap; // max 포인터 대입
20 		if(*ap < *min)
21 			min = ap; // min 포인터 대입
22 	}
23 	printf("\nmax = %d, min = %d\n", *max, *min);
24 	free(array);	//할당된 메모리 해제
25 }
```

#### ● 해설
* 6: 포인터 변수 및 다른 변수 선언
* 7: 포인터 변수 array에 메모리 할당. 메모리의 크기는 400바이트이며, 하나의 정수가 4바이트다. 따라서 이 메모리에는 100개의 정수가 저장될 수 있다. 배열 할당의 결과는 정수 포인터로 변환(캐스트)하여 포인터 변수에 저장한다.
* 10: 첫 번째 포인터가 변하면 그 위치를 찾기가 어려우므로 다른 포인터 ap를 이용하기 위해 포인터를 저장한다.
* 11\~12: 포인터에 값을 100개 무작위로 발생하여 저장한다. 이 때 ap를 증가하면서 포인터를 이동한다.
* 14: 다시 첫 위치를 ap에 저장한다.
* 15: 첫 데이터의 위치를 max와 min에 저장한다. 즉 max와 min의 초기값을 배열의 첫 번째 요소로 지정한다.
* 16: 포인터를 하나 대입하였으므로 ap의 포인터를 증가한다.
* 17\~22: 배열 요소 100개 중 최댓값과 최솟값의 위치를 찾아 max와 min에 대입한다. 실제 값을 대입하는 것이 아니라 포인터를 대입하여 그 위치만 알 수 있도록 한다.
* 23: max와 min이 가리키는 곳의 데이터를 출력한다.
* 24: 할당된 메모리를 해제한다. malloc로 할당된 메모리는 해제해 주어야 한다.
> [!NOTE]
> 난수를 발생하였으므로 실행할 때마다 위치가 바뀌게 되며, 그 값도 달라진다.

#### ● 유제 07-01-2 : 2차원 정수 포인터를 이용하여 2차원 포인터에 다시 1차원 포인터를 할당하라. 각 1차원 포인터의 크기는 정수 데이터 10개를 난수로 발생하여 저장한다.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 int main()
5 {
6 	int *array[5], *ap, **app, i, j;
7 	app = array;

13 		ap = *app++;
14 		for(j=0; j<10; j++)		// 각 행에 데이터 10개 생성 저장
```

#### ● 유제 07-02-1 : 실수(double)을 저장할 동적 메모리를 100개 할당하고, 0.0 ~ 1.0사이의 난수를 발생하여 저장한 뒤, 최댓값과 최솟값을 찾아 출력하는 프로그램을 작성하라.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 #include <time.h>
4 int main()
5 {
6 	double *array, *dp, *max, *min;
7 	int i;
8 	array = (double *)malloc(800); //double 100개 메모리 
9 	srand(time(NULL));
10 	dp = array;
11 	for(i=0; i<100; i++, dp++)	// 포인터를 증가하면서 
12 		printf("%f ", *dp = rand()/(double)RAND_MAX); 
13 		// 0.0~1.0 사이의 난수 100개 발생 
14 	dp = array;
15 	max = min = dp;		// max, min 초깃값을 배열의 첫 요소로 지정
16 	dp++;				// 포인터 증가
17 	for(i=1; i<100; i++, dp++) { // 1번 요소부터 끝까지 비교
18 		if(*dp > *max)
19 			max = dp;	// max 포인터 대입
20 		if(*dp < *min)
21 			min = dp;	// min 포인터 대입
22 	}
23 	printf("\n\nmax = %f, min = %f\n", *max, *min);
24 	free(array);		//할당된 메모리 해제
25 }
```

#### ● 해설
* 6: 실수(double)의 포인터 변수 선언
* 8: 100개의 double값이 저장될 장소를 메모리에서 확보
* 9: 난수 초기화
* 10: 배열의 시작 주소를 dp에 대입
* 11,12: double 포인터 변수를 증가하면서 난수를 발생하여 배열에 저장한다. 0.0 ~ 1.0사이의 난수를 발생 하는 방법을 보여준다. rand()함수는 0부터 RAND_MAX까지 정수 난수를 발생한다. 이를 RAND_MAX로 나누면 0.0에서 1.0사이의 실수 난수를 발생할 수 있다.
* 14: 다시 배열 array의 시작 주소를 dp에 대입
* 15: 배열의 첫 요소의 주소를 max, min에 대입한다.
* 16: 첫 요소가 대입되었으므로 포인터를 하나 증가한다.
* 17\~22: 최댓값과 최솟값의 위치를 찾아 포인터 변수 max와 min에 대입한다.
* 23: 결과를 출력한다.
* 24: 할당된 800바이트의 메모리를 해제한다.
> [!NOTE]
> 실제 값이 max, min에 저장되는 것이 아니라 위치만 저장된다.

#### ● 유제 07-04-3 : 정수(int) 포인터(double)와 실수 포인터를 적절히 활용하여 실수가 컴퓨터 내부에서 2진수로 어떻게 표현되는지 보여라. 힌트, 실수 포인터(8바이트)를 정수 포인터(4바이트)에 대입하고 두 번에 걸쳐서 2진수로 변환하면 된다. 또한 little endian이라는 것도 감안하여야 한다. 참고: 정수를 2진수 문자열로 변환하는 함수는 itoa(정수값, 문자열변수, 2)을 사용하면 간단하게 해결된다. itoa는 표준 함수는 아니지만 거의 대부분 컴파일러에서 지원한다.
```C
1 #include <stdio.h>
2 #include <stdlib.h>
3 int main()
4 {
5 	int *x;
6 	char buf[35];
7 	double *y;
8 	y=(double *)malloc(sizeof(double));
9 	*y = 0.625;
10 	x = (int *)y;		// double pointer를 int pointer로 변환 
11 	x++;				// 정수 포인터 증가 : little endian이므로 뒤에서부터 
12 	itoa(*x, buf, 2);	// 2진수 문자열로 변환
13 	printf("%032s", buf);	// 2진수 출력 
14 	x--;				// 정수 포인터 감소
15 	itoa(*x, buf, 2);
16 	printf("%032s", buf);
17 	free(y);			// 메모리 해제 
18 }
```

#### ● 해설
* 5: 정수 포인터 선언
* 6: 2진수를 저장할 문자 배열
* 7: 실수 포인터 선언
* 8: 실수 포인터 변수에 메모리 할당
* 10: 실수 포인터를 정수 포인터로 할당
* 11: little endian이므로 뒤의 4바이트부터 2진수로 바꾸기 위해 포인터를 하나(4바이트) 증가한다. 즉, int는 4바이트이고 double은 8바이트이므로 int 포인터로 변환하면 정수 인 것처럼 포인터가 이동한다.
* 12,13: 정수로 간주하고 2진수로 변환하여 출력한다.
* 14: 다시 앞으로 4바이트 이동, 즉, 포인터를 감소한다.
* 15,16: 정수로 간주하여 2진수로 변환하여 출력한다.
> [!NOTE]
> 실행결과를 보면 맨 좌측 비트가 부호 0이므로 양수이고, 지수는 11비트 값 01111111110이고, 지수에서 바이어스 01111111111을 빼면 –1이 된다. 가수는 010000... 인데 생략되어 보이지 않는 1.을 붙이면  1.01000... 이다. 결과적으로 임을 알 수 있다.
